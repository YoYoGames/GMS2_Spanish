array_length_2d


Descripción

Con esta función puede obtener la longitud (número de entradas) de una segunda dimensión de una matriz. Usted proporciona el número de entrada para la primera dimensión y la función devolverá el número de entradas de la segunda dimensión que tiene la matriz (para encontrar la longitud de la primera dimensión use la función array_height_2D ) La función devolverá 0 si la variable dada no es una matriz o 1 si la variable es una matriz 1D (ya que todavía hay 1 fila).


Sintaxis:

array_length_2d(array_index, n);


Argumentos:

Argumento Descripción
array_index El índice de la matriz para verificar
n La entrada de la matriz para obtener la longitud de.

Devoluciones:

Real


Ejemplo:

for (var i = array_height_2d(a) - 1; i > -1; i--;)
   {
   for (var j = array_length_2d(a, i) - 1; j > -1; j--;)
      {
      a[i, j] = -1;
      }
   }

El código anterior recorrerá una matriz 2D y establecerá cada entrada en -1.
