array_create


Descripción

Con esta función, puede crear una matriz de un tamaño determinado. Usted le dice a la función la longitud de la matriz para crear, y devolverá el "mango" para la matriz que luego puede asignar a una variable. Las matrices creadas de este modo tendrán cada entrada inicializada en 0 a menos que especifique un valor de inicialización (opcional). Si proporciona el valor extra para inicializar la matriz, todos los índices dentro de la nueva matriz se establecerán en lugar de 0, pero tenga en cuenta que la función tardará más tiempo en ejecutarse.


Sintaxis:

array_create(size, [value]);


Argumentos:

Argumento Descripción
size El tamaño de la matriz para crear.
value El valor a usar para inicializar todos los índices de matriz (opcional).

Devoluciones:

Array (handle)


Ejemplo:

instance_array = array_create(100, noone);

El código anterior creará una nueva matriz de 100 entradas, cada una establecida en la palabra clave noone, y luego asignarlo a la variable "instance_array".
