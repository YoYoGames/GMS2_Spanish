array_equals


Descripción

Con esta función puede verificar si dos matrices son iguales (equivalentes o iguales). Usted da las dos matrices para verificar, y la función regresará true si son iguales (ya sea el equivalente o el mismo) o false si no lo son Tenga en cuenta que esto no es lo mismo que comprobar si dos matrices son iguales usando ==, lo que no verificará si las dos matrices contienen valores equivalentes, pero solo para ver si las matrices hacen referencia a la misma matriz inicial. Por ejemplo:

var a = [1,2,3,4];
var b = [1,2,3,4];
if (a == b)
   {
   show_debug_message( "This will never fire, as a and be do not reference the SAME array" );
   }
if (array_equals(a, b))
   {
   show_debug_message( "This will fire now" );
   }


Sintaxis:

array_equals(var1, var2);


Argumentos:

Argumento Descripción
var1 El índice de la primera matriz.
var2 El índice de la segunda matriz.

Devoluciones:

Boolean


Ejemplo:

if !array_equals(inventory_array, item_array)
   {
   var len = array_length_1d(inventory_array);
   array_copy(item_array, 0, inventory_array, 0, len);
   }

El código anterior comprobará dos matrices para ver si contienen valores equivalentes, y si no lo hacen, el código copiará todo el contenido de la matriz "inventory_array" a la matriz "item_array".
