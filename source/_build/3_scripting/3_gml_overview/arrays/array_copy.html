array_copy


Descripción

Con esta función, puede copiar todo o parte de una matriz en otra matriz en cualquier posición. Debe proporcionar las matrices de origen y de destino (ambas deben haber sido creadas previamente), así como una posición dentro de la matriz de origen para copiar y una posición dentro de la matriz de destino para copiar. Finalmente, debe especificar la longitud de la matriz (o la longitud de la parte que desea) para copiar. Si los datos que se están copiando exceden la longitud de la matriz de destino, la matriz se ampliará para aceptar los datos.


Sintaxis:

array_copy(dest, dest_index, src, src_index, length);


Argumentos:

Argumento Descripción
dest El ID de la matriz para copiar.
dest_index El índice dentro de la matriz para copiar.
src La ID de la matriz para copiar desde.
src_index El índice con la matriz para comenzar a copiar.
length La longitud (número de índices de matriz) para copiar.

Devoluciones:

N/A


Ejemplo:

if !array_equals(inventory_array, item_array)
   {
   var len = array_length_1d(inventory_array);
   array_copy(item_array, 0, inventory_array, 0, len);
   }

El código anterior comprobará dos matrices para ver si contienen valores equivalentes, y si no lo hacen, el código copiará todo el contenido de la matriz "inventory_array" a la matriz "item_array".
