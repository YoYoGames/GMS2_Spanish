async_load


Sintaxis:

async_load;


Devoluciones: Real


Descripción

Esta variable tiene un alcance global y se usa para contener un mapa de DS cuando se usa en los eventos asíncronos, y -1 en todos los demás momentos. El contenido real del mapa de DS dependerá del tipo de evento asincrónico en el que esté poblado y de la función que se usó para desencadenar el evento, así que consulte las páginas individuales para esos eventos.


Ejemplo:

if ds_map_find_value(async_load, "id") == back
   {
   if ds_map_find_value(async_load, "status") >= 0
      {
      background_index[0] = back
      }
   }

El código anterior se toma del Evento Asíncrono de Image Loaded y primero verificará la ID del async_load El mapa de DS que se ha rellenado, luego verifica el estado de la devolución de llamada. Si el valor es mayor o igual a 0 (éxito de señalización), el resultado de la devolución de llamada se usará para establecer el índice de fondo en la imagen recién cargada.
