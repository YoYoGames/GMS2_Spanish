salud (obsoleta)


Sintaxis:

health;


Devoluciones: Real


Descripción

Esta variable tiene un alcance global y se usa para mantener un valor numérico que generalmente se usa para la salud del jugador. Tenga en cuenta que este no siempre es el caso; solo porque se llame "salud" no significa que deba usarse para la salud... puede contener cualquier valor real o entero que desee para su juego.

¡IMPORTANTE! Esta variable solo se incluye para admitir proyectos heredados de versiones anteriores de GameMaker y no debe utilizarse en proyectos nuevos, ya que está en desuso y en futuras versiones o actualizaciones se pueden eliminar por completo.


Ejemplo:

if place_meeting(x, y, obj_Enemy)
   {
   health -= 10;
   if health <= 0 instance_destroy();
   }

El código anterior verifica la instancia actual para ver si hay una colisión con la instancia del objeto indexado en "obj_Enemy", y si lo hay, elimina 10 puntos de la variable de salud global y luego verifica este valor para ver si es menos de o igual a 0. Si es la instancia se destruye a sí misma.
