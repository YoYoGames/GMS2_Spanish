Visión general del lenguaje GameMaker


Esta sección del manual contiene toda la información necesaria para comprender y utilizar el lenguaje GameMaker Studio 2 scripting GameMaker Studio 2 GML. El código en este idioma se puede usar para crear tus juegos y se agrega a objetos desde el Editor de objetos, normalmente cuando has elegido crear un nuevo Proyecto de idioma de GameMaker, aunque también se puede usar junto con DnD ™ (arrastrar y soltar) ) A continuación puede ver una imagen típica de un objeto con el editor de código abierto en un evento: Cada evento tiene su propia pestaña en el editor y puede agregar, editar o eliminar el código de ellos en cualquier momento (para obtener más información sobre eventos, consulte Eventos de objetos ). El código en sí debe tener una estructura básica y puede contener índices de recursos, variables, funciones, expresiones, palabras clave... etc.... todos los cuales se explican en las siguientes secciones. Si usted es un novato en programar o hacer el cambio de DnD ™, se recomienda que comience con la página en Estructura de código básica y luego lea todos, probando el código de cada una de las secciones dentro de GameMaker Studio 2.
