Asignaciones


Una tarea simplemente almacena un valor en una variable y tiene la forma:

<variable> = <expression>;

Una expresión puede ser un valor simple pero también puede ser más complicado, por lo que, en lugar de asignar un valor a una variable, también se puede agregar un valor al valor actual de la variable usando + =, por ejemplo:

a = 100;   // Assigning a simple value
b = 200;
c = 300;
a += b;    // Assigning with operation
a = b + c; // Assigning with expression

De manera similar, puede restar usando - =, multiplicar usando * =, dividir usando / =, o usar operadores bit a bit usando | =, & =, o ^ =. También puede agregar o restar uno de un valor usando ++, -. Para obtener más información, consulte la sección sobre Expresiones.

Tenga en cuenta que no puede hacer lo siguiente (o cualquier variación):

var a, b, c;
a = b = c = 4;

Y en su lugar, debe hacerse como:

a = 4;
b = 4;
c = 4;
