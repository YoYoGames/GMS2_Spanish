variable_instance_get_names


Descripción

Con esta función, puede recuperar una matriz rellenada con los nombres de las variables de instancia para una instancia, o las variables globales para un juego. Cuando se pasa un valor de ID de instancia, cada entrada en la matriz será una cadena del nombre de variable que corresponde a una variable de ámbito de instancia que se ha creado en la instancia. Sin embargo si pasas en la palabra clave global Cada entrada en la matriz será una cadena del nombre de la variable que corresponde a una variable de alcance global.

NOTA: Esta función es principalmente para uso con secuencias de comandos de compatibilidad para proyectos importados y el código convertido para las acciones de arrastrar y soltar.


Sintaxis:

variable_instance_get_names(instance_id/global);


Argumentos:

Argumento Descripción
instance_id/global El valor de ID único de la instancia para verificar o la palabra clave global

Devoluciones:

Array (each entry is a string)


Ejemplo:

var str = "";
var array = variable_instance_get_names(id);
show_debug_message("Variables for " + object_get_name(object_index) + string(id));
for (var i = 0; i < array_length_1D(array); i++;)
    {
    str = array[i] + ":" + string(variable_instance_get(id, array[i]));
    show_debug_message(str);
    }

El código anterior recuperará una matriz de todas las variables de alcance de instancia para la instancia que ejecuta el bloque de código y luego las mostrará junto con sus valores en la salida de depuración.
