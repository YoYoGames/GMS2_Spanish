variable_instance_set


Descripción

Con esta función, puede establecer el valor de una variable determinada en una instancia. Proporciona el valor único de ID de instancia (que se puede encontrar en Propiedades de instancia en el editor de sala o se devuelve cuando llama a la función instance_create_layer ), así como el nombre de la variable para establecer el valor de una cadena (ver código de ejemplo a continuación), y finalmente el valor para establecer (puede ser cualquier tipo de datos válido). Si la variable ya no existe en la instancia, se creará y se le asignará el valor.

NOTA: Esta función se usa principalmente con las secuencias de comandos de compatibilidad para proyectos importados y el código convertido para las acciones de arrastrar y soltar.


Sintaxis:

variable_instance_set(instance_id, name, val);


Argumentos:

Argumento Descripción
instance_id El valor de ID único de la instancia para usar
name El nombre de la variable a establecer (como una cadena)
val El valor para establecer la variable a

Devoluciones:

N/A


Ejemplo:

if !variable_instance_exists(id, "shields")
   {
   variable_instance_set(id, "shields", 0);
   }

El código anterior verificará si existe una variable de instancia en la instancia de llamada y si no lo está, se creará y establecerá en 0.
