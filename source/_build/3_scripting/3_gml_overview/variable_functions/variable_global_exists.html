variable_global_exists


Descripción

Con esta función puede verificar si existe una variable de ámbito global o no. Proporciona el nombre de la variable global para verificar como una cadena (ver ejemplo de código a continuación) y la función volverá true si existe una variable global con el nombre de pila o false de otra manera.

NOTA: Esta función se usa principalmente con las secuencias de comandos de compatibilidad para proyectos importados y el código convertido para las acciones de arrastrar y soltar.


Sintaxis:

variable_global_exists(name);


Argumentos:

Argumento Descripción
name El nombre de la variable global a verificar (como una cadena)

Devoluciones:

Boolean


Ejemplo:

if !variable_global_exists("enemy_num")) global.enemy_num = instance_number(obj_Enemey_Parent);

El código anterior verificará si la variable global llamada "número_enemigo" existe y, en caso contrario, se creará y establecerá un valor.
