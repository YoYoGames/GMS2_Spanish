variable_instance_exists


Descripción

Con esta función puede verificar si existe una variable de ámbito de instancia o no. Proporciona el valor único de ID de instancia (que se puede encontrar en Propiedades de instancia en el editor de sala o se devuelve cuando llama a la función instance_create_layer ), así como el nombre de la variable para verificar como una cadena (ver código de ejemplo a continuación). La función volverá true si existe una variable con el nombre dado para la instancia y false de otra manera.

NOTA: Esta función se usa principalmente con las secuencias de comandos de compatibilidad para proyectos importados y el código convertido para las acciones de arrastrar y soltar.


Sintaxis:

variable_instance_exists(instance_id, name);


Argumentos:

Argumento Descripción
instance_id El valor único de ID de la instancia para verificar
name El nombre de la variable a verificar (como una cadena)

Devoluciones:

Boolean


Ejemplo:

if !variable_instance_exists(id, "shields")) shields = 0;

El código anterior verificará si la variable llamada "escudos" existe en la instancia que ejecuta el código y, si no lo está, se creará e inicializará a 0.
