tipo de


Descripción

Esta función devuelve el tipo de datos de cualquier variable dada como una cadena. Los posibles valores de retorno se enumeran en la tabla a continuación:

Cuerda Descripción
number La variable tiene un número real (punto flotante)
string La variable contiene una cadena
array La variable hace referencia a una matriz
bool La variable tiene un valor booleano ( true / false )
int32 La variable contiene un entero de 32 bits
int64 La variable contiene un entero de 64 bits
ptr La variable tiene un puntero
undefined La variable no está definida
null La variable tiene un valor nulo (esto no debería verse normalmente)
vec3 La variable tiene un vector de 3 valores
vec4 La variable tiene un vector de 4 valores
unknown El valor es desconocido. Esto nunca debe verse y significa que algo ha salido mal en el nivel más básico, como una sobrescritura de memoria


Sintaxis:

typeof(variable);


Argumentos:

Argumento Descripción
variable La variable para obtener el tipo de datos de.


Devoluciones:

String (see table, above)


Ejemplo:

data_str = typeof(global.ExtensionInput);

El código anterior obtiene el tipo de datos que posee la variable global dada y devuelve la cadena a una variable de instancia.
