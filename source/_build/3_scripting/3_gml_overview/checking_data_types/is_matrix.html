is_matrix


Descripción

Esta función devuelve si una variable determinada se refiere a una matriz o no. En algunos casos, quiere verificar y ver qué tipo de datos tiene una variable en GameMaker Studio 2 y es entonces cuando usaría esta función. Vuelve true o false dependiendo de si el valor es una matriz o no. Para obtener más información sobre los tipos de datos, consulte aquí, y para obtener más información acerca de las matrices, consulte la sección sobre Funciones de la matriz.


Sintaxis:

is_matrix(n);


Argumentos:

Argumento Descripción
n El argumento para verificar


Devoluciones:

Boolean


Ejemplo:

if !is_matrix(val)
   {
   show_debug_message("Not a valid matrix!");
   }

El código anterior verifica la variable "val" para ver si contiene una matriz y si no lo está, muestra un mensaje en la consola de depuración.
