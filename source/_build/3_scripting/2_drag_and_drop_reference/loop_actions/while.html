Descripción

La acción While se usa para realizar bucles que solo finalizan cuando regresa una condición específica true. Por ejemplo, puede usar una variable y configurarla para false luego ejecute el ciclo While y haga que verifique esta variable, luego, en algún momento del ciclo, configure la variable como true y el ciclo terminará. Tenga en cuenta que no importa dónde establezca la condición dentro del ciclo while, todas las acciones dentro del ciclo se ejecutarán hasta el final antes de que el resto del script o evento continúe, cambiar la condición no saldrá instantáneamente del ciclo.

Cuando agrega una acción While a su código, debe proporcionar la variable para verificar, qué tipo de comprobación realizar (menor que, mayor que, igual, etc.) y el valor para verificar la variable (usted también puede marcar para que no se compruebe el ciclo While para ver si la condición no es verdadera). A continuación, agregue las acciones que se realizarán dentro del ciclo While, del mismo modo que agregaría una acción a un "si", es decir, colocándola al costado de la acción en lugar de debajo de ella:

Todas las acciones añadidas al lado se incluirán en el While y se realizarán antes de que el evento o script continúe con las acciones agregadas en (como se muestra en el ejemplo a continuación). Tenga en cuenta que puede finalizar cualquier ciclo prematuramente si llama a la acción Interrupción.
