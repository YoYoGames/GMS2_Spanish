Biblioteca Loop


La biblioteca de acciones Loop contiene aquellas acciones que se requieren para crear bucles en el código de acción del juego. Un bucle es simplemente una forma de iterar sobre el mismo conjunto de acciones 1 o más veces dependiendo de ciertas condiciones y es útil en cualquier situación donde necesite repetir las mismas acciones una y otra vez.

La biblioteca Loop tiene las siguientes acciones:

Lazo
Repetir
Mientras
por
Descanso
