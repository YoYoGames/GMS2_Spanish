Repetir


Descripción

Esta acción repetirá cualquier acción agregada al número de veces especificado. Usted da el número de veces que las acciones se van a repetir y luego agrega la acción al ciclo de repetición, de forma muy similar a como agregaría una acción a un "si", es decir, colocándolo al lado de la acción en lugar de hacerlo debajo:

Todas las acciones añadidas al lado se incluirán en la repetición y se realizarán antes de que el evento o el guión continúe con las acciones agregadas en (como se muestra en el ejemplo a continuación). Tenga en cuenta que puede finalizar cualquier ciclo prematuramente si llama a la acción Interrupción.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Times El número de veces para repetir (un valor entero entero)


Ejemplo:

El código de bloque de acción anterior obtiene el número de elementos en una lista y almacena el valor en una variable local temporal. Esta variable se usa para establecer el número de veces que se ejecuta un ciclo de repetición, y en el ciclo de repetición las entradas de la lista se leen en otra variable temporal que luego se usa para destruir las instancias a las que hacen referencia. Una vez que el ciclo ha terminado, la estructura de datos se libera.
