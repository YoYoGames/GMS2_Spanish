Gamepad Get Axis


Descripción

Puedes usar esta acción para obtener el valor de los diferentes ejes de un gamepad dado. Proporcionas el índice de gamepad (este es el número de "slot" que ocupa un gamepad) y seleccionas el eje para verificar desde el stick izquierdo o derecho, horizontal o vertical. El valor devuelto estará entre -1 y 1 para cada uno de los ejes horizontales y verticales disponibles: -1 es izquierda y 1 es derecha para el eje horizontal, y -1 está arriba y 1 está abajo para el eje vertical. Este valor devuelto se almacenará en la variable objetivo que usted suministra para uso futuro, que puede marcar como una variable "Temp", lo que significa que la acción creará una nueva variable temporal (local) para mantener el valor de retorno para usted, como se muestra en el ejemplo a continuación.

Argumentos:

Argumento Descripción
Gamepad El índice de gamepad.
