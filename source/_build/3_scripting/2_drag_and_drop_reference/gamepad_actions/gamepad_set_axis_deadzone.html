Establecer Gamepad Axis Deadzone


Descripción

Esta acción se puede usar para establecer la "zona muerta" del eje stick del gamepad. Especifica el índice de gamepad para establecer, y luego establece un valor de 0 a 1 y si la cantidad de entrada es menor que el valor dado, se considera que el eje stick del gamepad está en 0. Note que este es un ajuste global que afectará todos los ejes de la palanca conectados.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Gamepad El índice de gamepad.
Deadzone El valor de la zona muerta (0 - 1)


Ejemplo:

El código de bloque de acción anterior establece la zona muerta para cuatro índices de gamepad en 0.2.
