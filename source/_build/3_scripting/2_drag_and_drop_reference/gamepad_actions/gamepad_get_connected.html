Gamepad Get Connected


Descripción

Puedes usar esta acción para verificar y ver si una "ranura" de gamepad determinada tiene un gamepad conectado. Proporciona el índice de gamepad (este es el número de "slot" que ocupa un gamepad) y el valor devuelto se almacenará en la variable de destino que suministre para uso futuro (esto puede marcarse como una variable local temporal que cree para este evento o script).


Sintaxis de acción:


Argumentos:

Argumento Descripción
Gamepad El índice de gamepad para verificar.
Target La variable de destino para almacenar el valor devuelto en


Ejemplo:

El código de bloque de acción anterior ejecuta un for bucle para verificar y ver si un gamepad está conectado a cualquiera de las 12 "ranuras" disponibles. Si se descubre un gamepad, se establece una variable global en el número de ranura actual y el bucle se rompe.
