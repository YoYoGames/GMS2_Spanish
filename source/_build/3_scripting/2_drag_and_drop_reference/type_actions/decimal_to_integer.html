Decimal a Entero


Descripción

Esta acción tomará un número decimal y lo convertirá en un número entero. Usted da el valor para convertir y luego selecciona el tipo de redondeo que se debe hacer para convertirlo en un valor integr. Los tipos de redondeo disponibles son:

  • Redondo: toma un número decimal y lo redondea hacia arriba o hacia abajo al número entero más cercano. En el caso especial donde el número proporcionado es exactamente la mitad de un número entero (por ejemplo, 1.5, 17.5, -2.5, etc.), el número se redondeará al valor par más cercano, por ejemplo, 2.5 se redondeará a 2, mientras que 3.5 se redondearía a 4. Este tipo de redondeo se denomina redondeo bancario y en un gran número de iteraciones o al usar matemática de coma flotante, da un redondeo estadísticamente mejor que el más tradicional "redondear si más de.5 y redondear hacia abajo" "acercamiento"

  • Ceil: Esto tomará un valor decimal y se redondea al número entero más cercano. A diferencia de round (arriba) esto no toma en consideración la parte decimal del valor, por lo que un número como 5.0000001 se redondeará hasta 6. Vale la pena señalar que si está utilizando la acción aleatoria Obtenga un número aleatorio para generar un valor aleatorio valor decimal y luego llamar a esta acción para redondearlo, entonces puede obtener resultados inesperados. La acción aleatoria podría devolver 0, y usar "Ceil" en eso no dará 1 sino 0, así que si obtienes un decimal aleatorio de 0 a 9 y usas esta función, puedes esperar obtener enteros de 1 a 10, pero en la realidad podría obtener números enteros de 0 a 10. Esta es una posibilidad remota, pero debe tenerse en cuenta al usar esta acción.

  • Superficie: Esto tomará un valor decimal y se redondea hacia abajo al valor entero más cercano. A diferencia de la ronda (arriba), esto no tiene en cuenta la parte decimal del valor, por lo que un número como 5.9999999 seguirá siendo reducido a 5.

Una vez que haya seleccionado el tipo de redondeo, debe proporcionar el número a redondear (esto puede ser una variable), y luego dar una variable objetivo para devolver el nuevo valor entero (que puede marcarse como una variable local temporal).


Sintaxis de acción:


Argumentos:

Argumento Descripción
Rounding El tipo de redondeo a usar para obtener el número entero (ver descripción arriba)
Number El número decimal para redondear a un número entero
Target La variable de destino para almacenar el número entero devuelto
