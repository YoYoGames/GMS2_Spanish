Cadena al número


Descripción

Esta acción puede convertir una cadena en un número. Proporcione la cadena para cambiar y luego proporcione una variable objetivo para devolver el número real (esta variable se puede marcar como una variable local temporal). Tenga en cuenta que si la cadena tiene caracteres que no sean números y un punto decimal, no se convertirá, aunque si la cadena comienza con un número, se convertirán y se excluirá el resto de la cadena. Es importante tener en cuenta que al pasar decimales a esta acción, la cadena devuelta se redondeará a dos decimales solamente. A continuación se muestra una tabla de valores de entrada y lo que esperaría devuelto

Cuerda Valor devuelto
"123" 123
"123 Hola mundo" 123
"Hola Mundo" 0 (de forma predeterminada, si la cadena no se puede convertir, se devolverá 0)
"Hola mundo 123" 0
"123.45678" 123.46


Sintaxis de acción:


Argumentos:

Argumento Descripción
Value La cadena para convertir
Target La variable objetivo para mantener el valor devuelto


Ejemplo:

El código de bloque de acción anterior inicializa una variable global y luego abre un ini archivo. Se lee una cadena del archivo y se almacena en una variable temporal antes de convertirla en un número real y asignarla a la variable global. los ini luego se cierra
