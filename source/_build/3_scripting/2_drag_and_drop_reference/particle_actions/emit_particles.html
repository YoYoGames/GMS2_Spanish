Emitir partículas


Descripción

Esta acción se usa para crear partículas desde un emisor. Suministra la ID del sistema de partículas al que pertenece el emisor (como lo devuelve la acción Crear sistema de partículas ) y luego la ID del emisor real (como lo devuelve la acción Crear el emisor de partículas ) y finalmente el tipo de partículas para crear (tal como lo devuelve el acción Crear tipo de partícula ). Luego puede seleccionar "Ráfaga" o "Transmitir" las partículas del emisor, así como también dar el número que desea crear.

Cuando elige transmitir partículas, no necesita colocar esta acción en un evento de marco de juego como el evento de paso o el evento Dibujar, sino que simplemente lo llama una vez en el evento Crear (por ejemplo) y el emisor transmitirá partículas cada marco del juego hasta que vuelva a llamar a esta acción para detenerlo (estableciendo el valor de conteo en 0). Si eliges reventar las partículas, puedes llamar esto en cualquier momento que elijas y las partículas se crearán una vez.


Sintaxis de acción:


Argumentos:

Argumento Descripción
System La ID del sistema en el que está el emisor
Emitter La ID del emisor para usar
Type ID del tipo de partícula para crear
Event El tipo de evento para las partículas, ya sea "Ráfaga" o "Ráfaga"
Count El número de partículas para reventar o transmitir

El código de bloque de acción anterior creará un emisor de partículas, establecerá su región y luego lo configurará para emitir una corriente de partículas.
