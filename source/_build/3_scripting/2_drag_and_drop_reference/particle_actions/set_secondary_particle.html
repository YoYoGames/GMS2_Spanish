Establecer Partícula Secundaria


Descripción

Esta acción le permite usar un segundo tipo de partículas junto con la partícula primaria. Usted proporciona el valor de ID del tipo de partícula primaria (que debe ser el mismo que el utilizado al crear la partícula utilizando la acción Crear tipo de partícula ) y luego puede elegir que emita un tipo de partícula secundaria en cada cuadro de juego que exista. / o al final de la vida útil de cada partícula primaria. Debe dar otro valor de ID de partícula para el "Secundario" y luego un número que se creará en el "Conteo".

Si el Evento seleccionado es "Por Paso", entonces cada cuadro de juego que una partícula primaria exista en la pantalla emitirá el número dado de partículas secundarias; por ejemplo, podría tener un cohete con partículas de "llama", y cada partícula de llama emitiría una partícula secundaria de "humo". Si el evento está configurado como "Al morir", las partículas secundarias ahora solo se emitirán cuando una partícula del tipo primario llegue al final de su vida útil; por ejemplo, podría tener una partícula de "explosión" que emite un "polvo". "partícula al final de su vida".

Es importante tener en cuenta que no se puede usar el mismo ID de partícula como partícula primaria que para las partículas secundarias, y que puede encontrar rápidamente problemas de rendimiento al usar esta acción, ya que es fácil crear miles de partículas muy rápidamente de esta manera, por lo que Tenga cuidado de limitar la vida útil y la cantidad de partículas creadas por los tipos de partículas primarias y secundarias. También tenga en cuenta que puede asignar una partícula secundaria a cualquier tipo de partícula y así puede crear "cadenas de partículas" donde una partícula crea otra en cada paso y luego esta partícula crea otra cuando alcanza el final de su vida útil, etc... Aunque, de nuevo se debe tener cuidado al encadenar partículas de esta manera para asegurarse de que el rendimiento no sufra. También puede llamar a esta acción dos veces para un solo tipo de partícula y hacer que se transmita en un paso así como en ráfaga al morir.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Type ID El valor de ID para el tipo de partícula para dar la partícula secundaria a
Event Cuándo crear la partícula secundaria, ya sea "Por paso" o "Al morir"
Death Type El valor de ID para el tipo de partícula para emitir en el evento secundario elegido
Count La cantidad de partículas secundarias que se crearán en el Evento secundario elegido
