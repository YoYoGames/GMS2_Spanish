Destruye el sistema de partículas


Descripción

Esta acción "destruirá" el sistema de partículas dado (es decir, liberará la memoria utilizada por el sistema de partículas). Proporciona el valor de ID único para que el sistema lo destruya, tal como se devolvió cuando creó el sistema con la acción Crear sistema de partículas. Se debe llamar a esta acción siempre que ya no necesite un sistema de partículas en su juego, o cuando quiera volver a crear el sistema de partículas (por ejemplo, justo antes de llamar a un Juego de reinicio ). Tenga en cuenta que si el sistema de partículas no se ha marcado como persistente, entonces no necesita llamar a esta acción, ya que el sistema de partículas (y cualquier emisor asociado a él) se limpiará automáticamente en el extremo de la habitación.

¡IMPORTANTE! Esta función también destruirá y eliminará cualquier emisor que se haya creado y asociado con el sistema que se está destruyendo (consulte Crear un emisor de partículas para obtener más información).


Sintaxis de acción:


Argumentos:

Argumento Descripción
System El valor de ID exclusivo para el sistema que se destruirá


Ejemplo:

El código de bloque de acción anterior comprobará si se está presionando la tecla "Escape" y, si se detecta, eliminará el sistema de partículas y los diferentes tipos de partículas definidos en las variables globales dadas y luego reiniciará el juego.
