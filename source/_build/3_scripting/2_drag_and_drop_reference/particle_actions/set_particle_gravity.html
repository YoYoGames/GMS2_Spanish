Establecer la gravedad de partículas


Descripción

Esta acción simulará el efecto de la gravedad en todas las partículas del tipo dado. Primero debe especificar el valor de ID del tipo de partícula (que debe ser el mismo que el valor utilizado cuando creó la partícula usando la acción Crear tipo de partícula ) y luego puede establecer la fuerza de gravedad y la dirección. La fuerza es un valor acumulativo y se agregará al vector de velocidad en cada cuadro de juego, por lo que una gravedad de 1 agregará uno al valor de velocidad de cada cuadro de juego, mientras que el valor de dirección es simplemente la dirección en la que la gravedad debería "tirar" partícula. Tenga en cuenta que tanto para la fuerza como para la dirección puede usar valores negativos, y esa dirección en GameMaker Studio 2 se calcula con 0 ° hacia la derecha, 90 ° hacia arriba, 180 ° hacia la izquierda y 270 ° hacia abajo.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Type ID El valor de ID para el tipo de partícula para establecer la gravedad de
Force La fuerza de la gravedad (en píxeles por cuadro de juego)
Direction La dirección de la gravedad "tira"


Ejemplo:

El código de bloque de acción anterior creará un nuevo tipo de partícula y asignará su valor único de ID a una variable global. Luego procede a establecer todas las propiedades para el tipo de partícula.
