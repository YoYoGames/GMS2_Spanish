Establecer color de partícula


Descripción

Esta acción establecerá el color o los colores que se usarán cuando se cree su sprite y durante su vida útil. Primero debe especificar el valor de ID del tipo de partícula (que debe ser el mismo que el valor devuelto cuando creó la partícula utilizando la acción Crear tipo de partícula ) y luego puede establecer los colores para usar.

Puedes elegir tres colores para usar, y el sprite se mezclará entre ellos durante la vida de la partícula. Por ejemplo, si una partícula tiene colores rojo, verde y amarillo, y una vida de 60 marcos de juego, cuando se cree (cuadro 0) se tiñe de rojo, luego se desvanecerá hacia verde a la mitad de su vida útil ( cuadro 30), antes de desvanecerse en amarillo al final de su vida (cuadro 60). Si está utilizando un sprite personalizado en lugar de una forma de partícula, cualquier color que no sea blanco se combinará con el sprite utilizado.

NOTA: El componente alfa del color no se usa, solo los componentes RGB del selector de color. Para configurar el alfa usa la acción Set Particle Alpha.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Type ID El valor de ID exclusivo para el tipo de partícula para establecer el color de
Start El color inicial para usar
Middle El color intermedio para usar
End El color final para usar


Ejemplo:

El código de bloque de acción anterior creará un nuevo tipo de partícula y asignará su valor único de ID a una variable global. Luego procede a establecer todas las propiedades para el tipo de partícula.
