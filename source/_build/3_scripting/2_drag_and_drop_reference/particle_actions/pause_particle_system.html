Pause Particle System


Descripción

Esta acción pondrá en pausa el sistema de partículas dado. Usted proporciona el valor de identificación único para que el sistema pause, tal como se devolvió cuando creó el sistema con la acción Crear sistema de partículas. Un sistema de partículas en pausa seguirá dibujando las partículas que se han generado, pero ya no actualizarán su color, posición o alfa, ni completarán su vida útil y desaparecerán hasta que vuelvas a pausar el sistema. Tenga en cuenta que esto no pausa los emisores que están transmitiendo partículas, ni detendrá la explosión de un emisor de partículas; los emisores seguirán creando partículas cuando se los llama, pero cuando el sistema está en pausa, las partículas no harán nada y se dibujarán exactamente donde se crearon hasta que reinicie el estado en pausa o llame a Update Particle System.

Argumentos:
