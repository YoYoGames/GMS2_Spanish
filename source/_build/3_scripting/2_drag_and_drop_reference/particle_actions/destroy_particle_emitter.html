Destruye el Emisor de Partículas


Descripción

Esta acción "destruirá" el emisor de partículas dado, es decir: liberará la memoria utilizada por el emisor de partículas. Usted suministra la ID del sistema de partículas al que pertenece el emisor (como lo devuelve la acción Crear sistema de partículas ) y luego la ID del emisor real (como lo devuelve la acción Crear el emisor de partículas ). Se debe llamar a esta acción siempre que ya no necesite un emisor de partículas en su juego, o siempre que desee recrear el emisor de partículas (por ejemplo, justo antes de llamar a un reinicio del juego ). Tenga en cuenta que si el sistema de partículas al que se le asignó el emisor no ha sido marcado como Persistente, entonces no necesita llamar a esta acción, ya que el emisor se limpiará automáticamente en el extremo de la sala junto con el sistema de partículas.


Sintaxis de acción:


Argumentos:

Argumento Descripción
System El valor de ID del sistema que contiene el emisor
Emitter El valor de ID del emisor para destruir


Ejemplo:

El código de bloque de acción anterior comprobará una variable de instancia y, si es superior a 100, destruirá el emisor asignado a la instancia y luego irá a la habitación contigua.
