Crear un emisor de partículas


Descripción

Para poder ver tus partículas necesitas haber creado primero un Sistema de Partículas, luego haber definido algunos tipos de partículas y finalmente crear un emisor de partículas usando esta acción. Para crear el emisor, llame a esta acción y asigne el emisor a un sistema de partículas que haya creado previamente (consulte Sistema de partículas ) y luego proporcione una variable de destino para almacenar su valor único de ID para usar en otras acciones. Puede elegir marcar esta variable como una variable local temporal, pero eso significará que se descarta al final del script o evento de acción, lo que significa que el emisor seguirá existiendo, pero le dejará sin medios para acceder a él, por lo que se debe tener cuidado al usar esto (en general, los emisores nunca necesitarán almacenarse en una variable local temporal, pero para ciertas cosas, como extraer partículas a una superficie, puede ser necesario).

¡IMPORTANTE! Si su sistema de partículas es persistente, entonces debe destruir todos los emisores al final de cada habitación usando Destruir el Emisor de Partículas y volver a crearlos en la habitación contigua según sea necesario, o almacenar sus valores de ID en variables globales para que se pueda acceder en cualquier momento en cualquier habitación, o que los haya creado un objeto persistente, para que el objeto persista y pueda acceder a ellos a través de variables de instancia. También tenga en cuenta que si llama a la acción Reiniciar juego, debe destruir el sistema de partículas de antemano para que el sistema y los emisores asignados se eliminen de la memoria antes de volver a comenzar. Si no se eliminan adecuadamente los emisores de la memoria del sistema como se describe anteriormente, se producirá una pérdida de memoria. Esto sucede cuando un recurso se agrega a la memoria y luego se "desreferencia", lo que significa que GameMaker Studio 2 ya no tiene forma de interactuar con ese recurso. Esto lleva a que la memoria esté obstruida por recursos inaccesibles que a su vez pueden causar problemas con su juego.
