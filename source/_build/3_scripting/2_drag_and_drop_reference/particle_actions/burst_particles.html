Explosión de partículas


Descripción

Esta acción le permite reventar partículas en cualquier momento sin depender de un emisor. Proporciona el valor de ID exclusivo para que el sistema cree las partículas (como se devolvió cuando creó el sistema con la acción Crear sistema de partículas ) y la ID única del tipo de partículas para crear (como se devolvió cuando creó el tipo de partícula con el acción Crear tipo de partícula ), así como la posición x / y para crearlos en y el número para crear. La posición puede ser una posición absoluta dentro de la sala, o puede marcarla como una posición relativa a la instancia que llama a la acción.

Puedes llamar a esta acción cada cuadro de juego para replicar el efecto de transmisión de un emisor, o puedes invocarlo una vez en un evento (como el Destruir Evento) para simplemente estallar las partículas.


Sintaxis de acción:


Argumentos:

Argumento Descripción
System El valor de ID del sistema de partículas para usar
Type El valor de ID del tipo de partícula para reventar
X La posición x para reventar las partículas en
Y Posicionan para reventar las partículas en
Count La cantidad de partículas a reventar

El código de bloque de acción anterior verificará si la instancia ha cambiado de posición desde que se actualizó el último cuadro de juego y, si lo tiene, estallará algunas partículas en la nueva posición.
