Establecer el tamaño de partícula


Descripción

Esta acción se usa para establecer el tamaño de partícula, que puede ser estático o establecerse para aumentar / disminuir a lo largo del tiempo. Proporciona el valor de ID único para el tipo para establecer el tamaño (como se devolvió cuando creó el tipo con la acción Crear tipo de partícula ) y luego da un rango de tamaños de inicio inicial, luego el aumento y finalmente el "meneo". El tamaño en el que se crea una partícula es un valor escalar, donde 1 es del mismo tamaño que la imagen de partícula base, 0.5 sería la mitad del tamaño de la imagen base y un valor de 2 sería el doble. Si los tamaños mínimo y máximo no son iguales, entonces el tamaño inicial para cualquier partícula creada será un valor aleatorio entre ellos.

La opción de aumentar un tamaño de partícula le permite ingresar un valor que se agregará al valor escalar base en cada cuadro de juego de la vida de la partícula. Por lo tanto, establecer este valor en, por ejemplo, 0.1 agregará 0.1 al valor de escala en cada cuadro de juego. Tenga en cuenta que también puede usar valores negativos para hacer que la imagen de la partícula se reduzca también. Tenga en cuenta que la disminución de escala mínima que se acepta es -0.01 y el incremento mínimo es 0.01.

Finalmente puede configurar la opción "wiggle". Este es un valor que se generará aleatoriamente entre 0 y el valor que usted da y luego se suma o resta al tamaño de cada paso de la duración de las partículas, lo que hace que la imagen dibujada fluctúe en tamaño. Obviamente, los valores más grandes son más pronunciados que los más pequeños, y este valor puede ser incluso negativo, con un rango máximo entre -20 y 20.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Type El valor de ID exclusivo para el tipo de partícula para establecer el tamaño de
Min La escala mínima para el tipo de partícula (por defecto 0.1)
Max La escala máxima para el tipo de partícula (por defecto 1)
Increment El valor para agregar al tamaño de tipo de partícula en cada cuadro de juego (puede ser negativo, mínimo +/- 0.01)
Wiggle El valor máximo para sumar o restar aleatoriamente del tamaño de partículas por paso (+/- 20).
