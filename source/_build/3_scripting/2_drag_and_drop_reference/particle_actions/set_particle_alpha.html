Establecer Particle Alpha


Descripción

Esta acción establecerá el alfa para cuando crees la partícula así como también el alfa durante su vida útil. Primero debe especificar el valor ID de tipo de partícula (que debe ser el mismo que el valor devuelto cuando creó la partícula mediante la acción Crear tipo de partícula ) y luego puede establecer los valores alfa a usar.

Puedes elegir tres alfas para usar, y la partícula se mezclará entre ellos durante la vida de la partícula. Por ejemplo, si una partícula tiene 0.5, 1 y 0 como valores alfa, y una vida útil de 60 cuadros de juego, entonces cuando se crea (cuadro 0) lo hará a 0.5 alfa, luego se desvanecerá hacia un alfa completo de 1 (cuadro 30), antes de desvanecerse para ser invisible con un alfa de 0 (cuadro 60).

Argumentos:

Argumento Descripción
Type ID El valor de ID exclusivo para el tipo de partícula para establecer el color de
Start El alfa inicial para usar
Middle El alfa intermedio para usar
End El alfa final para usar
