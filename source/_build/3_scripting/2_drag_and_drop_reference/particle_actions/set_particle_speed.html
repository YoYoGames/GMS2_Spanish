Establecer velocidad de partícula


Descripción

Esta acción se usa para establecer la velocidad de la partícula, que puede ser estática o establecerse para aumentar / disminuir con el tiempo. Proporciona el valor de identificación único para el tipo para establecer el tamaño (como se devolvió cuando creó el tipo con la acción Crear tipo de partícula ) y luego da un rango de velocidades de inicio inicial, luego la cantidad de aumento y finalmente el "meneo". La velocidad a la que se crea una partícula se calcula como píxeles por cuadro de juego, por lo que una configuración mínima de 2 y una configuración máxima de 5 emitirá partículas que se mueven entre 2 y 5 píxeles por cuadro de juego.

La opción para aumentar la velocidad de una partícula le permite ingresar un valor que se agregará al valor de velocidad base en cada cuadro de juego de la vida útil de la partícula. Por lo tanto, establecer este valor en, por ejemplo, 0.1 agregará 0.1 a la velocidad de cada cuadro de juego. Tenga en cuenta que también puede usar valores negativos para hacer que la partícula también disminuya, aunque la velocidad nunca pasará por debajo de 0. Tenga en cuenta que la disminución mínima de velocidad que se acepta es -0.01 y el incremento mínimo es 0.01.

Finalmente puede configurar la opción "wiggle". Este es un valor que se generará aleatoriamente entre 0 y el valor que usted da y luego se suma o se resta a la velocidad de cada paso de la vida de las partículas, lo que hace que la imagen dibujada fluctúe en velocidad. Obviamente, los valores más grandes son más pronunciados que los más pequeños, y este valor puede ser incluso negativo, con un rango máximo entre -20 y 20.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Type El valor de ID exclusivo para el tipo de partícula para establecer el tamaño de
Min La velocidad mínima para el tipo de partícula (por defecto 0.1)
Max La velocidad máxima para el tipo de partícula (por defecto 1)
Increment El valor para agregar a la velocidad de tipo de partícula en cada cuadro de juego (puede ser negativo, mínimo +/- 0.01)
Wiggle El valor máximo para sumarse o restarse al azar de la velocidad de las partículas por paso (+/- 20).
