Referencia de arrastrar y soltar


Esta sección del manual enumera todas las diferentes acciones disponibles para usted cuando usa Arrastrar y soltar para codificar su juego. A continuación puede encontrar cada una de las diferentes Bibliotecas de Acción disponibles desde el objeto DnD™ Toolbox, y dentro de cada sección de la biblioteca se explican las acciones individuales, junto con ejemplos de uso.

  2. Acciones de Instancia
  3. Acciones de mouse y teclado
  4. Acciones de Gamepad

  6. Acciones de colisión
  7. Acciones de dibujo
  8. Acciones de mosaico
  9. Acciones de audio
  10. Acciones de bucle
  11. Cambiar acciones
  12. Acciones de estructura de datos
  13. Acciones de amortiguación
  14. Acciones de archivo
  15. Tipo de datos Acciones
  16. Acciones aleatorias
  17. Acciones de cámara
  18. Acciones en la habitación
  19. Acciones de ruta
  20. Acciones de línea de tiempo
  21. Acciones del juego
  22. Acciones de instancia de variable
  23. Acciones misceláneas
  24. Acciones de partículas

También tiene una biblioteca de Favoritos, que es una sección especial de la Caja de Herramientas donde puede arrastrar acciones y así mantener todas las que usa más útiles, y la biblioteca de Uso Reciente, que contendrá las últimas 5 acciones que haya usado.
