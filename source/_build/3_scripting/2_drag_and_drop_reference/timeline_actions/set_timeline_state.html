Establecer el estado de la línea de tiempo


Descripción

Esta acción se puede utilizar para iniciar, detener, pausar o reanudar una línea de tiempo. La línea de tiempo primero debe haberse establecido para la instancia utilizando la acción Establecer escala de tiempo de la instancia, y luego puede usar esta acción en cualquier momento para cambiar su estado de ejecución de las siguientes opciones

Opción Descripción
comienzo Comience la línea de tiempo asignada a la instancia (si la línea de tiempo se detuvo, esto establecerá la posición de inicio en el momento 0 de nuevo).
Currículum Reanudar una línea de tiempo en pausa (si la línea de tiempo se detuvo y no se detuvo, no hará nada).
Pausado Pausa la línea de tiempo asignada a la instancia (mientras está en pausa, los momentos en una línea de tiempo no avanzarán con los cuadros del juego).
Detener Detener la línea de tiempo asignada a la instancia.


Sintaxis de acción:


Argumentos:

Argumento Descripción
State El estado para poner la línea de tiempo (ver la tabla de arriba)


Ejemplo:

El código de bloque de acción anterior verificará el valor mantenido en una variable global y si es true entonces la línea de tiempo está en pausa, y si es false entonces la línea de tiempo se reanudará.
