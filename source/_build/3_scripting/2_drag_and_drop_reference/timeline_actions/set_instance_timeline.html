Establecer la línea de tiempo de la instancia


Descripción

Esta acción asigna una línea de tiempo desde el Árbol de recursos a la instancia. Usted asigna la línea de tiempo para asignar, y luego puede elegir tener la línea de tiempo en bucle cuando llega al último momento (el siguiente fotograma después de que se ejecute el último momento volverá al primer momento de la línea de tiempo) y también si comienza la línea de tiempo de inmediato o no. Si no inicia la línea de tiempo inmediatamente, puede usar la acción Establecer estado de la línea de tiempo para iniciarla más adelante en otro evento o secuencia de comandos.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Timeline El recurso timleine para asignar a una instancia
Loop Marcar para tener el ciclo de la línea de tiempo después del último momento (el valor predeterminado está marcado)
Start Timeline Marcar para que la línea de tiempo comience inmediatamente o no (el valor predeterminado está marcado)


Ejemplo:

El código de bloque de acción anterior asignará una línea de tiempo a la instancia que llama a las acciones, así como establecerá el momento inicial en 10 y luego la velocidad de la línea de tiempo en 2.
