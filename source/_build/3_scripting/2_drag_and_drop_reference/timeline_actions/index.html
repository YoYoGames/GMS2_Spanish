Biblioteca de línea de tiempo


Las líneas de tiempo son un mecanismo poderoso para controlar los eventos en su juego, y se basan en "momentos", donde un "momento" es simplemente un paso de un solo juego. Estas líneas de tiempo generalmente se crean a partir del Árbol de recursos de manera similar a un objeto, es decir: usted crea una línea de tiempo en el Editor de línea de tiempo, luego en cada momento que lo requiere para realizar una acción agrega algún código o una acción DnD™ para para realizar.

Una vez que se ha creado un recurso de línea de tiempo, puede detenerlo, iniciarlo y manipularlo mediante las siguientes acciones:

Establecer la línea de tiempo de la instancia
Establecer el momento de la línea de tiempo
Establecer la velocidad de la línea de tiempo
Establecer el estado de la línea de tiempo
