Establecer el momento de la línea de tiempo


Descripción

Esta acción se usa para omitir una línea de tiempo del momento actual a uno nuevo. Las líneas de tiempo avanzan a través de diferentes "momentos", y cada momento típicamente ocupa un cuadro de juego (como el evento de paso de un objeto). Una vez que ha comenzado una línea de tiempo, progresará a través de los momentos hasta que llegue al final, en cuyo punto retornará al primer momento en la línea de tiempo o se detendrá por completo. Sin embargo, este enfoque lineal de las líneas de tiempo se puede modificar utilizando esta acción para establecer un momento diferente en cualquier momento, lo que le permite crear sus propios ciclos de línea de tiempo, o saltear secciones de una línea de tiempo en determinadas circunstancias, etc. Con esta acción simplemente establezca el momento para saltar dentro de la línea de tiempo y el próximo cuadro de juego se ejecutará en ese momento (tenga en cuenta que debe haber asignado una línea de tiempo a la instancia utilizando la acción Establecer línea de tiempo de la instancia ).
