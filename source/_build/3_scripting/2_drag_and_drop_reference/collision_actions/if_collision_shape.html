Si el objeto en


Descripción

Esta acción se usa para verificar y ver si hay una colisión con una instancia de un objeto dado dentro de un área determinada o a lo largo de una línea. Usted da la forma (rectángulo, elipse o línea), el objeto a verificar y luego proporciona las coordenadas para las esquinas superior izquierda e inferior derecha de la forma que se está utilizando, o en el caso de una línea, el inicio y el posiciones finales. Estas pueden ser posiciones absolutas dentro de la sala de juegos, o posiciones relativas a la instancia que realiza la acción. Puedes ver cómo funciona esto en la imagen a continuación:

Como puede ver, los resultados dependerán de la máscara de colisión de la instancia que se comprueba, lo que significa que la instancia del objeto que está buscando debe tener un sprite asignado al sprite_index, o un sprite asignado a mask_index. Si no lo hace, no será detectado por estas funciones, independientemente de si está dibujando un sprite o no. Vale la pena señalar que cualquier instancia que caiga completamente dentro de los límites establecidos para la elipse o la forma del rectángulo también devolverá una colisión.

La acción devolverá la palabra clave noone si no se devuelve una colisión, o devolverá el valor ID único de la instancia que se encuentra en colisión. También puede comprobar el "No" bandera para comprobar si no hay una colisión en la posición dada. Tenga en cuenta que si se encuentran múltiples instancias en colisión, se le devolverá un único valor de ID que podría corresponderse con cualquiera de las instancias (pero solo una de ellas). Puede marcar la variable de destino como una variable temporal (local), lo que significa que la acción creará esta variable para mantener el valor de retorno hasta el final del evento.

Argumentos:

Argumento Descripción
Shape La forma de usar
Object El objeto para verificar instancias de
x1 La posición x para comenzar desde
y1 La posición y para comenzar desde
x2 La posición x para finalizar en
y2 La posición y para terminar en
Target La variable a destino para el valor de retorno de la acción
