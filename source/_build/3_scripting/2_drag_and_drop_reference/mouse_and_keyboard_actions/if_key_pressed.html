Si presionó la tecla


Descripción

Con esta acción puede sondear el teclado para ver si alguna tecla está siendo presionada actualmente. La acción volverá true cada paso que se presiona (y mantiene presionada) la tecla del teclado, o false de lo contrario, aunque si hace clic en not modificador puede verificar si la tecla no se está presionando, es decir: la acción volverá true mientras no se presiona ninguna tecla false si hay. Si solo necesita verificar que se presione una sola tecla, use la acción si presiona la tecla.

Tenga en cuenta que para agregar acciones al bloque "if", deben soltarse al lado de la acción, como se muestra en la imagen siguiente:
Cualquier acción que se elimine en otro lugar se realizará después del bloque "if".


Sintaxis de acción:


Argumentos:

Argumento Descripción
Key El nombre de la clave para verificar
Not Negar el cheque (verdadero se convierte en falso y viceversa)


Ejemplo:

El código de bloque de acción anterior sondea el estado del teclado en cada paso y si se mantiene presionada una tecla, comprueba si la tecla inicial presiona hacia abajo. Si el cheque es true en la pulsación inicial inicial, se reproduce un sonido, luego, mientras se mantiene presionada la tecla, la mezcla de sprites se cambia a rojo. Si no se mantiene presionada la tecla, se realiza una comprobación en la liberación de la tecla para restablecer el color de la mezcla a blanco.
