Si se muestra el teclado virtual


Descripción

Esta acción se puede usar para verificar si el teclado virtual del SO se muestra o no. La acción evaluará a true cuando el teclado está abierto o abriéndose, o false Si el teclado virtual está cerrado o cerrando. Si habilitas la not Modificador, esta acción se invertirá y podrá almacenar la devolución. true / false valor en una variable de destino, que se puede marcar como temp (local).
