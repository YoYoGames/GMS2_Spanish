Si presiona el mouse


Descripción

Con esta acción puede sondear el mouse para ver si se está presionando algún botón. La acción volverá true cada paso que se presiona el botón del mouse (y se mantiene presionado) hacia abajo, o false de lo contrario, aunque si hace clic en not modificador puede verificar si el botón del mouse no se está presionando, es decir: la acción volverá true mientras no se presiona ningún botón false si hay. Si solo necesita comprobar si presiona un solo mouse, utilice la acción si mouse_down.

Tenga en cuenta que para agregar acciones al bloque "if", deben soltarse al lado de la acción, como se muestra en la imagen siguiente:

Estas acciones ahora se ejecutarán si el "si" se evalúa como true, mientras que cualquier acción que se elimine en otro lugar se realizará después del bloque "if".

Argumentos:

Argumento Descripción
Mouse Button El nombre del botón del mouse para verificar (izquierda, centro, derecha).
Not Negar el cheque (verdadero se convierte en falso y viceversa)


Ejemplo:

El código de bloque de acción anterior sondea el estado del botón del mouse en cada paso y, si se mantiene presionada, busca el mouse inicial presionado. Si la comprobación está en la pulsación inicial inicial, se reproduce un sonido, luego, mientras el botón está presionado, la mezcla de sprites cambia a rojo. Si el mouse no se mantiene presionado, se realiza una comprobación en el lanzamiento del mouse para restablecer el color de fusión a blanco.
