Si se lanza el mouse


Descripción

Esta acción busca cuándo se ha soltado un botón del mouse. Regresará true en el tick de un solo juego que el botón del mouse se lanza y false en todos los demás momentos. Si habilita el not modificador, esta acción se invertirá y verificará si el botón del mouse no se ha liberado, regresando false en el tic del juego que se ha lanzado y true en todos los demás momentos.

Tenga en cuenta que para agregar acciones al bloque "if", deben soltarse al lado de la acción, como se muestra en la imagen siguiente:

Estas acciones ahora se ejecutarán si el "si" se evalúa como true, mientras que cualquier acción que se elimine en otro lugar se realizará después del bloque "if".


Sintaxis de acción:


Argumentos:

Argumento Descripción
Mouse Button El nombre del botón del mouse para verificar (izquierda, centro, derecha).
Not Negar el cheque (verdadero se convierte en falso y viceversa)


Ejemplo:

El código de bloque de acción anterior sondea el estado del botón del mouse en cada paso y, si se mantiene presionada, busca el mouse inicial presionado. Si el cheque es true en la pulsación inicial inicial se reproduce un sonido, luego, mientras el botón está presionado, la mezcla de sprites cambia a rojo. Si el mouse no se mantiene presionado, se realiza una comprobación en el lanzamiento del mouse para restablecer el color de fusión a blanco.
