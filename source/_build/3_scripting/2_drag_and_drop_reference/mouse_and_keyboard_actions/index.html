Biblioteca de acciones de mouse y teclado


La biblioteca de acción del mouse y el teclado es donde puede encontrar las acciones necesarias para sondear y procesar la entrada desde el teclado o los dispositivos del mouse. Las acciones disponibles se enumeran a continuación:

Si presiona el mouse
Si Mouse Down
Si se lanza el mouse
Si el teclado está presionado
Si teclado abajo
Si el teclado se lanzó
