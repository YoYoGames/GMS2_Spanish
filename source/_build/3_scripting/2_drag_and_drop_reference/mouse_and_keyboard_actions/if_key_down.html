Si llave hacia abajo


Descripción

Con esta acción, puede sondear el teclado para ver si alguna tecla se está manteniendo presionada. La acción volverá. true cada paso que se presiona la tecla del teclado, o false De lo contrario, aunque si hace clic en el botón not modificador, puede verificar si la clave no está siendo de ayuda, es decir: la acción regresará true mientras que ninguna tecla está abajo y false si hay. Si solo necesita verificar una sola tecla, presione la tecla Si presionó la tecla Si.

Tenga en cuenta que para agregar acciones al bloque "si", se deben colocar al lado de la acción, como se muestra en la siguiente imagen:
Cualquier acción eliminada en otro lugar se realizará después del bloque "if".


Sintaxis de acción:


Argumentos:

Argumento Descripción
Key El nombre de la tecla del teclado para comprobar.
Not Negar el cheque (verdadero se vuelve falso y viceversa)


Ejemplo:

El código de bloqueo de acción anterior sondea el estado del teclado en cada paso y, si se mantiene presionada una tecla, verifica la tecla inicial presionada. Si el cheque es true en la presión inicial hacia abajo, se reproduce un sonido, luego, mientras se mantiene presionada la tecla, la mezcla de sprites cambia a rojo. Si la tecla no se mantiene presionada, se realiza una comprobación en la liberación de la tecla para restablecer el color de fusión a blanco.
