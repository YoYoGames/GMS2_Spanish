Establecer valor de mapa


Descripción

Con esta acción, puede agregar un par de clave / valor en la estructura de datos del mapa. Usted suministra la variable que contiene el índice del mapa (como lo devuelve la acción Crear mapa ) y luego proporciona una "clave" (que es el identificador dentro del mapa para un valor), y luego el "valor" que se asociará con esa clave. Ambas claves pueden ser valores enteros (como 0, 200, 36712, etc.) o cadenas (como "Nombre", "Escudo", etc.), pero el valor puede ser cualquier tipo de datos válido.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Map El índice (almacenado en una variable) del mapa para establecer
Key La clave para establecer (entero o cadena)
Value El valor que se almacenará con la clave dada


Ejemplo:

El código de bloque de acción anterior crea una nueva estructura de datos de mapa y luego establece tres entradas de clave / valor en ella.
