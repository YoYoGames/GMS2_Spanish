Si existe una estructura de datos


Descripción

Con esta acción puede verificar una variable para ver si indexa una estructura de datos o no. Primero debe elegir el tipo de estructura de datos para verificar (ya sea una Lista, un Mapa, una Grilla o una Pila) y luego debe proporcionar la variable que contiene el índice de estructura de datos (como lo devuelven las acciones Crear Lista, Crear Mapa, Crear cuadrícula y Crear pila ). La acción volverá true si la estructura de datos existe, y false si no es así Tenga en cuenta que puede marcar "no" en la acción para verificar si una variable no contiene un índice de estructura de datos.
