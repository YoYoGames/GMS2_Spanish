Empujar hacia la pila


Descripción

Con esta acción puede insertar un valor en una estructura de datos de pila. Proporciona la variable que contiene el índice de pila (como lo devuelve la acción Crear pila ) y luego da el valor para empujar hacia arriba. Con estructuras de datos de pila, los valores siempre se agregan a la "parte superior" de la pila, y nunca al final o en el medio (si necesita esta funcionalidad, utilice una estructura de datos de lista).


Sintaxis de acción:


Argumentos:

Argumento Descripción
Stack El índice (almacenado en una variable) de la pila para usar
Value El valor para presionar en la pila


Ejemplo:

El código de bloque de acción anterior crea una variable de instancia y una nueva estructura de datos de pila. El índice de la pila se almacena en la nueva variable, y luego se realiza un ciclo que crea 1o instancias y empuja sus valores únicos de ID a la pila.
