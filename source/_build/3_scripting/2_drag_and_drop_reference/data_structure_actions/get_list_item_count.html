Obtener recuento de elementos de lista


Descripción

Puede usar esta acción para averiguar cuántas entradas hay en una lista. Proporciona la variable que contiene el índice de lista (como lo devuelve la acción Crear lista ) y la variable de destino para almacenar el número devuelto de elementos en la lista (que se puede marcar como una variable local temporal para usar hasta el final del secuencia de comandos o evento).


Sintaxis de acción:


Argumentos:

Argumento Descripción
List El índice (almacenado en una variable) para obtener el recuento de elementos
Target La variable de destino para almacenar el recuento de elementos devueltos


Ejemplo:

El código de bloque de acción anterior obtiene el tamaño de la lista a la que se hace referencia en la variable global y luego utiliza un bucle de repetición para recorrerlo y destruir las instancias con los ID almacenados en la lista. Una vez finalizado el ciclo de repetición, la lista se libera de la memoria.
