Insertar en la lista


Descripción

Con esta acción puede recuperar la posición del índice de un artículo dentro de la lista dada. Usted suministra la variable que almacena el índice de la lista (como lo devuelve la acción Crear lista ) y el valor que desea encontrar la posición del índice dentro de la lista, así como una variable objetivo para almacenar la posición del índice devuelto para el artículo (que se puede marcar como una variable local temporal para usar hasta el final del script o evento). Cuando inserta un valor en una lista, todos los índices superiores al índice dado para insertar se moverán uno arriba, por lo tanto, si tiene una lista con 4 elementos (posición de índice de 0 a 3) y coloca un nuevo elemento en la posición de índice 2, el artículo anterior 2 se convertirá en el elemento 3 y el elemento anterior 3 se convertirá en el elemento 4. Tenga en cuenta que si el valor que busca no existe dentro de la lista, entonces no habrá una posición de índice de lista para devolver, y por lo tanto la devolución el valor será -1.


Sintaxis de acción:


Argumentos:

Argumento Descripción
List El índice (almacenado en una variable) de la lista para insertar en
Index La posición del índice dentro de la lista para insertar un valor
Value El valor que se almacenará en la posición
