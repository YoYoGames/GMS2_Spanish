Borrar estructura de datos


Descripción

Esta acción borrará todos los valores de la estructura de datos dada. Primero debe seleccionar el tipo de estructura de datos para borrar y luego proporcionar el índice de la estructura de datos (el índice es el valor que se devuelve y se almacena en una variable cuando creó originalmente la estructura de datos). Una estructura de datos borrada se considera "vacía" (es decir, no tiene entradas), excepto las cuadrículas. Con las cuadrículas, esta acción simplemente borrará todas las celdas de la grilla a 0.

NOTA: Esto no libera la estructura de datos de la memoria y solo borra cualquier información. Para liberar la memoria asignada a la estructura de datos, debe usar la acción Estructura de datos gratuita.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Structure El tipo de estructura de datos que está borrando (Lista, Mapa, Cuadrícula o Pila)
Variable La variable que contiene el índice de la estructura de datos para borrar
