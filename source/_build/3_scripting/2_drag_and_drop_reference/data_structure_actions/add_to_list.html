Agregar a la lista


Descripción

Esta acción se puede usar para agregar un nuevo valor (de cualquier tipo de datos) a la lista, y este valor se agregará al final. Usted proporciona la variable que almacena el índice de la lista (como lo devuelve la acción Crear lista ) y el valor para almacenar.


Sintaxis de acción:


Argumentos:

Argumento Descripción
List El índice (almacenado en una variable) de la lista para agregar a
Value El valor para agregar a la lista


Ejemplo:

El código de bloque de acción anterior crea una variable de ámbito global y luego crea una nueva estructura de datos de lista, asignando su valor de índice a la variable global. El alcance se cambia para que todas las instancias del objeto "obj_Enemy_Parent" agreguen su valor único de ID de instancia en la lista.
