Obtener índice del elemento de la lista


Descripción

Con esta acción puede recuperar la posición del índice de un artículo dentro de la lista dada. Usted suministra la variable que almacena el índice de la lista (como lo devuelve la acción Crear lista ) y el valor que desea encontrar la posición del índice dentro de la lista, así como una variable objetivo para almacenar la posición del índice devuelto para el artículo (que se puede marcar como una variable local temporal para usar hasta el final del script o evento). Tenga en cuenta que si el valor que busca no existe dentro de la lista, entonces no habrá una posición de índice de lista para devolver, por lo que el valor de retorno será -1.


Sintaxis de acción:


Argumentos:

Argumento Descripción
List El índice (almacenado en una variable) de la lista para eliminar de
Value El valor para verificar la lista de
Target La variable de destino para almacenar el índice devuelto


Ejemplo:

El código de bloque de acción anterior comprueba una colisión en la posición de instancia y, si se encuentra uno, el valor de ID exclusivo para la instancia se almacena en una variable temporal y luego se verifica para ver si existe en la estructura de datos de la lista. Si existe, entonces el artículo se elimina de la lista y la instancia se destruye.
