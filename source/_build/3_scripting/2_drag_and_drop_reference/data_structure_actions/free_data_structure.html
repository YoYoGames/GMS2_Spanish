Estructura de datos gratuita


Descripción

Esta acción liberará la estructura de datos de la memoria y se deberá invocar siempre que ya no se requiera una estructura de datos, por ejemplo, en Destruir incluso una instancia o evento de fin de sala, etc. Liberar una estructura de datos significa que no hay más acciones se puede usar para referenciarlo porque ya no existirá, aunque también puede verificarlo con la acción Si existe estructura de datos.

Tenga en cuenta que a menos que la estructura de datos tenga un índice asignado a una variable de ámbito global, no se transmitirá de una habitación a otra. Por ejemplo, si crea una instancia de jugador y se asigna una estructura de datos de lista a una variable en esa instancia, la lista deberá liberarse si el jugador es destruido o si la habitación cambia con la instancia de jugador en él (a menos que el jugador la instancia se marca como persistente), por lo que la instancia del jugador tendrá dos acciones de estructura de datos gratuitas para ambos eventos. Es importante que todas las estructuras de datos creadas se destruyan de esta manera cuando ya no se requiera para evitar fugas de memoria que pueden causar problemas de rendimiento o incluso bloquear su juego.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Type El tipo de estructura de datos que está liberando (Lista, Mapa, Cuadrícula o Pila)
Variable La variable que contiene el índice de la estructura de datos para liberar


Ejemplo:

El código de bloque de acción anterior obtiene el tamaño de la lista a la que se hace referencia en la variable global y luego utiliza un bucle de repetición para recorrerlo y destruir las instancias con los ID almacenados en la lista. Una vez finalizado el ciclo de repetición, la lista se libera de la memoria.
