Eliminar entrada de mapa


Descripción

Con esta acción, puede eliminar una clave (y su valor correspondiente) de una estructura de datos de mapa. Usted suministra la variable que contiene el índice del mapa (como lo devuelve la acción Crear mapa ) y luego proporciona una "clave" (que es el identificador dentro del mapa para un valor), y la acción eliminará esa clave del mapa. Tenga en cuenta que esta acción no devuelve nada, y si necesita el valor para la clave dada, debe usar la acción Obtener valor del mapa antes de eliminarla.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Map El índice (almacenado en una variable) del mapa para eliminar el valor de
Key La clave para usar (real o cadena)


Ejemplo:

El código de bloque de acción anterior recupera el valor de una clave específica de la estructura de datos del mapa y luego verifica el valor para ver si es menor o igual a 0. Si se elimina la clave del mapa.
