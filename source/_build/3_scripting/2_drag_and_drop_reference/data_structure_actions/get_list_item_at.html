Obtiene el elemento de la lista en


Descripción

Con esta acción puede recuperar la posición del índice de un artículo dentro de la lista dada. Proporciona la variable que almacena el índice de lista (como lo devuelve la acción Crear lista ) y la posición del índice dentro de la lista para obtener el valor del artículo, así como una variable objetivo para almacenar el valor del elemento devuelto (que puede marcarse como una variable local temporal que se utilizará hasta el final del script o evento).

NOTA: si asigna una posición que está fuera del tamaño de la lista dada (es decir, la posición 11 en una lista de valores 10), la acción volverá. undefined. Puede verificar esto usando la acción If Undefined.

Argumentos:

Argumento Descripción
List El índice (almacenado en una variable) de la lista para eliminar de
Index La posición del índice dentro de la lista para obtener el elemento de
Target La variable de destino para almacenar el valor devuelto


Ejemplo:

El código de bloque de acción anterior obtiene el tamaño de la lista a la que se hace referencia en la variable global y luego utiliza un bucle de repetición para recorrerlo y destruir las instancias con los ID almacenados en la lista. Una vez finalizado el ciclo de repetición, la lista se libera de la memoria.
