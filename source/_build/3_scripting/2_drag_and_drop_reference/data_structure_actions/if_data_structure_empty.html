Si la estructura de datos está vacía


Descripción

Con esta acción puede verificar a para ver si la estructura de datos dada está vacía o no. Primero debe elegir el tipo de estructura de datos para verificar (ya sea una Lista, un Mapa o una Pila, sin grillas ya que ni siquiera pueden estar vacíos) y luego proporciona la variable que contiene el índice de estructura de datos (tal como lo devuelve las acciones Crear lista, Crear mapa y Crear pila ). La acción volverá true si la estructura de datos está vacía, y false si no es. Observe que puede bandera "No" en la acción para comprobar si una estructura de datos hace contener datos comprobando si no está vacío.
