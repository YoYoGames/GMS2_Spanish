Obtener valor de cuadrícula


Descripción

Con esta acción, puede establecer una celda específica dentro de la estructura de datos de la grilla en un valor. Usted suministra la variable que contiene el índice de la cuadrícula (como lo devuelve la acción Crear cuadrícula ) y luego le da a la posición x e y (la columna y la fila) la celda para establecer, y finalmente el valor para esa celda. También debe establecer una variable de destino (que puede marcarse como una variable local temporal) para almacenar el valor de celda devuelto, y el valor puede ser cualquier tipo de datos válido. Tenga en cuenta que la posición xey de la celda debe ser enteros enteros dentro de los límites de la grilla.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Grid El índice (almacenado en una variable) de la grilla para obtener
x La posición de la celda a lo largo del eje x para obtener (la columna)
y La posición de la celda a lo largo del eje y para obtener (la fila)
Target La variable de destino a usar para almacenar el valor devuelto


Ejemplo:

El código de bloque de acción anterior obtiene el valor en la cuadrícula almacenado en una variable global que corresponde a la posición de instancias dentro de la sala. Este valor se verifica y si es un valor de ID de instancia, la instancia de llamada se destruye.
