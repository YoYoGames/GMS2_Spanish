Crear pila


Descripción

Esta acción creará una nueva estructura de datos de pila y devolverá el valor de índice para que luego pueda acceder a la pila a través de las otras acciones de la Estructura de datos. El índice de pila se devolverá a la variable de destino que proporcione, que puede haberse creado anteriormente utilizando Asignar variable o Declarar temperatura, o puede marcar la casilla de verificación "Temp" para nombrar y crear una variable local temporal para almacenar el valor hasta el final del guión o evento. Una estructura de datos de pila recién creada se considera "vacía", es decir, no contiene entradas. Tenga en cuenta que puede crear acumulaciones de DS adicionales haciendo clic en el ícono más al lado de la acción, y seleccionando otra variable para contener la ID de la pila.

¡Importante! La creación de cualquier estructura de datos consume memoria en la plataforma objetivo y, como tal, todas las estructuras de datos deberían estar libres cuando ya no es necesario utilizar la estructura de datos gratuita; de lo contrario, se produce una pérdida de memoria que puede afectar el rendimiento de los juegos o incluso colisionar. Esto es particularmente relevante cuando se usan variables locales temporales para almacenar índices de estructura de datos, ya que estas variables se eliminan al final del código o evento, ¡pero eso no significa que la estructura de datos también se elimine! La estructura de datos seguirá existiendo, solo que no tendrá forma de referencia, así que use una variable de instancia y libere la estructura en un momento posterior, o libere la estructura antes del final del evento o secuencia de comandos si su índice está almacenado en una variable temporal.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Target La variable de destino para almacenar el índice de pila en


Ejemplo:

El código de bloque de acción anterior crea una variable de instancia y una nueva estructura de datos de pila. El índice de la pila se almacena en la nueva variable, y luego se realiza un ciclo que crea 1o instancias y empuja sus valores únicos de ID a la pila.
