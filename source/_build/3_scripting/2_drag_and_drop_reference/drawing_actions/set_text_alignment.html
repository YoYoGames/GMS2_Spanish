Establecer alineación de texto


Descripción

Esta acción establecerá la alineación de fuente para todas las acciones de dibujo de texto subsiguientes. Puede configurar la alineación horizontal como izquierda, centro o derecha, y la alineación vertical para que sea superior, intermedia o inferior y el texto se alineará con relación a la posición x / y de la acción de dibujar, como se muestra en las imágenes a continuación:

El texto está alineado horizontalmente a la izquierda
El texto está alineado horizontalmente con el centro
El texto está alineado horizontalmente a la derecha
El texto está alineado verticalmente a la parte superior
El texto está alineado verticalmente con el medio
El texto está alineado verticalmente con la parte inferior

Tenga en cuenta que esto establecerá la alineación que se utilizará para todas las llamadas de texto de sorteo posteriores, por lo que puede llamarse, por ejemplo, una vez al comienzo del juego, en cualquier caso, no solo el Evento de sorteo, y todas las acciones de dibujo de texto usarán la alineación establecida. Sin embargo, si necesita varias acciones de dibujar texto para alinearlas de diferentes maneras, deberá llamar a esta acción antes de dibujar cada acción de texto que desee alinear de manera diferente.


Sintaxis de acción:


Argumentos:

Argumento Descripción
H.Align El valor de alineación horizontal (Izquierda, Centro, Derecha)
V.Align El valor de alineación vertical (Superior, Medio, Inferior)


Ejemplo:

El código de bloque de acción anterior establece el color de dibujo, la fuente y la alineación para algún texto que se dibuja.
