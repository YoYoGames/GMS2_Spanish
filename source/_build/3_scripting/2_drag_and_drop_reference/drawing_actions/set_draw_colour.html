Establecer color de dibujo


Descripción

Con esta acción puede configurar el color que se usará para cualquier forma (rectángulo / elipse / línea) o para cualquier texto que se dibuje posteriormente. Tenga en cuenta que esto no tiene que invocarse antes de cada acción de robo, y solo se puede establecer una vez al comienzo del juego. Por ejemplo, si todo el texto va a ser amarillo, puede llamar a esta acción en el evento Crear de la primera sala del juego, y todo el texto (y las formas) se dibujarán en amarillo a partir de ese momento. Sin embargo, si necesita utilizar varios colores diferentes, deberá llamar esto antes de cada texto o forma que requiera un color diferente.

La acción tiene una casilla de verificación en la parte inferior para Usar alfa. Esto permite que el componente alfa del color se use para establecer el dibujo alfa también (más o menos lo mismo que usar Set Draw Alpha. Si no quieres incluir el componente alpha y solo tienes un color sólido, esto puede ser deshabilitado ( está habilitado por defecto).
