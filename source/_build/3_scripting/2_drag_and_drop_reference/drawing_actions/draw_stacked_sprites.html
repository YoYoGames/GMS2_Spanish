Dibujar Sprites apilados


Descripción

Esta acción dibujará un sprite un número dado de sprites uno tras otro en una posición dada dentro de la sala. Le da al sprite para dibujar y el orden de la pila, que puede ser Fila (horizontalmente, de izquierda a derecha), o columna (verticalmente, de arriba a abajo), así como también el número de sprites para dibujar y la posición. La posición puede ser una posición absoluta dentro de la sala, o una relativa a la posición de la instancia que realiza el dibujo, y el espacio entre las imágenes se basará en el ancho o la altura del elemento. Tenga en cuenta que esto simplemente dibuja una imagen estática, la imagen individual inicial (fotograma 0) del sprite dado, y cualquier fotograma adicional será ignorado, al igual que cualquier transformación que se haya agregado al cambiar las variables de instancia (como image_xscale o image_blend )

NOTA: Esta acción solo se puede usar en los diversos eventos Draw, y no dibujará nada si se usa en otro lugar.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Sprite El sprite para dibujar
Stack Order El orden para dibujar (Fila o Columna)
Number La cantidad de sprites para dibujar
x La posición x para dibujar en
y La posición y para dibujar en


Ejemplo:

El código de bloque de acción anterior obtiene el número de instancias del objeto "obj_Player" y luego usa esto para dibujar un número de sprites marcadores en la pantalla.
