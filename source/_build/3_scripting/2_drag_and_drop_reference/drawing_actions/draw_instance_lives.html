Dibujar vidas de instancias


Descripción

Esta acción dibujará un sprite para representar cada una de las vidas que la instancia tiene actualmente. El número para dibujar se toma del valor actual de la variable de instancia local lives, que es una variable especial que se crea automáticamente dentro de la instancia si usa alguna de las acciones que lo requieren. Le das al sprite para dibujar para las vidas y el orden de la pila, que puede ser Fila (horizontalmente, de izquierda a derecha), o columna (verticalmente, de arriba a abajo), así como la posición. La posición puede ser una posición absoluta dentro de la sala, o una relativa a la posición de la instancia que realiza el dibujo, y el espacio entre las imágenes se basará en el ancho o la altura del elemento. Tenga en cuenta que esto simplemente dibuja una imagen estática, la imagen individual inicial (fotograma 0) del sprite dado, y cualquier fotograma adicional será ignorado, al igual que cualquier transformación que se haya agregado a la instancia cambiando las variables de instancia (como image_xscale o image_blend ) El texto del título se dibujará precediendo el valor de las vidas y si es texto, debe estar envuelto con "", y ambos se formatearán utilizando la fuente, el color y la alineación del dibujo actual.

NOTA: Esta acción solo se puede usar en los diversos eventos Draw, y no dibujará nada si se usa en otro lugar.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Caption La cadena para dibujar junto con el valor del puntaje (anterior y puede dejarse en blanco)
Stack Order El orden para dibujar (Fila o Columna)
x La posición x para dibujar las vidas en
y La posición y para dibujar la vida en
