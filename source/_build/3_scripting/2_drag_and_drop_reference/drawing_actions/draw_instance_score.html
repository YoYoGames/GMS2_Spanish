Draw Instance Score


Descripción

Con esta acción puede dibujar el valor de puntaje de la instancia. Cuando utiliza cualquier acción de puntaje en GameMaker Studio 2 la instancia que lo llama tendrá una nueva variable de ámbito de instancia llamada score adicional. Esta variable se puede verificar y establecer como lo haría con cualquier otra variable. En el caso de esta acción, la variable se dibujará en la pantalla en la posición dada, junto con cualquier texto de título que desee proporcionar. El texto del título se dibujará precediendo el valor del puntaje y si es texto, se debe envolver con "", y ambos se formatearán utilizando la fuente, el color y la alineación del dibujo actual.

NOTA: Esta acción solo se puede usar en los diversos eventos Draw, y no dibujará nada si se usa en otro lugar.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Caption La cadena para dibujar junto con el valor del puntaje (anterior y puede dejarse en blanco)
x La posición x para dibujar la puntuación en
y La posición y para dibujar el puntaje en

El código de bloque de acción anterior dibuja el puntaje usando el formato de dibujo dado.
