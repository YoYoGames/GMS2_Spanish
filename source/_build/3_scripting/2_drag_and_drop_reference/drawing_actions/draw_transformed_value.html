Dibujar valor transformado


Descripción

Con esta acción, puede dibujar un título de cuerda y / o una variable o valor en la pantalla. El argumento de título es una cadena que desea agregar antes del valor, y puede dejarse en blanco si es necesario, y el valor puede ser una variable, un valor fijo u otra cadena, en cuyo caso debe estar entre comillas, y también se puede dejar en blanco si es necesario. A continuación, indica la posición en la que se debe dibujar la cadena, ya sea como una posición absoluta o como una posición relativa a la instancia que dibuja el texto. Además, puede establecer la escala del texto, estirándolo a lo largo del eje horizontal o vertical, así como también la rotación (los ángulos para girar el texto son antihorario, por lo que si lo ajusta a 90 ° lo girará hacia la izquierda y -90 a la derecha). El escalado se realiza por el factor dado, por lo que una escala de 0.5 será la mitad del ancho del texto y una escala de 2 duplicará el ancho, siendo 1 el valor predeterminado para no escalar. Tenga en cuenta que la apariencia del texto cuando se dibuje dependerá de la fuente utilizada, el color utilizado y la alineación. Todas estas cosas se pueden cambiar antes de dibujar, como se muestra en el siguiente ejemplo.

NOTA: Esta acción solo se puede usar en los diversos eventos Draw, y no dibujará nada si se usa en otro lugar.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Caption El título para dibujar (una cadena), se puede dejar en blanco
Value El valor para dibujar (real, variable o cadena)
x La posición x para dibujar en
y La posición y para dibujar en
xscale El factor de escala x (horizontal), por defecto es 1
yscale El factor de escala y (vertical), por defecto es 1
Rotation El ángulo de rotación para dibujar el texto


Ejemplo:

El código de bloque de acción anterior establece un color, una fuente y luego alinea el texto para dibujar desde el centro antes de dibujar realmente el valor "current_time" junto con el título "TIME", escalado para duplicar el tamaño y "vibrar" ligeramente debido al cambio de ángulo aleatorio.
