Dibujar rectángulo de degradado


Descripción

Esta acción dibujará un rectángulo en una posición determinada dentro de la sala, utilizando un conjunto de colores de fusión para crear un degradado. Usted da la posición superior izquierda y la posición inferior derecha y el rectángulo se dibujará entre ellos, y la posición puede ser una posición absoluta dentro de la habitación, o una posición relativa a la instancia que llama a la acción. A continuación, configura los colores para mezclar desde cada una de las esquinas y también puede establecer si el rectángulo puede rellenarse o delimitarse marcando el cuadro de relleno en la parte inferior.

NOTA: Esta acción solo se puede usar en los diversos eventos Draw, y no dibujará nada si se usa en otro lugar.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Left La posición izquierda para comenzar a dibujar desde
Top La posición superior para comenzar a dibujar desde
Right La posición correcta para finalizar el dibujo de
Bottom La posición inferior para finalizar el dibujo de
Top Left El color para mezclar desde la esquina superior izquierda
Top Right El color para mezclar desde la esquina superior derecha
Bottom Left El color para mezclar desde la esquina inferior izquierda
Bottom Right El color para mezclar desde la esquina inferior derecha


Ejemplo:

El código de bloque de acción anterior dibuja un rectángulo de degradado relativo a la posición de instancia con algún texto sobre él.
