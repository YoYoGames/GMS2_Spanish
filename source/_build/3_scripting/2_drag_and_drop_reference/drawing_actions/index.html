Dibujar acciones biblioteca


La biblioteca de acciones de Dibujo es donde puede encontrar las acciones necesarias para dibujar sprites, texto o formas, así como establecer ciertas propiedades de dibujo. La mayoría de estas acciones son solo para uso en los diversos eventos de dibujo de un objeto y pueden no funcionar si se usan fuera del evento de dibujo. Las excepciones a esto son las acciones de Establecer, que se pueden agregar a cualquier evento y afectarán a todos los dibujos para todas las instancias posteriores.

Es importante tener en cuenta que si agrega alguna acción en el Evento principal de Draw de un objeto, no dibujará el sprite que se ha asignado a la instancia a menos que indique explícitamente a GameMaker Studio 2 que dibuje, utilizando una acción como Dibujar El yo Básicamente, GameMaker Studio 2 dibujará por defecto cualquier sprite asignado a una instancia, solo si no hay nada más en el Evento de Draw.
