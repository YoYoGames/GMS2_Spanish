Obtener Draw Color


Descripción

Con esta acción puede obtener el color que se utiliza para cualquier forma (rectángulo / elipse / línea) o para cualquier texto que se dibuje. La acción devolverá el color a la variable de destino que especifique, que puede haberse creado previamente o puede ser una nueva temporal (si marca la casilla de verificación "Temp").


Sintaxis de acción:


Ejemplo:

El bloque de acción anterior obtiene el color de dibujo actual y si no está configurado para $ffffff (blanco) se establece en ese color.
