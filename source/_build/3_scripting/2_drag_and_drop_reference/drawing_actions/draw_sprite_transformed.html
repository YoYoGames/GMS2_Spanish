Dibujar Sprite transformado


Descripción

Esta función dibujará el sprite dado similar a la acción Draw Sprite pero con opciones adicionales para cambiar la escala, la combinación, la rotación y el marco del sprite que se está dibujando. Cambiar estos valores no modifica el recurso de ninguna manera (solo cómo se dibuja), y puede usar cualquiera de las variables de propiedades de sprites disponibles en lugar de valores directos para todos los argumentos en la función. La imagen a continuación ilustra cómo diferentes valores afectan el dibujo del sprite:

NOTA: La combinación de colores solo se recomienda para el destino HTML5 cuando WebGL está habilitado, aunque aún puede establecer el color de fusión si no está habilitado y combinará el sprite de la forma habitual. Sin embargo, todas las mezclas de esta manera crean un elemento gráfico duplicado que luego se almacena en la memoria caché y se utiliza cuando es necesario. Esto está lejos de ser óptimo y si usa múltiples cambios de color, reducirá el rendimiento de su juego a menos que active WebGL. si no desea usar WebGL, puede configurar el tamaño de la memoria caché de fuentes para intentar limitar esto si fuera necesario utilizando la función sprite_set_cache_size.
NOTA: Esta acción solo se puede usar en los diversos eventos Draw, y no dibujará nada si se usa en otro lugar.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Sprite El sprite para dibujar
Frame El marco del sprite para dibujar (usa la variable incorporada image_index para el cuadro actual)
X La posición x para dibujar dentro de la habitación
Y La posición y para dibujar dentro de la habitación
Scale X El factor de escala para el eje X (use la variable incorporada) image_xscale para la escala X actual)
Scale Y El factor de escala para el eje Y (use la variable incorporada) image_yscale para la escala Y actual)
Rotation El ángulo de estiramiento para el sprite (de 0 a 360, girando en sentido antihorario, donde 0 está a la derecha, usa la variable incorporada image_angle para la rotación actual)
Colour El color para combinar con el sprite (el valor predeterminado es blanco, y puedes usar la variable incorporada image_blend para el color actual)
