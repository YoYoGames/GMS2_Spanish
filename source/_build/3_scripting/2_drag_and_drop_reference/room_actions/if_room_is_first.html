Si la habitación es primero


Descripción

Esta acción verificará si la sala actual es la primera que figura en el árbol de recursos. La acción volverá true si la habitación actual es la primera en el árbol de recursos y false si no es. Puede marcar la casilla de verificación no para convertir esto en "Si la habitación NO ES primera".

Tenga en cuenta que para agregar acciones al bloque "if", deben soltarse al lado de la acción, como se muestra en la imagen siguiente:

Estas acciones ahora se ejecutarán si el "si" se evalúa como true, mientras que cualquier acción que se elimine en otro lugar se realizará después del bloque "if".
