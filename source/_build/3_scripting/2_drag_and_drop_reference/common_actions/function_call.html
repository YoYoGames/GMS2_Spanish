Llamada de función


Descripción

Esta acción se puede usar para llamar a cualquier función GML. Le da a la función GML para llamar y luego proporciona cualquier argumento adicional que requiera, es decir: cualquier valor (cadenas, enteros, índices de recursos, etc.) que desee ingresar en la función para que pueda usarlos. Puede agregar entradas de argumento adicionales haciendo clic en el ícono más al lado de la acción, y debe asegurarse de que su conteo de argumentos coincida con el de la función que se está utilizando (es decir: si la función requiere 3 argumentos, debe proporcionar 3 argumentos). Algunas funciones también pueden devolver un valor para su uso posterior en la lista de bloqueo de acciones, por lo que también puede especificar la variable de destino que debe almacenar este valor de retorno, que puede haber sido creado previamente o puede ser un nuevo temporal (si marca el Casilla de verificación "Temp"). Tanto el campo de destino como el argumento pueden dejarse en blanco si no son necesarios.

Argumentos:

Argumento Descripción
Script La función GML para llamar.
Argument0... ArgumentX Los diferentes argumentos (valores) que se deben pasar a la función.
Target El nombre de la variable que se debe segmentar para cualquier valor devuelto (se puede dejar en blanco).
