Si no está definido


Descripción

Esta acción se usa para verificar si una variable o expresión devuelve un valor de "indefinido", y si es así, esta acción volverá true y si no lo es, volverá false. "Indefinido" puede devolverse por varias razones, y un valor indefinido puede causar un bloqueo o un error grave en su proyecto, por lo que puede usar esto para verificarlo y detectarlo antes de que suceda. Si marca el argumento "No", entonces está verificando si un valor no está "indefinido".
