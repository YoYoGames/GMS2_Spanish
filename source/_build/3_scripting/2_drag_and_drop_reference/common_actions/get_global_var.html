Obtener variable global


Descripción

Esta acción se usa para recuperar el valor de una variable global y asignarla a otra instancia o variable temporal (local). La variable a la que asigna el valor global puede ser una que se haya creado previamente, o puede marcar la casilla de verificación "Temp" y dar un nuevo nombre de variable para que la acción cree una nueva variable temporal (local) específicamente para contener el retorno valor.

Tenga en cuenta que puede agregar variables adicionales en la misma acción haciendo clic en el ícono más al lado de la acción, y dando otro nombre de variable global y nombre de variable local temporal. Para obtener información más avanzada sobre las variables, consulte la sección sobre Variables y alcance variable.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Name El nombre de la variable global para obtener el valor de.
Target La variable objetivo para mantener el valor devuelto: debería haber sido creado previamente si "Temp" no está marcado (ver a continuación).
Temp Cree una variable temporal (local) para mantener el valor devuelto (desactivado de manera predeterminada)
