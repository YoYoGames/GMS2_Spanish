Aplicar para...


Descripción

Esta acción se puede usar para cambiar el Alcance de la acción para todas las acciones posteriores encadenadas a esta. Al igual que con una acción if, si deja caer más acción a la derecha de la acción Aplicar a... entonces se ejecutará desde el objeto o instancia de ámbito. El alcance se elige desde el menú emergente que se muestra al hacer clic en la flecha pequeña en la parte superior derecha de la acción, y de esta manera puede indicar a un conjunto diferente de objetos, o incluso a una sola instancia si tiene la ID de instancia, que ejecutar varias acciones antes de continuar con el código del objeto actual.

Tenga en cuenta que para agregar acciones al bloque "Se aplica a", deben soltarse al lado de la acción, como se muestra en la imagen a continuación:

Estas acciones se ejecutarán ahora desde el objeto o instancia al que apunta el bloque "Se aplica a", mientras que las acciones que se eliminen en otro lugar se realizarán después del bloque "Se aplica a".
