Descripción

Esta acción se usa para verificar el estado de una variable dada contra un valor dado usando una expresión específica. Una instrucción "if" siempre devolverá true o false y las expresiones para verificar pueden ser cualquiera de las siguientes:

  • Igual a - La variable y el valor son exactamente iguales
  • Menos de - La variable es menor que el valor
  • Mayor que - La variable es mayor que el valor
  • Menor que o igual a - La variable es menor o igual que el valor
  • Mayor que o igual a - La variable es mayor o igual que el valor

Si marca el argumento "No", lo anterior será expresiones negadas, por ejemplo "igual a" se convierte en " no es igual a". También tenga en cuenta que ni la variable ni los argumentos de valor están limitados a valores numéricos y pueden ser una cadena, un puntero, una identificación de recurso o cualquier otra cosa que una función pueda devolver o usar (consulte la sección sobre tipos de datos para obtener más información). no puede comparar dos tipos de datos diferentes sin obtener un error; por ejemplo, si la variable contiene una cadena e intenta verificar si es igual a un número, obtendrá un error de sintaxis.

NOTA IMPORTANTE: debido a problemas de precisión de punto flotante, puede verificar si dos valores son exactamente iguales. false, ya que uno puede ser exactamente 1, mientras que el otro puede ser 1.00000000000001. Esto se puede evitar utilizando la acción Decimal a Integer antes de verificar o usar las expresiones "mayor que" o "menor que".
