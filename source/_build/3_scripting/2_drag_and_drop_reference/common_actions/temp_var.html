Declarar variable temporal


Descripción

Esta acción se usa para declarar y dar un valor inicial a una variable temporal (también llamada variable local ), que es una variable que solo existe durante la duración del evento o script en el que fue declarada. Las variables temporales son muy útiles para almacenar valores que solo son necesarios para una sola operación y para evitar que sus objetos se "saturen" con variables que solo se pueden usar una vez. También son muy útiles para desglosar operaciones complejas o para establecer valores en otras instancias cuando se cambia el alcance de la acción. Esto se debe a que las variables declaradas como temporales no se asignan a ninguna instancia en particular, sino que se asignan al evento o al script en sí, de modo que si crea una variable temporal y luego cambia el ámbito de acción con la acción Aplicar a..., aún puede compruebe la variable temporal sin problemas y sin cambiar su alcance de acción.

Vale la pena señalar que cuando hablamos de "valores", no solo nos referimos a valores numéricos, ya que una variable puede ser una cadena, un puntero, una identificación de recurso o cualquier otra cosa que una función pueda devolver o usar, incluso una expresión. Consulte la sección sobre tipos de datos para obtener más información.

Tenga en cuenta que muchas acciones tienen una casilla de verificación "Temp" que le permite crear una nueva variable local temporal para mantener el valor de retorno de la acción, y se puede acceder a esta variable en el script o evento de acción como lo haría con cualquier otra variable a partir de entonces. También tenga en cuenta que las variables locales temporales tienen un alcance para el script o evento en el que se crean, y para que puedan ser utilizadas por otras instancias en el mismo bloque cuando se utiliza Aplicar a.

También puede agregar variables adicionales en la misma acción haciendo clic en el ícono más al lado de la acción, y dando otro nombre y valor de variable local temporal. Para obtener información más avanzada sobre las variables, consulte la sección sobre Variables y alcance variable.
