Establecer variable global


Descripción

Esta acción se usa para asignar un valor a una variable global (crearla en el proceso) o para cambiar el valor de una variable global existente. Las variables globales son variables que "pertenecen" a ninguna instancia específica, sino a todo el proyecto, y como tal se puede acceder y modificar mediante cualquier instancia, secuencia de comandos o código. Para obtener información más avanzada sobre variables globales, consulte la sección sobre Variables y alcance variable.

Usted da el nombre de la variable global y luego el valor, que puede agregarse en relación con el valor original si es necesario. Si le da un nuevo nombre de variable (es decir, una variable que nunca ha sido asignada previamente en todo el juego), esto creará la nueva variable global y le asignará el valor que le dé, mientras que si le da el nombre de una variable global existente, simplemente cambiará el valor al nuevo. Tenga en cuenta que al verificar Is Relative agregará el valor al valor de variable global existente (use un número negativo para restar) y también puede usar expresiones y variables creadas previamente como el valor.

Vale la pena señalar que cuando hablamos de "valores", no solo nos referimos a valores numéricos, ya que una variable puede ser una cadena, un puntero, una identificación de recurso o cualquier otra cosa que una función pueda devolver o usar. En el caso de una cadena, marcar Is Relative concatenará el valor con cualquier variable que haya sido guardada previamente, por lo tanto, si su variable fue "Hello" y le asigna "World" utilizando la verificación relativa, el valor de la variable final será "Hello" Mundo". Sin embargo, se debe tener cuidado para garantizar que los valores de las variables sean del mismo tipo de datos, ya que intentar agregar, por ejemplo, un entero y una cadena, dará como resultado un error del compilador.

También puede agregar variables adicionales en la misma acción haciendo clic en el ícono más al lado de la acción, y dando otro nombre y valor de variable global. Para obtener información más avanzada sobre las variables, consulte la sección sobre Variables y alcance variable.

Argumento Descripción

Value El valor que debe tener la variable global.
Is Relative Verifique esto para establecer el valor relativo al anterior.
