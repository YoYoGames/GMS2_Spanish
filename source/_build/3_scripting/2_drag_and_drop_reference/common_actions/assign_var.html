Asignar variable


Descripción

Esta acción se usa para asignarle a la variable un valor (crearlo en el proceso) o para cambiar el valor de una variable existente. Usted da el nombre de la variable y luego el valor, que se puede agregar en relación con el valor original si es necesario. Si le da un nuevo nombre de variable (es decir, una variable que nunca se haya asignado previamente en la instancia), esto creará la nueva variable y le asignará el valor que le dé, mientras que si le da el nombre de una variable existente, lo hará simplemente cambie el valor al nuevo. Tenga en cuenta que al verificar Is Relative agregará el valor al valor de la variable existente (use un número negativo para restar) y también puede usar expresiones y variables creadas previamente como el valor.

Vale la pena señalar que cuando hablamos de "valores", no solo nos referimos a valores numéricos, ya que una variable puede ser una cadena, un puntero, una identificación de recurso o cualquier otra cosa que una función pueda devolver o usar. En el caso de una cadena, marcar Is Relative concatenará el valor con cualquier variable que haya sido guardada previamente, por lo tanto, si su variable fue "Hello" y le asigna "World" utilizando la verificación relativa, el valor de la variable final será "Hello" Mundo". Sin embargo, se debe tener cuidado para garantizar que los valores de las variables sean del mismo tipo de datos, ya que intentar agregar, por ejemplo, un entero y una cadena, dará como resultado un error del compilador.

Tenga en cuenta que puede agregar variables adicionales en la misma acción haciendo clic en el ícono más al lado de la acción, y dando otro nombre y valor a la variable. Para obtener información más avanzada sobre las variables, consulte la sección sobre Variables y alcance variable.

Argumentos:

Argumento Descripción
Name El nombre de la nueva variable a asignar o el nombre de una variable existente para cambiar.
Value El valor que la variable debería contener.
Is Relative Verifique para establecer el valor relativo al anterior.
