Descripción

La acción "else" solo se usa junto con If Variable (o cualquier otra acción "si"), y es una forma de definir un conjunto de acciones que se deben realizar si la evaluación "si" regresa false. Por ejemplo, podría tener una acción "si" que verifica si la variable de salud del jugador es menor o igual a 0, y si eso se evalúa como verdadero, usted destruye la instancia. Sin embargo, si desea realizar un conjunto diferente de acciones de código si la salud del jugador no es menor que 0, puede agregar un "else" y luego especificar las acciones que se realizarán mientras el if se evalúa como false.

Tenga en cuenta que para agregar acciones al bloque "else", deben soltarse al lado de la acción, como se muestra en la imagen a continuación: Cualquier acción que se elimine en otro lugar se realizará después del bloque "else".
