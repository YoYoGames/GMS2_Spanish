Establecer la posición de audio


Descripción

Con esta acción, puede configurar un sonido para reproducir en un momento específico. Usted le da el sonido para jugar, y el tiempo para iniciarlo, pero esta acción no reproducirá el sonido para usted, simplemente establecerá la posición de reproducción para el sonido. Para utilizar esta acción, primero debe establecer la posición y luego reproducir el sonido con la acción Reproducir audio ; de lo contrario, no cambiará de posición hasta la próxima vez que reproduzca el mismo recurso de sonido (que comenzará en la nueva posición). La posición se establece en segundos, puede usar valores decimales, y puede obtener la duración total del sonido con la acción Obtener longitud de audio, aunque si proporciona un valor mayor que la duración del sonido deseado, se ajustará de forma que el sonido tiene 10 segundos de duración y da 12 segundos como posición de inicio, el sonido comenzará a los 2 segundos en adelante.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Sound El recurso de sonido para establecer la posición de reproducción de
Time El tiempo para establecer (en segundos)


Ejemplo:

El código de bloque de acción anterior obtiene la longitud de reproducción del recurso de sonido dado y lo almacena en una variable local temporal. Esto se usa para generar un valor aleatorio entre 0 y la duración del sonido, que luego se almacena en una variable local temporal diferente. Este nuevo valor aleatorio se usa para establecer la posición de inicio para el sonido y luego se reproduce el sonido.
