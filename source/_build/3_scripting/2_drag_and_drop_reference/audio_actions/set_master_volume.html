Establecer volumen principal


Descripción

Con esta acción puede configurar el volumen para todo el sistema de sonido. El valor de volumen que proporcione puede estar entre 0 (silencioso) y 1 (volumen completo) y la escala es lineal, de modo que un valor de 0.5 sería la mitad del volumen. Tenga en cuenta que esta acción afectará todas las instancias de todos los sonidos que se estén reproduciendo actualmente en la sala, y afectará todas las instancias futuras de todos los sonidos reproducidos. Esta acción también es independiente del volumen establecido para cada sonido individual, por lo que si tiene un sonido configurado para reproducir a 0,5 volúmenes y luego ajusta el volumen principal a 0,5, el sonido se reproducirá a 0,25 volumen.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Volume El nuevo volumen maestro para todo el sonido (de 0 a 1, por defecto 1)


Ejemplo:

El código de bloque de acción anterior comprueba una variable global para ver si es true o falso. Si esto es true luego, el volumen maestro para el sistema de sonido se recupera y se almacena en una variable local temporal. Luego se verifica el valor para ver si no es igual a 0.5, y si no lo está, entonces el volumen maestro del sistema de sonido se establece en 0.5. Si la variable global se evalúa como false, entonces el volumen maestro está establecido en 1.
