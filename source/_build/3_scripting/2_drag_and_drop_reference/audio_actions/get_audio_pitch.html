Obtener Audio Pitch


Descripción

Esta acción recuperará el tono del recurso de sonido que proporcione. El valor de retorno será un multiplicador para el tono grabado real de la muestra, donde un valor de 1 no cambia. Para más información sobre el lanzamiento, vea la acción Set Audio Pitch.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Sound El recurso de sonido para obtener el tono de
Target La variable objetivo para almacenar el tono devuelto


Ejemplo:

El código de bloque de acción anterior primero obtiene el tono para el sonido dado y lo almacena en una variable local temporal. Luego comprueba si se presiona una tecla, y si se detecta una, el valor del tono se verifica para ver si es menor que 2. Si lo es, el sonido se detiene, la variable de tono tiene 0.1 agregado a él, y luego, esta variable se usa para establecer el tono del sonido nuevamente antes de reproducirlo.
