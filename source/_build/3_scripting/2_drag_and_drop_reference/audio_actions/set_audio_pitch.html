Establecer tono de audio


Descripción

Esta acción se puede usar para establecer el tono de un sonido. Usted suministra el activo de sonido desde el árbol de recursos, y luego le da el tono deseado para cambiarlo, y todas las instancias adicionales de ese sonido que juegue cambiarán su tono. Sin embargo, esta acción no cambiará el tono de ningún sonido que se esté reproduciendo actualmente, y debe usarse antes de utilizar la acción Reproducir audio para el sonido.

El valor de tono es un multiplicador de tono, en el que el valor de entrada multiplica el tono actual por esa cantidad, por lo que el valor predeterminado de 1 no es el cambio de tono, mientras que un valor de menos de 1 reducirá el tono y más de 1 elevará el tono tono. Lo mejor es usar pequeños incrementos para esta función, ya que cualquier valor por debajo de 0 o por encima de 5 puede no ser audible de todos modos. Vale la pena señalar que el cambio de tono total permitido está fijado a (1/256) - 256 octavas, por lo que cualquier valor por encima o por debajo de este no se registrará.

NOTA: El valor fijado anteriormente es a lo que GameMaker Studio 2 intenta sujetar el rango, pero este valor no está garantizado en todas las plataformas de destino. iOS, por ejemplo, abrazaderas a (1/256) - 8, por lo que puede necesitar experimentar en cada plataforma de destino y tener diferentes versiones de un recurso de sonido, cada una de ellas previamente desplazada, si necesita valores de octava más altos o más bajos.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Sound El recurso de sonido para establecer el tono de
pitch El tono para establecer el sonido (por defecto es 1)


Ejemplo:

El código de bloque de acción anterior primero obtiene el tono para el sonido dado y lo almacena en una variable local temporal. Luego comprueba si se presiona una tecla, y si se detecta una, el valor del tono se verifica para ver si es menor que 2. Si lo es, el sonido se detiene, la variable de tono tiene 0.1 agregado a él, y luego, esta variable se usa para establecer el tono del sonido nuevamente antes de reproducirlo.
