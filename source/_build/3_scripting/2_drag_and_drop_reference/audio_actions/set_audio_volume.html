Establecer el volumen de audio


Descripción

Con esta acción puede configurar el volumen de un sonido determinado. Selecciona el sonido del explorador de activos y luego establece el valor del volumen. El volumen puede estar entre 0 (silencioso) y 1 (volumen completo) y la escala es lineal, de modo que un valor de 0.5 sería la mitad del volumen. Tenga en cuenta que esta acción afectará a todas las instancias del sonido que se estén reproduciendo actualmente en la sala, y afectará todas las instancias futuras del sonido reproducido.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Sound El recurso de sonido para establecer el volumen de
Volume El nuevo volumen para el sonido (de 0 a 1, por defecto 1)


Ejemplo:

El código de bloque de acción anterior comprueba una variable global para ver si es true o falso. Si esto es true luego, el volumen del sonido dado se recupera y se almacena en una variable local temporal. El valor se verifica luego para ver si no es igual a 0.5, y si no lo es, entonces el volumen del sonido se establece en 0.5. Si la variable global se evalúa como false, entonces el sonido tiene su volumen establecido en 1.
