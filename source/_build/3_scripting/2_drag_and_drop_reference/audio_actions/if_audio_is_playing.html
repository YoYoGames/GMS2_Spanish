Si el audio está sonando


Descripción

Esta acción se puede usar para verificar si una instancia de cualquier recurso de sonido se está reproduciendo actualmente en tu juego. Usted proporciona el recurso de sonido para verificar desde el explorador de activos y la función regresará true si está jugando y falso si no lo está, aunque si marca la bandera "no", esto se invierte y la función volverá true si el sonido no se está reproduciendo y false de otra manera.

Tenga en cuenta que para agregar acciones al bloque "if", deben soltarse al lado de la acción, como se muestra en la imagen siguiente:

Estas acciones ahora se ejecutarán si el "si" se evalúa como true, mientras que cualquier acción que se elimine en otro lugar se realizará después del bloque "if".
