Establecer variable de vista


Descripción

Esta acción le permite establecer ciertas variables integradas relacionadas con la forma en que el puerto de visualización mostrará las vistas de la habitación de la cámara. Seleccione la variable de puerto de vista para editar, luego dele el puerto de vista al objetivo (de 0 a 7) y finalmente el nuevo valor para la variable. El puerto de vista es el área de la pantalla que se usará para dibujar una vista de cámara determinada (como se configuró en el editor de sala), y las diferentes variables que puede establecer de esta manera se describen en la tabla a continuación:

Variable Descripción
Cámara Esto es para configurar la ventana gráfica para que muestre una vista de cámara diferente. Debe proporcionar el valor de identificación único de la cámara para cambiar a (consulte aquí para obtener más información).
Visibilidad Esta es la palanca de visibilidad para el puerto de visualización y espera un valor de true o false donde configurarlo para true hará visible el puerto de visualización (es decir, dibujará en la pantalla), y un valor de false no dibujará nada en la pantalla.
Ver la Coordenada del puerto X Esta es la posición X del puerto de vista dentro de la ventana del juego. Sin embargo, tenga en cuenta que esto solo es realmente necesario cuando hay más de un puerto de vista activo, por ejemplo, para crear un efecto de pantalla dividida, y si se usa cuando solo hay un puerto activo, se obtienen resultados inesperados.
Ver el puerto Y Coordinate Esta es la posición Y del puerto de visualización dentro de la ventana del juego. Sin embargo, tenga en cuenta que esto solo es realmente necesario cuando hay más de un puerto de vista activo, por ejemplo, para crear un efecto de pantalla dividida, y si se usa cuando solo hay un puerto activo, se obtienen resultados inesperados.
Ver el ancho del puerto Este es el ancho (en píxeles) del puerto de visualización. El ancho del puerto de vista (o puertos de vista combinados si hay más de uno activo) define el ancho de la ventana del juego o del lienzo de fondo al inicio del juego, por lo que cambiar este valor una vez que el juego haya comenzado no tendrá efecto visible en el tamaño de la ventana del juego a menos que se llame junto con la función GML window_set_size. También tenga en cuenta que si tiene un tamaño de puerto más grande o más pequeño que el asignado a la cámara, entonces la vista de la cámara se reducirá o hacia arriba para ajustarse.
Ver la altura del puerto Esta es la altura (en píxeles) del puerto de visualización. La altura del puerto de vista (o puertos de vista combinados si hay más de uno activo) define la altura de la ventana del juego o del lienzo de fondo al inicio del juego, por lo que cambiar este valor una vez que el juego haya comenzado no tendrá efecto visible en el tamaño de la ventana del juego a menos que se llame junto con la función GML window_set_size. También tenga en cuenta que si tiene un tamaño de puerto más grande o más pequeño que el asignado a la vista de la cámara, entonces la vista de la cámara se reducirá o hacia arriba para ajustarse.
Ver ID de la superficie del puerto Con esta variable puede establecer el contenido de un puerto de vista determinado para dibujar en una superficie, con el valor que suministra es el de la superficie que se utilizará para dibujar el puerto de vista. Para obtener más información, consulte aquí.
