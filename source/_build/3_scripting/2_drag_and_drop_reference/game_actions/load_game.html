Cargar juego


Descripción

Esta acción cargará un juego que se ha guardado previamente con la acción Guardar juego. Usted da el nombre de archivo del archivo para cargar (como una cadena, e incluye la extensión) y el juego se cargará y continuará ejecutándose desde el punto guardado. Tenga en cuenta que esto no está diseñado como un sistema universal de guardar / cargar, y tratar de cargar un juego guardado de una ejecución previa del proyecto puede dar errores (especialmente si ha utilizado cosas como estructuras de datos o partículas ), y como tal debería solo intente y cargue juegos que se hayan guardado en el progreso de un juego único (para cosas como puntos de control). También tenga en cuenta que el archivo no se cargará hasta el final del evento o secuencia de comandos actual, por lo que se seguirán realizando todas las acciones posteriores a la acción de carga. Para obtener un enfoque más completo para cargar datos de juegos guardados, consulte las Acciones de archivo.
