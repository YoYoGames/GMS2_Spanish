Salir del juego


Descripción

Con esta acción, puedes finalizar el juego, lo que activará el Evento de Fin del Otro Juego. Tenga en cuenta que esto no ocurre en el momento en que se llama a la acción, sino al final del evento o script actual, por lo que cualquier acción que tenga después de que se haya llamado a esta acción se ejecutará hasta que se llegue al final del evento / script.

Tenga en cuenta que esta acción tiene las siguientes restricciones:

  • Informará un error en el objetivo de iOS, ya que está en contra de las condiciones de su mercado.
  • No tendrá ningún efecto cuando se Windows UWP objetivos HTML5 y Windows UWP.
  • Funcionará en Windows (incluidos los juegos creados para Steam ), Android, Ubuntu (Linux) y Mac.


Sintaxis de acción:


Ejemplo:

El código de bloque de acción anterior verificará una variable global y si regresa true el juego saldrá
