Obtener un número aleatorio


Descripción

Esta acción generará un número aleatorio dentro de los límites de un conjunto de valores mínimos y máximos. Puede elegir el tipo de número que va a generar (ya sea decimal o entero) y luego establecer el rango para el número que se generará dentro proporcionando un valor mínimo y máximo. El valor aleatorio generado se devolverá a la variable objetivo (que puede marcarse como una variable local temporal).

NOTA: Al generar un entero aleatorio, la acción devolverá un valor entero entre el valor mínimo y el valor máximo inclusive, por lo que un rango de 5 a 10 podría devolver 5 o 10, así como cualquier número entero entre ellos. Sin embargo, generar un decimal aleatorio nunca devolverá el valor máximo, por lo que el rango de 5 a 10 puede devolver 5 pero no devolverá 10 (aunque podría devolver 9.999999999999).

Argumentos:

Argumento Descripción
Type El tipo de números a usar (entero o decimal)
Minimum El número mínimo para el rango aleatorio
Maximum El número máximo para el rango aleatorio
Target La variable de destino para almacenar el valor devuelto
