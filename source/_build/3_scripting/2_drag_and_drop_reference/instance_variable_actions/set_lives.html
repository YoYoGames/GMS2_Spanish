Establecer vidas


Descripción

Con esta acción puede crear o establecer una variable de instancia para vidas. Usted proporciona el valor para establecer la variable de vidas en o para agregar a la variable de vidas (si marca la casilla relativa, el valor se agregará al valor actual de "vidas"). Tenga en cuenta que, a diferencia de las variables de instancias regulares, el nombre de la variable "lives" se almacena internamente, por lo que para recuperar el valor de la puntuación debe usar la acción Get Lives.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Lives La cantidad para establecer las vidas (o para agregar a las vidas si el pariente está marcado)
