Si vive


Descripción

Esta acción se usa para verificar el valor de la variable de instancia lives usando una expresión específica. Da el tipo de expresión para verificar y el valor para verificar la corriente lives en contra, y la instrucción "si" siempre regresará true o false dependiendo de las expresiones y los valores utilizados. Las expresiones disponibles son:

  • Igual a - La variable y el valor son exactamente iguales
  • Menos de - La variable es menor que el valor
  • Mayor que - La variable es mayor que el valor
  • Menor que o igual a - La variable es menor o igual que el valor
  • Mayor que o igual a - La variable es mayor o igual que el valor

Si marca el argumento No, entonces las expresiones anteriores serán negadas, por ejemplo "igual a" se convierte en " no es igual a", por lo que estaría comprobando si el lives el valor no es igual al valor dado.

NOTA IMPORTANTE: debido a problemas de precisión de punto flotante, puede verificar si dos valores son exactamente iguales. false, ya que uno puede ser exactamente 1, mientras que el otro puede ser 1.00000000000001. Esto se puede evitar utilizando la acción Decimal a Integer antes de verificar o usar las expresiones "mayor que" o "menor que".
