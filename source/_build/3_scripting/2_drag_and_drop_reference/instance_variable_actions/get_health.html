Obtener salud


Descripción

Puede usar esta acción para recuperar el valor de la variable de instancia creada para el "estado" del objeto. Proporciona la variable de destino para almacenar el valor de "salud" devuelto (que puede marcarse como una variable local temporal), pero tenga en cuenta que si la variable no se ha establecido previamente (utilizando Establecer salud ), entonces la variable se creará para usted y la acción devolverá 0 (a diferencia de las variables de instancia regulares que darán un error si intenta acceder a ellas sin haberlas configurado primero).


Sintaxis de acción:


Argumentos:

Argumento Descripción
Target La variable de destino para almacenar el valor de "salud" devuelto (puede marcarse como una variable local temporal)


Ejemplo:

El código de bloque de acción anterior recuperará el valor de "estado" y lo almacenará en una variable local temporal. Esta variable se usa para dibujar una barra de estado para la instancia.
