Copia Buffer


Descripción

Esta acción se puede usar para copiar todos los datos almacenados en un búfer a otro. Cuando utilice dos almacenamientos intermedios y copie de uno a otro, ambos almacenamientos intermedios deben haberse creado previamente utilizando la acción Crear memoria intermedia, y debe proporcionar la identificación de almacenamiento intermedio única devuelta como argumentos desde y hacia.

NOTA: No puede copiar al mismo buffer.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Buffer El índice del búfer (almacenado en una variable) para copiar de
Copy To El índice del buffer (almacenado en una variable) para copiar a


Ejemplo:

El código de bloque de acción anterior creará un nuevo búfer y almacenará su ID en una variable temporal. Luego verifica si el búfer indexado en la variable global dada existe, y si lo hace, entonces se copia en el búfer temporal y la posición de lectura / escritura se restablece al inicio, y si no existe, entonces es creado.
