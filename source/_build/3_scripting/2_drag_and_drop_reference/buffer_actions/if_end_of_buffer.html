If End Of Buffer


Descripción

Esta acción se puede usar para verificar si la posición de lectura / escritura ha llegado al final del búfer. Usted proporciona el valor único de ID de buffer devuelto por la acción Create Buffer y luego la acción volverá true si la posición de lectura / escritura ha llegado al final de todos los datos disponibles en el búfer, o false si no es así Puede marcar "no" para comprobar la acción si la posición de lectura / escritura no está al final del búfer.
