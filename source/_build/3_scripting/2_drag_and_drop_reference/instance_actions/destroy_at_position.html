Destruir en posición


Descripción

Este bloque de acción destruirá todas las instancias encontradas en el punto indicado, llamando al Evento Destruir y luego al Evento de Limpieza para todas las instancias encontradas. Tenga en cuenta que esta acción se basa en la máscara de colisión de las instancias, y si alguna parte de la máscara se superpone con el punto objetivo, la función destruirá esa instancia (si las instancias no tienen sprite o máscara aplicada, entonces lo harán). no ser destruido).


Sintaxis de acción:


Argumentos:

Argumento Descripción
x La posición x en la habitación para usar.
y La posición y en la habitación para usar.


Ejemplo:

El código de bloque de acción anterior busca una pulsación de tecla (la tecla de espacio) y, si se detecta una, destruirá todas las instancias encontradas en la posición relativa de las instancias.
