Obtener variable de instancia


Descripción

Con esta acción puede obtener cualquiera de las variables integradas listadas. Todas las instancias de todos los objetos tienen una cantidad de variables ya creadas dentro de ellas para contener información esencial como la posición x / y actual, la velocidad, la dirección, la visibilidad y muchas otras cosas. Las variables de instancia que puede obtener son:

  • X Coordinate - La posición a lo largo del eje x en la habitación.

  • Coordenada Y: la posición a lo largo del eje y en la sala.

  • Dirección - La dirección de movimiento para la instancia (por defecto es 0, donde la derecha es 0 °, arriba es 90 °, la izquierda es 180 ° y abajo es 270 °).

  • Velocidad: la velocidad (en píxeles por cuadro de juego) a la que se está moviendo la instancia.

  • Velocidad horizontal: la velocidad horizontal (en píxeles por cuadro de juego) a la que se mueve la instancia. Una velocidad negativa significa que se mueve hacia la izquierda, una velocidad positiva es la correcta y el valor predeterminado es 0.

  • Velocidad vertical: la velocidad vertical (en píxeles por cuadro de juego) en la que se mueve la instancia. Una velocidad negativa significa que se mueve hacia arriba, una velocidad positiva está bajada y el valor predeterminado es 0.

  • Visible: la variable de alternancia de visibilidad (el valor predeterminado es true a menos que se establezca lo contrario en el Editor de objetos, configure en false hacer invisible). Tenga en cuenta que una instancia invisible aún ejecuta todos sus eventos, excepto el evento Draw.

  • Persistente: la variable de alternancia de persistencia (el valor predeterminado es false a menos que se establezca lo contrario en el Editor de objetos, configure en true para hacer persistente). Esto hará que una instancia "persista" de una habitación a otra, es decir: la instancia no se limpiará en el extremo de la sala, sino que se trasladará a la sala contigua. Tenga en cuenta que cuando se persiste una instancia, no llamará a su evento Crear ni Destruir, sino que llamará a los eventos de Limpieza, Final de la sala y Inicio de la sala.

  • Layer: la capa a la que se ha asignado la instancia.

  • Profundidad: el valor de profundidad asignado a la capa.

  • Sprite: el recurso de sprite asignado a la instancia.

  • Índice de imagen - El índice de imagen del sprite asignado. Este valor obtiene el marco del sprite asignado a la instancia. El valor devuelto estará entre 0 y el número de imágenes del sprite, y tenga en cuenta que si la velocidad de la imagen no es exactamente una, entonces el valor devuelto puede ser un decimal.

  • Ángulo de imagen: el ángulo en el que la instancia está configurada actualmente (por defecto es 0, donde la derecha es 0 °, arriba es 90 °, la izquierda es 180 ° y abajo es 270 °).

  • Image Alpha - El valor alfa actual para la instancia (por defecto es 1) de 0 a 1, donde 0 es completamente transparente y 1 es completamente opaco.

  • Mezcla de imagen: el color actual se mezcla con el sprite de instancia (el valor predeterminado es blanco).

  • Velocidad de imagen: la velocidad de animación actual para el sprite.

  • Escala horizontal: la escala horizontal en la que se dibuja el sprite de la instancia (esto también afecta al cuadro delimitador de la instancia y las colisiones). Una escala de 1 indica que no hay escala (1: 1), los valores más pequeños se reducen (0.5, por ejemplo, tendrá la mitad del ancho del sprite), los valores más grandes se amplían y los valores negativos reflejan el sprite y lo escalan a menos que el valor devuelto es exactamente -1 (en cuyo caso el sprite solo se refleja en el eje y sin escalar).

  • Escala vertical: la escala vertical en la que se dibuja el sprite de instancia (esto también afecta el cuadro delimitador de la instancia y las colisiones). Una escala de 1 indica que no hay escala (1: 1), los valores más pequeños se reducen (0.5, por ejemplo, tendrá la mitad de la altura del sprite), los valores más grandes se amplían y los valores negativos lo hacen girar y escalar a menos que el valor devuelto es exactamente -1 (en cuyo caso el sprite simplemente se está volteando a lo largo del eje y sin escalar).

  • Máscara: el índice de máscara para la instancia. El valor devuelto por defecto es -1 (lo que significa que la instancia está utilizando la máscara de colisión para el índice de sprite actualmente asignado), pero si se establece en cualquier otro recurso de sprite, el valor devuelto será el índice de sprite para el recurso utilizado.

  • Fricción: la fricción que se usa en cada cuadro de juego para ralentizar la instancia. El valor predeterminado es 0.

  • Gravedad: el valor devuelto es la fuerza de gravedad constante que se aplica a la instancia. El valor predeterminado es 0.

  • Dirección de gravedad: la dirección en la que la gravedad está "arrastrando" la instancia. El valor devuelto estará entre 0 y 360 donde la derecha es 0 °, arriba es 90 °, la izquierda es 180 ° y abajo es 270 °. El valor predeterminado es 270.

  • Puntuación - Obtener el score variable.

  • Vidas - Obtener el lives variable.

  • Salud - Obtener el health variable.

Una vez que selecciona la variable incorporada para obtener, entonces establece la variable objetivo para mantener el valor devuelto. Si marca esta variable como variable temporal, se creará para usted en la acción y estará disponible solo hasta el final del evento o secuencia de comandos de acción. También puede recuperar los valores de variables adicionales en la misma acción haciendo clic en el ícono más al lado de la acción, y seleccionando otra variable de instancia y dando una variable acompañante para almacenar el valor devuelto.
