Evento de usuario de llamada


Descripción

Con esta acción puede llamar a cualquiera de los 12 Otros eventos de usuario disponibles. Un evento de usuario es aquel en el que puede agregar scripts o código de acción y luego llamar en cualquier momento con esta acción, convirtiéndolos en eventos de objetos esencialmente personalizados.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Event El número del evento del usuario para llamar (0 - 11).


Ejemplo:

El código de bloque de acción anterior comprueba una variable y luego llama a Evento de usuario [0] o Evento de usuario [1] dependiendo de si la verificación se evalúa como true o false.
