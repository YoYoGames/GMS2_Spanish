Establecer velocidad de animación


Descripción

Este bloque de acción establece el image_speed (velocidad de animación) para el sprite asignado a la instancia. Establece los fotogramas ideales por segundo en el Editor de Sprites y luego puede usar este valor para modificarlo. La velocidad de animación predeterminada se establece en 1, lo que significa que el juego intentará mantener el número de fotogramas por segundo que haya definido para el sprite, pero puede usar valores superiores a 1 para animar más rápido (los fotogramas pueden omitirse), o valores decimales para animar más lento (los cuadros se mostrarán en varios pasos) o incluso valores negativos para invertir la velocidad de la animación.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Speed El nuevo valor de velocidad del modificador.


Ejemplo:

El código de bloque de acción anterior establece un nuevo sprite así como varias otras propiedades de cómo se debe mostrar el sprite, incluida la configuración para animar a la mitad de la velocidad definida.
