Si existe instancia


Descripción

Esta acción se puede usar para verificar si existe una instancia de un objeto específico en la sala de juegos, o si existe una instancia específica. Le das el objeto al explorador de activos, y si existe alguna instancia en la sala, la acción volverá true de lo contrario, volverá false. Tenga en cuenta que aquí puede usar una variable con el ID exclusivo para una instancia específica (por ejemplo, como lo devuelve la acción Crear instancia de objeto ) o incluso el valor de Id. De instancia tal como lo devolvió el editor de sala cuando agregó la instancia.

Si marca el argumento "No", la acción verificará si no existe ninguna instancia y si no se encuentra ninguna, volverá. true y si existe uno, regresará false.
