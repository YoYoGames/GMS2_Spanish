Establecer escala de instancia


Descripción

Este bloque de acción establece el image_xscale y image_yscale valores para la instancia. Los valores de entrada aquí son modificadores que se aplicarán al sprite asignado a la instancia, donde una escala de 1 (el valor predeterminado) indica que no hay escala (1: 1), los valores más pequeños se reducirán (0.5, por ejemplo, será la mitad el ancho del sprite) y los valores más grandes se escalarán. Puedes usar valores negativos para voltear el sprite y escalarlo a menos que el valor utilizado sea exactamente -1 (en cuyo caso el sprite solo se voltea o refleja sobre su origen sin escalar). Puede optar por establecer la escala horizontal o vertical en relación con los valores actuales, en cuyo caso agregará o restará el nuevo valor de los valores de escala actuales.

Tenga en cuenta que cambiar estos valores también cambiará la forma en que la instancia detecta colisiones, a menos que haya suministrado un mask_index (máscara de colisión) en el Editor de objetos. Tenga en cuenta que para que los cambios en esta variable sean visibles, la instancia no debería tener ningún evento de sorteo (y por lo tanto, GameMaker Studio 2 dibujará el sprite de forma predeterminada) o se dibujará utilizando Draw Self action.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Horizontal El factor de escala vertical a usar (por defecto es 1).
Vertical El factor de escala horizontal a usar (por defecto es 1).


Ejemplo:

El código de bloque de acción anterior establece un nuevo sprite así como varias otras propiedades de cómo se debe mostrar el sprite, incluida la configuración para escalar a tres cuartos de su tamaño de base a lo largo de ambos ejes.
