Obtenga el conteo de instancias


Descripción

Con esta acción puede encontrar el número de instancias de cualquier objeto dado actualmente activo dentro de la sala. Puede seleccionar cualquier objeto de la lista de recursos o puede usar la palabra clave especial all para obtener el número de todas las instancias activas. También debe proporcionar una variable objetivo para devolver el valor, de modo que pueda usarlo en otras acciones.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Object El nombre del objeto a verificar.
Target La variable a apuntar para el valor de retorno.


Ejemplo:

El código de bloque de acción anterior crea una variable local (temporal) y luego le asigna el número de instancias del objeto "obj_Squirrel". Este valor se verifica y la instancia se destruye si el número de instancias es mayor a 100.
