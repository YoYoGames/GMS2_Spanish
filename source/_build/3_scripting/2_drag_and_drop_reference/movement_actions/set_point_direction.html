Establecer la dirección del punto


Descripción

Esta acción se usa para establecer una dirección desde un vector formado por la posición actual de la instancia y la posición dada. Proporciona las posiciones xey para "apuntar" a - estas pueden marcarse como relativas a la posición de instancia - y la dirección de la instancia a ese punto se usará para la instancia.


Sintaxis de acción:


Argumentos:

Argumento Descripción
x El componente x del vector de dirección.
y El componente y del vector de dirección.


Ejemplo:

El código de bloque de acción anterior comprueba las comprobaciones para ver si existe una instancia del objeto "obj_Player" en la sala y, si lo hace, establece la dirección de la instancia para que apunte hacia la posición x / y de "obj_Player".
