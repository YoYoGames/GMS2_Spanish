Establecer dirección aleatoria


Descripción

Esta acción se usa para establecer la dirección de movimiento de la instancia a una o más direcciones aleatorias elegidas entre los 8 puntos cardinales. Puede hacer clic en una o más de las 8 flechas direccionales dentro de la acción y la acción elegirá una de ellas al azar para usarla como la dirección de la instancia. Si no selecciona ninguna de las flechas, la acción no tendrá efecto en la dirección actual de la instancia.


Sintaxis de acción:


Ejemplo:

El código de bloque de acción anterior genera un valor aleatorio entre 0 y 10 y lo almacena en una variable temporal (local). Luego se verifica esta variable y, si es mayor que cinco, se elige un conjunto de direcciones aleatorias y si es menor que 5, se elige otro conjunto.
