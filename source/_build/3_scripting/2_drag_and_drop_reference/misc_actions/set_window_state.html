Establecer estado de ventana


Descripción

Con esta acción puede establecer el estado de la ventana del juego en pantalla completa o ventana. Aquí hay tres opciones:

  • Alternar: cuando se selecciona esta opción, cada vez que llame a la acción, se alternará la ventana entre los estados de pantalla completa y ventana
  • Pantalla completa: hará que la ventana pase a pantalla completa desde un estado de ventana
  • Windowed - Esto hará que la ventana vuelva a un estado de ventana si está en pantalla completa

Tenga en cuenta que la apariencia del juego mientras está en pantalla completa dependerá de las Opciones de juego: configuración de gráficos para sus plataformas de destino, ya que si el juego no tiene algún mecanismo para ajustar a diferentes proporciones y resoluciones, puede obtener estiramientos (si el juego la opción está configurada en Escala completa ), o puede agregar barras negras agregadas (si la opción del juego está configurada para Mantener la relación de aspecto ). En general, si no tiene nada en su lugar para manejar diferentes resoluciones, debe usar Mantener relación de aspecto para las plataformas de destino que desea compilar.


Sintaxis de acción:


Argumentos:

Argumento Descripción
State El estado de la ventana para establecer (ver arriba para las opciones)


Ejemplo:

El código de bloque de acción anterior alternará el estado de la ventana entre pantalla completa y ventana cada vez que se presiona la tecla F4.
