Establecer Tile At Pixel


Descripción

Con esta acción puede establecer un mosaico en una capa dada en un nuevo índice. La capa se especifica a partir del nombre de la capa dada (una cadena, como se usa para nombrar la capa en el Editor de la sala), y luego le da una posición xey para obtener la configuración del mosaico. La posición xey se puede marcar como relativa a la posición de la instancia llamante, y la tesela que se establece será aquella en la que la posición dada cae dentro de la sala y el conjunto de teselas utilizado será el asignado a la capa en el editor de la sala Finalmente, proporcione el índice del mosaico para establecer el mosaico (los mosaicos se indexan de izquierda a derecha de arriba a abajo, con 0 en la parte superior izquierda). Tenga en cuenta que siempre puede establecer el índice del mosaico en 0 para borrar el mosaico.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Layer La capa para configurar el mosaico
x La posición x para verificar
y La posición y para verificar
Tile El nuevo índice de la ficha para usar


Ejemplo:

El código de bloque de acción anterior comprueba si se presiona el botón izquierdo del mouse y si se detecta uno obtiene el índice de la celda en la que se encuentra la posición del mouse. Este índice de mosaico se verifica y si no es 0, se establece en 0.
