Obtener variable de Tilemap


Descripción

Con esta acción puede recuperar cualquiera de una cantidad dada de variables para el elemento tilemap en una capa. Cuando creas una capa de mosaico en el editor de sala, esta capa contiene un elemento de mosaico que luego se rellena con teselas de un conjunto de teselas. Este elemento de mapa de mosaico tiene ciertos valores asociados, como una posición de desplazamiento, el conjunto de mosaicos que se usa, el ancho, alto, etc. La lista completa de valores que puede obtener es:

  • Coordenada X: la posición a lo largo del eje x en la capa donde se ha colocado el elemento de mapa de mosaicos.

  • Coordenada Y: la posición a lo largo del eje y en la capa donde se ha colocado el elemento de mosaico.

  • Columnas: el ancho (definido como el número de columnas de celdas de mosaico) del elemento de mosaico de teselas.

  • Filas: la altura (definida como el número de filas de celdas de mosaico) del elemento de mosaico de teselas.

  • Ancho de mosaico: el ancho de una sola celda de mosaico.

  • Altura de mosaico: la altura de una sola celda de mosaico.

  • Recurso de conjunto de mosaico: el recurso de conjunto de mosaico que se ha asignado al elemento de mosaico para el dibujo.

  • Marco actual: el marco actual que se dibuja para los mosaicos animados.

  • Máscara: los datos de máscara para el elemento de mapa de mosaico.

  • Máscara global: la información de máscara global para todos los elementos de mosaico.

Al utilizar esta acción, proporciona el nombre de la capa (una cadena, como se define en el Editor de la sala) para obtener los datos del elemento del mapa de azulejos, luego el tipo de datos que desea recuperar (como se muestra en la lista anterior). El valor devuelto se almacenará en la variable de destino que se haya creado previamente o puede ser una nueva temporal (si marca la casilla de verificación "Temp"). Tenga en cuenta que puede recuperar valores adicionales haciendo clic en el ícono más al lado de la acción, y seleccionando otra variable y dando otra variable para almacenar el valor devuelto.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Layer La capa para obtener datos de
Variable La variable para recuperar el valor de (como se menciona arriba)
