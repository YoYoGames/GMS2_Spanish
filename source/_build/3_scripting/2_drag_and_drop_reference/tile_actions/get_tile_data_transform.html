Obtener transformación de datos de mosaico


Descripción

Con esta acción puede averiguar si los datos de mosaico de un mosaico se han transformado de una de estas tres formas: volteado, duplicado o girado. La tiledata es simplemente un valor que refleja el índice del mosaico junto con las transformaciones aplicadas, y normalmente querrá obtener los datos del mosaico primero usando Obtener datos del mosaico en la celda o Obtener datos del mosaico en píxeles antes de usar esta acción. La acción volverá true si la transformación elegida se ha aplicado a los datos de mosaico, o false de lo contrario, y el valor devuelto se almacenará en la variable de destino que se haya creado previamente o puede ser una nueva temporal (si marca la casilla "Temp"). Para obtener más información sobre tiledata, consulte la sección GML en Tilemaps - Funciones de mosaico.

Ejemplo:

El código de bloque de acción anterior obtiene los datos de mosaico en la posición del mouse y luego comprueba si el mosaico tiene una transformación de rotación aplicada. Si tiene, entonces la transformación se restablece y los datos del azulejo establecen el mosaico en la posición nuevamente.
