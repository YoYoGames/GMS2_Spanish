Obtener datos de mosaico en Pixel


Descripción

Con esta acción, puede obtener los datos de mosaico para una celda de mosaico en una capa de mosaico de azulejos desde una posición. Los datos de mosaico son simplemente un valor que refleja el índice del mosaico junto con los valores de volteo, espejo y rotación, así como también cualquier máscara de mosaico personalizada que haya utilizado. Con esta acción, primero proporciona la capa al objetivo utilizando el nombre de la capa (una cadena) como se define en el editor de sala, y luego los ejes xey en la sala para obtener los datos de mosaico. El valor de azulejo devuelto coincidirá con la celda de mosaico que se encuentre en ese punto. Por ejemplo, si sus mosaicos de mosaico son 16x16, entonces la capa de mosaico de azulejos contendrá mosaicos en celdas de 16x16 píxeles, por lo que si usa esta acción para apuntar a la posición de habitación (8, 8), obtendrá los datos de mosaico para la celda ( 0, 0) del mapa de mosaico.

NOTA: La capa seleccionada debe haberse definido como una capa de Tilemap en el editor de sala, de lo contrario, puede recibir errores.


El valor devuelto se almacenará en la variable de destino que se haya creado previamente o puede ser una nueva temporal (si marca la casilla de verificación "Temp"). Para obtener más información sobre tiledata, consulte la sección GML en Tilemaps - Funciones de mosaico.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Layer La capa con el mapa de azulejos al objetivo
X La posición a lo largo del eje horizontal (X) para establecer los datos de mosaico para
Y La posición a lo largo del eje vertical (Y) para establecer los datos del mosaico


Ejemplo:

El código de bloque de acción anterior prueba un clic del mouse y, cuando se recibe, recupera los datos de mosaico para la celda que coincide con la posición del mouse. Estos datos luego se modifican y se devuelven a la capa de mosaico de azulejos, rotando el mosaico en esa celda.
