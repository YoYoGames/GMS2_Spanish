Establecer transformación de datos de mosaico


Descripción

Con esta acción, puede cambiar una de las tres transformadas de mosaico en los datos de mosaico. Proporciona la variable que contiene los datos de mosaico para el mosaico, luego seleccione la transformación para cambiar. Al hacer clic en la casilla de verificación "Aplicar" se aplicará que la transformación y anulación de la comprobación lo eliminará. El tiledata es simplemente un valor que refleja el índice del mosaico junto con las transformaciones aplicadas, así como cualquier máscara de mosaico personalizada que haya utilizado, y esta acción le permite cambiar tres de esos valores: Voltear, duplicar y rotar. Tenga en cuenta que normalmente querrá obtener los datos de mosaico primero usando Obtener datos de teselas en celda o Obtener datos de teselas en píxeles, luego aplicar las transformaciones requiridas y luego establecer el mosaico en el mapa de azulejos usando Establecer datos de teselas en celdas o conjuntos Datos de mosaico en Pixel, ya que esta acción solo establece un valor pero no lo aplica al mosaico del que se tomó. Para obtener más información sobre tiledata, consulte la sección GML en Tilemaps - Funciones de mosaico.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Tile Los datos de mosaico para que el mosaico cambie
Transform La transformación para aplicar / eliminar


Ejemplo:

El código de bloque de acción anterior obtiene los datos de mosaico en la posición del mouse y luego comprueba si el mosaico tiene una transformación de rotación aplicada. Si tiene, entonces la transformación se restablece y los datos del azulejo establecen el mosaico en la posición nuevamente.
