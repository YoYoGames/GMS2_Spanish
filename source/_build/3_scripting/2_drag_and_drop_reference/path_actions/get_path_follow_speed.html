Obtén la velocidad de seguimiento de ruta


Descripción

Esta acción obtendrá la velocidad, en píxeles por trama de juego, de que una instancia esté configurada para seguir una ruta (tenga en cuenta que debe haber llamado a la acción Iniciar ruta siguiente para asignar una ruta a la instancia para que esto devuelva un valor distinto de 0). El valor devuelto se almacenará en la variable de destino que proporcione, o puede marcar la variable como una variable local temporal para crearla y asignar el valor.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Target La variable objetivo para mantener la ruta devuelta sigue la velocidad


Ejemplo:

El código de bloque de acción anterior obtiene la velocidad actual de la instancia a lo largo de la ruta asignada y, si es mayor que 4, se establece en 4.
