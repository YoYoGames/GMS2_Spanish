Establecer la velocidad de seguimiento de ruta


Descripción

Esta acción establecerá la velocidad, en píxeles por cuadro de juego, que una instancia debe seguir una ruta (tenga en cuenta que debe haber llamado a la acción Comenzar siguiendo camino para asignar una ruta a la instancia para que esto tenga algún efecto). Puede usar valores negativos para indicar que la instancia debe seguir la ruta inversa y vale la pena señalar que la velocidad que establezca aquí puede modificarse mediante el modificador de velocidad establecido en el editor de ruta para cada punto de ruta, por lo que si establece una velocidad de ruta a 2 y en el camino uno de los puntos tiene un modificador de velocidad de 200%, los píxeles reales por velocidad de cuadro de juego en ese punto serán 4.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Speed La velocidad (en píxeles por cuadro de juego) que la instancia debe seguir la ruta
