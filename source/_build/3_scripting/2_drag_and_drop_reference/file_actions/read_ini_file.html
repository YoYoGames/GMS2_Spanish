Leer archivo Ini


Descripción

Con esta acción, puede leer un dato de un archivo Ini. Debes haber abierto previamente el archivo Ini para leer / escribir usando la acción Abrir archivo Ini, y luego con esta acción le das el tipo de datos para leer (ya sea un número real o una cadena encerrada en ""), así como encabezado de sección y la clave dentro de esa sección para obtener los datos. También puede proporcionar un valor predeterminado que puede ser un número real de una cadena, y el tipo debe coincidir con el "Tipo" establecido en la acción. La acción devolverá esta acción predeterminada en caso de que ocurra cualquiera de los siguientes:

  • El archivo Ini que has abierto no existe
  • El encabezado de sección que hace referencia no existe
  • La clave de la que está tratando de obtener datos no existe

En cualquiera de las circunstancias anteriores, obtendrá el valor predeterminado devuelto en lugar de un valor del archivo Ini. Cualquier valor que se devuelva se almacenará en la variable de destino que especifique, que también se puede crear en ese momento si se marca como una variable local temporal.

¡IMPORTANTE! Intentar leer datos de un archivo Ini que no se haya abierto previamente con la acción Abrir Archivo Ini causará un error grave y bloqueará tu juego.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Type El tipo de datos para leer, ya sea string (entre comillas ""), o real number
Section La sección del archivo Ini para leer (como una cadena)
Key La clave dentro de la sección para leer (como una cadena)
Default El valor predeterminado para devolver debe ser que el archivo, la sección o la clave no existan
Target La variable objetivo para mantener el valor de retorno de la acción


Ejemplo:

El código de bloque de acción anterior abrirá un archivo Ini para leer y luego utilizará datos leídos para establecer 3 variables globales diferentes antes de volver a cerrar el archivo.
