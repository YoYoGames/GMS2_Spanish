Descripción

Esta acción creará un nuevo caso predeterminado dentro de un Switch. El caso predeterminado es uno que se realizará si el valor de entrada no coincide con ninguna otra constancia de caso. Las acciones que se agregan a un caso predeterminado se deben descartar al lado de la acción, del mismo modo que se agregaría una acción a un "si":


Sintaxis de acción:


Ejemplo:

El código de bloque de acción anterior obtiene el valor almacenado en una variable global y luego lo compara con los diferentes casos posibles. Si alguno de los casos es igual al valor, entonces se realizará la acción en ese caso; de lo contrario, se evaluará el siguiente caso. Si ningún caso coincide con el valor de entrada, se ejecutará el caso predeterminado.
