Caso


Descripción

Esta acción creará un nuevo caso dentro de un Switch. Usted da el valor para el caso y se comparará con el valor de entrada para el Interruptor, y si coinciden, se realizarán las acciones agregadas bajo el Caso. Las acciones que se agregan a un caso para que se realicen solo en ese caso se deben descartar al lado de la acción, del mismo modo que se agregaría una acción a un "si":

Tenga en cuenta que un valor de Caso debe ser una constante, que puede ser cualquier tipo de datos (real, cadena, etc.), siempre que evalúe un solo valor en la compilación, de modo que no pueda usar expresiones que dependan de variables. También tenga en cuenta que Case no se puede usar fuera de un Switch e intentar hacerlo le dará un error.


Sintaxis de acción:


Argumentos:

Argumento Descripción
Constant El valor constante a evaluar contra el valor de entrada del interruptor


Ejemplo:

El código de bloque de acción anterior obtiene el valor almacenado en una variable y luego lo compara con los diferentes casos posibles. Si alguno de los casos es igual al valor, entonces se realizará la acción en ese caso; de lo contrario, se evaluará el siguiente caso. Después de que el interruptor se haya ejecutado, una alarma se establece en 30.
