object_get_sprite

Devuelve el sprite_index del objeto especificado.

Sintaxis:

object_get_sprite(obj);


Argumento Descripción
obj El índice del objeto a verificar


Devoluciones: reales


Descripción

Esta función le dirá si el objeto que está revisando tiene un sprite o no, y si lo hace, devolverá el índice de ese sprite, o -1 si no lo hace. Tenga en cuenta que esta no es una función de instancia. Puede tener un objeto sin sprite, mientras que una instancia de ese mismo objeto puede tener uno y viceversa, o incluso pueden tener diferentes sprites. Puede establecer un sprite de instancias individuales usando sprite_index Instancia variable.


Ejemplo:

if sprite_index != object_get_sprite(object_index) sprite_index = object_get_sprite(object_index);

El ejemplo anterior verificará el índice de sprites de la instancia contra el sprite del object_index de la instancia. Si no son lo mismo, asignará el mismo sprite que el índice de objeto a la instancia.
