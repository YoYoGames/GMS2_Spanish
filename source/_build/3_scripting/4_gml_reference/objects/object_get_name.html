object_get_name

Devuelve el nombre del objeto especificado como se define en el árbol de recursos.

Sintaxis:

object_get_name(obj);


Argumento Descripción
obj El índice del objeto a verificar.


Devoluciones: cadena


Descripción

Esta función devolverá el nombre como una cadena del objeto especificado. Este nombre es el que se ha especificado para el objeto en el árbol de recursos de la ventana principal de GameMaker Studio 2. Tenga en cuenta que esto es solo una cadena y no se puede usar para hacer referencia directamente al objeto, para eso necesitaría el índice del objeto. Sin embargo, puede usar esta cadena para obtener el índice del objeto utilizando la cadena devuelta junto con la función asset_get_index().


Ejemplo:

str = object_get_name(object_index);

El código anterior obtendrá el nombre del índice del objeto para la instancia que ejecuta el código y almacenará el valor de retorno en la variable "str".
