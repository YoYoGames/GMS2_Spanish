object_set_persistent

Establece la persistencia del objeto dado.

Sintaxis:

object_set_persistent(index, pers);


Argumento Descripción
index El índice del objeto a cambiar.
pers Si el objeto debe marcarse como persistente ( true ) o no ( false )


Devoluciones: N / A


Descripción

Con esta función puede establecer la persistencia de un objeto específico. Un objeto persistente se "transfiere" de una habitación a otra y mantiene sus propiedades en todo momento (para obtener más información sobre la persistencia, consulte Definición de objetos ). Establecer la persistencia del objeto significa que todas las instancias de este objeto que se crean después de que se haya cambiado la persistencia se crearán persistentes, mientras que las instancias que ya están en la sala pueden no verse afectadas.

Tenga en cuenta que esta no es una función de instancia. Puede establecer el indicador persistente de instancias individuales usando el persistent variable y así tienen diez instancias todas marcadas como persistentes aunque el objeto en cuestión tenga el indicador establecido en falso en el IDE, e incluso si cambia el indicador persistente del objeto a verdadero usando esta función, todas las instancias que actualmente están en la sala permanecerán como estaban, y solo las instancias creadas después de llamar a la función comenzarán con persistencia marcada como verdadera.


Ejemplo:

if room == rm_final object_set_persistent(obj_Player, 0); instance_create_layer(32, 32, "Instances", obj_Player);

El código anterior comprueba para ver en qué sala se encuentra actualmente la instancia y si está en el "rm_final" especificado, establece la persistencia del objeto "obj_Player" en false y luego crea la instancia.
