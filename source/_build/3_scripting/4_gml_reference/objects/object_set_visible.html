object_set_visible

Establece la visibilidad del objeto dado.

Sintaxis:

object_set_visible( index, vis );


Argumento Descripción
index El índice del objeto a cambiar.
vis La nueva visibilidad del objeto (verdadero = visible, falso = invisible).


Devoluciones: N / A


Descripción

Con esta función, puede marcar un objeto específico como visible o no (para obtener más información sobre el indicador "visible", consulte Definición de objetos ). Esto significa que todas las instancias de este objeto que se crean después de que se haya modificado se crearán con este nuevo estado, mientras que las instancias que ya están en la sala pueden no verse afectadas.

Tenga en cuenta que esta no es una función de instancia. Puede establecer el indicador visible de instancias individuales usando el visible variable y así tienen diez instancias todas marcadas como visibles aunque el objeto en cuestión tenga el indicador establecido en falso en el IDE, e incluso si cambia el indicador visible del objeto a verdadero usando esta función, todas las instancias que actualmente están en la sala permanecerán como estaban, y solo las instancias creadas después de llamar a la función comenzarán con visible marcado como verdadero.


Ejemplo:

if !object_get_visible(obj_Block) object_set_visible(obj_Block, true);

El código anterior verifica la bandera visible del objeto "obj_Block" y si es falso, se cambia para que se marque como verdadero.
