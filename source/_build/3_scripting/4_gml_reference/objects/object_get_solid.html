object_get_solid

Devuelve si un objeto tiene un sólido marcado como verdadero o no.

Sintaxis:

object_get_solid(obj);


Argumento Descripción
obj El índice del objeto a verificar.


Devoluciones: Boolean


Descripción

Esta función le dirá si el objeto que está revisando ha sido marcado como "sólido" o no. Un objeto sólido genera un evento de colisión especial cuando se usa el sistema de colisión tradicional (es decir, el mundo de la física está apagado). Tenga en cuenta que esta no es una función de instancia. Entonces, puedes tener un objeto sólido y una instancia normal del mismo objeto y viceversa. Puede establecer un indicador sólido de instancias individuales usando el solid Instancia variable.


Ejemplo:

if !solid && object_get_solid(object_index) solid=true;

El código anterior verificará la instancia que lo ejecuta para ver si es sólido o no tan bien como para verificar el índice del objeto de la instancia para ver si está marcado como sólido o no. Si la instancia no es sólida y el índice del objeto se marca como sólido, establecerá "sólido" en verdadero para esa instancia.
