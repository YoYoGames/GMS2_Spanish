object_get_persistent

Devuelve si el objeto especificado está marcado como persistente o no.

Sintaxis:

object_get_persistent(obj);


Argumento Descripción
obj El índice del objeto a verificar.


Devoluciones: Boolean


Descripción

Esta función le dirá si el objeto que está revisando ha sido marcado como "persistente" o no. Un objeto persistente es aquel que provocará que cualquier instancia de este se transmita de una habitación a otra a menos que se destruyan explícitamente. Tenga en cuenta que esta no es una función de instancia. Entonces, puede tener un objeto persistente y una instancia no persistente del mismo objeto y viceversa. Puede establecer un indicador persistente de instancias individuales utilizando el persistent Instancia variable.


Ejemplo:

if !persistent && object_get_persistent(object_index) persistent=true;

El código anterior verificará la instancia que lo ejecuta para ver si es persistente o no, así como también verifica el índice de objeto de la instancia para ver si está marcado como persistente o no. Si la instancia no es persistente pero el índice del objeto se marca como persistente, establecerá "persistente" en verdadero para esa instancia.
