event_user

Llama a uno de los 16 eventos definidos por el usuario.

Sintaxis:

event_user(numb);


Argumento Descripción
numb El número de evento para llamar, entre 0 y 15.


Devoluciones: N / A


Descripción

Con esta función le dice a la instancia que ejecute las acciones o el código que se ha colocado dentro de uno de los 16 eventos definidos por el usuario. Estos eventos solo se pueden invocar de esta forma, o utilizando el event_perform función.


Ejemplo:

event_user(4);

Esto llamaría Evento 4 Definido por el Usuario.
