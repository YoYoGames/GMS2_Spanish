event_perform

Realiza un evento determinado de la instancia de llamada.

Sintaxis:

event_perform(type, numb);


Argumento Descripción
type El tipo de evento a realizar. Vea abajo.
numb La constante o valor del evento específico, si es necesario (de lo contrario, simplemente use 0).


Devoluciones: N / A


Descripción

Esta función realizará el evento especificado, con el argumento designado, para la instancia que ejecuta el código. Aquí hay muchas opciones que permiten la simulación completa de todos los eventos posibles, pero tenga en cuenta que esto literalmente solo realiza todo el código del evento y el juego no modificará nada para que se active manualmente, por ejemplo, si elige ejecutar un teclado presione evento, el evento se activará pero la tecla correspondiente no se reconocerá como presionada. O si realiza un evento de alarma, el contador de alarma no se configurará en -1 sino que continuará la cuenta regresiva.

Ejemplo extendido

Para ayudar a comprender cómo se puede utilizar esta función, consulte los siguientes ejemplos:

event_perform(ev_keypress, ord("W"));

Esto llevaría a cabo el evento asociado con Keyboard Check Presionada la tecla "W" (sin generar realmente una pulsación del teclado).

event_perform(ev_step, ev_step_begin);

Esto llevaría a cabo el evento Begin Step (si se llama desde cualquiera de los eventos de paso, provocaría que el código de evento begin step se ejecute dos veces).

event_perform(ev_create, 0);

Esto llevaría a cabo el evento Crear de la instancia.
