event_perform_object

Realiza un evento dado del objeto especificado.

Sintaxis:

event_perform_object(obj, type, numb);


Argumento Descripción
obj El objeto que debería tener su evento activado.
type El tipo de evento a realizar. Vea abajo.
numb El evento específico, si es necesario (de lo contrario, simplemente use 0).


Devoluciones: N / A


Descripción

Esta función funciona igual que event_perform() excepto que esta vez puede especificar eventos de otro objeto. Aquí hay muchas opciones que permiten la simulación completa de todos los eventos posibles, pero tenga en cuenta que esto literalmente solo realiza todo el código del evento y el juego no modificará nada para que se active manualmente, por ejemplo, si elige ejecutar un teclado presione evento, el evento se activará pero la tecla relevante no se reconocerá como presionada. O si realiza un evento de alarma, el contador de alarma no se establecerá en -1 sino que continuará la cuenta regresiva. Puede encontrar una lista completa de las constantes disponibles que esta función requiere de la página para la función event_perform.

NOTA: Las acciones en el evento llamado con esta función se aplican a la instancia actual, y no a las instancias del objeto dado.


Ejemplo

event_perform_object(obj_Player, ev_keypress, ord("W"));

Esto llevaría a cabo el evento asociado con Keyboard Check Pressed> W key del objeto "obj_Player" en la instancia actual.
