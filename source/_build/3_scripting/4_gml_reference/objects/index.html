Objetos

En GameMaker Studio 2 tienes objetos y tienes instancias. Los objetos son esencialmente la plantilla base para una instancia y, como tales, nunca están presentes en una sala directamente... solo se colocan instancias del objeto en la sala. Esto significa que si quiere cambiar algo para todas las instancias que va a crear, puede hacerlo cambiando el objeto.

¡IMPORTANTE! Cambiar cualquier cosa sobre una operación no cambiará ninguna instancia actualmente presente en la sala, solo aquellas que se crean después del cambio.

También puede acceder a información sobre un objeto base y usarlo para dictar comportamientos o cambios en el juego. Por ejemplo, puede hacer un chequeo para una identificación de objeto padre, y en el siguiente código si se encuentra una instancia con ese padre, entonces puede verificar su object_index para decidir qué se debe hacer.

GameMaker Studio 2 le brinda una serie de funciones que le permiten obtener varios detalles sobre un objeto. Tenga en cuenta que un objeto no es una instancia (una entidad en el juego), es un recurso puramente a partir del cual se crean todas las instancias. Esto significa que los valores de retorno de estas funciones pueden ser diferentes a los valores reales que muestran las instancias en la sala, ya que una vez que se crea una instancia, se puede cambiar a través de código y acciones de DnD™.

  • object_exists
  • object_index
  • object_get_mask
  • object_get_name
  • object_get_parent
  • object_get_persistent
  • object_get_solid
  • object_get_sprite
  • object_get_visible
  • object_get_physics
  • object_is_ancestor

También hay una serie de funciones que le permiten establecer las propiedades de un objeto. Se debe tener en cuenta que cualquier instancia de este objeto que ya exista en la sala puede no verse afectada por estas funciones, pero todas las instancias nuevas de este objeto creadas en la sala serán, por lo que se recomienda que nunca cambie las propiedades de un objeto cuando instancias de ese objeto están presentes en la sala actual.

  • object_set_mask
  • object_set_persistent
  • object_set_solid
  • object_set_sprite
  • object_set_visible
