cursor_sprite

Sintaxis:

cursor_sprite;


Devoluciones: Real


Descripción

Establecer esta variable le indicará a GameMaker Studio 2 que use el sprite designado como un cursor (básicamente estableciéndolo en la posición x / y actual del mouse en cada paso). El valor predeterminado es -1, que no es sprite para el cursor, pero puede asignar cualquier índice de sprite de los recursos del juego o que se haya importado de un recurso externo. Tenga en cuenta que no hay forma de controlar la velocidad de la animación o image_index, por lo que si el sprite tiene subimágenes, éstas se ciclarán a la misma velocidad que la velocidad de la habitación. Para eliminar el cursor sprite, puede establecer esta variable en -1 de nuevo.

También vale la pena señalar que esta variable no reemplaza el cursor de la ventana del juego, y que todavía se dibujará de la forma habitual. Para evitar esto, puede usar la función window_set_cursor y establecerlo en la constante cr_none que hará que el cursor estándar sea invisible.


Ejemplo:

cursor_sprite = spr_CustomCursor;

El código anterior configurará el sprite indexado en la variable "spr_CustomCursor" para que sea el cursor sprite del juego.
