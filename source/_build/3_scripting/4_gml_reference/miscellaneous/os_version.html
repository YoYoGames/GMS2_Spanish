versión del sistema operativo

Esta variable contiene el número de versión del sistema operativo que ejecuta el juego.

Sintaxis:

os_version


Devoluciones: Real


Descripción

Esta variable le dirá el número de versión del sistema operativo que ejecuta su juego. Por ejemplo, si lo está ejecutando en un iPad con iOS 4 instalado, os_version será igual a 83890177.

NOTA: si necesita más información sobre el sistema operativo Windows, puede usar environment_get_variable.


Ejemplo:

if os_type = os_android && os_version >10 global.GFX = 1;

El código anterior comprueba el tipo de sistema operativo y el número de versión y ambos son correctos, entonces la variable global se establece en 1.
