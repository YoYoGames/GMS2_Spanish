os_browser

Esta variable tiene una constante diferente dependiendo del navegador en el que se está ejecutando el juego.

Sintaxis:

os_browser


Devoluciones: constante


Descripción

Esta variable contiene uno de los varios GameMaker constantes : Studio tiene que decirle en qué navegador está ejecutando actualmente el juego (si lo hay). Se pueden devolver las siguientes constantes:

Constante Descripción
browser_not_a_browser El juego no se juega en un navegador
browser_unknown Navegador desconocido
browser_ie explorador de Internet
browser_ie_mobile Internet Explorer en un dispositivo móvil
browser_firefox Mozilla Firefox
browser_chrome Google Chrome
browser_safari Safari
browser_safari_mobile Safari en un dispositivo móvil
browser_opera Ópera
browser_tizen Navegador de dispositivo móvil Tizen
browser_windows_store Aplicación Windows 8

Ejemplo:

if os_browser == browser_not_a_browser
   {
   global.Config = 0;
   }
else
   {
   global.Config = 1;
   }

El código anterior comprueba si el juego se está ejecutando en un navegador o no y establece una variable global a un valor según el resultado de la comprobación.
