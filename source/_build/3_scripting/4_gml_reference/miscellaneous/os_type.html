os_type

Esta variable mantiene una constante diferente dependiendo del sistema operativo con el que se está ejecutando el juego actualmente.

Sintaxis:

os_type;


Devoluciones: Constante


Descripción

Esta variable mantiene una de varias constantes de GameMaker Studio 2 para decirte en qué sistema operativo se ha creado el juego. Tenga en cuenta que esto no es necesariamente el mismo que el sistema operativo del dispositivo que lo ejecuta, ya que, por ejemplo, su juego podría estar ejecutándose en un sistema operativo Amazon Fire, pero se habrá creado para la plataforma Android (en cuyo caso os_type estarán os_android ).

Se pueden devolver las siguientes constantes:

Constante Descripción
os_windows Sistema operativo Windows
os_uwp Windows 10 Universal Windows Platform
os_linux Linux
os_macosx Mac OS X
os_ios iOS (iPhone, iPad, iPod Touch)
os_android Android
os_ps4 Sony PlayStation 4
os_xboxone Microsoft XBox One
os_unknown SO desconocido


Ejemplo:

switch (os_type)
   {
   case os_windows: global.Config = 0; break;
   case os_android: global.Config = 1; break;
   case os_linux: global.Config = 2; break;
   case os_macosx: global.Config = 3; break;
   case os_ios: global.Config = 4; break;
   case os_winphone: global.Config = 5; break;
   }

El código anterior comprueba el sistema operativo que ejecuta el juego y establece una variable global en consecuencia.
