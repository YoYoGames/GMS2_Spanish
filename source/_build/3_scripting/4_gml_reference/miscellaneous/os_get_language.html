os_get_language

Esta función devuelve un valor diferente según el navegador o el idioma del dispositivo.

Sintaxis:

os_get_language()


Devoluciones: cadena


Descripción

Esta función devuelve una cadena con el código de dos letras para el sistema operativo que ejecuta el juego, como lo establece el estándar ISO639. Si la información no está disponible, contendrá simplemente una cadena vacía "" o "en" para el idioma "inglés". Tenga en cuenta que algunos idiomas también tienen un código regional relevante, por lo que para distinguir entre las diferentes regiones del mismo país, utilice la función os_get_region().

La siguiente tabla muestra algunos de los principales códigos de dos letras según lo definido por ISO 639:

Idioma Código
Arábica Arkansas
chino Z h
danés da
Inglés es
francés fr
alemán Delaware
griego el
italiano eso
japonés ja
noruego no
polaco m pl
portugués pt
ruso ru
Español es
sueco sv



NOTA: ¡ Este no es el código de país de ubicación que se devuelve, sino el código de idioma del sistema operativo!


Ejemplo:

switch (os_get_language())
   {
   case "es": ini_open("spanish.ini"); break;
   case "fr": ini_open("french.ini"); break;
   case "it": ini_open("italian.ini"); break;
   default: ini_open("english.ini"); break;
   }

El código anterior verifica el idioma del sistema operativo y abre un *.ini archivo dependiendo del valor devuelto.
