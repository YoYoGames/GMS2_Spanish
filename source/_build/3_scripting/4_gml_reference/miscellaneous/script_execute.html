script_execute

Sintaxis:

script_execute(scr, arg0, arg1, arg2, ...);


Argumento Descripción
scr el nombre de la secuencia de comandos que desea llamar
arg0... arg14 Los diferentes argumentos que desea pasar a la secuencia de comandos


Devoluciones: N / A


Descripción

Hay momentos en los que es posible que desee pasar un script a una instancia de una manera dinámica, y para estos casos proporcionamos el script_execute función. Ejecutará un recurso de script creado por el usuario y pasará hasta 15 argumentos a ese script. Para obtener más información sobre scripts, vea Scripts.


Ejemplo:

script_execute(choose(scr_Up, scr_Down, scr_Left, scr_Right), irandom(5));

El código de ejemplo anterior usará script_execute para elegir uno de los cuatro recursos de script creados por el usuario (en este caso relacionados con el movimiento) y pasarle un entero aleatorio como argumento0.
