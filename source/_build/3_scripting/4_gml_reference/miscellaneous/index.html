Funciones Misceláneas

Las siguientes funciones y variables son funciones especiales que admite GameMaker Studio 2 pero que no encajan en ninguna categoría importante en particular:

Al crear juegos multiplataforma, a menudo es de vital importancia que obtenga los detalles del sistema operativo que utiliza el dispositivo que ejecuta el juego. Cosas como el idioma, la versión o el estado de la red se pueden usar para adaptar su juego al sistema que lo ejecuta y hacer que la experiencia del usuario final sea lo mejor posible, por lo que GameMaker Studio 2 tiene una serie de funciones especiales que se pueden usar para Obtener la información necesaria.

Existen las siguientes funciones que puede utilizar para obtener ciertos detalles sobre el sistema operativo o el navegador que ejecuta su juego:

  • os_browser
  • os_device
  • os_type
  • versión del sistema operativo
  • os_is_paused
  • os_is_network_connected
  • os_get_config
  • os_get_language
  • os_get_region
  • os_get_info
  • os_powersave_enable
  • os_lock_orientation

Para ayudarlo a crear y mantener puntuaciones altas para sus juegos, GameMaker Studio 2 crea una matriz global de puntuaciones altas a las que puede acceder, agregar y cambiar para crear sus propias listas personalizadas de puntuaciones altas sin demasiado problema. Esta funcionalidad es local al juego y funciona en todas las plataformas, por lo que puede almacenar y mantener fácilmente una tabla de puntaje básico usando estas funciones junto con las Funciones de manejo de archivos.

Existen las siguientes funciones que se ocupan de la lista interna de puntuación más alta.

  • puntuación alta_add
  • puntuación más alta
  • highscore_value
  • highscore_clear

GameMaker Studio 2 también incluye una función para dibujar las puntuaciones más altas en la pantalla como una lista, sin embargo, es muy básico y se recomienda que lo uses para depurar más que cualquier otra cosa, ya que hacer tu propia tabla de puntuaciones es fácil y te dará tienes mucho más control sobre cómo se muestra:

  • dibujar_puntuación más alta

Las siguientes funciones solo se usan cuando se llaman scripts:

  • script_exists
  • script_get_name
  • script_execute

Ciertas funciones se compilan previamente cuando se ejecuta el juego (es decir, se usan para definir cómo se compilará el juego final), y algunas también se usan cuando el juego se ha compilado para obtener detalles específicos sobre el entorno de ejecución. Aquí puedes encontrar la lista completa de estas funciones:

  • GM_build_date
  • GM_version
  • GM_runtime_version
  • gml_release_mode
  • gml_pragma
  • Parameter_Count
  • Parametrización de cadena
  • environment_get_variable

Cuando trabaje con Extensions, debe poder definir y llamar a funciones externas, por lo que usaría lo siguiente:

  • external_define
  • external_call
  • free_free

Las siguientes funciones son para uso general y simplemente no se ajustan a ninguna de las otras categorías en este documento:

  • cursor_sprite
  • set de alarmas
  • alarm_get
