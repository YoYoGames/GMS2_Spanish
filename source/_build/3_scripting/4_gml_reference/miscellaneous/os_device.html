os_device

Esta variable tiene una constante diferente dependiendo del dispositivo en el que se está ejecutando el juego.

Sintaxis:

os_device


Devoluciones: constante


Descripción

Esta variable contiene uno de los varios GameMaker constantes : Studio tiene que decirle en qué dispositivo está ejecutando actualmente el juego. Se pueden devolver las siguientes constantes:

Constante Descripción
device_ios_ipad iPad
device_ios_ipad_retina IPad más nuevo con tamaño de pantalla Retina de 2048 x 1536
device_ios_iphone6 iPhone6 ​​con un tamaño de pantalla de 1334 x 750
device_ios_iphone6plus Más grande iPhone 6 con pantalla 1920 x 1080
device_ios_iphone5 iPhone5 con tamaño de pantalla 640 x 1136)
device_ios_iphone Older iPhone / iPod Touch (pantalla de 480 x 320) o teléfono Android
device_ios_iphone_retina Nuevo iPhone / iPod Touch con pantalla Retina de 960 x 640
device_emulator El dispositivo es en realidad un emulador ( Windows Phone o Android )
device_tablet Tableta Android
device_ios_unknown Desconocido o no iOS

Ejemplo:

if os_browser = browser_not_a_browser
   {
   switch (os_device)
      {
      case device_ios_ipad: global.Config = 2; break;
      case device_ios_iphone: global.Config = 3; break;
      case device_ios_iphone_retina: global.Config = 4; break;
      case device_ios_unknown: global.Config = 5; break;
      }
   }
else
   {
   global.Config = 1;
   }

El código anterior comprueba si el juego se está ejecutando en un navegador o no y establece una variable global a un valor según el resultado de la comprobación.
