GM_runtime_version

Sintaxis:

GM_runtime_version;


Devoluciones: cadena


Descripción

Esta constante mantiene el número de versión de tiempo de ejecución como se define en las Preferencias como el tiempo de ejecución que se utiliza para construir el proyecto. El valor se almacena como una cadena.


Ejemplo:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);
draw_text(32, 96, "Runtime " + GM_runtime_version);

El código anterior toma la fecha GM, la versión y las constantes de tiempo de ejecución y los dibuja en la pantalla.
