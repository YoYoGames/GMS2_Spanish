external_call


Sintaxis:

external_call(id, args[0...15]);


Argumento Descripción
id El nombre de la función a la que desea llamar
args[0...10] Los diferentes argumentos que desea pasar a la función externa


Devoluciones:

Value (the type of value returned will depend on the defined function)


Descripción

Si ha creado una llamada de función externa a un dll o dylib utilizando external_define, puedes usar esta función para llamarla. Usted proporciona el nombre de la función definida previamente, así como también cada uno de los argumentos que requiere (cada argumento debe ser del tipo correcto, ya sea real o de cadena) y la función devuelve el resultado de la llamada externa.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

my_function = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);
var _a = external_call(my_function, x, y);

El código de ejemplo anterior llama a una función externa previamente definida y almacena el valor devuelto en una variable local.
