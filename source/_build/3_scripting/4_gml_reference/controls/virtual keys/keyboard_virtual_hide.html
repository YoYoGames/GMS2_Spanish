keyboard_virtual_show


Descripción

Esta función se puede usar para ocultar el teclado virtual en el dispositivo que ejecuta el juego. Al llamar a esta función se generará un Evento Asíncrono del Sistema, en el que el async_load El mapa DS se llenará con los siguientes pares clave / valor:

  • " event_type ": este será el " estado del teclado virtual " cuando sea activado por las funciones del teclado virtual.

  • " screen_height ": la altura del teclado virtual (en píxeles). Esto será 0 si el teclado es invisible.

  • " keyboard_status ": el estado actual del teclado, devuelto como una de las siguientes cadenas:

    • "ocultación"
    • "oculto"
    • "demostración"
    • "visible"


Sintaxis

keyboard_virtual_hide();


Devoluciones:

N/A


Ejemplo:

if input == true
    {
    input = false;
    keyboard_virtual_hide();
    }

El código anterior ocultará el teclado virtual del sistema operativo si la variable dada no está configurada para false.
