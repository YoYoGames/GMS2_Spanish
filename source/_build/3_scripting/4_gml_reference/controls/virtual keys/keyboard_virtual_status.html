keyboard_virtual_status


Descripción

Esta función se puede utilizar para obtener el estado del teclado virtual en el dispositivo que ejecuta el juego. La función volverá true si el teclado virtual del SO está visible / se muestra o false Si está escondido / escondido.


Sintaxis

keyboard_virtual_status();


Devoluciones:

Boolean


Ejemplo:

var _status = keyboard_virtual_status();
if _status == false
    {     keyboard_virtual_show(kbv_type_numbers, kbv_returnkey_default, kbv_autocapitalize_none, false);
    }

El código anterior mostrará el teclado virtual del SO si el estado actual es false.
