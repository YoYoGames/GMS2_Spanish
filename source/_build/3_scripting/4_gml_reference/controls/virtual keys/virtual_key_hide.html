virtual_key_hide


Descripción

Una vez que haya creado su clave virtual para sus dispositivos, es posible que tenga que depurar y probar que esté correctamente posicionada y funcionando correctamente y para eso debería llamar a la función virtual_key_show. Sin embargo, es posible que deba activar y desactivar esta vista, de modo que también pueda ocultar la clave virtual con esta función, utilizando el índice de la clave virtual que desea ocultar (creado y almacenado anteriormente con virtual_key_add ). Una vez que esté satisfecho con las cosas, se recomienda que cree y coloque su propio gráfico en su área.


Sintaxis

virtual_key_hide(index);

Argumento Descripción
index El índice de la clave virtual para ocultar.


Devoluciones:

N/A


Ejemplo:

if global.Debug
   {
   virtual_key_show(global.Left);
   }
else
   {
   virtual_key_hide(global.Left);
   }

El código anterior comprueba la variable global "global.Debug" y, si se prueba que es verdadero, la clave virtual indexada en la variable "global.Left" se dibujará en la pantalla, y si es falsa, la clave se ocultará.
