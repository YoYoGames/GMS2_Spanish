virtual_key_hide

Oculta una clave virtual creada que se muestra previamente en la pantalla del dispositivo.

Sintaxis:

virtual_key_hide(index);


Argumento Descripción
index El índice de la clave virtual para ocultar.


Devoluciones: N / A


Descripción

Una vez que haya creado su clave virtual para sus dispositivos, es posible que deba depurar y probar que está correctamente posicionada y funcionando correctamente y para eso llamaría a la función virtual_key_show. Sin embargo, es posible que necesite activar y desactivar esta vista, de modo que también puede ocultar la clave virtual con esta función, utilizando el índice de la clave virtual que desea ocultar (previamente creado y almacenado usando virtual_key_add ) Una vez que esté satisfecho con las cosas, le recomendamos que cree y coloque su propio gráfico en su área.


Ejemplo:

if global.Debug
   {
   virtual_key_show(global.Left);
   }
else
   {
   virtual_key_hide(global.Left);
   }

El código anterior comprueba la variable global "global.Debug" y si se prueba verdadero, la clave virtual indexada en la variable "global.Left" se dibujará en la pantalla, y si es falsa, la clave se ocultará.
