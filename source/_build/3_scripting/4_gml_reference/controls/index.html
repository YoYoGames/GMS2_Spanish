Controles


Lo fundamental que hace que un juego sea un juego es la interacción con el jugador. Esto se logra mediante una variedad de medios, con el teclado y el mouse en las computadoras y mediante el toque y la inclinación en los dispositivos móviles. La forma estándar de hacer esto en GameMaker es poner acciones en eventos de mouse o teclado, pero a veces se necesita más control, especialmente cuando se crea algo para dispositivos móviles o HTML5. Para eso, puede usar las funciones del mouse y teclado para obtener los botones que se presionaron o soltaron, así como recuperar los toques en una pantalla usando las funciones de gestos, y hay un par de funciones específicas relacionadas con los controles de inclinación que la mayoría de los dispositivos móviles tener también

Las siguientes secciones tratan sobre los principales dispositivos de entrada de usuario para desarrollar juegos con GameMaker Studio 2:

  1. Entrada de teclado
  2. Entrada de mouse
  3. Entrada de dispositivo
  4. Entrada de gesto
  5. Entrada de GamePad
  6. Llaves virtuales
