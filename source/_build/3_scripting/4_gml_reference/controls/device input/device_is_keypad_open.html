device_is_keypad_open

Verifica el dispositivo para ver si un teclado está abierto o no.

Sintaxis:

device_is_keypad_open


Devoluciones: Boolean


Descripción

Esto verifica el dispositivo de un teclado y, si hay uno disponible, lo devuelve true de lo contrario, regresa false. Tenga en cuenta que esta función es principalmente para usar con dispositivos Android.

Para aquellos usuarios con un Sony Xperia Play, hay un conjunto de botones / configuraciones de mapas clave en GameMaker Studio 2, por lo que puede usar las constantes del teclado vk_up, vk_down, vk_left, vk_right para las teclas del joypad y Triangle es ord("T"), Square es ord("S"), Circle es vk_alt + vk_backspace mientras que el botón Atrás es simplemente vk_backspace, Cross es vk_space, Select es vk_return y Start es vk_rshift, el L Trigger es ord("L") y R Trigger es ord("R").


Ejemplo:

if device_is_keypad_open()
   {
   global.Setting = 2;
   }
else
   {
   global.Setting = 1;
   }

El código anterior comprueba si hay un teclado y luego cambia una variable global según el valor devuelto.
