device_mouse_x

Devuelve la posición x del dispositivo cuando se presiona.

Sintaxis:

device_mouse_x(device);


Argumento Descripción
device El dispositivo (de 0 - n ) que se está verificando.


Devoluciones: Real


Descripción

Esta función devuelve la posición x de un toque en el dispositivo. Si está ejecutando esto en los módulos HTML5 o PC y Mac, este valor se actualiza constantemente, siempre que el dispositivo (por lo general, un mouse) esté enchufado; sin embargo, para dispositivos móviles, esto solo se actualizará mientras se esté visualizando la pantalla. tocado y tenga en cuenta que la cantidad máxima de toques que se pueden detectar dependerá en gran medida del dispositivo que se utilice y del sistema operativo que ejecute.

Ejemplo:

if device_mouse_check_button(0, mb_left)
   {
   if device_mouse_x(0) > x-32 && device_mouse_x(0) < x+32
      {
      pressed = true;
      }
   else
      {
      pressed = false;
      }
   }

El código anterior comprueba si el dispositivo está siendo presionado y, de ser así, sondea la posición del dispositivo x para ver si está dentro de los parámetros. Si es así, establece la variable "presionada" para true, de otro modo, lo configura para false.
