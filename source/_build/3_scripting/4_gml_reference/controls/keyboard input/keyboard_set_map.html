keyboard_set_map

Mapas una tecla en el teclado a otro.

Sintaxis:

keyboard_set_map(key1, key2);


Argumento Descripción
key1 Esta es la clave que se debe asignar a key1 para
key2 Esta es la clave que se debe mapear


Devoluciones: Boolean


Descripción

A veces, al hacer un juego, puede desear que una clave haga lo mismo que otra. Por ejemplo, muchas personas usan las teclas WASD para moverse, pero ¡muchas personas también usan las teclas de flecha! ¿Entonces lo que hay que hacer? Bueno, podría codificar el sistema de movimiento dos veces, pero eso es un poco complicado y afortunadamente redundante ya que esta función le permite "asignar" una clave a otra y, por lo tanto, cualquier entrada de cualquiera de ellas se interpretará como la misma. Para ello elige una clave que desea asignar (clave2 - esta será la clave que se escribe el código para) y una clave que desea que se asigna a (key1). Después de eso, GameMaker Studio 2 interpretará las pulsaciones de tecla en cualquiera de las teclas como procedentes de key2. También puede usar esta función para diseñar un sistema en el que el usuario pueda definir sus propias teclas para jugar simplemente asignando la clave de entrada del usuario a la clave que ha codificado en el juego.


Ejemplo:

keyboard_set_map(ord("A"), vk_left);

El código de ejemplo anterior asignará la tecla "A" a la tecla de flecha izquierda. Esto significa que el jugador puede usar la tecla "A" o la tecla de flecha izquierda, y que todo el código escrito para la flecha izquierda también responderá a la tecla "A" que se usa en su lugar.
