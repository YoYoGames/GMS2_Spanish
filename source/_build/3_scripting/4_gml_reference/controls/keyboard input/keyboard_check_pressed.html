keyboard_check_pressed


Descripción

Con esta función puede verificar si una tecla ha sido presionada o no. A diferencia del keyboard_check función, esta función solo se ejecutará una vez por cada vez que se presione la tecla, por lo que para que se dispare nuevamente, primero debe soltar la tecla y luego presionar nuevamente. La función tomará un valor de código clave como lo devuelve la función ord() (sólo letras mayúsculas A a la Z o números de 0-9), o cualquiera de los vk_* constantes enumeradas en la página principal de entrada del teclado.


Sintaxis:

keyboard_check_pressed(key);

Argumento Descripción
key La tecla para comprobar el estado presionado de.


Devoluciones:

Boolean


Ejemplo:

if keyboard_check_pressed(vk_anykey)
   {
   room_goto_next();
   }

El código anterior avanzará a la siguiente habitación si el jugador presiona cualquiera de las teclas del teclado (funciona como un mensaje de "Presione cualquier tecla para continuar").
