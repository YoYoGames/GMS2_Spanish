keyboard_check_direct


Descripción

Esta función volverá true si se presiona la tecla con el código clave particular, o false Si no lo está, comprobando directamente el hardware. Permite realizar algunas comprobaciones más, en particular, puede usar los códigos clave vk_lshift, vk_lcontrol, vk_lalt, vk_rshift, vk_rcontrol y vk_ralt para verificar si se presiona la tecla de dirección izquierda o derecha, control o alt. La función tomará un valor de código clave como lo devuelve la función ord() (sólo letras mayúsculas A a la Z o números de 0-9), o cualquiera de los vk_* constantes enumeradas en la página principal de entrada del teclado.

NOTA: Esta función solo está disponible para el destino estándar de Windows y el resultado es independiente de qué aplicación se enfoca.


Sintaxis:

keyboard_check_direct(key);

Argumento Descripción
key La clave para comprobar el estado hacia abajo de.


Devoluciones:

Boolean


Ejemplo:

if keyboard_check_direct(vk_ralt) || keyboard_check_direct(vk_lalt)
   {
   crouch = true;
   }

El código anterior verificará si se han presionado las teclas alt izquierda o derecha, y si lo tienen, establece la variable "crouch" en verdadero.
