keyboard_check_direct

Devuelve si la tecla con el código clave particular se presiona al verificar el hardware directamente.

Sintaxis:

keyboard_check_direct(key);


Argumento Descripción
key La clave para verificar el estado inactivo de.


Devoluciones: Boolean


Descripción

Esta función regresará true si se presiona la tecla con el código clave particular, o false si no es así, verificando el hardware directamente. Permite algunas comprobaciones más, en particular puede usar los códigos clave vk_lshift, vk_lcontrol, vk_lalt, vk_rshift, vk_rcontrol y vk_ralt para verificar si se presionó la tecla shift, control o alt a la izquierda o derecha.

NOTA: Esta función solo está disponible para el objetivo estándar de Windows y el resultado es independiente de qué aplicación tiene foco.


Ejemplo:

if keyboard_check_direct(vk_ralt) || keyboard_check_direct(vk_lalt)
   {
   crouch = true;
   }

El código anterior verificará si se presionaron las teclas alt izquierda o derecha, y si lo tienen establece la variable "crouch" en verdadero.
