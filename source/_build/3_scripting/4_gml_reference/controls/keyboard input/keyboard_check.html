keyboard_check


Descripción

Con esta función puede verificar si una tecla está presionada o no. A diferencia del keyboard_check_pressed o keyboard_check_released funciones que solo se activan una vez cuando se presiona o suelta la tecla, esta función se activa en cada paso para el que se mantiene presionada la tecla. La función tomará un valor de código clave como lo devuelve la función ord() (sólo letras mayúsculas A a la Z o números de 0-9), o cualquiera de los vk_* constantes enumeradas en la página principal de entrada del teclado.


Sintaxis:

keyboard_check(key);

Argumento Descripción
key La clave para comprobar el estado hacia abajo de.


Devoluciones:

Boolean


Ejemplo:

if keyboard_check(vk_left)
   {
   x -= 5;
   }

El código anterior verificará si se está presionando la tecla de flecha y moverá la instancia 5 píxeles a la izquierda en cada paso que devuelva verdadero.
