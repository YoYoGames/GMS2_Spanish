Entrada de gestos


Las funciones de Entrada de dispositivo están muy bien para tratar con simples toques en pantalla o para obtener orientación del dispositivo, pero a veces querrá que sus proyectos detecten cosas más complejas, como poder arrastrar o "deslizar" una instancia a través de una habitación. Estas cosas se pueden hacer con las funciones del dispositivo, pero para facilitar las cosas, GameMaker Studio 2 tiene una gama completa de eventos gestuales para detectar toques dobles, arrastres, películas, pellizcos y rotaciones, tanto a nivel de instancia como a nivel global, así como Devuelve información sobre el evento. Todas las funciones enumeradas aquí están diseñadas para brindarle un mayor control sobre cómo y cuándo se activarán estos eventos.

¡IMPORTANTE! Estas funciones no funcionarán en la plataforma de destino HTML5. No darán un error si se usan, sino que simplemente serán ignorados.


Las siguientes secciones enumeran todas las diferentes funciones de gestos:

Las siguientes funciones se pueden usar para obtener los diferentes valores que se utilizan para calcular el movimiento rápido, el arrastre, el pellizco, etc...:

  1. Gestion_get_drag
  2. gestos_de_trabajo_distancia
  3. Gestion_get_flick_speed
  4. Gestion_get_double_tap_time
  5. gesto_get_doble_tap_distancia
  6. gesto_get_tap_count
  7. Gestion_get_pinch_distance
  8. Gestion_get_pinch_angle_towards
  9. Gestion_get_pinch_angle_away
  10. gestos_get_rotate_tiempo
  11. Gestion_get_rotate_angle

Las siguientes funciones se pueden usar para configurar los diferentes valores que se utilizan para calcular el movimiento rápido, el arrastre, el pellizco, etc...:

  1. gesto_tiempo_
  2. gesto_diractivo_distancia
  3. gesto_flick_speed
  4. gesto_tiempo_tapa_tiempo
  5. gesto_doble_tap_distancia
  6. gestos_cuenta_cuenta
  7. gesto_pinch_distancia
  8. gesto_pinch_angle_towards
  9. gesto_pinch_angle_away
  10. gesto_rotate_tiempo
  11. gesto_rotate_angle
