gesto_get_tap_count


Descripción

Esta función se usa para verificar si el conteo de tomas está habilitado o deshabilitado. La función volverá true si está habilitado, y false de lo contrario, puede habilitar o inhabilitar el conteo de tomas usando la función gesture_tap_count. Cuando está habilitado, cada evento de toque tendrá una entrada de mapa DS "tapcount" adicional que habrá registrado el número de toques, y cuando esté habilitado, significa que se activarán todos los eventos de toque, es decir: dos toques activarán el evento de toque único y el evento de doble toque, con el número de tomas del evento de un solo toque es 1 y el número de toques del evento de doble toque es 2. El valor del conteo de toques se restablecerá a 0 una vez transcurrido el tiempo establecido para la detección de doble toque (consulte la función gesture_double_tap_time ). Si el conteo de tomas está deshabilitado, la toma inicial no se registrará hasta que haya transcurrido el tiempo de doble toque y no se haya detectado una segunda pulsación. Tenga en cuenta que esto está habilitado por defecto.


Sintaxis:

gesture_get_tap_count();


Devoluciones:

Bool


Ejemplo:

if !gesture_get_tap_count()
    {
    getsure_tap_count(true);
    }

El código anterior verifica si el conteo de tomas está habilitado y si no lo está, entonces está encendido.
