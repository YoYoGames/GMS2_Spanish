mouse_clear


Descripción

Esta función borrará el estado de los botones del mouse actualmente en uso. Esto significa que no se devolverán los controles para mantenerlos presionados. true hasta que el jugador suelta los botones y los reprime. Usted proporciona el botón del mouse para verificar a partir de una de las siguientes constantes:

Constante Descripción
mb_left El botón izquierdo del mouse
mb_middle El botón central del mouse (esto puede no ser válido para todas las plataformas de destino)
mb_right El botón derecho del mouse
mb_none Sin botón de mouse
mb_any Cualquiera de los botones del mouse


Sintaxis:

mouse_clear(button);


Argumento Descripción
button ¿Qué botón del mouse es constante para verificar?


Devoluciones:

Boolean


Ejemplo:

mouse_clear(mb_any);

El código anterior borrará el estado inactivo de todos los botones del mouse.
