Entrada de mouse


La entrada del mouse se acepta en todas las plataformas (en dispositivos móviles se acepta como un solo toque. Si necesita usar multitáctil, debe usar las funciones específicas del dispositivo ) y tiene una serie de constantes que se usan para especificar los botones. siendo presionado. Estas constantes se muestran en la siguiente tabla:

Constante Descripción
mb_left El botón izquierdo del mouse
mb_middle El botón central del mouse (esto puede no ser válido para todas las plataformas de destino)
mb_right El botón derecho del mouse
mb_none Sin botón de mouse
mb_any Cualquiera de los botones del mouse


Las siguientes funciones existen para los controles de botón del mouse estándar:

  1. Botón del ratón
  2. mouse_check_button
  3. mouse_check_button_pressed
  4. mouse_check_button_released
  5. mouse_clear
  6. mouse_last_button
  7. mueve hacia arriba la ruleta del ratón
  8. mouse_wheel_down
  9. mouse_x
  10. pardusco

Tenga en cuenta que también hay un conjunto de funciones de ventana relacionadas con el uso del mouse en los objetivos de escritorio:

  1. window_mouse_get_x
  2. window_mouse_get_y
  3. window_mouse_set
  4. window_view_mouse_get_x
  5. window_view_mouse_get_y
  6. window_views_mouse_get_x
  7. window_views_mouse_get_y
