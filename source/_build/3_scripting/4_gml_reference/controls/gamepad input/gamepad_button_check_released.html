gamepad_button_check_released


Descripción

Esta función volverá true o false Dependiendo de si el botón del gamepad dado se detecta como liberado o no. Tenga en cuenta que esta función solo se activará una vez para el botón en el momento en que se suelte. Para que se active nuevamente, primero se debe presionar el botón y luego soltarlo una vez más.


Sintaxis:

gamepad_button_check_released(device, button);

Argumento Descripción
device Qué dispositivo gamepad "slot" para comprobar.
button Qué botón del gamepad es constante para comprobar.


Devoluciones:

Boolean


Ejemplo:

if gamepad_button_check_released(0, gp_select)
   {
   audio_play_sound(snd_Button, 0, false);
   global.Pause = !global.Pause;
   }

El código anterior detectará si se presionó o no el botón "seleccionar" del gamepad conectado al dispositivo "slot" 0 y cambiar la variable global "Pausa".
