gamepad_get_device_count

Devuelve la cantidad de juegos conectados o el número de espacios disponibles para juegos.

Sintaxis:

gamepad_get_device_count();


Devoluciones: Real


Descripción

Esta función te dirá una de dos cosas. Ya sea la cantidad de almohadillas de juegos conectadas, o la cantidad de "ranuras" disponibles para las almohadillas de juegos a las que se va a conectar. El valor real de retorno dependerá de la plataforma y la configuración interna de esa plataforma y, como tal, esta función se debe utilizar junto con la función gamepad_is_connected para asegurarse de la cantidad exacta de almohadillas conectadas en cualquier momento.


Ejemplo:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
   {
   if gamepad_is_connected(i) global.gp[i] = true else global.gp[i] = false;
   }

El código anterior recorre las almohadillas disponibles (o las ranuras del gamepad) y luego comprueba cada uno de ellos para un gamepad conectado. el valor devuelto se usa para establecer una matriz global para true o false para usar en futuros controles.
