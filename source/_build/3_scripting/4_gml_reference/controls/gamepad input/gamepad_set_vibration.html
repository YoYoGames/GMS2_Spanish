gamepad_set_vibration

Controla la vibración del dispositivo

Sintaxis:

gamepad_set_vibration(device, left_motor, right_motor);


Argumento Descripción
device Qué dispositivo de gamepad "ranura" para verificar.
left_motor La cantidad de vibración del motor izquierdo de 0 a 1.
right_motor La cantidad de vibración del motor derecho de 0 a 1.


Devoluciones: N / A


Descripción

Con esta función puede configurar la vibración de los motores del gamepad, con cualquier motor usando un valor de 0 (sin vibración) a 1 (vibración completa). Tenga en cuenta que no hay un límite de tiempo para esta función, por lo que necesitará usar una variable o una alarma para apagar la vibración (configure los motores a 0) después de que haya transcurrido un tiempo determinado; de lo contrario, el gamepad continuará vibrando indefinidamente.

NOTA: Esta función solo está disponible actualmente para los módulos de destino estándar de Windows, PS4 y Xbox One.


Ejemplo:

if lives = 0
   {
   gamepad_set_vibration(0, 1, 1);
   alarm[0] = room_speed / 2;
   }

El código anterior se usaría (por ejemplo) en un evento de colisión para hacer que el gamepad enchufado en "slot" 0 vibre durante medio segundo, con la alarma configurada siendo utilizada para apagarla nuevamente una vez que ese tiempo haya pasado.
