gamepad_set_vibration


Sintaxis:

gamepad_set_vibration(device, left_motor, right_motor);

Argumento Descripción
device Qué dispositivo gamepad "slot" para comprobar.
left_motor La cantidad de vibración del motor izquierdo de 0 a 1.
right_motor La cantidad de vibración del motor derecho de 0 a 1.


Devoluciones:

N/A


Descripción

Con esta función, puede establecer la vibración de los motores del gamepad, ya sea que el motor utilice un valor de 0 (sin vibración) a 1 (vibración completa). Tenga en cuenta que no hay límite de tiempo para esta función, por lo que deberá usar una variable o una alarma para apagar la vibración (configurar los motores a 0) después de que haya transcurrido un tiempo determinado, de lo contrario, el gamepad seguirá vibrando indefinidamente.

NOTA: esta función actualmente solo está disponible para los módulos estándar de Windows, PS4 y Xbox One.


Ejemplo:

if lives = 0
   {
   gamepad_set_vibration(0, 1, 1);
   alarm[0] = room_speed / 2;
   }

El código anterior se usaría (por ejemplo) en un evento de colisión para hacer que el gamepad conectado a la "ranura" 0 vibre por medio segundo, con la alarma que se está configurando para apagarlo nuevamente una vez que haya pasado ese tiempo.
