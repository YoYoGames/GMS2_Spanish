gamepad_axis_value


Descripción

Puede usar esta función para obtener el valor de los diferentes ejes de una ranura de gamepad dada. Usted proporciona el índice de la ranura del gamepad para verificar, junto con una constante de índice de eje (como se indica aquí ) o un valor entero entre 0 y gamepad_axis_count() -1. El valor de retorno estará entre -1 y 1 para cada uno de los ejes horizontal y vertical disponibles.


Sintaxis:

gamepad_axis_value(device, axisIndex);

Argumento Descripción
device Qué dispositivo gamepad "slot" para comprobar.
axisIndex El índice del eje a verificar (ver la lista de constantes ).


Devoluciones:

real


Ejemplo:

var haxis = gamepad_axis_value(0, gp_axislh);
var vaxis = gamepad_axis_value(0, gp_axislv);
direction = point_direction(0, 0, haxis, vaxis);
speed = point_distance(0 ,0, haxis, vaxis) * 5;

El código anterior establece la velocidad y la dirección de la instancia según los valores devueltos del controlador analógico izquierdo del gamepad conectado al dispositivo "slot" 0.
