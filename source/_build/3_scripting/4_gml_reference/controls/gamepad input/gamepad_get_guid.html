gamepad_get_guid


Descripción

Con esta función, puede recuperar el GUID para el gamepad conectado al índice de ranura dado. Usted suministra la ranura del gamepad para verificar (de 0 a 11), y la función devolverá una de las siguientes cadenas:

  • " ninguno " si no hay GUID disponible o no hay un gamepad conectado
  • " índice de dispositivo fuera de rango " si el índice de ranura de gamepad está fuera de rango
  • La cadena GUID, algo así como "050000005e040000fd020000ffff3f00", si la ranura tiene un gamepad asignado y puede obtener el GUID para ello

Esta función se usaría generalmente en conjunto con gamepad_get_description() para generar parte de la cadena SDL requerida para volver a asignar los controladores en Windows, Android y macOS.


Sintaxis:

gamepad_get_guid(index);

Argumento Descripción
index Qué gamepad "ranura" índice para comprobar


Devoluciones:

String


Ejemplo:

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

El código anterior obtiene el GUID y las cadenas de descripción del nombre, luego las concatena y almacena la cadena final en una variable global para referencia de sutura.
