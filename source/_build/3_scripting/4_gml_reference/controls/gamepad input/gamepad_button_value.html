gamepad_button_value


Descripción

Con esta función puede obtener el valor actual de un botón analógico, de 0 a 1, donde 0 es sin presión y 1 es presión total. Usted proporciona el índice de la ranura del gamepad para verificar, junto con una constante de botón (como se indica aquí ) o un valor entero entre 0 y gamepad_button_count() -1.


Sintaxis:

gamepad_button_value(device, button);

Argumento Descripción
device Qué dispositivo gamepad "slot" para comprobar.
button Qué botón del gamepad es constante para comprobar.


Devoluciones:

Real


Ejemplo:

speed = gamepad_button_value(0, gp_shoulderrb) * 4;

El código anterior usa el valor de disparo analógico del gamepad conectado al dispositivo "slot" 0 para establecer la velocidad de la instancia.
