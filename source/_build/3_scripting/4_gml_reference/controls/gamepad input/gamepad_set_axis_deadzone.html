gamepad_set_axis_deadzone


Descripción

Esta función se puede utilizar para establecer la "zona muerta" del eje del joystick. Especifica la ranura del dispositivo para establecer, y luego establece un valor de 0 a 1 y si la cantidad de entrada es más baja que el valor dado, se considera que el eje del joystick está en 0. Tenga en cuenta que esta es una configuración global que afectará a todos Eje de todos los joysticks conectados a la ranura del dispositivo especificado.


Sintaxis:

gamepad_set_axis_deadzone(device, deadzone);

Argumento Descripción
device Qué dispositivo gamepad "slot" para comprobar.
deadzone El valor de la zona muerta de 0 a 1.


Devoluciones:

N/A


Ejemplo:

if gamepad_is_connected(0) gamepad_set_axis_deadzone(0, 0.05);
if gamepad_is_connected(1) gamepad_set_axis_deadzone(1, 0.05);
if gamepad_is_connected(2) gamepad_set_axis_deadzone(2, 0.05);
if gamepad_is_connected(3) gamepad_set_axis_deadzone(3, 0.05);

El código anterior establecerá la zona muerta del eje del joystick en cualquiera de los 4 dispositivos conectados a 0.05.
