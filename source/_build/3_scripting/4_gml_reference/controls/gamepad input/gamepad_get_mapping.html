gamepad_get_mapping


Descripción

Esta función se puede utilizar para recuperar la cadena de mapeo para el gamepad. Esta cadena será:

  • La cadena de mapa actual para el gamepad, por ejemplo:

    "050000005e040000fd020000ffff3f00, Xbox Wireless Controller, a: b0, b: b1, inicio: b4, dpdown: h0.4, dpleft: h0.8, dpright: h0.2, dpup: h0.1, guía: b6, leftshoulder: b9, leftstick: b7, lefttrigger: a4, leftx: a0, zurdo: a1, rightshoulder: b10, rightstick: b8, righttrigger: a5, rightx: a2, righty: a3, x: b2, y: b3, platform: android "


  • Una cadena con "sin asignación" si no hay ningún conjunto de asignación (esto no es lo mismo que cuando no hay asignación disponible)

  • Una cadena con "índice de dispositivo fuera de rango" si el índice de ranura no es un índice de gamepad válido

  • Una cadena vacía "" si no hay una asignación disponible para el gamepad conectado

Para obtener más información sobre el formato de la cadena de mapa devuelta, consulte la función gamepad_test_mapping().


Sintaxis:

gamepad_get_mapping(index);

Argumento Descripción
index De qué gamepad índice "ranura" para obtener el mapeo de.


Devoluciones:

String


Ejemplo:

var _gpMap = gamepad_get_mapping(global.PadIndex);
show_debug_message("Gamepad Mapping = " + _gpMap);

El código anterior obtiene la cadena del mapa para la ranura del gamepad dada y luego la envía a la consola para la depuración.
