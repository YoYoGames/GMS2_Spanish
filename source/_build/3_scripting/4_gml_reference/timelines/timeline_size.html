timeline_size

Devuelve la cantidad de momentos activos de una línea de tiempo determinada.

Sintaxis:

timeline_size(ind);


Argumento Descripción
ind El índice de la línea de tiempo obtiene la información de.


Devoluciones: N / A


Descripción

Con esta función, puede obtener el número total de momentos activos para una línea de tiempo (un momento "activo" es uno que tiene un código o DnD ™ agregado a él). Esto puede ser útil al crear líneas de tiempo dinámicas usando el timeline_moment_add_script() y timeline_moment_clear() funciones, ya que puede verificar si una determinada línea de tiempo tiene la cantidad correcta de momentos activos o ninguno.


Ejemplo:

if timeline_size(global.tl) == 0
   {
   timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), scr_spawn_enemy);
   }

El código anterior verifica el tamaño de línea de tiempo dado, y si devuelve 0 (es decir: la línea de tiempo no tiene momentos activos) agregará una secuencia de comandos para ser utilizada en un momento aleatorio dentro de la línea de tiempo indexada en la variable global "tl".
