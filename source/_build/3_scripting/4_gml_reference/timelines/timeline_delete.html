timeline_delete

Elimina una línea de tiempo determinada.

Sintaxis:

timeline_delete( ind );


Argumento Descripción
ind El índice de la línea de tiempo para eliminar.


Devoluciones: N / A


Descripción

Con esta función puedes eliminar cualquier línea de tiempo de tu juego. si esta línea de tiempo se creó en el árbol de recursos de GameMaker Studio 2, tenga en cuenta que se elimina por completo del juego e intentar llamar de nuevo a la línea de tiempo indexada después de usar esta función provocará un error. Si la línea de tiempo se creó dinámicamente con el timeline_add función, entonces esta función debe usarse en el momento en que la línea de tiempo ya no sea necesaria para evitar cualquier pérdida de memoria que pueda ralentizar (y finalmente bloquear) su juego.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

if timeline_exists(global.tl) timeline_delete(global.tl);

El código anterior comprueba si existe una línea de tiempo y está indexada en la variable global "tl" y, de ser así, elimina esa línea de tiempo.
