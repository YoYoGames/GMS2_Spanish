timeline_get_name

Obtiene el nombre de una línea de tiempo determinada.

Sintaxis:

timeline_get_name( ind );


Argumento Descripción
ind El índice de la línea de tiempo para verificar el nombre de.


Devoluciones: cadena


Descripción

Esta función se puede usar para obtener el nombre de una línea de tiempo como una cadena. si la línea de tiempo ha sido creada dinámicamente usando el timeline_add función, el nombre devuelto tendrá el formato "__newtimeline N " donde " N " es el número de la línea de tiempo (comenzando desde 0). Tenga en cuenta que esto es solo una cadena y no se puede usar para hacer referencia directamente a la línea de tiempo, para eso necesitaría el índice de línea de tiempo. Sin embargo, puede utilizar esta cadena para obtener el índice de línea de tiempo usando la cadena devuelta junto con la función asset_get_index().


Ejemplo:

str = timeline_get_name(global.tl);

El código anterior almacenará el nombre de la línea de tiempo indexada en la variable "global.tl".
