Líneas de tiempo

Las líneas de tiempo son un mecanismo poderoso para controlar los eventos en su juego, y se basan en "momentos", donde un "momento" es simplemente un paso de un solo juego. Estas líneas de tiempo se crean generalmente desde el Árbol de recursos usando el Editor de línea de tiempo de forma similar a un objeto, es decir: usted crea una línea de tiempo, luego, en cada momento que lo requiere para realizar una acción, agrega algún código o un icono DnD™ para para realizar. Una vez creado, puede detener, iniciar y manipular líneas de tiempo a través del código. Incluso puede modificar momentos individuales definiendo un guión y agregándolo dinámicamente a la línea de tiempo desde un objeto controlador (por ejemplo), lo que lo convierte en una herramienta muy poderosa y versátil al crear sus juegos.

Las siguientes funciones son para agregar y manipular Timelines desde tu juego:

  • Timeline_exists
  • timeline_get_name
  • timeline_add
  • timeline_delete
  • timeline_moment_add_script
  • timeline_moment_clear
  • timeline_clear
  • timeline_size
  • timeline_max_moment

Las siguientes variables están "incorporadas" a todas las instancias para facilitar el trabajo con Timelines y darle control sobre:

  • timeline_index
  • Timeline_running
  • timeline_speed
  • timeline_position
  • Timeline_loop

Tenga en cuenta que puede encontrar más información sobre variables de instancia aquí: Variables de instancia.
