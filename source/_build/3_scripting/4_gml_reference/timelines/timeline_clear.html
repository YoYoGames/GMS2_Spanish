timeline_clear

Borra la línea de tiempo de todos los momentos.

Sintaxis:

timeline_clear(ind);


Argumento Descripción
ind El índice de la línea de tiempo para borrar.


Devoluciones: N / A


Descripción

Con esta función puede borrar una línea de tiempo específica de "momentos", eliminando todos los códigos y acciones para esa línea de tiempo y dejándola vacía.


Ejemplo:

if timeline_position > 200
   {
   timeline_clear(global.tl);
   timeline_index = -1;
   }

El código anterior borrará la línea de tiempo especificada indexada por la variable "global.tl" de todos los momentos cuando se haya pasado un momento específico.
