Timeline_exists

Comprueba si existe una línea de tiempo determinada.

Sintaxis:

timeline_exists(ind);


Argumento Descripción
ind El índice de la línea de tiempo para verificar.


Devoluciones: Boolean


Descripción

Con esta función puede verificar y ver si existe una línea de tiempo (devoluciones) true ) o no (devoluciones) false ) Esto es particularmente útil cuando se crean líneas de tiempo dinámicamente con el timeline_add función, pero debe tener en cuenta que si busca una variable no inicializada (que de otro modo se asignaría al índice de una línea de tiempo) se producirá un error.


Ejemplo:

if timeline_exists(global.tl) timeline_delete(global.tl);

El código anterior comprueba si existe una línea de tiempo y está indexada en la variable global "tl" y, de ser así, elimina esa línea de tiempo.
