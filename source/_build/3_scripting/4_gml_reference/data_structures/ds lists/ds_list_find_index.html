ds_list_find_index

Sintaxis:

ds_list_find_index(id, val);


Argumento Descripción
id La identificación de la lista a usar.
val El valor para encontrar.


Devoluciones: Real


Descripción

Con esta función puede verificar la lista dada para un valor y se devolverá la posición dentro de la lista para ese valor. Tenga en cuenta que si hay más de una entrada en la lista con el mismo valor, se puede devolver la posición de cualquiera de ellas, y si el valor no existe, se devolverá -1. Tenga en cuenta que el valor puede ser una matriz, que puede verificar con la función is_array.


Ejemplo:

pos = ds_list_find_index(list, "Player1");

El código anterior verifica la lista indexada en la variable "list" para el valor "Player1" y almacena la posición devuelta en la variable "pos".
