ds_list_create

Sintaxis:

ds_list_create();


Devoluciones: Real


Descripción

Esta función creará una nueva estructura de datos de lista y devolverá el valor del índice. Este valor debe almacenarse en una variable y usarse en todas las funciones adicionales relacionadas con la lista. Tenga en cuenta que si intenta acceder a un valor en la lista antes de que se complete, es decir, la lista está "vacía", entonces el valor de retorno puede ser 0, ya que internamente, las primeras entradas de la lista se configuran cuando se crean para minimizar los problemas de rendimiento. cuando se agregan inicialmente elementos a la lista (aunque ds_list_size() La función todavía devolverá 0 en una lista recién creada). Si desea asegurarse de que la lista esté "realmente" vacía en la creación, debe llamar ds_list_clear() después de crear la lista, lo que significará que cualquier valor devuelto para las ranuras de lista no pobladas será undefined.

¡IMPORTANTE! Cuando creas una estructura de datos, el valor del índice para identificarlo es un valor entero que comienza en 0. Esto significa que las estructuras de datos de diferentes tipos pueden tener el mismo valor de índice, por lo que si tienes dudas debes utilizar el ds_exists Funciona antes de acceder a ellos. También tenga en cuenta que los índices se reutilizan, por lo que un valor nuevo de índice de estructura de datos destruido puede ser utilizado por uno recién creado posteriormente.


Ejemplo:

list = ds_list_create();

Esto creará una nueva lista y asignará su ID de índice a la variable de instancia "lista".
