ds_list_set

Sintaxis:

ds_list_set(id, pos, val);


Argumento Descripción
id La identificación de la lista para agregar a.
pos La posición dentro de la lista para establecer.
[val2,... max_val] El valor para agregar a la lista.


Devoluciones: N / A


Descripción

Esta función se puede usar para configurar una entrada de lista previamente agregada. Usted proporciona la Id. De la lista (como se devolvió al crear la lista) y la posición dentro de la lista para establecer, así como el valor para establecerlo. Tenga en cuenta que si la entrada configurada está fuera de los límites de la lista (es decir, establece la entrada de lista 20 pero la lista actual solo contiene 10 entradas), la lista se completará en la posición determinada y cada entrada se establecerá en 0.


Ejemplo:

for (var i = 0; i < ds_list_size(list); i++;)
    {
    ds_list_set(list, i, -1);
    }

El código anterior agregará el valor almacenado en la variable "puntuación" a la lista indexada en la variable "sc_list".
