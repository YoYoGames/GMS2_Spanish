ds_list_find_value

Sintaxis:

ds_list_find_value(id, pos);


Argumento Descripción
id El id de la lista a utilizar.
pos La posición a mirar, donde 0 corresponde al comienzo de la lista y la posición final es ds_list_size (id) -1.


Devoluciones: Real, String o Undefined


Descripción

Con esta función, puede verificar la posición de la lista dada y se devolverá el valor mantenido dentro de la lista para esa posición. Tenga en cuenta que si asigna una posición que está fuera del tamaño de la lista dada (es decir, la posición 11 en una lista de 10 valores), la función puede regresar undefined o 0. Esto se debe a que cuando crea la lista, internamente, las primeras entradas de la lista se configuran en 0 para minimizar los problemas de rendimiento cuando se agregan inicialmente elementos a la lista (aunque ds_list_size() La función todavía devolverá 0 en una lista recién creada). Si desea asegurarse de que la lista esté "realmente" vacía en la creación, debe llamar ds_list_clear() después de crear la lista, lo que significará que cualquier valor devuelto para las ranuras de lista no pobladas será undefined, que luego puede verificar de manera consistente usando is_undefined() función.


Ejemplo:

val = ds_list_find_value(list, ds_list_size(list) - 1);

El código anterior verifica la lista indexada en la variable "lista" en la última posición de la lista y almacena el valor devuelto en la variable "val".
