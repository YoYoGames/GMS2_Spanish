ds_queue_head

Lee el valor de la cabecera de la cola.

Sintaxis:

ds_queue_head(id);


Argumento Descripción
id El id de la estructura de datos para leer.


Devoluciones: Real, Cadena, undefined


Descripción

Esta función solo leerá el primer valor de la cola (el que está "en la cabecera"). No eliminará de la cola el valor, lo que significa que aún se puede leer en el futuro con esta función o el ds_queue_dequeue. Si la cola está vacía, la función devolverá la constante undefined De lo contrario, devolverá el valor real o de cadena contenido en la cola.


Ejemplo:

num = ds_queue_head(control_queue);

El código anterior leerá el valor de la cola indexada en la variable "control_queue" y almacenará el valor de retorno en la variable "num".
