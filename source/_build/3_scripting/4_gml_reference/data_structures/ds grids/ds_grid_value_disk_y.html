ds_grid_value_disk_y

Encuentra la posición y de un valor encontrado dentro de una región circular de una grilla.

Sintaxis:

ds_grid_value_disk_y(index, xm, ym, r, val);


Argumento Descripción
index El índice de la grilla
xm La posición x del disco en la cuadrícula.
ym La posición y del disco en la cuadrícula.
r El radio del disco en la grilla.
val El valor para encontrar.


Devoluciones: Real


Descripción

Con esta función puede obtener la coordenada y (dentro de la región en forma de disco de cuadrícula dada) del valor que se busca. Tenga en cuenta que si el valor que se busca no existe, la función devolverá 0, por lo que es una buena idea verificar si los valores existen primero con ds_grid_value_disk_exists luego ejecute esta función (junto con ds_grid_value_disk_x función) para obtener la posición exacta dentro del disco que se busca.


Ejemplo:

if ds_grid_value_disk_exists(grid, 5, 5, 5, val)
   {
   xpos = ds_grid_value_disk_x(grid, 5, 5, 5, val);
   ypos = ds_grid_value_disk_y(grid, 5, 5, 5, val);
   }

El código anterior comprueba un ds_grid para un valor específico dentro de una región de disco. si se encuentra, almacena la posición xey del valor en dos variables para su uso posterior.
