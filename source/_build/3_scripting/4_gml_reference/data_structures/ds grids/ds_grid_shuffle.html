ds_grid_shuffle


Descripción

Esta función se puede usar para aleatorizar las posiciones de todos los valores en todas las celdas dentro de una cuadrícula.

NOTA: Esta función cambiará las celdas de la cuadrícula a las mismas posiciones cada vez que el juego se ejecute de nuevo debido al hecho de que GameMaker Studio 2 genera la misma inicialización aleatoria inicial cada vez para hacer que la depuración del código sea una tarea mucho más fácil. Para evitar este comportamiento utiliza. randomise al comienzo de tu juego. Esto solo es cierto cuando se prueba y depura el juego, ya que el paquete ejecutable final no mostrará este comportamiento y será aleatorio en cada juego.


Sintaxis:

ds_grid_shuffle(index);

Argumento Descripción
index El índice de la rejilla a barajar.


Devoluciones:

N/A


Ejemplo:

ds_grid_shuffle(grid)

Esto tomaría todos los valores de ds_grid indexados en la variable "grilla" y los ordenaría aleatoriamente para obtener un orden diferente.
