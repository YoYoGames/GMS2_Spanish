ds_grid_value_y


Descripción

Con esta función, puede obtener la coordenada y (dentro de la región de la cuadrícula rectangular dada) del valor que se está buscando. Usted da el índice de cuadrícula DS (como lo devuelve ds_grid_create ) junto con las posiciones x / y de las esquinas superior izquierda e inferior derecha de la región de la cuadrícula para buscar, antes de proporcionar el valor a buscar. Si el valor que se busca no existe, la función devolverá -1.


Sintaxis:

ds_grid_value_y(index, x1, y1, x2, y2, val);

Argumento Descripción
index El índice de la cuadrícula.
x1 La posición x de la izquierda de la región en la cuadrícula, de 0 a (ancho de cuadrícula - 1).
y1 La posición y de la parte superior de la región en la cuadrícula, de 0 a (altura de cuadrícula - 1).
x2 La posición x de la derecha de la región en la cuadrícula, de 0 a (ancho de cuadrícula - 1).
y2 La posición y de la parte inferior de la región en la cuadrícula, de 0 a (altura de cuadrícula - 1)
val El valor a encontrar.


Devoluciones:

Real


Ejemplo:

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
   {
   xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);
   ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val);
   }

El código anterior verifica un ds_grid para un valor específico dentro de una región rectangular. si se encuentra, luego almacena las posiciones x e y del valor en dos variables para su uso posterior.
