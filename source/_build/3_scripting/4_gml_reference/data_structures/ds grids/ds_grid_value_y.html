ds_grid_value_y

Devuelve la posición y de un valor dentro de una región de una grilla.

Sintaxis:

ds_grid_value_y(index, x1, y1, x2, y2, val);


Argumento Descripción
index El índice de la grilla
x1 La posición x de la izquierda de la región en la cuadrícula.
y1 La posición y de la parte superior de la región en la cuadrícula.
x2 La posición x de la derecha de la región en la cuadrícula.
y2 La posición y de la parte inferior de la región en la cuadrícula.
val El valor para encontrar.


Devoluciones: Real


Descripción

Con esta función puede obtener la coordenada y (dentro de la región de cuadrícula rectangular dada) del valor que se busca. Tenga en cuenta que si el valor que se busca no existe, la función devolverá 0, por lo que es una buena idea verificar si los valores existen primero con ds_grid_value_exists luego ejecute esta función (junto con ds_grid_value_x función) para obtener la posición exacta dentro de la región que se busca.


Ejemplo:

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
   {
   xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);
   ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val);
   }

El código anterior comprueba una ds_grid para un valor específico dentro de una región rectangular. si se encuentra, almacena la posición xey del valor en dos variables para su uso posterior.
