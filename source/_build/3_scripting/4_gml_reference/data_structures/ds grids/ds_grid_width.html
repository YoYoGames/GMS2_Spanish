ds_grid_width

Devuelve el ancho de una grilla.

Sintaxis:

ds_grid_width(index);


Argumento Descripción
index Este índice de la grilla para encontrar el ancho de.


Devoluciones: Real


Descripción

Esta función devolverá el ancho de la grilla dada. Este valor es el número de celdas que la grilla tiene a lo largo del eje x y siempre es un número entero, como se muestra en la siguiente imagen:


Ejemplo:

var i, j;
for (i = 0; i < ds_grid_width(grid); i += 1)
   {
   for (j = 0; j < ds_grid_height(grid); j += 1)
      {
      if ds_grid_get(grid, i, j) = 1
         {
         instance_create_layer(i * 32, j * 32, "Walls", obj_Wall);
         }
      }
   }

El código anterior recorrerá el ds_grid indexado en la variable "grid" y si el valor encontrado en cualquier celda específica es igual a 1, creará una instancia de "obj_Wall" en la posición apropiada dentro de la sala.
