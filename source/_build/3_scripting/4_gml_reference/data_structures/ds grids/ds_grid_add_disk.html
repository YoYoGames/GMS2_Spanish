ds_grid_add_disk

Agrega a todas las celdas de una región circular en una cuadrícula un valor indicado (puede ser tanto un número como una cadena).

Sintaxis:

ds_grid_add_disk(index, xm, ym, r, val);


Argumento Descripción
index El índice de la grilla
xm La posición x del disco en la cuadrícula.
ym La posición y del disco en la cuadrícula.
r El radio del disco en la grilla.
val El valor para agregar a las celdas dentro del disco.


Devoluciones: N / A


Descripción

Esta función se puede usar para agregar un valor dado (real o cadena) a todos los valores de las celdas que se encuentran dentro del área de disco definida de una grilla. El valor que se debe agregar debe ser del mismo tipo que el que se mantiene dentro de las celdas de la cuadrícula, es decir: no se puede agregar una cadena a un real o viceversa, y para cadenas esto corresponde a la concatenación.


Ejemplo:

ds_grid_add_disk(grid, 7, 6, 5, 2);

Esto agregaría 2 a todos los valores contenidos en las celdas dentro del área de disco definida de la grilla ds_ a la que hace referencia la variable "grilla".
