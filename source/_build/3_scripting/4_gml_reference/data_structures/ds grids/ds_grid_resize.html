ds_grid_resize

Cambia el tamaño de la grilla a un nuevo ancho y alto.

Sintaxis:

ds_grid_resize(index, w, h);


Argumento Descripción
index Este índice de la grilla para cambiar el tamaño.
w El nuevo ancho de la grilla
h La nueva altura de la grilla.


Devoluciones: N / A


Descripción

Con esta función, puede cambiar el tamaño de la ds_grid dada para que tenga un ancho y / o altura diferente. Si el tamaño de la cuadrícula es mayor que la cuadrícula actual, las celdas nuevas tendrán un valor base de 0, y si el tamaño es más pequeño, los valores retenidos en las celdas que ya no están dentro del nuevo tamaño se perderán. Todas las demás celdas quedarán intactas.


Ejemplo:

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1);

El código anterior cambiará el tamaño del ds_grid indexado en la variable global "Grid" y luego lo borrará para que cada celda tenga el valor -1.
