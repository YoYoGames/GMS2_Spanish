ds_grid_write


Descripción

Esta función se puede utilizar para convertir el ds_grid dado en una cadena, que luego se puede almacenar en un archivo externo (por ejemplo). Puede leer la cadena devuelta de esta función nuevamente en un ds_grid usando la función ds_grid_read().

NOTA: La cadena devuelta no es una cadena legible para el ser humano, sino un volcado del contenido de la estructura de datos.


Sintaxis:

ds_grid_write(index);

Argumento Descripción
index El índice de la cuadrícula a escribir.


Devoluciones:

String


Ejemplo:

ini_open("Save.ini");
ini_write_string("Save", "0", ds_grid_write(mygrid));
ini_close()

El código anterior abrirá un archivo ini (creándolo si aún no existe) y luego escribe el ds_grid dado como una cadena para ese archivo.
