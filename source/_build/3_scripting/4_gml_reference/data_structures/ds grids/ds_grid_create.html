ds_grid_create


Descripción

Con esta función, puede crear una nueva estructura de datos ds_grid del ancho y alto de celda especificados. Esta función devuelve un ID que debe utilizarse en todas las funciones adicionales que se ocupan de este ds_grid.

¡IMPORTANTE! Cuando creas una estructura de datos, el valor del índice para identificarlo es un valor entero que comienza en 0. Esto significa que las estructuras de datos de diferentes tipos pueden tener el mismo valor de índice, por lo que si tienes dudas debes utilizar el ds_exists Funciona antes de acceder a ellos. También tenga en cuenta que los índices se reutilizan, por lo que un valor nuevo creado por un índice de estructura de datos destruido puede ser utilizado posteriormente.


Sintaxis:

ds_grid_create(w, h);

Argumento Descripción
w El ancho de la rejilla a crear.
h La altura de la rejilla a crear.


Devoluciones:

Real


Ejemplo:

mygrid = ds_grid_create(10, 10)

Esto crea una cuadrícula de 10 celdas de alto y 10 celdas de ancho.
