ds_grid_read


Descripción

Esta función se puede utilizar para convertir una cadena creada previamente por la función ds_grid_write() de nuevo en un ds_grid. El ds_grid debe haberse creado previamente (vea el ejemplo a continuación). Tenga en cuenta que si la estructura de datos se creó con versiones anteriores de GameMaker, debe agregar el argumento opcional "heredado", configurándolo true como el formato de cadena ha cambiado con esta versión.


Sintaxis:

ds_grid_read(index, string [, legacy]);

Argumento Descripción
index El índice de la cuadrícula para leer.
string La cadena para leer en el ds_grid.
legacy ( optional ) Pueden ser cualquiera de los dos true o false u omitido por completo.


Devoluciones:

N/A


Ejemplo:

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open("Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", ""));
ini_close()

El código anterior crea un ds_grid basado en el tamaño de la habitación (cada cuadrado de píxeles de 32x32 representa una celda de la cuadrícula) y luego lee un conjunto previamente guardado de datos de cuadrícula de un archivo ini en la nueva cuadrícula de ds.
