ds_exists

Verifique si existe una estructura de datos de un tipo determinado.

Sintaxis:

ds_exists(ind, type);


Argumento Descripción
ind El índice variable para verificar la estructura de datos
type El tipo de estructura de datos a verificar (ver la lista de constantes a continuación)


Devoluciones: Boolean


Descripción

Con esta función puede verificar si existe una estructura de datos del tipo especificado. Usted suministra el valor "índice" (como se mantiene en una variable) y el "tipo" ds, que puede ser cualquiera de las constantes enumeradas a continuación, y la función regresará true si la estructura de datos existe y false de otra manera.

Constante Descripción
ds_type_map Una estructura de datos de mapa
ds_type_list Una estructura de datos de lista
ds_type_stack Una estructura de datos de pila
ds_type_grid Una estructura de datos de cuadrícula
ds_type_queue Una estructura de datos en cola
ds_type_priority Una estructura de datos de prioridad


Ejemplo:

if !ds_exists(ai_grid, ds_type_grid)
   {
   ai_grid = ds_grid_create(room_width / 32, room_height / 32);
   }

El código anterior verifica la variable (inicializada previamente) "ai_grid" para ver si indexa una estructura de datos de tipo ds_grid, y si no lo hace, crea una y almacena su índice en la variable.
