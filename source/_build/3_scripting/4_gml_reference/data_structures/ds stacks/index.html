DS Stacks

Una ds_stack es una estructura de datos de último en entrar primero en salir.

Una estructura de datos de pila es la denominada estructura de último en entrar primero en salir (LIFO). Puede insertar valores en una pila y eliminarlos de nuevo sacándolos de la pila, y el valor que se insertó en la pila más recientemente es el primero que se saca de allí (solo piense en una pila de monedas, donde cada la moneda que agrega debe ser removida primero antes de que pueda llegar al resto de las monedas debajo). Las pilas se usan a menudo cuando hay interrupciones que manejar, o cuando se tienen funciones recursivas, o incluso cuando se construye una IA rudimentaria para tus juegos.

NOTA: Al igual que con todos los recursos dinámicos, las estructuras de datos ocupan la memoria y, por lo tanto, siempre deben destruirse cuando ya no son necesarias para evitar fugas de memoria que disminuirán la velocidad y, finalmente, bloquearán su juego.

Las siguientes funciones existen que se ocupan de DS Stacks.

  1. ds_stack_create
  2. ds_stack_destroy
  3. ds_stack_clear
  4. ds_stack_empty
  5. ds_stack_size
  6. ds_stack_copy
  7. ds_stack_top
  8. ds_stack_pop
  9. ds_stack_push
  10. ds_stack_read
  11. ds_stack_write
NOTA: si necesita verificar si existe una estructura de datos, puede usar ds_exists() función.
