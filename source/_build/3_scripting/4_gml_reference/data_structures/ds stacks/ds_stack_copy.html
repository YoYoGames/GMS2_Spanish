ds_stack_copy

Copia el contenido de una pila a otra.

Sintaxis:

ds_stack_copy(id, source);


Argumento Descripción
id La identificación de la NUEVA pila.
source La pila ORIGINAL para copiar a id.


Devoluciones: N / A


Descripción

Esta función se puede usar para copiar el contenido de una pila en otra. Tenga en cuenta que esto NO elimina el contenido de la pila original, ni destruye la pila original. Cuando se utiliza esta función, la pila que se está copiando debe haberse creado previamente y si contenía algún elemento antes de la copia, estos se borrarán primero (lo que significa que esta información se perderá).


Ejemplo:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
   {
   stack = ds_stack_create();
   ds_stack_copy(stack, other.stack);
   }

La función anterior crea una nueva instancia y luego en esa instancia crea una nueva ds_stack y copia el contenido de la pila en la instancia que ejecuta el bloque de código, en la pila de instancia recién creada.
