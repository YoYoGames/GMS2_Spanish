Colas de prioridad de DS

Una ds_priority es una estructura de datos basada en el valor de importancia de la información que contiene.

Una cola ds_priority es una estructura de datos especial similar a una ds_queue, solo aquí la cola de valores está ordenada por las prioridades (siempre números reales) que los valores han sido asignados. Esta estructura puede ser muy útil para construir tablas de clasificación o listas de información donde se puede ponderar la importancia de cada entrada.

NOTA: Al igual que con todos los recursos dinámicos, las estructuras de datos ocupan la memoria y, por lo tanto, siempre deben destruirse cuando ya no son necesarias para evitar fugas de memoria que disminuirán la velocidad y, finalmente, bloquearán su juego.

Existen las siguientes funciones que se ocupan de colas ds_priority:

  1. ds_priority_create
  2. ds_priority_destroy
  3. ds_priority_clear
  4. ds_priority_empty
  5. ds_priority_size
  6. ds_priority_add
  7. ds_priority_change_priority
  8. ds_priority_delete_max
  9. ds_priority_delete_min
  10. ds_priority_delete_value
  11. ds_priority_find_max
  12. ds_priority_find_min
  13. ds_priority_find_priority
  14. ds_priority_copy
  15. ds_priority_read
  16. ds_priority_write
NOTA: si necesita verificar si existe una estructura de datos, puede usar ds_exists() función.
