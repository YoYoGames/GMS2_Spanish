ds_priority_add

Agrega un valor priorizado a la cola.

Sintaxis:

ds_priority_add(id, val, priority);


Argumento Descripción
id La identificación de la cola de prioridad para agregar a.
val El valor para agregar a la cola de prioridad.
priority La prioridad del valor a agregar.


Devoluciones: N / A


Descripción

Con esta función puede agregar un valor (ya sea un número real o una cadena) a una cola de prioridad, al mismo tiempo que le asigna un valor de prioridad.


Ejemplo:

ds_priority_add(ai_priority, scr_AI_Search, 5);

El código anterior agrega un script a la cola de prioridad indexada en la variable "ai_priority" y le asigna una prioridad de 5.
