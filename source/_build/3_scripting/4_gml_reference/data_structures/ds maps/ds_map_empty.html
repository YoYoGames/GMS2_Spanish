ds_map_empty

Devuelve si el mapa dado está vacío de pares clave / valor o no.

Sintaxis:

ds_map_empty(id);


Argumento Descripción
id La identificación de la estructura de datos para verificar.


Devuelve: booleano.


Descripción

Esta función simplemente regresará false si el ds_map especificado (creado previamente) contiene pares de clave / valor, o true si no es así


Ejemplo:

if ds_map_empty(inventory)
   {
   weight = 0;
   }

El código anterior comprueba si el ds_map indexado en la variable "inventario" tiene pares de clave / valor y, si no lo hace, establece la variable "peso" en 0.
