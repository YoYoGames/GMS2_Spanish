ds_map_copy

Copia el contenido de un mapa completo en otra identificación.

Sintaxis:

ds_map_copy(id, source);


Argumento Descripción
id La identificación del mapa que está copiando a </>
source El id del mapa está copiando desde


Devoluciones: N / A


Descripción

Puede usar esta función para copiar el contenido de un mapa en otro que haya creado previamente usando ds_map_create (). Si el ds_map que se está copiando a no está vacío, entonces esta función se aclarará en primer lugar antes de copiar. El ds_map original no cambia por este proceso.


Ejemplo:

inventory_2 = ds_map_create();
ds_map_copy(inventory_2, inventory_1);

El código anterior creará un nuevo mapa y lo asignará a la variable "inventory_2". Luego copiará el contenido del ds_map indexado en la variable "inventory_1" a este nuevo mapa.
