ds_map_add_list

Agrega una ds_list previamente creada a un ds_map determinado.

Sintaxis:

ds_map_add_list(id, key, value)


Argumento Descripción
id La identificación del mapa a usar.
key La clave para la lista agregada.
value La identificación de la lista para agregar.


Devoluciones: N / A


Descripción

Con esta función puedes asignar un (previamente creado) ds_list a una clave dentro del ds_map dado. Esta función está diseñada para crear mapas compatibles con JSON que luego codificaría usando json_encode y solo debe usarse junto con esa funcionalidad. Si un ds_map tiene una lista agregada de esta manera, la destrucción del mapa principal también destruirá las listas contenidas y liberará su memoria.


Ejemplo:

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
var j_map = ds_map_create();
ds_map_add_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_map_destroy(j_map);

El código anterior creará una lista y la poblará con los diversos valores de variables globales. Esta lista se "anida" dentro de un ds_map, y el mapa se codifica en una cadena JSON, antes de que se destruya el mapa, eliminándolo, y cualquier lista que contenga, de la memoria.
