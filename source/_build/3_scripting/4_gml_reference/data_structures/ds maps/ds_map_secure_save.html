ds_map_secure_save

Esta función guardará el ds_map dado en una ubicación segura en la plataforma de destino.

Sintaxis:

ds_map_secure_save(map, filename);


Argumento Descripción
map El id de la estructura de datos a utilizar.
filename El nombre del archivo para guardar el mapa a


Devoluciones: Boolean


Descripción

Esta función guardará el contenido del ds_map dado en un archivo. El archivo en sí puede tener casi cualquier extensión (por ejemplo, *.dat, *.json, *.bin, etc...) y se cifrarán y almacenarán en una ubicación segura en la plataforma de destino. De esta manera, puede almacenar información confidencial, como detalles de compra dentro de la aplicación o contraseñas, de manera local y segura. A continuación, puede volver a cargar el ds_map utilizando la función ds_map_secure_load(). Tenga en cuenta que si el mapa de DS que se está guardando contiene una matriz, esta matriz se convertirá en una lista de DS cuando se guarde.

¡IMPORTANTE! El archivo de mapa DS seguro guardado solo se puede cargar en el dispositivo que lo creó, y si intenta cargar un archivo que se guardó en un dispositivo diferente, entonces no funcionará.


Ejemplo:

ds_map_secure_save(purchase_map, "p_data.dat");

El código anterior guardará el ds_map indexado en la variable "p_data" en el archivo dado para su posterior recuperación.
