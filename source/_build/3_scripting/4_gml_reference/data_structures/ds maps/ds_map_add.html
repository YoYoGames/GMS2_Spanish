ds_map_add

Agrega el valor dado y la clave asociada en el mapa.

Sintaxis:

ds_map_add(id, key, val);


Argumento Descripción
id La identificación del mapa a agregar a.
key La clave del valor para agregar.
val El valor para agregar al mapa.


Devoluciones: Boolean


Descripción

Esta función se debe usar para agregar conjuntos de pares clave / valor en el ds_map especificado. Puede verificar esta función para ver si fue exitosa o no, ya que puede fallar si ya existe la misma clave en el ds_map o si especifica un ds_map inexistente como el id del mapa para agregar. Tanto las claves como los valores pueden estar formados por enteros o cadenas, por lo que todos los siguientes son aceptables:

ds_map_add(map, 5, 1);
ds_map_add(map, "level", 100);
ds_map_add(map, 89, "hello world");
ds_map_add(map, "fish", "good");

También puede agregar a un mapa utilizando el descriptor de acceso "?", Como se muestra a continuación:

map[? 5] = 1;
map[? "level"] = 100;
map[? 89] = "hello world";
map[? "fish"] = "good";



NOTA: A diferencia de otras estructuras de datos en GameMaker Studio 2, esta clave no irá al inicio (ni al final) del ds_map, sino que irá al ds_map en algún lugar.


Ejemplo:

inventory = ds_map_create();
ds_map_add(inventory, "hp potion", 1);
ds_map_add(inventory, "gold", 100);

Esto creará un nuevo mapa, asignando su id a la variable "inventario". Luego agrega dos nuevas claves al mapa, "hp potion" y "gold", y establece sus valores iniciales como 1 y 100.
