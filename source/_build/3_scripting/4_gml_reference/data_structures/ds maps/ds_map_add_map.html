ds_map_add_map

Agrega un ds_map previamente creado a un ds_map determinado.

Sintaxis:

ds_map_add_map(id, key, value)


Argumento Descripción
id La identificación del mapa a usar.
key La clave para el mapa agregado.
value La identificación del mapa para agregar.


Devoluciones: N / A


Descripción

Con esta función puedes asignar un (previamente creado) ds_map a una clave dentro del ds_map dado. Esta función está diseñada para crear mapas compatibles con JSON que luego codificaría usando json_encode y solo debe usarse junto con esa funcionalidad. Si un ds_map tiene otro mapa agregado de esta manera, la destrucción del mapa principal también destruirá los mapas contenidos y liberará su memoria.


Ejemplo:

var j_map = ds_map_create();
var j_list = ds_list_create();
var sub_map = ds_map_create();
ds_map_add_list(sub_map, "list", j_list);
ds_map_add(sub_map, "array", j_array);
ds_map_add_map(j_map, "map", sub_map);
var j = json_encode(j_map);
ds_map_destroy(j_map);

El código anterior creará dos ds_maps y luego rellenará uno con una lista y una matriz antes de agregarlo al segundo, que luego se codifica en una cadena JSON. El mapa se destruye para eliminarlo de la memoria y de cualquier otro mapa o lista que contenga.
