ds_map_replace_list

Reemplace una lista con otra lista después de que se haya agregado a un ds_map.

Sintaxis:

ds_map_replace_list(id, key, value)


Argumento Descripción
id La identificación del mapa a usar.
key La clave para reemplazar
value La id de la lista ds para usar para reemplazar la que estaba almacenada previamente en la clave dada.


Devoluciones: N / A


Descripción

Con esta función puedes reemplazar un ds_list que ha sido almacenado en la "clave" dada con otra lista que ha sido creada previamente. Esta función está diseñada para crear mapas compatibles con JSON que luego codificaría usando json_encode y solo debe usarse junto con esa funcionalidad.


Ejemplo:

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
ds_map_replace_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_list_destroy(j_list);

El código anterior creará un ds_list y llenarlo con los valores de varias variables globales antes de reemplazar una lista previamente almacenada en el ds_map "j_map".
