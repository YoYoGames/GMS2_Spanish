ds_map_replace

Reemplaza el valor de una tecla determinada en el mapa con un nuevo valor.

Sintaxis:

ds_map_replace( id, key, val );


Argumento Descripción
id La identificación del mapa para cambiar
key La clave con el valor que debe ser reemplazado por el nuevo
val El nuevo valor para reemplazar el valor dado con


Devoluciones: N / A.


Descripción

Con esta función puede cambiar el valor de la clave dada dentro del ds_map (creado previamente). Tenga en cuenta que si la clave que debe reemplazarse no existe, se crea y se le asigna el valor dado.


Ejemplo:

ds_map_replace(inventory, "torso", 55);

El código anterior busca el ds_map para la clave "torso" y cuando lo encuentra (o se crea si no existe), el valor actual se reemplaza por el especificado.
