ds_map_find_first

Encuentra y devuelve la primera clave, almacenada por la computadora, en ds_map.

Sintaxis:

ds_map_find_first(id);


Argumento Descripción
id La identificación del mapa a usar.


Devoluciones: Real o String


Descripción

Esta función devuelve la primera clave almacenada en el ds_map. ¡Esta no es la primera clave en el orden en que los agregó! Ds_maps no se almacenan en forma lineal, para eso usan ds_lists, por lo que todo lo que hace es encontrar la primera clave almacenada por la computadora. Esto puede ser útil si tiene que recorrer el mapa ds buscando algo, pero debe evitarse si es posible, ya que puede ser lento.


Ejemplo:

var size = ds_map_size(inventory) ;
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
   {
   if key != "gold"
      {
      key = ds_map_find_next(inventory, key);
      }
   else break;
   }

El código anterior crea algunas variables temporales y luego obtiene el tamaño ds_map y encuentra la primera clave almacenada por la computadora en el mapa. Luego usa un for loop para iterar a través del ds_map buscando el valor clave "gold". Si lo encuentra, rompe el ciclo.
