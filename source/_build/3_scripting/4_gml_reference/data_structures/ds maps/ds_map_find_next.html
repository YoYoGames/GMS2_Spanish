ds_map_find_next

Encuentra y devuelve la siguiente clave en el ds_map después de la especificada en la función.

Sintaxis:

ds_map_find_next(id, key);


Argumento Descripción
id La identificación del mapa a usar.
key La clave para encontrar el siguiente.


Devoluciones: Real, cadena o indefinido


Descripción

Esta función devuelve la siguiente clave almacenada en el ds_map después de la especificada en la función. Esto puede ser útil si tiene que recorrer el mapa ds buscando algo, pero debe evitarse si es posible, ya que puede ser lento. Si no existe tal clave, entonces la función regresará <undefined>. Siempre debe verificar esto usando el is_undefined() función.



Ejemplo:

var size, key, i;
size = ds_map_size(inventory);
key = ds_map_find_first(inventory);
for (i = 0; i < size; i++;)
   {
   if key != "gold" key = ds_map_find_next(inventory, key) else break;
   }

El código anterior crea algunas variables temporales y luego obtiene el tamaño ds_map y encuentra la primera clave almacenada por la computadora en el mapa. Luego usa un for loop para iterar a través del ds_map buscando el valor clave "gold". Si lo encuentra, rompe el ciclo.
