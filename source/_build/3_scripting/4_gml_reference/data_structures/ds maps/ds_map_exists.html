ds_map_exists

Devuelve si la clave dada existe o no en el ds_map especificado.

Sintaxis:

ds_map_exists(id, key);


Argumento Descripción
id la identificación de la estructura de datos para verificar
key la clave para verificar


Devuelve: booleano


Descripción

Esta función regresará true si la clave especificada existe en el ds_map (creado previamente), y false si no es así


Ejemplo:

if !ds_map_exists(inventory, "potions")
   {
   ds_map_add(inventory, "potions", 1);
   }

El código anterior verificará el ds_map indexado en la variable "inventario" para la clave "pociones" y, si no existe, lo agregará al mapa.
