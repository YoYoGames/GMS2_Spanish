string_delete


Descripción

Puede usar esta función para eliminar una parte específica de una cadena. Entonces, usted suministra la cadena de entrada y la posición de inicio y final dentro de esa cadena para eliminar caracteres (el índice comienza en 1) y la función devolverá una nueva cadena sin esa sección en ella.


Sintaxis:

string_delete(str, index, count);

Argumento Descripción
str La cadena para copiar y eliminar de.
index La posición del primer personaje para eliminar.
count La cantidad de caracteres para eliminar.


Devoluciones:

String


Ejemplo:

str1 = "Helloo World";
str2 = string_delete(str1, 5, 1);

Esto establecerá str2 a "Hello World", ya que elimina la "o" extra de "Hello", es decir: la cadena cuenta a lo largo de la primera letra 5 lugares, luego elimina 1 carácter).
