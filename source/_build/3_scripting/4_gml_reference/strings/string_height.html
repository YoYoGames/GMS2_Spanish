string_height


Descripción

Esta función devolverá la altura (en píxeles) de la cadena de entrada. Es muy útil para calcular distancias entre elementos de texto en función de la letra más alta que compone la cadena tal como se dibujaría con draw_text usando la fuente actualmente definida.


Sintaxis:

string_height(string);

Argumento Descripción
string La cuerda para medir la altura de.


Devoluciones:

Real


Ejemplo:

var hh;
hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ");
draw_text(32, 32, string(score));
draw_text(32, 32 + hh, string(lives);

El código anterior obtendrá la altura de la cadena dada y luego dibujará dos líneas de texto, utilizando la altura de la cadena devuelta como un separador.
