string_lettersdigits


Descripción

Esta función devolverá una copia de una cadena dada con todo excepto sus letras y dígitos, lo que significa que puede usarse para eliminar cualquier carácter no deseado (como "#" o "?") De, por ejemplo, un nombre de inicio de sesión o un contraseña.

NOTA: Esta función solo devuelve el alfabeto inglés de 26 letras de A - Z (mayúscula o minúscula) y los números del 0 al 9.


Sintaxis:

string_lettersdigits(string);

Argumento Descripción
string La cadena para convertir a letras y dígitos.


Devoluciones:

String


Ejemplo:

if string_length(password) > string_length(string_lettersdigits(password))
   {
   draw_text(32,32,"Invalid Password! Only Letters and numbers please!");
   }

El código anterior verificará la longitud de una cadena contra la longitud de la misma cadena, pero con todos los símbolos eliminados. Si no son lo mismo, se dibuja un mensaje en la pantalla.
