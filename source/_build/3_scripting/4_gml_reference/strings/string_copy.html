string_copy


Descripción

Con esta función, puede seleccionar fácilmente una cantidad de caracteres dentro de una cadena para copiar a otra. El primer carácter de una cadena siempre se numera como "1", por lo que para copiar (por ejemplo) los primeros cinco caracteres de cadena que tendría string_copy(str, 1, 5). Un ejemplo adicional se proporciona a continuación.


Sintaxis:

string_copy(str, index, count);

Argumento Descripción
str La cadena para copiar desde.
index La posición del primer caracter en la cadena para copiar desde (numerado de 1)
count El número de caracteres, comenzando desde la posición del primero, para copiar


Devoluciones:

String


Ejemplo:

name = keyboard_string;
if string_length(name) > 15 )
      {
      keyboard_string = string_copy(name, 1, 15 );
      }

El código anterior le permitirá al jugador ingresar una cadena a través del teclado que luego se almacena en la variable name. Esta variable se verifica luego para ver si excede la longitud máxima de 15 caracteres y si lo hace, la cadena del teclado se reemplaza por una copia de 15 caracteres del name cuerda. Esto efectivamente limita la entrada del jugador a 15 caracteres.
