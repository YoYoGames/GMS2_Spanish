string_width_ext


Descripción

Esta función devolverá el ancho máximo (en píxeles) de la cadena de entrada, teniendo en cuenta la separación de línea y el ancho de salto de línea (que se define como el número de píxeles que la cadena puede ocupar antes de insertar un salto de línea). Es muy útil para calcular distancias entre elementos de texto en función del ancho máximo de una cadena que se divide en varias líneas como se dibujaría con draw_text_ext usando la fuente actualmente definida. La separación y el ancho se pueden establecer en -1 para obtener el espaciado predeterminado.


Sintaxis:

string_width_ext(string, sep, w);

Argumento Descripción
string La cuerda para medir el ancho de.
sep La distancia en píxeles entre líneas de texto como si se estuviera dibujando la cadena.
w El ancho máximo (en píxeles) de la cadena antes de un salto de línea como si la cadena fuera dibujada.


Devoluciones:

Real


Ejemplo:

var ww;
ww = string_width_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32 + ww, 32, str_Story_Text[2], -1, 100);

El código anterior obtendrá el ancho de la cadena dada, teniendo en cuenta la separación de línea y el ancho de salto de línea, y luego dibujará dos líneas de texto, utilizando el ancho de cadena total devuelto como separador.
