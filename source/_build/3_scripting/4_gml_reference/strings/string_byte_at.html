string_byte_at


Descripción

Devuelve el valor del byte sin formato como un valor real en una posición dada en la cadena dada.


Sintaxis:

string_byte_at(str, index);

Argumento Descripción
str La cadena para verificar
index La posición para obtener el byte de.


Devoluciones:

Real


Ejemplo:

newbyte = string_byte_at("Hello World", 5);

Esto establecerá newbyte al valor de bytes crudos de la sexta letra de "Hello World".
