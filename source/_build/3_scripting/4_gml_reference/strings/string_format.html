string_format


Descripción

Convierte un número real en una cadena utilizando su propio formato, donde puede elegir cuántos "lugares" se guardan en la cadena y cuántos decimales se guardan también. Ambos pueden ser muy útiles, algunos juegos prefieren mostrar una puntuación como un número determinado de dígitos, mientras que el control sobre los lugares decimales puede ser bueno para una gran precisión, los dos decimales de string() no puedo proporcionar Si el número de lugares especificados es mayor que el valor que se mostrará y / o el número más los lugares decimales que se han especificado es menor que el total de lugares, se agregarán espacios antes del valor para compensar la diferencia (consulte el ejemplo a continuación). Se agregarán ceros a la derecha del punto decimal si el valor proporcionado es menor que el total y la cantidad de decimales que se incluirán.


Sintaxis:

string_format(val, tot, dec);

Argumento Descripción
val El número real que se convertirá en una cadena.
tot El número total de lugares del número principal que se mostrará. Zeroes o espacios se insertarán para que coincida con esto.
dec La cantidad de decimales que se incluirán.


Devoluciones:

String


Ejemplo:

str1 = string_format(1234, 8, 0);
str2 = string_format(pi, 1, 10);
str3 = string_format(pi, 5, 5);

Esto establecerá str1 a "1234", str2 a "3.1415926535" y str3 a "3.14159".
