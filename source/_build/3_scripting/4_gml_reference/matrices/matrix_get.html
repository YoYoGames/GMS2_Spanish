matrix_get


Descripción

Esta función devuelve una matriz 1D de 16 valores, correspondientes al tipo de matriz 4x4 dado, donde la fila 1 son los elementos [0 - 3], la fila 2 son los elementos [4 -7], etc... (ver la imagen en la página principal) página ). Las matrices disponibles son view, projection y world, para las cuales usaría una de las siguientes constantes:

Constante Descripción
matrix_view La matriz de vista actual
matrix_projection La matriz de proyección actual
matrix_world La matriz mundial actual


Sintaxis:

matrix_get(type);


Argumentos:

Argumento Descripción
type El tipo de matriz para obtener los valores de (ver las constantes enumeradas arriba)


Devoluciones:

Array (1D, 16 elements)


Ejemplo:

v_array = matrix_get(matrix_view);

El código anterior obtendrá los valores de la matriz de vista actual y puebla la matriz "v_array" con ellos.
