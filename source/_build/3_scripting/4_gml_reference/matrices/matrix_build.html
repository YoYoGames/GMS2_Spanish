matrix_build


Descripción

Esta función se puede usar para crear su propia matriz personalizada y devolverá un valor de índice para esta nueva matriz que se debe almacenar en una variable para futuras referencias y uso. La matriz en sí se crea como una matriz de 16 valores de 1D donde los primeros 4 elementos son la fila 1, los segundos 4 elementos son la fila 2, etc... de una matriz de 4x4.

NOTA: Cuando construye una nueva matriz de esta manera, el orden de operación es YXZ.


Sintaxis:

matrix_build(x, y, z, xrotation, yrotation, zrotation, xscale, yscale, zscale);


Argumentos:

Argumento Descripción
x El componente x del vector de traducción.
y El componente y del vector de traducción.
z El componente z del vector de traducción.
xrotation El ángulo para girar alrededor del eje x (en grados °).
yrotation El ángulo para girar alrededor del eje y (en grados °).
xzotation El ángulo para girar alrededor del eje z (en grados °).
xscale La cantidad de la escala x
yscale El monto de la escala y.
zscale La cantidad de la escala z


Devoluciones:

4x4 Matrix


Ejemplo:

t_matrix = matrix_build(x, y, 0, 0, 90, 0, 1, 2, 1);

El código anterior creará una nueva transformación de matriz y almacenará el índice de matriz resultante en la variable "t_matrix".
