audio_get_listener_info


Descripción

Esta función creará un ds_map y llenarlo con información para el oyente dado.

NOTA: usted es responsable de la destrucción de los devueltos ds_map usando la función apropiada.

los ds_map contendrá las siguientes claves:

  1. "nombre": el nombre del oyente, como una cadena, con "predeterminado" como nombre de escucha estándar en la mayoría de las plataformas de destino
  2. "máscara" - La máscara de bits para el oyente
  3. "índice" - El valor de índice único del oyente

El valor de la máscara se puede usar para configurar un sonido o un emisor para reproducir desde múltiples oyentes a la vez, simplemente usando el bit a bit or "|" para generar una máscara para el sonido (vea el código de ejemplo a continuación), mientras que el índice se usa para establecer las propiedades como posición o velocidad para un oyente dado utilizando funciones como audio_listener_set_position().


Sintaxis:

audio_get_listener_info(num);


Argumento Descripción
num El número de oyente para obtener los datos.


Devoluciones:

ds_map


Ejemplo:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
   {
   var info = audio_get_listener_info(i);
   var m = audio_listener_get_data(info[? "mask"]);
   mask = mask | m;
   ds_map_destroy(info);
   }
audio_set_listener_mask(mask);

El código anterior comprueba la cantidad de oyentes disponibles, luego los bucles obtienen sus bits de máscara, que luego se combinan para crear una sola máscara de bits que se aplica al oyente global.
