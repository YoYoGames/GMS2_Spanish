audio_sound_get_gain


Descripción

Esta función devolverá el valor de ganancia actual para el sonido dado. El sonido puede ser uno referenciado desde un índice para un sonido individual que se está reproduciendo y que ha sido almacenado en una variable cuando se usa el audio_play_sound o audio_play_sound_at funciones, o un activo de sonido real del árbol de recursos. La ganancia generalmente se calcula como un valor de 0 a 1, pero en algunas plataformas puede tener una ganancia superior a 1, aunque un valor de 1 se considera "volumen completo" y cualquier cosa mayor puede introducir recorte de audio.


Sintaxis:

audio_sound_get_gain(index);


Argumento Descripción
index El índice del sonido para obtener la ganancia de.


Devoluciones:

Real


Ejemplo:

if audio_sound_get_gain(snd_Music) != 1
   {
   audio_sound_gain(snd_Music, 1, 0);
   }

El código anterior cambiará la ganancia del audio reproducido del sonido indexado como "snd_Music" si su ganancia no es igual a 1.
