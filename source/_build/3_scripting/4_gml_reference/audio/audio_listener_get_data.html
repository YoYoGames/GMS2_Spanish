audio_listener_get_data


Descripción

Esta función creará un ds_map y poblarlo con los valores de posición, velocidad y orientación para el oyente dado. El índice de escucha predeterminado es 0, pero puede usar la función audio_get_listener_info para obtener los diferentes índices disponibles para la plataforma objetivo. Si proporciona un índice de escucha incorrecto, la función devolverá -1.

NOTA: usted es responsable de la destrucción de los devueltos ds_map usando la función apropiada.

los ds_map contendrá las siguientes claves:

  1. "x" - La posición x del oyente
  2. "y" - La posición y del oyente
  3. "z" - La posición z del oyente
  4. "vx" - La velocidad del eje x del oyente
  5. "vy" - La velocidad del eje y del oyente
  6. "vz" - La velocidad del eje z del oyente
  7. "lookat_x" - El componente x de la mirada en el vector del oyente
  8. "lookat_y" - El componente y de la mirada en el vector del oyente
  9. "lookat_z" - El componente z de la mirada al vector del oyente
  10. "up_x" - El componente x del vector ascendente del oyente
  11. "up_y" - El componente y del vector ascendente del oyente
  12. "up_z" - El componente z del vector ascendente del oyente


Sintaxis:

audio_listener_get_data(index);


Argumento Descripción
index El oyente para obtener los datos (por defecto 0).


Devoluciones:

ds_map


Ejemplo:

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
   {
   var info = audio_get_listener_info(i);
   var data = audio_listener_get_data(info[? "index"]);
   if data[? "x"] != 0
      {
      audio_listener_set_position(info[? "index"], 0, 0, 0);
      }
   ds_map_destroy(info);
   ds_map_destroy(data);
   }

El código anterior comprueba el número de oyentes disponibles y luego pasa a través de ellos y si su posición x no es 0, establece su posición en 0, 0, 0.
