audio_get_recorder_count


Descripción

Esta función devolverá el número de fuentes de grabación de audio (como micrófonos, etc...) actualmente disponibles para su juego. Entonces, si tiene un valor de retorno de, por ejemplo, cuatro, entonces tendrá entrada de audio en las fuentes 0, 1, 2 y 3, siendo uno de estos valores el que usa para comenzar a grabar usando la función audio_start_recording(). Este valor puede cambiar en cualquier momento a medida que las personas conectan / desconectan micrófonos u otras fuentes de entrada al dispositivo. Tenga en cuenta que puede utilizar la función audio_get_recorder_info Para obtener información sobre cada dispositivo conectado.

¡IMPORTANTE! Esta función siempre devolverá 0 cuando se use en el destino HTML5.


Sintaxis:

audio_get_recorder_count();


Devoluciones:

Integer


Ejemplo:

if audio_get_recorder_count() > 0
   {
   channel_index = audio_start_recording(0);
   }

El código anterior verifica las grabadoras de audio disponibles y si hay 1 o más, comienza la grabación desde la primera indexada (fuente 0).
