audio_pause_sound


Descripción

Con esta función, puede pausar cualquier sonido que se esté reproduciendo actualmente. El sonido puede ser una instancia única de un sonido (el índice de los sonidos individuales que se reproducen se puede almacenar en una variable cuando se usa el audio_play_sound o audio_play_sound_at funciones) o un activo de sonido, en cuyo caso se pausarán todas las instancias del sonido dado.


Sintaxis:

audio_pause_sound(index);


Argumento Descripción
index El índice del sonido para pausar.


Devoluciones:

N/A


Ejemplo:

if keyboard_check_pressed(ord("P"))
   {
   global.Pause = !global.Pause;
   if global.Pause
      {
      audio_pause_sound(snd_Waterfall);
      }
   else
      {
      audio_resume_sound(snd_Waterfall);
      }
   }

El código anterior comprueba si se presiona la tecla "P" del teclado y, si detecta uno, establece la variable global "Pausa" para true o falso y luego pausa el sonido indexado en la variable "snd_Waterfall" o reanuda el sonido desde su estado de pausa.
