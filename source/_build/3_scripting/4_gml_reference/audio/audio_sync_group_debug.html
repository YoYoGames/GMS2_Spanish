audio_sync_group_debug


Descripción

Esta función se puede usar para mostrar información de depuración sobre cualquier grupo de sincronización dado (el índice de grupo es el valor devuelto cuando el grupo se creó usando la función audio_create_sync_group() ). Puede llamar a la función con un valor de -1 para desactivar la superposición.

Cuando esté depurando los grupos de sincronización de audio, la superposición mostrará qué grupos de sincronización está depurando

  1. Cuantos canales hay en el
  2. Cuántas veces ha fallado el grupo de sincronización (esto significa que se ha intentado hacer demasiado trabajo)
  3. Una lista de los canales.
    • Rojo significa que este canal no está haciendo ningún trabajo.
    • Amarillo significa que el canal está en estado inicial (aún no se está decodificando o reproduciendo)
    • Verde significa que la pista está actualmente en búsqueda / decodificación
    • Púrpura significa que el canal se ha quedado sin datos.
  5. Una barra de perfil que muestra qué trabajo está haciendo el hilo de decodificación
    • azul significa que se está leyendo un archivo por primera vez
    • rojo significa que el hilo está leyendo un archivo para saber dónde leer
    • verde significa que el hilo está decodificando datos
    • púrpura significa que el hilo está durmiendo hasta que necesita trabajar

Tenga en cuenta que para la depuración general del audio, también tiene la función audio_debug().

NOTA: Esta funcionalidad no está disponible para la plataforma de destino HTML5.


Sintaxis:

audio_sync_group_debug(group_index);


Argumento Descripción
group_index El índice de grupo a depurar, o -1 para desactivar la depuración.


Devoluciones:

N/A


Ejemplo:

if (debug_mode)
   {
   audio_sync_group_debug(sg);
   }
else
   {
   audio_sync_group_debug(-1);
   }

El código anterior activará o desactivará la superposición de depuración del grupo de sincronización de audio que muestra el grupo de sincronización indexado en la variable "sg" dependiendo de si el juego se ejecuta en modo de depuración o no.
