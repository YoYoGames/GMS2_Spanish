audio_free_play_queue


Descripción

Esta función se usa para liberar la memoria asociada con la cola de audio dada. El índice de cola es el valor devuelto cuando creó la cola con la función audio_create_play_queue(), y se debe llamar a esta función cuando ya no se requiere la cola para evitar fugas de memoria. Al liberar la cola, se detendrá cualquier sonido que se esté reproduciendo, y no podrá eliminar el búfer desde el que se transmite el sonido hasta que se haya liberado la cola a la que está asignado. Esta función desencadenará un evento asíncrono de reproducción de audio, y en este caso un evento especial ds_map se creará en la variable async_load con los siguientes pares clave / valor:

  • "queue_id": la cola que finalizó la reproducción de datos
  • "buffer_id": el búfer que ya no se reproduce desde
  • "queue_shutdown": se establece en 0 durante la reproducción normal y 1 cuando se recibe el evento porque ha liberado la cola con esta función. Cuando se establece en 1, no desea poner en cola ningún dato adicional.
¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Sintaxis:

audio_free_play_queue(queueIndex);


Argumento Descripción
queueIndex El índice de la cola para liberar.


Devoluciones:

N/A


Ejemplo:

var q_id = async_load[? "queue_id"]; if q_id == audio_queue    {
   audio_free_play_queue(audio_queue);
   buffer_delete(audio_buffer);
   }

El código anterior iría en el evento de Reproducción de audio asíncrono y verificará si el audio que ha terminado de reproducirse proviene de "audio_queue" y si lo está, libera la cola y el búfer asociado.
