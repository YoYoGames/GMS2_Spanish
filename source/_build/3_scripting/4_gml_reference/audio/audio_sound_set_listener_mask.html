audio_sound_set_listener_mask


Descripción

Esta función se puede usar para configurar la máscara de bits para un sonido de modo que solo se reproduzca desde los oyentes especificados. Puede crear una máscara de bits usando el audio_get_listener_info() y luego usando el bit a bit or ("|") para crear una máscara personalizada con los oyentes que requiere el sonido para reproducir, y luego aplicar esta máscara personalizada al sonido después de haberlo reproducido. Esta máscara anulará la máscara predeterminada o la que haya configurado con la función audio_set_listener_mask().

NOTA: Esta función solo se puede usar en un sonido una vez que se ha configurado para reproducirse y requiere que el ID de sonido sea devuelto por la función utilizada para reproducir el sonido.


Sintaxis:

audio_sound_set_listener_mask(soundID, mask);


Argumento Descripción
soundID La ID única del sonido para establecer la máscara de
mask Los datos de máscara de bits para establecer para el sonido


Devoluciones:

N/A


Ejemplo:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
audio_sound_set_listener_mask(snd, global.PlayerMask);

El código anterior reproduce un sonido y luego establece el / los oyente / s para reproducir utilizando los datos de máscara almacenados en una variable global.
