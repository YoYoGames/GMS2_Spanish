audio_emitter_velocity


Descripción

Esta función se puede usar para dar un efecto Doppler del emisor y simular el movimiento del audio en función del vector que se resuelve desde las posiciones relativas x, yyz dadas. Si el emisor en sí no se va a mover, normalmente no necesitaría establecer estos valores, pero, por ejemplo, si está haciendo un juego de disparos en el que los enemigos vienen de la derecha y se desplazan hacia la izquierda, debe configurar esto. tener una velocidad de (hspeed, 0, 0) en el evento create de los enemigos (y actualizar el emisor de cada instancia en el evento step usando audio_emitter_position ) para dar a cualquier sonido emitido por las instancias enemigas el Doppler correcto a medida que pasan la instancia del jugador (que a su vez usaría la función) audio_listener_position para establecer al oyente en la posición correcta).


Sintaxis:

audio_emitter_velocity(emitter, vx, vy, vz);


Argumento Descripción
emitter El índice del emisor para cambiar.
vx El valor del vector x (predeterminado 0).
vy El valor del vector y (predeterminado 0).
vz El valor del vector z (predeterminado 0).


Devoluciones:

N/A


Ejemplo:

s_emit = audio_emitter_create();
audio_emitter_position(s_emit, room_width, 0, 0);
audio_emitter_velocity(s_emit, -5, 0, 0);

El código anterior crea un emisor de audio y asigna su índice a la variable "s_emit". Este emisor se coloca entonces en una posición dentro de la habitación y se le da una velocidad de 5 píxeles por paso a lo largo del eje x (se doblará correctamente en relación con el oyente como si tuviera una velocidad horizontal de 5 píxeles por paso).
