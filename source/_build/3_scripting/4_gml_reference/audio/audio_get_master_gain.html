audio_get_master_gain


Descripción

Con esta función, puede obtener el valor absoluto del volumen global de todos los sonidos y la música de un oyente específico. El índice de escucha predeterminado es 0, pero puede usar la función audio_get_listener_info para obtener los diferentes índices disponibles para la plataforma objetivo. El valor de ganancia devuelto se basa en una escala lineal de 0 (silencioso) a 1 (volumen completo). Tenga en cuenta que en algunas plataformas puede tener una ganancia superior a 1, aunque un valor de 1 se considera "volumen completo" y cualquier cosa mayor puede introducir recorte de audio.


Sintaxis:

audio_get_master_gain(listenerIndex);


Argumento Descripción
listenerIndex El índice del oyente para obtener la ganancia de.


Devoluciones:

Real


Ejemplo:

var num = audio_get_listener_count();
for( var i = 0; i < num; ++i;)
   {
   var info = audio_get_listener_info(i);
   var ind = info[? "index"];
   if audio_get_master_gain(ind) != 1
      {
      audio_set_master_gain(info[? "index"], 1);
      }
   ds_map_destroy(info);
   }

El código anterior recorre los oyentes disponibles, verificando si su ganancia es 1 o no, y si no lo está establece en 1 para cada uno de ellos.
