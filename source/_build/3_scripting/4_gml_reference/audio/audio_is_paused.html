audio_is_paused


Descripción

Esta función verificará el sonido dado para ver si está actualmente en pausa. El sonido puede ser una instancia única de un sonido (el índice de los sonidos individuales que se reproducen se puede almacenar en una variable cuando se usa el audio_play_sound o audio_play_sound_at funciones) o un activo de sonido, en cuyo caso se verificarán todas las instancias del sonido dado y si alguna de ellas se pausa, la función volverá true de lo contrario, volverá false.


Sintaxis:

audio_is_paused(index);


Argumento Descripción
index El índice del sonido para verificar.


Devoluciones:

Boolean


Ejemplo:

if audio_is_paused(snd_Waterfall)
   {
   audio_resume_sound(snd_Waterfall);
   }

El código anterior comprueba si el sonido indexado en la variable "snd_Waterfall" está actualmente en pausa y si regresa true entonces se reanudará la reproducción del sonido.
