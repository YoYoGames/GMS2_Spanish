audio_sound_gain


Descripción

Con esta función, puede fundir o atenuar un sonido durante un período de tiempo determinado, o puede usarse para establecer la ganancia de sonido al instante. El tiempo se mide en milisegundos, y la función requiere que ingrese un nivel final de ganancia para que el sonido haya llegado al final de ese tiempo. Esta ganancia puede estar entre 0 (silencio) y 1 (volumen completo) y la escala es lineal, de modo que un valor de 0.5 sería la mitad del volumen. Para cambiar instantáneamente la ganancia, simplemente configure el argumento de tiempo en 0.

Esta función afectará todas las instancias del sonido que se estén reproduciendo actualmente en la sala si el índice es un recurso de sonido, y el volumen final será el volumen en el que se reproducirán todas las demás instancias del sonido. Sin embargo, si ha utilizado el índice devuelto por una función como audio_play_sound solo afectará esa única instancia del sonido.


Sintaxis:

audio_sound_gain(index, volume, time);


Argumento Descripción
index El índice del sonido para establecer la ganancia.
volume Valor para el volumen de la música.
time La duración del cambio en la ganancia en milisegundos.


Devoluciones:

N/A


Ejemplo:

if val
   {
   var snd = audio_play_sound(snd_fountain);
   audio_sound_gain(snd, 0, 0);
   audio_sound_gain(snd, 1, 5000);
   }

El código anterior verifica una variable y si regresa true luego asignará el índice de un sonido para ser reproducido a la variable local "snd". Esta variable se usa para reducir el volumen de ese sonido específico a 0 y fundirse hasta volumen completo durante 5 segundos.
