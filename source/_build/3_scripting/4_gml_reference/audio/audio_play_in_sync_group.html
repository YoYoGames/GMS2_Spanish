audio_play_in_sync_group


Descripción

Con esta función puede asignar un sonido a un grupo de sincronización creado previamente. El índice de grupo es el valor devuelto cuando creó el grupo usando la función audio_create_sync_group(), y el índice de sonido es el índice de cualquier sonido comprimido ( *.ogg o *.mp3 ) del árbol de recursos. Esta función solo agrega el sonido al grupo, pero no reproduce el sonido, y será un valor> -1 si tiene éxito y -1 si falla, donde un valor correcto indica el orden de los sonidos en el grupo de sincronización (por lo que 0 sería el primer sonido agregado, 1 sería el segundo, etc...).

NOTA: Esta funcionalidad no está disponible para la plataforma de destino HTML5.


Sintaxis:

audio_play_in_sync_group(group_index, sound_index);


Argumento Descripción
group_index El índice de grupo para asignar el sonido a.
sound_index El índice de sonido a asignar al grupo.


Devoluciones:

Index


Ejemplo:

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_play_in_sync_group(sg, sound3);
audio_play_in_sync_group(sg, sound4);
audio_start_sync_group(sg);

Lo anterior crea un nuevo grupo de sincronización y asigna el índice del grupo a la variable "sg". Luego se agregan cuatro sonidos al grupo, y la ganancia para tres de ellos se establece en 0. Finalmente, se reproduce el grupo de sincronización.
