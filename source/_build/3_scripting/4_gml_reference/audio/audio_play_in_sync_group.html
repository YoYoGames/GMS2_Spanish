audio_play_in_sync_group


Descripción

Con esta función, puede asignar un sonido a un grupo de sincronización previamente creado. El índice de grupo es el valor devuelto cuando creó el grupo utilizando la función audio_create_sync_group(), y el índice de sonido es el índice de cualquier sonido comprimido ( *.ogg o *.mp3 ) del árbol de recursos. Esta función solo agrega el sonido al grupo, pero no reproduce el sonido. La función devuelve un valor de ID único para el sonido que se ha agregado (distinto del índice de sonido) que luego se puede utilizar como siempre en todas las otras funciones de audio.


Sintaxis:

audio_play_in_sync_group(group_index, sound_index);


Argumento Descripción
group_index El índice de grupo para asignar el sonido.
sound_index El índice de sonido que se asignará al grupo.


Devoluciones:

Index


Ejemplo:

sg = audio_create_sync_group(true);
snd[0] = audio_play_in_sync_group(sg, sound1);
snd[1] = audio_play_in_sync_group(sg, sound2);
snd[2] = audio_play_in_sync_group(sg, sound3);
snd[3] = audio_play_in_sync_group(sg, sound4);
audio_sound_gain(snd[1], 0, 0);
audio_sound_gain(snd[2], 0, 0);
audio_sound_gain(snd[3], 0, 0);
audio_start_sync_group(sg);

Lo anterior crea un nuevo grupo de sincronización y asigna el índice del grupo a la variable "sg". Luego se agregan cuatro sonidos al grupo, con la ganancia de tres de ellos en 0. Finalmente, se reproduce el grupo de sincronización.
