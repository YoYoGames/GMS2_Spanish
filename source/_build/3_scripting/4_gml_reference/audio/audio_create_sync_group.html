audio_create_sync_group


Descripción

Crea un grupo de sincronización y devuelve un valor de ID único para él que luego se debe usar en todas las llamadas de funciones de audio adicionales para este grupo. Si quieres que el grupo repita el ciclo, pasa true De lo contrario, pase false, pero tenga en cuenta que si desea que se repitan, todas las pistas agregadas posteriormente deben ser de la misma longitud. Tenga en cuenta que cuando crea un grupo de sincronización, deberá liberar la memoria y los sonidos asociados con él cuando no esté en uso utilizando el audio_destroy_sync_group() función, por ejemplo, en los eventos Room End o Destroy.


Sintaxis:

audio_create_sync_group(loop);


Argumento Descripción
loop Si las pistas del grupo deberían repetirse ( true ) o no ( false )


Devoluciones:

index


Ejemplo:

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_sound_gain(sound2, 0, 0);
audio_play_in_sync_group(sg, sound3);
audio_sound_gain(sound3, 0, 0);
audio_play_in_sync_group(sg, sound4);
audio_sound_gain(sound4, 0, 0);
audio_start_sync_group(sg);

Lo anterior crea un nuevo grupo de sincronización y asigna el índice del grupo a la variable "sg". Luego se agregan cuatro sonidos al grupo, con la ganancia para tres de ellos configurados en 0. Finalmente, se reproduce el grupo de sincronización.
