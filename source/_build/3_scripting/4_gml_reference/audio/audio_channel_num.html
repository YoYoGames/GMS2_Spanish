audio_channel_num


Descripción

Con esta función puede establecer cuántos canales de audio están disponibles para reproducir audio en GameMaker Studio 2. Lo que esto significa básicamente es que das la cantidad de sonidos simultáneos que se pueden reproducir al mismo tiempo, y si el número excede la cantidad, los sonidos con una prioridad más baja se detienen para liberar un canal para los sonidos con una prioridad más alta.. Puede utilizar esta función para optimizar su juego para dispositivos, ya que cuanto menor sea el número de canales de audio, mejor será el rendimiento, pero tenga en cuenta que esto también puede causar un corte notable de ciertos sonidos si se tocan muchos a la vez.


Sintaxis:

audio_channel_num(num);


Argumento Descripción
num Cantidad de canales de audio disponibles (por defecto es 128).


Devoluciones:

N/A


Ejemplo:

switch (os_browser)
   {
   case browser_not_a_browser:
      switch (os_type)
         {
         case os_windows:
         case os_macos:
            audio_channel_num(200);
            break;
         default:
            audio_channel_num(64);
            break;
         }
      break;
   default:
      audio_channel_num(16);
      break;
   }

El código anterior comprueba la plataforma en la que se está ejecutando el juego y cambia la cantidad de canales de sonido disponibles para aumentar el rendimiento.
