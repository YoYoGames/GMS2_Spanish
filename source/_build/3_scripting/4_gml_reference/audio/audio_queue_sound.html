audio_queue_sound


Descripción

Esta función agregará los datos de un buffer a la cola de audio que creó previamente usando la función audio_create_play_queue. Especifique el índice de cola para agregar y la ID del búfer para obtener los datos, así como la posición (desplazamiento) dentro del búfer y la cantidad de bytes que se agregarán. Una vez que haya agregado el audio de un búfer a una cola, el búfer no podrá eliminarse hasta que haya liberado por primera vez la cola con la función audio_free_play_queue y las propiedades del búfer deben coincidir con las de la cola en la que está agregando el sonido.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Sintaxis:

audio_queue_sound(queueIndex, bufferId, bufferOffset, bufferLength);


Argumento Descripción
queueIndex El índice de la cola para agregar a.
bufferId El buffer para agregar a la cola.
bufferOffset El desplazamiento dentro del buffer de origen para comenzar.
bufferLength La longitud del búfer (el número de bytes en el búfer).


Devoluciones:

N/A


Ejemplo:

len = async_load[? "data_len"];
audio_buff = buffer_create(len, buffer_fast, 1);
buffer_copy(async_load[? "buffer_id"], 0, len, buff, 0);
audio_queue_sound(audio_queue, audio_buff, 0, len);
audio_play_sound(audio_queue, 10, 0);

El código anterior se invocará en el evento de grabación de audio asíncrona y asignará algunos datos grabados a un búfer, que luego se agregará a una cola de audio. Esto luego se juega.
