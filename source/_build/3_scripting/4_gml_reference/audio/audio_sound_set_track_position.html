audio_sound_set_track_position


Descripción

Esta función establecerá la posición (en segundos) dentro del archivo de sonido para reproducir el sonido. El sonido puede ser una instancia única de un sonido (el índice de los sonidos individuales que se reproducen se puede almacenar en una variable cuando se usa el audio_play_sound o audio_play_sound_at funciones) o un activo de sonido. Si es una ID de sonido única que usa, la posición cambiará inmediatamente a la nueva hora dada, y si es un activo de árbol de recursos, todas las reproducciones adicionales del sonido dado comenzarán en el nuevo horario. La función ignorará cualquier valor dado que sea más largo que la duración del sonido dado para el valor de "índice".


Sintaxis:

audio_sound_set_track_position(index, time);


Argumento Descripción
index El índice del sonido para cambiar.
time El tiempo (en segundos) para establecer el punto de inicio a.


Devoluciones:

N/A


Ejemplo:

var snd = audio_play_sound(snd_MainTrack, 0, false);
audio_sound_set_track_position(snd, 32);

El código anterior reproduce un sonido y luego utiliza el valor ID devuelto para establecer la posición de inicio para el sonido en 32 segundos.
