audio_sound_pitch


Descripción

Esta función se puede usar para cambiar el tono de un sonido dado. El sonido puede ser uno referenciado desde un índice para un sonido individual que se está reproduciendo y que ha sido almacenado en una variable cuando se usa el audio_play_sound o audio_play_sound_at funciones, o un activo de sonido real del árbol de recursos. Si es un índice de un sonido de reproducción, solo se cambiará esa instancia; sin embargo, cuando se utiliza un activo de sonido del árbol de recursos, se modificarán todas las instancias adicionales que se reproduzcan con ese sonido.

El argumento de tono es un multiplicador de tono, en el que el valor de entrada multiplica el tono actual por esa cantidad, por lo que el valor predeterminado de 1 no es el cambio de tono, mientras que un valor de menos de 1 reducirá el tono y más de 1 elevará el tono tono. Lo mejor es usar pequeños incrementos para esta función, ya que cualquier valor por debajo de 0 o por encima de 5 puede no ser audible de todos modos. Vale la pena señalar que el cambio de tono total permitido está fijado a (1/256) - 256 octavas, por lo que cualquier valor por encima o por debajo de este no se registrará.

NOTA: El valor fijado anteriormente es a lo que GameMaker Studio 2 intenta sujetar el rango, pero este valor no está garantizado en todas las plataformas de destino. iOS, por ejemplo, abrazaderas a (1/256) - 8, por lo que puede necesitar experimentar en cada plataforma de destino y tener diferentes versiones de un recurso de sonido, cada una de ellas previamente desplazada, si necesita valores de octava más altos o más bajos.


Sintaxis:

audio_sound_pitch(index, pitch);


Argumento Descripción
index El índice del sonido para cambiar.
pitch El multiplicador de tono (predeterminado 1).


Devoluciones:

N/A


Ejemplo:

var s_engine = audio_play_sound(snd_CarEngine, 10, false);
switch (gear)
   {
   case 1: audio_sound_pitch(s_engine, 0.8); break;
   case 2: audio_sound_pitch(s_engine, 0.9); break;
   case 3: audio_sound_pitch(s_engine, 0.95); break;
   case 4: audio_sound_pitch(s_engine, 1); break;
   case 5: audio_sound_pitch(s_engine, 1.2); break;
   }

El código anterior cambiará el tono del audio reproducido del sonido indexado en la variable "s_engine" en función del valor de la variable "engranaje".
