audio_get_name


Descripción

Esta función devolverá el nombre de un recurso de audio dado como una cadena. El valor de "índice" puede ser el del recurso en sí (como se ve en el árbol de recursos) o el valor de ID único que se da cuando se reproduce el sonido usando, por ejemplo, audio_play_sound(). Tenga en cuenta que la cadena devuelta no es la misma que la ID del recurso y no se puede usar para acceder al recurso en sí, por lo que solo se debe usar para visualizar o verificar errores.


Sintaxis:

audio_get_name(index);


Argumento Descripción
index El índice del sonido para verificar.


Devoluciones:

String


Ejemplo:

var snd = audio_play_sound(choose(snd_One, snd_Two, snd_Three), 0, false);
var name = audio_get_name(snd);
show_debug_message("Sound = " + name);

El código anterior reproduce un sonido aleatorio elegido entre tres recursos de sonido diferentes y luego muestra un mensaje de depuración con su nombre.
