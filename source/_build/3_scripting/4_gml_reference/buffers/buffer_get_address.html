buffer_get_address

Sintaxis:

buffer_get_address(buffer);


Argumento Descripción
buffer El índice del buffer a usar.


Devuelve: puntero


Descripción

Con esta función, puede obtener un puntero a la dirección de búfer sin formato y alineada. Esto se utiliza principalmente con extensiones ya que puede pasarles este valor, lo que les permite acceder a los datos del búfer. Tenga en cuenta que hay algunas cosas que no puede hacer con el búfer al acceder desde una extensión:

  • No puedes liberar la memoria.
  • No puedes cambiar el tamaño de la memoria.
  • No puede escribir más allá de la dirección del búfer más el tamaño del búfer (puede usar el buffer_get_size función para esto) o obtendrá un error fuera de límites.


Ejemplo:

var b_address = buffer_get_address(buff_model);
var b_end_address = b_address + buffer_get_size(buff_model);

El código anterior obtendrá la dirección de memoria del buffer indexado en la variable "buff_model" y luego obtendrá la dirección de memoria para el final del buffer usando el buffer_get_size() función, almacenando ambos valores en variables locales para su uso posterior.
