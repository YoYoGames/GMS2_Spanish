buffer_get_type

Sintaxis:

buffer_get_type(buffer);


Argumento Descripción
buffer El índice del buffer para verificar.


Devoluciones: constante


Descripción

Con esta función, recupera el tipo de memoria intermedia a la que hace referencia un valor de ID de memoria intermedia. El valor de retorno será una de las siguientes constantes:

Constante Descripción
buffer_fixed Un buffer de tamaño fijo.
buffer_grow Un buffer que "crecerá" dinámicamente a medida que se agreguen los datos
buffer_wrap Un buffer donde los datos se "ajustarán". Cuando los datos que se agregan alcanzan el límite del tamaño del búfer, la sobrescritura se volverá a colocar al inicio del búfer, y la escritura continuará desde ese punto.
buffer_fast Buffer especial "stripped" que es extremadamente rápido para leer / escribir. Solo se puede usar con buffer_u8 tipos de datos, y deben estar alineados a 1 byte.
buffer_vbuffer


Ejemplo:

type = buffer_get_type(buff);

El código anterior obtendrá el tipo de buffer del valor indexado en la variable "buff" y lo almacenará en una variable.
