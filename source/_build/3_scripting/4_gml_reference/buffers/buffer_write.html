buffer_write

Sintaxis:

buffer_write(buffer, type, value)


Argumento Descripción
buffer El índice del búfer para escribir.
type El tipo de datos que se escribirán en el búfer (consulte la lista de constantes a continuación).
value Los datos a escribir.


Devoluciones: N / A


Descripción

Esta función se puede utilizar para escribir datos en un búfer creado previamente. Los datos que escriba deben estar de acuerdo con el argumento "tipo" de esta función, lo que significa que no puede intentar escribir una cadena como un entero sin signo de 16 bits, por ejemplo. Las siguientes constantes se pueden usar para definir el tipo de datos:

Constante Descripción
buffer_u8 Un entero de 8 bits sin signo. Este es un valor positivo de 0 a 255.
buffer_s8 Un entero de 8 bits con signo. Este puede ser un valor positivo o negativo de -128 a 127 (0 se clasifica como positivo).
buffer_u16 Un entero sin signo, de 16 bits. Este es un valor positivo de 0 - 65,535.
buffer_s16 Un entero de 16 bits con signo. Este puede ser un valor positivo o negativo de -32,768 a 32,767 (0 se clasifica como positivo).
buffer_u32 Un entero de 32 bits sin signo. Este es un valor positivo de 0 a 4,294,967,295.
buffer_s32 Un entero de 32 bits con signo. Este puede ser un valor positivo o negativo de -2,147,483,648 a 2,147,483,647 (0 se clasifica como positivo).
buffer_u64 Un entero sin signo de 64 bits.
buffer_f16 Un flotador de 16 bits. Este puede ser un valor positivo o negativo dentro del mismo rango que un entero con signo de 16 bits. (Actualmente no es compatible)
buffer_f32 Un flotador de 32 bits. Este puede ser un valor positivo o negativo dentro del mismo rango que un entero con signo de 32 bits.
buffer_f64 Un flotador de 64 bits.
buffer_bool Un valor booleano. Solo puede ser 1 o 0 ( true o false )
buffer_string Una cadena de cualquier tamaño, finalizada con un carácter de terminación nulo.
buffer_text Una cadena de cualquier tamaño, sin el carácter de terminación nulo final.


Ejemplo:

buffer_seek(buff, buffer_seek_start, 0);
buffer_write(buff, buffer_s16, 0);
buffer_write(buff, buffer_s16, x);
buffer_write(buff, buffer_s16, y);

El código anterior encuentra el inicio del búfer con el ID almacenado en la variable "buff" y le escribe una serie de valores enteros de 16 bits con signo.
