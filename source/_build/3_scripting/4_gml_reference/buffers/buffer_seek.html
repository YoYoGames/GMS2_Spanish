buffer_seek

Sintaxis:

buffer_seek(buffer, base, offset);


Argumento Descripción
buffer El índice del buffer a usar.
base La posición de base para buscar.
offset El valor de compensación de datos.


Devoluciones: N / A


Descripción

Esta función se puede usar para desplazarse a través de un búfer, encontrar el inicio, el final o una posición relativa a la última utilizada al leer o escribir datos. El valor "offset" es el desplazamiento (en bytes) para agregar a la posición de búsqueda dada, por ejemplo, si la base es relativa y el desplazamiento es 4, entonces la posición del búfer se moverá a lo largo de 4 bytes desde su posición actual. Tenga en cuenta lo siguiente:

  • Puede usar valores negativos para el desplazamiento para buscar a través del búfer, así como valores positivos.

  • Si el buffer es del tipo "wrap" y se desplaza más allá del final del buffer, la posición de búsqueda también se ajustará.

  • Si el búfer no es del tipo "wrap", la búsqueda se fijará al principio o al final del búfer, incluso cuando el desplazamiento tomaría la búsqueda fuera de los límites del búfer.

Las siguientes constantes se aceptan como el argumento "base" para buscar:

Constante Descripción
buffer_seek_start El comienzo del buffer
buffer_seek_relative Una posición relativa a la posición actual de lectura / escritura
buffer_seek_end El final del buffer


Ejemplo:

buffer_seek(buff, buffer_seek_start, 0);
buffer_write(buff, buffer_s16, 0);
buffer_write(buff, buffer_s16, x);
buffer_write(buff, buffer_s16, y);

El código anterior encuentra el inicio del búfer con el ID almacenado en la variable "buff" y escribe una serie de valores enteros de 16 bits con signo.
