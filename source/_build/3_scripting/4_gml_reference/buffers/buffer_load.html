buffer_load

Sintaxis:

buffer_load(filename);


Argumento Descripción
filename El nombre del archivo para cargar.


Devoluciones: reales


Descripción

Esta función cargará los datos del búfer que se guardaron previamente utilizando el buffer_save funciona en un nuevo buffer. Este nuevo búfer se crea mediante la función como un búfer de "crecimiento" (ver aquí ) con una alineación de bytes de 1, y se devuelve la nueva identificación del búfer. Esta ID debe almacenarse en una variable y usarse en todas las llamadas a funciones adicionales a este búfer.

NOTA: Es importante que elimine de la memoria todos los recursos creados dinámicamente de esta manera cuando ya no los necesite para evitar pérdidas de memoria, de modo que cuando haya terminado con el búfer que ha creado, debería liberarlo nuevamente usando buffer_delete.


Ejemplo:

player_buffer = buffer_load("Player_Save.sav");

El código anterior cargará un búfer previamente guardado en la memoria, creando un nuevo búfer. El índice de este nuevo buffer se almacena en la variable "player_buffer".
