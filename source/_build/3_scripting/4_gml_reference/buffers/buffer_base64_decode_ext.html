buffer_base64_decode_ext

Sintaxis:

buffer_base64_decode_ext(buffer, string, offset);


Argumento Descripción
buffer El índice del buffer para decodificar la cadena en.
string La cadena codificada en base64 para decodificar.
offset El valor de compensación de datos.


Devoluciones: N / A


Descripción

Con esta función puede decodificar una cadena codificada en base64 (creada usando buffer_base64_encode() función) en un buffer. A diferencia de la función buffer_base64_decode(), esto no creará un búfer para ti, sino que deberías haber creado el búfer (ver buffer_create ), cuya identificación usaría luego con esta función. El "desplazamiento" es la posición dentro del búfer para decodificar la cadena dada (en bytes).


Ejemplo:

buff = buffer_create(16384, buffer_grow, 2);
ini_open("Save.ini");
var str = ini_read_string("Save", "Slot1", "");
buffer_base64_decode_ext(buff, str, 0);
ini_close();

El código anterior creará un búfer y almacenará el ID único para él en la variable "buff", luego abrirá un archivo ini y leerá una cadena de él en la variable local "str". Esta cadena se decodifica en el búfer recién creado antes de volver a cerrar el archivo ini.
