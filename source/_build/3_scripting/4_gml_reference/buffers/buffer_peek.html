buffer_peek

Sintaxis:

buffer_peek(buffer, offset, type);


Argumento Descripción
buffer El índice del buffer a usar.
offset La posición de desplazamiento (en bytes) dentro del búfer para leer los datos dados.
type El tipo de datos que se leerán del búfer (consulte la lista de constantes aquí ).


Devoluciones: Real / String


Descripción

Con el buffer_read función, puede leer datos del búfer dado en la posición actual de "búsqueda", con cada dato que se lee avanzando esta posición por los bytes que se leen o escriben. Sin embargo, puede ser necesario que lea un dato dado sin querer cambiar la posición de búsqueda actual, y es entonces cuando usaría esta función. Simplemente proporcione la función con una identificación de búfer y luego una posición de compensación (desde el inicio del búfer) dentro de ese búfer para leer, así como también el tipo de datos que desea leer.

NOTA: ¡ Usar el tipo de datos incorrecto para los datos que se leen dará como resultado valores erróneos!


Ejemplo:

var red = buffer_peek(buff, 1, buffer_u8);
var green = buffer_peek(buff, 2, buffer_u8);
var blue = buffer_peek(buff, 3, buffer_u8);
image_blend = make_colour_rgb(red, green, blue);

El código anterior obtendrá tres valores de tres posiciones diferentes dentro del búfer indexado en la variable "buff" y luego usará esos valores para establecer la combinación de imágenes de la instancia.
