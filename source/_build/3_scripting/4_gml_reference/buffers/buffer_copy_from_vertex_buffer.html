buffer_copy_from_vertex_buffer

Sintaxis:

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset);


Argumento Descripción
vertex_buffer El índice del búfer de vértices para copiar desde.
start_vertex El vértice inicial.
num_vertices La cantidad total de vértices a usar.
dest_buffer El índice del buffer para copiar a.
dest_offset La posición de desplazamiento para copiar los datos a (en bytes).


Devoluciones: N / A


Descripción

Esta función se puede usar para copiar algunos (o todos) los datos de vértice almacenados en un búfer de vértices en un búfer regular creado previamente. Al copiar desde un búfer de vértices en un búfer regular con esta función, ambos búferes deben haberse creado previamente (utilizando el vertex_create_buffer() y buffer_create funciones, por ejemplo). Puede especificar el rango de datos de vértice que desea copiar en el búfer, donde el vértice de inicio puede estar entre 0 y el número de vértices -1, y puede dar el número de vértices desde ese punto para copiar. Puedes usar la función vertex_get_number() en el buffer de vértice para obtener el número total de vértices almacenados. Por último, proporcione el índice del búfer para copiar los datos del vértice en, así como un desplazamiento de datos para definir la posición para copiar los datos del vértice en el búfer de destino.


Ejemplo:

var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0);

El código anterior copiará los datos de vértice almacenados en el búfer de vértices indexados en la variable "model_buffer", y luego los pegará en el búfer indexado en la variable "player_buffer".
