buffer_compress


Sintaxis:

buffer_compress(buffer, offset, size);


Argumento Descripción
buffer El índice del búfer para comprimir.
offset El offset dentro del buffer a comprimir (en bytes).
size El tamaño del área de búfer para comprimir (en bytes).


Devoluciones: Índice de Buffer


Descripción

Con esta función puede comprimir parte (o todo) de un búfer usando la compresión zlib. Usted proporciona el ID del búfer para comprimir (como lo devuelve buffer_create() ), el desplazamiento dentro del búfer para usar en bytes, y el tamaño de los datos del búfer para comprimir (también en bytes). La función devolverá un nuevo valor de ID de búfer para el búfer comprimido, o un valor menor que 0 si ha fallado por algún motivo. Esta función no alterará el búfer original.


Ejemplo:

var srcBuff = buffer_create(1024, buffer_grow, 1);
buffer_write(srcBuff, global.DataString);
var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff));
buffer_save(cmpBuff, "Player_Save.sav");
buffer_delete(srcBuff);
buffer_delete(cmpBuff);

El código anterior creará un búfer y luego lo llenará con los datos de una cadena. Este búfer se comprime y se guarda, y se eliminan los búferes de origen y comprimidos
