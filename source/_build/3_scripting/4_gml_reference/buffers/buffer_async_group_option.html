buffer_async_group_option

Sintaxis:

buffer_async_group_option(option, value);


Argumento Descripción
option La opción de configurar.
value El valor a establecer (puede ser cadena o real, dependiendo de la opción).


Devoluciones: N / A


Descripción

Con esta función, puede configurar algunas opciones específicas de la plataforma para el grupo de búfer que se está guardando. Las opciones disponibles son las siguientes:

  • " subtitle " o " slottitle ": el valor de esta opción sería una cadena, que se mostrará al usuario cuando administre sus datos guardados en el sistema operativo. Esto solo es importante al guardar datos, no al cargar de nuevo.

  • " showdialog ": el valor para esta opción es un valor booleano true o false. Si se establece en true, cuando guarde o cargue GameMaker Studio 2 se mostrará la interfaz de usuario del sistema, de lo contrario hará una rápida / rápida carga sin la interfaz de usuario mostrada. Por lo general, solo necesita esto si admite varias ranuras de guardado para permitir que el usuario elija una ranura, pero si solo admite una ranura por usuario, configure esto en false.

  • " savepadindex ": para esta opción, debe especificar el índice de almohadilla del reproductor que está guardando o cargando y el sistema escribirá y leerá los datos de la carpeta de guardado de este reproductor.

  • " saveslotsize ": esta opción requiere que especifique el tamaño real en bytes que desea guardar (para que pueda hacer un buffer_seek y buffer_tell para conseguir eso, por ejemplo). Tenga en cuenta que no es obligatorio proporcionar este valor, ya que todos los guardados están asignados previamente a un espacio mínimo, que normalmente es de aproximadamente 10 MB.

IMPORTANTE: esta función solo es válida actualmente para los módulos de destino PSVita, PS4 y XBox One. En todos los otros objetivos no hará nada.


Ejemplo:

buffer_async_group_begin("save_folder_name");
buffer_async_group_option("showdialog", false);
buffer_async_group_option("slottitle", "Catch The Haggis Save");
buffer_async_group_option("subtitle", "All your haggis are saved here!");
save = buffer_save_async(buff, "Player_Save.sav", 0, 16384);
buffer_async_group_end();

El código anterior inicia un grupo de búfer y luego establece las opciones del grupo antes de configurar 4 archivos para guardar de forma asíncrona. La definición de grupo finaliza (en cuyo punto comenzará el guardado).
