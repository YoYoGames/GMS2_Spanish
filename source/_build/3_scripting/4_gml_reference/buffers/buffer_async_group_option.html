buffer_async_group_option

Sintaxis:

buffer_async_group_option(option, value);


Argumento Descripción
option La opción de establecer
value El valor para establecer (puede ser de cadena o real, dependiendo de la opción).


Devoluciones: N / A


Descripción

Con esta función puede establecer algunas opciones específicas de la plataforma para el grupo de búferes que se está guardando. Las opciones disponibles son las siguientes:

  • " subtítulo " o " slottitle ": el valor de esta opción sería una cadena, que se mostrará al usuario al administrar sus datos guardados en el sistema operativo. Esto solo es importante cuando se guardan datos, no cuando se cargan nuevamente.

  • " showdialog " - El valor para esta opción es un booleano true o false. Si está configurado para true Cuando guarde o cargue, GameMaker Studio 2 mostrará la IU del sistema, de lo contrario, realizará una salvaguarda / carga rápida sin que se muestre la IU. Normalmente solo necesita esto si admite varias ranuras de guardado para permitir que el usuario elija una ranura, pero si solo admite una ranura por usuario, configúrelo para false.

  • " savepadindex ": para esta opción, debe especificar el índice del panel del reproductor que está guardando o cargando y el sistema escribirá y leerá datos en la carpeta de guardado de este jugador.

  • " saveslotsize ": esta opción requiere que especifique el tamaño real en bytes que desea guardar (para que pueda hacer una buffer_seek y buffer_tell para obtener eso, por ejemplo). Tenga en cuenta que no es obligatorio proporcionar este valor, ya que todas las grabaciones están preasignadas en un espacio mínimo, que normalmente es de unos 10 MB más o menos.

IMPORTANTE: esta función actualmente solo es válida para los módulos de destino PSVita, PS4 y XBox One. En todos los otros objetivos no hará nada.


Ejemplo:

buffer_async_group_begin("save_folder_name");
buffer_async_group_option("showdialog", false);
buffer_async_group_option("slottitle", "Catch The Haggis Save");
buffer_async_group_option("subtitle", "All your haggis are saved here!");
save = buffer_save_async(buff, "Player_Save.sav", 0, 16384);
buffer_async_group_end();

El código anterior inicia un grupo de almacenamiento intermedio y luego establece las opciones de grupo antes de establecer 4 archivos para guardar de forma asíncrona. La definición del grupo se termina (en ese momento comenzará el ahorro).
