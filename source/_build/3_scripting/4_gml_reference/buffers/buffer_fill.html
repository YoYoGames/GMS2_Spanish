buffer_fill

Sintaxis:

buffer_fill(buffer, offset, type, value, size);


Argumento Descripción
buffer El índice del buffer para llenar.
offset El valor de compensación de datos (en bytes).
type El tipo de datos que se escriben en el búfer (consulte la lista de constantes aquí ).
value Los datos para escribir
size El tamaño del búfer (en bytes) que desea llenar.


Devoluciones: N / A


Descripción

Esta función se puede usar para llenar un buffer previamente creado con un tipo y valor de datos dados. Los datos con los que llena el búfer deben estar de acuerdo con el argumento "tipo" de esta función, lo que significa que no puede tratar de rellenar con una cadena y usar el tipo entero de 16 bits sin signo, por ejemplo. Las constantes de tipo son las mismas que las utilizadas por buffer_read y buffer_write funciones. El "tamaño" es el tamaño del búfer (en bytes) que desea llenar, mientras que el desplazamiento es el valor de compensación (también en bytes) desde el inicio del búfer para iniciar el llenado.


Ejemplo:

map_buffer = buffer_create(16384, buffer_fixed, 0);
buffer_fill(map_buffer, 0, buffer_u16, 0, 16384);

El código anterior encuentra el inicio del búfer con el ID almacenado en la variable "buff" y escribe una serie de valores enteros de 16 bits con signo.
