buffer_read

Sintaxis:

buffer_read(buffer, type)


Argumento Descripción
buffer El índice del buffer para leer.
type El tipo de datos que se deben leer desde el búfer (consulte la lista de constantes a continuación).


Devoluciones: Real / String


Descripción

Esta función se puede utilizar para leer datos de un búfer creado previamente. El valor de retorno dependerá del tipo de datos que esté leyendo, que en sí mismo está definido por las siguientes constantes:

Constante Descripción
buffer_u8 Un entero sin signo de 8 bits. Este es un valor positivo de 0 a 255.
buffer_s8 Un entero de 8 bits con signo. Este puede ser un valor positivo o negativo de -128 a 127 (0 se clasifica como positivo).
buffer_u16 Un entero sin signo de 16 bits. Este es un valor positivo de 0 - 65,535.
buffer_s16 Un entero firmado de 16 bits. Este puede ser un valor positivo o negativo de -32,768 a 32,767 (0 se clasifica como positivo).
buffer_u32 Un entero sin signo de 32 bits. Este es un valor positivo de 0 a 4.294.967.295.
buffer_s32 Un entero firmado de 32 bits. Este puede ser un valor positivo o negativo de -2,147,483,648 a 2,147,483,647 (0 se clasifica como positivo).
buffer_u64 Un entero de 64 bits sin signo. Esto puede ser un valor positivo o negativo de - (2 63 ) a 2 63 - 1.
buffer_f16 Un flotador de 16 bits. Esto puede ser un valor positivo o negativo dentro del rango de +/- 65504. (¡Actualmente no es compatible!)
buffer_f32 Un flotador de 32 bits. Este puede ser un valor positivo o negativo dentro del rango de +/- 16777216.
buffer_f64 Un flotador de 64 bits. Este puede ser un valor positivo o negativo de - (2 52 ) a 2 52 - 1.
buffer_bool Un valor booleano. Solo puede ser 1 o 0 ( true o false )
buffer_string Una cadena de cualquier tamaño.
buffer_text Una cadena de cualquier tamaño, sin el carácter de terminación nulo final.

NOTA: ¡ Usar el tipo de datos incorrecto para los datos que se leen dará como resultado valores erróneos!


Ejemplo:

var cmd = buffer_read(buff, buffer_S16 );

El código anterior se lee desde el búfer con el ID almacenado en la variable "buff" un valor de 16 bits firmado en la variable local "cmd".
