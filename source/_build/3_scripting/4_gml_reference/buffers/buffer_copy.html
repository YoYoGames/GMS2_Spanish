buffer_copy

Sintaxis:

buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset);


Argumento Descripción
src_buffer El índice del buffer para copiar desde.
src_offset La compensación de datos para comenzar a copiar desde (en bytes).
size El tamaño de los datos para copiar (en bytes).
dest_buffer El índice del buffer para copiar a.
dest_offset La posición de desplazamiento para copiar los datos a (en bytes).


Devoluciones: N / A


Descripción

Esta función se puede usar para copiar un segmento (o todos) de los datos almacenados en un búfer a otro. Cuando utilice dos almacenamientos intermedios y copie de uno a otro, ambos almacenamientos intermedios deben haberse creado previamente utilizando el buffer_create función (por ejemplo), y puede especificar un desplazamiento de datos (en bytes) para el punto de inicio de los datos que se copiarán del búfer de origen en relación con el inicio del búfer, así como otro desplazamiento de datos para definir la posición para copiar los datos en el búfer de destino.

NOTA: No puede copiar al mismo buffer.


Ejemplo:

buffer_copy(buff1, 0, 2048, buff2, 2048);

El código anterior copiará los datos almacenados en el búfer indexado en la variable "buff1", y luego los pegará en el búfer indexado en la variable "buff2", pero se compensará con 2048 bytes.
