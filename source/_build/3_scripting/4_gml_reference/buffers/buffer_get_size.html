buffer_get_size

Sintaxis:

buffer_get_size(index);


Argumento Descripción
index El índice del búfer para obtener el tamaño de.


Devoluciones: Real


Descripción

Con esta función puede obtener el tamaño del búfer dado en bytes.


Ejemplo:

var _size = buffer_get_size(player_data);
var _temp = buffer_create(_size, buffer_fixed, 0);

El código anterior creará un nuevo búfer y almacenará su índice en la variable local "_temp", y el tamaño de este nuevo búfer será el mismo que el del búfer indexado en la variable "player_data".
