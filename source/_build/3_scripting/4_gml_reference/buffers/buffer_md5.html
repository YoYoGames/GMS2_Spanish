buffer_md5

Sintaxis:

buffer_md5(buffer, offset, size);


Argumento Descripción
buffer El índice del buffer a usar.
offset El valor de compensación de datos.
size El tamaño del buffer


Devoluciones: cadena


Descripción

En criptografía, MD5 (Message-Digest algorithm 5) es una función hash criptográfica ampliamente utilizada con un valor hash de 128 bits y se ha empleado en una amplia variedad de aplicaciones de seguridad. También se usa comúnmente para verificar la integridad de los datos. Esta función tomará los datos de entrada de un búfer determinado (o parte del búfer) y devolverá el hash hexadecimal MD5 de 32 caracteres que es exclusivo de esos datos. De esta forma, puede generar una clave segura que puede almacenarse y usarse para verificar la integridad de la información que se envía (o recibe de) un servidor externo (por ejemplo). Al aplicar esto a los buffers usando esta función, debe especificar el ID del buffer del buffer a usar, luego un valor de compensación (en bytes) para dónde comenzar, y luego un tamaño (de nuevo en bytes) para la región que se va a aplicar hash.


Ejemplo:

check_string = buffer_md5(buff, 0, buffer_get_size(buff));

El código anterior creará un hash md5 para todos los datos almacenados en el búfer indexado por la variable "buff", y almacenará el hash devuelto en la variable "check_string".
