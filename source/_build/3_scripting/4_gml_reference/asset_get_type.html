asset_get_type


Descripción

Con esta función puede obtener el tipo de activo al que se hace referencia desde su nombre (una cadena). Se devolverá una de las constantes enumeradas a continuación.


Sintaxis:

asset_get_type(name);

Argumento Descripción
name El nombre del activo del juego para obtener el tipo de (una cadena).


Devoluciones:

Constant

Constante Descripción
asset_object El nombre de pila se refiere a un objeto.
asset_sprite El nombre de pila se refiere a un sprite.
asset_sound El nombre de pila se refiere a un sonido.
asset_room El nombre de pila se refiere a una habitación.
asset_tile El nombre de pila se refiere a un conjunto de fichas.
asset_path El nombre de pila se refiere a un camino.
asset_script El nombre de pila se refiere a una secuencia de comandos.
asset_font El nombre de pila hace referencia a una fuente.
asset_timeline El nombre de pila se refiere a una línea de tiempo.
asset_shader El nombre de pila se refiere a un sombreador.
asset_unknown El nombre de pila hace referencia a un activo que no existe o no figura en alguno de los anteriores.


Ejemplo:

if asset_get_type("pth_Path_" + string(global.Game)) == asset_unknown
   {
   show_debug_message("Path doesn't exist!!!");
   }
else
   {
   path_index = asset_get_index("pth_Path_" + string(global.Game));
   }

El código anterior comprueba un nombre de activo creado dinámicamente para ver si el activo es del tipo correcto. Si no es así, se mostrará un mensaje de depuración, de lo contrario, el nombre del activo se utilizará para asignar el activo a la instancia.
