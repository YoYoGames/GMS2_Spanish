show_debug_message


Descripción

Esta función creará un mensaje de depuración personalizado que se muestra en la ventana del compilador en el tiempo de ejecución. Tenga en cuenta que el mensaje puede ser una cadena o un número real, pero si necesita tanto a continuación, el número tendrá que ser convertido a la cadena por primera vez el string función (consulte el ejemplo a continuación) y que si el número tiene más de dos lugares decimales, entonces debe usar string_format para mostrarlos como por decimales predeterminados se redondean a los dos lugares decimales más cercanos (para que "1.2468" se muestre como "1.25" en la ventana de salida).

Los mensajes de depuración que se muestran con esta función se mostrarán en la Ventana de salida del compilador en la parte inferior del IDE, así como en la Vista de gráfico del depurador al ejecutar el juego en Modo de depuración. Si solo quiere ver mensajes en modo de depuración, probablemente debería estar usando debug_event en lugar.


Sintaxis:

show_debug_message(string);

Argumento Descripción
string El mensaje de depuración personalizado para mostrar.


Devoluciones:

N/A


Ejemplo:

if !instance_exists(obj_Exit)
   {
   show_debug_message("Exit not created!");
   show_debug_message("Error Num: " + string(global.error));
   game_end();
   }

El código anterior comprueba si existe una instancia y, si no, se muestra un mensaje de depuración en la ventana de compilación y el juego finaliza.
