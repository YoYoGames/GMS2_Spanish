debug_get_callstack


Descripción

Esta función genera una matriz de cadenas, donde el script actual se enumera primero. El formato exacto de la cadena variará según la plataforma de destino elegida, pero en su mayoría tendrá el nombre del script / evento, luego dos puntos: y luego el número de línea, algo como esto:

"gml_Script_script2:1"
"gml_Script_script1:32"
"gml_Script_script0:64"
"gml_Object_object0_Create_0:1"


Sintaxis:

debug_get_callstack()


Devoluciones:

Array


Ejemplo:

if debug_mode
    {
    if keyboard_check(vk_escape)
        {
        var _a = debug_get_callstack();
        for (var i = 0; i < array_length_id(_a); ++i;)
            {
            show_debug_message(_a[i]);
            }
        }
    }

El código anterior verifica si el modo de depuración está habilitado y si lo está, verifica si se está presionando un ios clave. Si es así, envía la pila de llamadas actual a la consola,
