get_string


Descripción

Esto crea una ventana emergente que muestra un mensaje estándar, con un botón etiquetado "Ok", que solicita al usuario que ingrese una cadena. La función devolverá la cadena de entrada o el valor predeterminado si no se ha ingresado nada.

NOTA: Esta función es para uso de depuración solo en destinos de escritorio, pero está en desuso en todos los demás destinos.


Sintaxis:

get_string(str, def);

Argumento Descripción
str La cadena para mostrar en el mensaje emergente.
def La cadena por defecto en el cuadro de texto.


Devoluciones:

String


Ejemplo:

global.test_name = get_string("Test highscore name:", "Anonymous");

El código anterior le pedirá al usuario que dé un nombre que luego se almacenará en la variable global "nombre de prueba". Si no se ingresa nada y el usuario simplemente presiona "Aceptar", se devolverá el valor predeterminado, "Anónimo".
