steam _get_quota_total

Sintaxis:

steam_get_quota_total();


Devoluciones: Real


Descripción

Al usar Steam Cloud para almacenar y sincronizar archivos, debe configurar la cuota de espacio que su juego necesitará. Esta cuota se aplica en cada juego habilitado para la nube, por usuario y por juego, por lo que, por ejemplo, si la cuota para el juego X es de 1 megabyte, cada cuenta de Steam que posee el juego X puede almacenar, como máximo, 1 megabyte de datos asociados con ese juego en la nube. Cualquier otro juego habilitado para la nube que el usuario posea (por ejemplo, el Juego Y) no se verá afectado por los datos almacenados en el Juego X. La cuota predeterminada para los nuevos juegos de Steam es de un gigabyte, pero puede cambiar esto desde el control de Steam Works panel para tu juego.

NOTA: Una vez que se agote la cuota, las escrituras de los archivos fallarán. Si cree que es posible que la cuota se agote para el usuario de su juego, debe crear un código para manejarlo, ya que al no hacer nada deja a los usuarios en una situación en la que no pueden arreglar las cosas y eso los conducirá a una mala experiencia de tu juego.


Ejemplo:

if steam_is_cloud_enabled_for_app()
   {
   quota = steam_get_quota_total();
   }

El código anterior comprueba si la nube de steam está habilitada para el juego y, de ser así, obtiene el tamaño de la cuota de almacenamiento y la almacena en una variable.
