steam _create_leaderboard

Sintaxis:

steam_create_leaderboard(lb_name, sort_method, display_type);


Argumento Descripción
lb_name El nombre de la tabla de clasificación que estás creando (una cadena).
sort_method El método para clasificar las entradas de la tabla de clasificación (ver las constantes que figuran a continuación).
display_type La forma de mostrar la tabla de clasificación al usuario (ver las constantes que figuran a continuación).


Devoluciones: Real


Descripción

Con esta función puedes crear una nueva tabla de clasificación para tu juego. El primer argumento es una cadena que define el nombre de su tabla de clasificación, y este nombre debe usarse en cualquier llamada de función adicional relacionada con la tabla de clasificación que se está creando. A continuación, puede definir el método de clasificación (consulte las constantes enumeradas a continuación), así como la forma en que se muestra la información (una vez más, una de las constantes enumeradas a continuación). La función devolverá un valor de -1 si no puede crear la tabla de clasificación o un valor mayor que -1 si ha tenido éxito.

NOTA: Si previamente has creado una tabla de clasificación con el mismo nombre (ya sea a través del código o en tu página de Steam para el juego) esta función se ignorará.

Sort Constant Descripción
lb_sort_none Sin clasificación. La información se mostrará "tal cual".
lb_sort_ascending Ordena la tabla de clasificación en orden ascendente.
lb_sort_descending Clasifica la tabla de clasificación en orden descendente.
Display constante Descripción
lb_disp_none Muestre la tabla de clasificación "tal como está".
lb_disp_numeric Muestra la tabla de clasificación como una pantalla numérica.
lb_disp_time_sec Muestre los valores de la tabla de líderes como veces, con el valor base como segundos.
lb_disp_time_ms Muestre los valores de la tabla de líderes como veces, con el valor base en milisegundos.


Ejemplo:

steam_create_leaderboard("Game Times", lb_sort_ascending, lb_disp_time_sec);

El código anterior creará una tabla de clasificación llamada "Tiempos de juego" y la configurará para que muestre los resultados en orden ascendente y con una pantalla en segundos.
