steam _get_stat_float

Sintaxis:

steam_get_stat_float(stat_name);


Argumento Descripción
stat_name El nombre de la estadística para obtener (una cadena).


Devoluciones: Real


Descripción

Con esta función puede obtener el valor de un estadístico de coma flotante específico. La estadística debería haberse definido previamente en la página de cuentas del panel de control de las obras de Steam para su juego y la cadena que se pasa a la función debe coincidir con la utilizada como nombre de API en el panel de control.


Ejemplo:

var dist_pc = (dist / dist_max) * 100;
if steam_get_stat_float("Travelled") < dist_pc
   {
   steam_set_stat_int("Travelled", dist_pc);
   }

El código anterior calcula un porcentaje basado en la distancia recorrida variable "dist" y la distancia máxima que puede viajar "dist_max". Luego sondea el valor actual para la estadística "Viajado" y si es menor que el valor calculado, establece la estadística nuevamente.
