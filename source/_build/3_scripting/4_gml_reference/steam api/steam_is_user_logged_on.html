steam _is_user_logged_on

Sintaxis:

steam_is_user_logged_on();


Devoluciones: cadena


Descripción

Esta función regresará true si el cliente Steam actualmente tiene una conexión en vivo a los servidores de Steam. Si regresa false, significa que no hay conexión activa debido a un problema de red en la máquina local, o que el servidor Steam está inactivo o ocupado.


Ejemplo:

if steam_is_user_logged_on()
   {
   steam_publish_workshop_file("SaveData.txt", "", "Save Text", "Workshop saved text file");
   }

El código anterior verificará si el usuario está conectado al servidor Steam y, de ser así, publicará un archivo en el Taller.
