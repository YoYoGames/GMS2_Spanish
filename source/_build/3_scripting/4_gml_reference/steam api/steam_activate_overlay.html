steam _activate_overlay

Sintaxis:

steam_activate_overlay(overlayindex);


Argumento Descripción
overlayindex El índice de la página de la interfaz de usuario de Steam API para mostrar (ver las constantes que figuran a continuación).


Devoluciones: N / A


Descripción

La superposición de Steam es una parte de la interfaz de usuario de Steam que puede activarse sobre casi cualquier juego lanzado a través de Steam. Permite al usuario acceder a la lista de amigos, al navegador web, al chat y a la compra de DLC dentro del juego. La tecla predeterminada para que un usuario acceda a la superposición mientras está en un juego es MAYÚS + TAB, pero también puede abrir cualquier página de la superposición usando esta función. Toma una de las seis constantes que se enumeran a continuación:

Constante Descripción
ov_friends La página de amigos para el usuario actual
ov_community La página de la comunidad para tu juego
ov_players La página que muestra a otras personas que están jugando el mismo juego o con las que jugaste recientemente
ov_settings La configuración de superposición del cliente de Steam
ov_gamegroup Abre el navegador web de Steam Community para el grupo oficial de juegos para este juego
ov_achievements La página de logros para tu juego


Ejemplo:

var key = keyboard_lastkey;
switch (key)
   {
   case vk_f1: steam_activate_overlay(ov_friends); break;
   case vk_f2: steam_activate_overlay(ov_community); break;
   case vk_f3: steam_activate_overlay(ov_players); break;
   case vk_f4: steam_activate_overlay(ov_settings); break;
   case vk_f5: steam_activate_overlay(ov_gamegroup); break;
   case vk_f6: steam_activate_overlay(ov_achievements); break;
   }

El código anterior sondea la última tecla del teclado presionada y si es alguna de las teclas de función del 1 al 6, abrirá la página correspondiente de la superposición de Steam.
