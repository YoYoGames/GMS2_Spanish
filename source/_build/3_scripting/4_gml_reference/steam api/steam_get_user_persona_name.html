steam _get_user_persona_name

Sintaxis:

steam_get_user_persona_name(steamID);


Argumento Descripción
steam ID El ID de Steam exclusivo para un usuario.


Devoluciones: Real


Descripción

Esta función se puede usar para recuperar el nombre de usuario (nombre de usuario) para cualquier ID de usuario específico. Usted le otorga la ID de usuario y luego la función devuelve un valor único de identificación asíncrona que se puede usar para verificar los detalles correctamente en el evento Async de Steam. Este evento tendrá el async_load Mapa DS que se completará con los siguientes pares clave / valor:

  1. " id ": la identificación asíncrona devuelta por la función que activó el evento

  3. " event_type " - La cadena " user_persona_name "

  5. " steam id" "- El valor de Steam ID utilizado para obtener el nombre de usuario.

  7. " nombre_personal ": el nombre de usuario para la ID de Steam proporcionada


Ejemplo:

request = steam_get_user_persona_name(global.UGC_UserID);

El código anterior solicitará el nombre de usuario de la ID de usuario almacenada en la variable global "UGC_UserID", almacenando el valor devuelto en una variable para analizar en el evento Async.
