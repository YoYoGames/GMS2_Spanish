steam _ugc_get_item_install_info

Sintaxis:

steam_ugc_get_item_install_info(published_file_id, info_map);


Argumento Descripción
published_file_id El único identificador para el UGC que se actualizará.
info_map Un índice de mapa DS (creado previamente).


Devoluciones: Boolean


Descripción

Esta función se puede utilizar para recuperar información sobre cualquier elemento de archivo publicado que se haya suscrito y descargado en el área de almacenamiento local de Steam para su juego. Le da la identificación del artículo y proporciona el índice a un mapa de DS vacío que luego se completará con los siguientes pares clave / valor si el artículo existe:

  1. " size_on_disk ": el tamaño del archivo en el disco (en bytes)

  3. " legacy_item " - Será true o false dependiendo de si es un archivo heredado o no.

  5. " carpeta ": esta es la ruta completa al contenido instalado (consulte "Instalación de elementos" en los documentos de Steam SDK, ya que los elementos "heredados" cargados con el método anterior se tratan de manera diferente).

Si el artículo existe, la función regresará true y poblar el mapa; de lo contrario, volverá false y el mapa permanecerá vacío.


Ejemplo:

var item_map = ds_map_create();
steam_ugc_get_item_install_info(global.fileID, item_map);

El código anterior consultará el estado de instalación del elemento indexado en la variable global "fileID", utilizando un ds_map para almacenar la información.
