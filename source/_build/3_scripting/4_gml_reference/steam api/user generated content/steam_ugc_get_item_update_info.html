steam _ugc_get_item_update_info

Sintaxis:

steam_ugc_get_item_update_info(published_file_id, info_map);


Argumento Descripción
published_file_id La identificación de archivo único para el UGC a verificar.
info_map Un índice de mapa DS (creado previamente).


Devoluciones: Boolean


Descripción

Esta función se puede usar para recuperar información sobre el estado de descarga actual para la identificación de archivo dada. Le da la identificación del artículo y proporciona el índice a un mapa de DS vacío que luego se completará con los siguientes pares clave / valor si el artículo existe:

  1. " needs_update " - Volverá true o false dependiendo de si el artículo necesita una actualización o no.

  3. " is_downloading " - Volverá true o false dependiendo de si el artículo se está descargando o no.

  5. " bytes_downloaded ": la cantidad de bytes que se ha descargado.

  7. " bytes_total ": el tamaño total (número de bytes) requerido para el elemento en el disco.

Si el artículo existe, la función regresará true y poblar el mapa; de lo contrario, volverá false y el mapa permanecerá vacío.


Ejemplo:

var info_map = ds_map_create();
var info = steam_ugc_get_item_update_info(global.fileID, info_map);
if info
   {
   draw_text(32, 15, "needs_update: " + string(info_map[? "needs_update"]));
   draw_text(32, 30, "is_downloading: " + string(info_map[? "is_downloading"]));
   draw_text(32, 45, "bytes_downloaded: " + string(info_map[? "bytes_downloaded"]));
   draw_text(32, 60, "bytes_total: " + string(info_map[? "bytes_total"]));
   }

El código anterior consultará el estado de descarga del elemento indexado en la variable global "fileID", utilizando un ds_map para almacenar la información.
