steam _ugc_set_item_visibility

Sintaxis:

steam_ugc_set_item_visibility(ugc_update_handle, visibility);


Argumento Descripción
ugc_update_handle El único identificador para el UGC que se actualizará.
visibility La visibilidad (una constante, que se detalla a continuación) que se utilizará para el artículo.


Devoluciones: Boolean


Descripción

Esta función establecerá la visibilidad del elemento dado, configurado mediante una de las siguientes constantes:

Constante Descripción
ugc_visibility_public Establecer el elemento para ser visible públicamente
ugc_visibility_friends_only Establezca el elemento para que solo sea visible para las personas en la lista de amigos de los usuarios
ugc_visibility_private Establecer el elemento para ser privado



El controlador de actualización es el valor devuelto cuando llamó a la función steam_ugc_start_item_update y la función volverá true si la API fue accedida con éxito y false si hubo un problema




Ejemplo:

var app_id = steam_get_app_id();
var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID);
steam_ugc_set_item_title(updateHandle, "My workshop item(3)!");
steam_ugc_set_item_description( updateHandle, "testing workshop...");
steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public);
var tagArray;
tagArray[0] = "Test";
tagArray[1] = "New";
steam_ugc_set_item_tags(updateHandle, tagArray);
steam_ugc_set_item_preview(updateHandle, "promo.jpg");
steam_ugc_set_item_content(updateHandle, "WorkshopContent1");
requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2");

El código anterior obtiene el ID del juego, luego lo usa junto con un ID de archivo publicado previamente almacenado para generar un identificador de actualización para el elemento. Este identificador se utiliza para actualizar varias piezas de información antes de enviar la actualización a los servidores del taller.
