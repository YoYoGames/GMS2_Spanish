steam _ugc_submit_item_update

Sintaxis:

steam_ugc_submit_item_update(ugc_update_handle, change_note);


Argumento Descripción
ugc_update_handle El único identificador para el UGC que se actualizará.
change_note Las notas de cambio (una cadena) que se utilizarán para el artículo.


Devoluciones: identificación Async


Descripción

Esta función enviará el elemento UGC inexacto por el controlador dado a los servidores de Steam Workshop, agregando las notas de cambio que se utilizarán para el artículo dado. Las notas de cambio deben ser una cadena y el manejador de actualización es el valor devuelto cuando llamó a la función steam_ugc_start_item_update. Al usar esta función, se devolverá un valor de identificación asíncrona que luego se puede analizar cuando se desencadena el evento Steam Asynchronous para informar la creación del elemento. El evento contendrá los siguientes valores de clave / mapa en async_load ds_map:

  1. " id ": la identificación asíncrona devuelta por la función de llamada

  3. " resultado " - El resultado de la operación (un valor real). Esta será la constante GML ugc_result_success o algún otro número real. Por lo tanto, debe verificar esta constante para asegurarse de que la llamada fue exitosa y, si no, algo no funcionó correctamente. El resto de los posibles valores devueltos se muestran como resultado del valor "EResult" de Steam y debería ver steamclientpublic.h en los encabezados del SDK para los 89 valores posibles.

  5. " event_type " - Esta tecla mantendrá el valor " ugc_update_item "

  7. " legal_agreement_required " - Será true o false (mira los documentos de Steam para más detalles)


Ejemplo:

var app_id = steam_get_app_id();
var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID);
steam_ugc_set_item_title(updateHandle, "My workshop item(3)!");
steam_ugc_set_item_description( updateHandle, "testing workshop...");
steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public);
var tagArray;
tagArray[0] = "Test";
tagArray[1] = "New";
steam_ugc_set_item_tags(updateHandle, tagArray);
steam_ugc_set_item_preview(updateHandle, "promo.jpg");
steam_ugc_set_item_content(updateHandle, "WorkshopContent1");
requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2");

El código anterior obtiene el ID del juego, luego lo usa junto con un ID de archivo publicado previamente almacenado para generar un identificador de actualización para el elemento. Este identificador se utiliza para actualizar varias piezas de información antes de enviar la actualización a los servidores del taller.
