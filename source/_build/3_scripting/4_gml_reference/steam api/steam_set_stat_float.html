steam _set_stat_float

Sintaxis:

steam_set_stat_float(stat_name, value);


Argumento Descripción
stat_name El nombre de la estadística para establecer (una cadena).
value El valor (un flotante de 32 bits) para establecer la estadística.


Devoluciones: N / A


Descripción

Con esta función puede establecer una estadística específica a un nuevo valor de punto flotante. La estadística debería haberse definido previamente en la página de cuentas del panel de control de las obras de Steam para su juego y la cadena que se pasa a la función debe coincidir con la utilizada como nombre de API en el panel de control. Los ejemplos de cuándo podrías usar esto son para saber cuánto ha recorrido tu jugador o qué porcentaje del juego se ha completado.


Ejemplo:

var dist_pc = (dist / dist_max) * 100;
steam_set_stat_float("Travelled", dist_pc);

El código anterior calcula un porcentaje basado en la distancia recorrida variable "dist" y la distancia máxima que puede viajar "dist_max" y luego establece la estadística "Viajado" al nuevo valor.
