distance_to_object

Sintaxis:

distance_to_object( obj );


Argumento Descripción
obj El objeto a verificar


Devoluciones: Real.


Descripción

Esta función calcula la distancia desde el borde del cuadro delimitador de la instancia de llamada al borde más cercano de la instancia más cercana del objeto especificado. El objeto puede ser un índice de objeto o una identificación de instancia específica, así como la palabra clave other, y la distancia se devuelve en píxeles. Tenga en cuenta que si alguno de los objetos no tiene sprite o no hay máscara definida, los resultados serán incorrectos.


Ejemplo:

if distance_to_object(obj_Player) < range
   {
   canshoot = true;
   }

El código anterior verificará la distancia al objeto del jugador y si es menor que el valor almacenado en la variable "rango", la variable "canasta" se establece en verdadero.
