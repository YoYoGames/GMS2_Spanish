angle_difference

Sintaxis:

angle_difference(ang1, ang2)


Argumento Descripción
ang1 El primer ángulo para usar.
ang2 El segundo ángulo para usar.


Devoluciones: Real


Descripción

Esta función devolverá la diferencia de ángulo más pequeña entre dos ángulos como un valor entre -180 y 180 grados (donde un ángulo positivo es antihorario y un ángulo negativo en el sentido de las agujas del reloj).


Ejemplo:

var pd = point_direction(x, y, mouse_x, mouse_y);
var dd = angle_difference(image_angle, pd);
image_angle -= min(abs(dd), 10) * sign(dd);

El código anterior obtendrá el ángulo de dirección desde la instancia hasta el cursor del mouse, luego obtendrá la diferencia entre ese ángulo y el actual image_angle, usando este valor para girar lentamente hacia el mouse.
