random_set_seed


Descripción

Para generar un número aleatorio, GameMaker Studio 2 comienza con un número inicial aleatorio. Con esta función, puede establecer esa semilla en un valor conocido y, por lo tanto, "forzar" el resultado de todos los eventos aleatorios posteriores para que sea el mismo cada vez que se ejecute el programa. Por ejemplo, esta función se puede utilizar junto con random_get_seed para crear contenido generado de manera procesal y guardar los resultados sin tener grandes partidas guardadas (solo guardas la semilla, no necesitas nada más). Si necesita resultados verdaderamente aleatorios para todo, debe utilizar el randomise() función.

NOTA: Si bien esta semilla dará resultados consistentes en cada plataforma de destino, los resultados pueden variar entre plataformas debido a la forma diferente en que funciona cada objetivo.


Sintaxis:

random_set_seed(val);


Argumento Descripción
val La semilla para establecer.


Devoluciones:

N/A


Ejemplo:

if debug random_set_seed(1);

El código anterior establece la semilla aleatoria en 1 solo si la variable "debug" es verdadera.
