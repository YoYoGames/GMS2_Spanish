random_set_seed

Establece la semilla a partir de la cual se toman los números aleatorios y los resultados.

Sintaxis:

random_set_seed(val);


Argumento Descripción
val La semilla para establecer


Devoluciones: N / A


Descripción

Para generar un número aleatorio, GameMaker Studio 2 comienza con un número de semilla aleatorio. Con esta función puede establecer esa semilla a un valor conocido y así "forzar" el resultado de todos los eventos aleatorios para que sea el mismo cada vez que se ejecuta el programa. Por ejemplo, esta función se puede usar junto con random_get_seed para crear contenido generado por procedimientos y guardar los resultados sin tener grandes partidas guardadas (solo guarda la semilla, no necesita otra cosa). Si necesita resultados verdaderamente aleatorios para todo, debe usar el randomise() función.

NOTA: Si bien esta semilla dará resultados consistentes en cada plataforma objetivo, los resultados pueden variar entre las plataformas debido a la forma diferente en que funciona cada objetivo.


Ejemplo:

if debug random_set_seed(1);

El código anterior establece la semilla aleatoria en 1 solo si la variable "depuración" es verdadera.
