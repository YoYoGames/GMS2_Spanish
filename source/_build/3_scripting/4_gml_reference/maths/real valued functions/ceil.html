hacer techo


Descripción

Esta función toma cualquier número real y lo redondea al entero más cercano. Se debe tener cuidado con esta función, ya que un error común es usarlo para redondear un valor aleatorio y esperar que siempre sea mayor que 1, es decir:

int = ceil(random(5));

Ahora, esperaría que este código siempre dé un número entero entre 1 y 5, pero puede que no siempre sea así, ya que existe una posibilidad muy pequeña de que la función aleatoria devuelva 0, y al redondear a 0 todavía le da 0. Esto es una posibilidad remota, pero debe tenerse en cuenta al utilizar esta función.


Sintaxis:

ceil(val);


Argumento Descripción
x El número a cambiar.


Devoluciones:

Integer


Ejemplo:

val = ceil( 3.4 );

Esto establecerá val en 4.
