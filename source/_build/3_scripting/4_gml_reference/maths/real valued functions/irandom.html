Descripción

Esta función muy útil solo devuelve números enteros (números enteros). Así, por ejemplo, para obtener un número aleatorio de 0 a 9 puede usar irandom(9) y devolverá un número de 0 a 9 inclusive. Tenga en cuenta que también se pueden usar flotantes, pero el valor superior después del punto se excluirá, por lo que irandom(9.7) devolverá un valor de 0 a 9 solamente. La función tiene un límite superior de $7fffffffffffffff LL, por lo que se debe tener cuidado si se utilizan números muy grandes.

NOTA: esta función devolverá el mismo valor cada vez que el juego se ejecute de nuevo debido al hecho de que GameMaker Studio 2 genera la misma inicialización aleatoria inicial cada vez que la tarea de depurar el código es mucho más fácil. Para evitar este comportamiento utiliza. randomise al comienzo de tu juego.


Sintaxis:

irandom(n);


Argumento Descripción
n El rango superior desde el cual se seleccionará el número aleatorio.


Devoluciones:

Integer


Ejemplo:

if irandom(9) = 1
   {
   score += 100;
   }

Esto producirá una probabilidad de uno en diez (ya que se incluye 0) de agregar 100 a la puntuación.
