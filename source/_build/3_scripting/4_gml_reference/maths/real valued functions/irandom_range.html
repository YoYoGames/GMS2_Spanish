irandom_range

Devuelve un entero aleatorio entre n1 y n2.

Sintaxis:

irandom_range(n1, n2);


Argumento Descripción
n1 El extremo inferior del rango desde el que se seleccionará el número aleatorio.
n2 El extremo superior del rango desde el que se seleccionará el número aleatorio.


Devoluciones: Entero


Descripción

Esta función es similar a random_range() solo con valores enteros como entrada. Asi que irandom_range(10, 35) devolverá un número entero entre 10 y 35. Como con el irandom() función, se pueden usar números reales, en cuyo caso se redondearán al entero más cercano EG: irandom_range (6.2,9.9) dará un valor entre 6 y 9.

NOTA: Esta función devolverá el mismo valor cada vez que se vuelva a ejecutar el juego debido al hecho de que GameMaker Studio 2 genera la misma semilla aleatoria inicial cada vez para hacer que el código de depuración sea una tarea mucho más sencilla. Para evitar este comportamiento, use randomise al comienzo de tu juego


Ejemplo:

score += irandom_range(500, 600);

Esto agregará entre 500 y 600, un valor entero, al puntaje total.
