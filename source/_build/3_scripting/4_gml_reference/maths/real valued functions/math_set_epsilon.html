math_set_epsilon


Descripción

Epsilon es un valor que se utiliza para determinar si dos números sujetos al error de redondeo están lo suficientemente cerca para ser considerados "iguales". Es útil cuando se trata de matemáticas de punto flotante, ya que puede reducir los "errores de redondeo" que hacen que ciertas operaciones devuelvan valores que parecen incorrectos o contrarios a lo que esperamos. Por ejemplo, es posible que hayamos agregado un valor al índice de imagen de un sprite y esperemos que el resultado sea un solo entero, pero debido a la naturaleza de las matemáticas de coma flotante, el valor final real termina siendo algo así como 5.0000002, por lo que cuando tener la siguiente comprobación:

if image_index == 5 {do something}

El código no se comporta como se espera y devuelve. false. Sin embargo, si establecemos el valor épsilon en 0.000001, el valor de image_index se redondeará al número real más cercano que es +/- 0.000001 del valor original, haciendo que la comparación anterior sea verdadera.

El valor épsilon se utilizará para todos los siguientes operadores de enteros:

  • <: Menos que
  • >: Mayor que
  • ==: Igual a
  • <=: Menor o igual que
  • > =: Mayor o igual que
  • ! =: No es igual a

Tenga en cuenta que establecer un valor épsilon de 0 deshabilitará todo el redondeo, y usar un valor de 1 dará un error.


Sintaxis:

math_set_epsilon(epsilon);


Argumento Descripción
epsilon El nuevo valor épsilon (de 0 a 0.999999999).


Devoluciones:

Real


Ejemplo:

math_set_epsilon(0.0001);

Esto establecerá el valor épsilon para todas las operaciones de punto flotante adicionales.
