math_set_epsilon

Establezca el "épsilon de la máquina" para los cálculos de coma flotante.

Sintaxis:

math_set_epsilon(epsilon);


Argumento Descripción
epsilon El nuevo valor epsilon (de 0 a 0,999999999).


Devoluciones: Real


Descripción

Epsilon es un valor utilizado para determinar si dos números sujetos a error de redondeo son lo suficientemente cercanos para ser considerados "iguales". Es útil cuando se trata de matemática de punto flotante ya que puede reducir los "errores de redondeo" que hacen que ciertas operaciones devuelvan valores que parecen incorrectos o contrarios a lo que esperamos. Por ejemplo, podemos haber agregado un valor al índice de imagen de un sprite y esperar que el resultado sea un entero único, pero debido a la naturaleza de las matemáticas de punto flotante, el valor final real termina siendo algo así como 5.0000002, por lo que cuando tener la siguiente verificación:

if image_index == 5 {do something}

El código no se comporta como se esperaba y regresa false. Sin embargo, si establecemos el valor épsilon en 0.000001, el valor image_index se redondeará al número real más cercano que sea +/- 0.000001 del valor original, haciendo que la comparación anterior sea verdadera.

El valor epsilon se usará para todos los siguientes operadores enteros:

  • <: Menos de
  • >: Mayor que
  • ==: Igual a
  • <=: Menor que o igual a
  • > =: Mayor que o igual a
  • ! =: No es igual a

Tenga en cuenta que establecer un valor épsilon de 0 deshabilitará todo redondeo, y usar un valor de 1 dará un error.


Ejemplo:

math_set_epsilon(0.0001);

Esto establecerá el valor épsilon para todas las demás operaciones de coma flotante.
