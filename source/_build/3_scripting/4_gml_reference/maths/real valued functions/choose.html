Descripción

A veces desea especificar algo distinto a los números para una selección aleatoria, o los números que desea no están en ningún orden real o dentro de ningún rango establecido. En estos casos utilizarías choose() Para generar un resultado aleatorio. Por ejemplo, digamos que desea crear un objeto con un sprite aleatorio al principio, entonces podría usar esta función para establecer el índice de sprites en uno de un conjunto de sprites dados. Tenga en cuenta que puede tener tantos como necesite (tenga en cuenta que más argumentos significarán que la función será más lenta de analizar).

NOTA: esta función devolverá el mismo valor cada vez que el juego se ejecute de nuevo debido al hecho de que GameMaker Studio 2 genera la misma inicialización aleatoria inicial cada vez que la tarea de depurar el código es mucho más fácil. Para evitar este comportamiento utiliza. randomise al comienzo de tu juego.


Sintaxis:

choose(val0, val1, val2... max_val);


Argumento Descripción
val0... max_val Un valor de entrada que puede ser cadena, entero, variable o constante.


Devoluciones:

One of the given arguments


Ejemplo:

sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub);
hp = choose(5, 8, 15, 32, 40);
name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen");

Los usos del código anterior eligen establecer una serie de propiedades para la instancia.
