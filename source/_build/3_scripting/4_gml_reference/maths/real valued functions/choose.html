escoger

Elige al azar uno de los conjuntos de argumentos dados.

Sintaxis:

choose(val0, val1, val2... max_val);


Argumento Descripción
val0... max_val Un valor de entrada que puede ser de cadena, entero, variable o constante.


Devoluciones: uno de los argumentos dados.


Descripción

Algunas veces quiere especificar algo que no sean números para una selección aleatoria, o los números que desea no están en ningún orden real o dentro de un rango establecido. En estos casos, usarías choose() para generar un resultado aleatorio. Por ejemplo, supongamos que quieres crear un objeto con un sprite al azar al principio, luego puedes usar esta función para establecer el índice de sprites en uno de un conjunto de sprites dados. Tenga en cuenta que puede tener tantas como necesite (tenga en cuenta que más argumentos significarán que la función será más lenta de analizar).

NOTA: Esta función devolverá el mismo valor cada vez que se vuelva a ejecutar el juego debido al hecho de que GameMaker Studio 2 genera la misma semilla aleatoria inicial cada vez para hacer que el código de depuración sea una tarea mucho más sencilla. Para evitar este comportamiento, use randomise al comienzo de tu juego


Ejemplo:

sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub);
hp = choose(5, 8, 15, 32, 40);
name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen");

El código anterior usa elegir establecer un número de propiedades para la instancia.
