rango_aleatorio

Devuelve un número real aleatorio entre n1 y n2.

Sintaxis:

random_range(n1, n2);


Argumento Descripción
n1 El extremo inferior del rango desde el que se seleccionará el número aleatorio.
n2 El extremo superior del rango desde el que se seleccionará el número aleatorio.


Devoluciones: Real


Descripción

Esta función devuelve un número aleatorio entre el rango especificado, y este valor de retorno no necesita ser un número entero. Por ejemplo, random_range(20,50) devolverá un número aleatorio de 20 a 50, pero el valor puede ser un número real como 38,65265. Los números reales también se pueden usar como argumentos de entrada.

NOTA: Esta función devolverá el mismo valor cada vez que se vuelva a ejecutar el juego debido al hecho de que GameMaker Studio 2 genera la misma semilla aleatoria inicial cada vez para hacer que el código de depuración sea una tarea mucho más sencilla. Para evitar este comportamiento, use randomise al comienzo de tu juego


Ejemplo:

score += random_range(500, 600);

Esto agregará entre 500 y 600, a la puntuación total.
