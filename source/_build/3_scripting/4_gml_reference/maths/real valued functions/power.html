poder

Devuelve el número dado a la potencia dada; el número x multiplicado por sí mismo n número de veces.

Sintaxis:

power(x, n);


Argumento Descripción
x El número para cambiar
n Cuantas veces multiplicar x solo.


Devoluciones: Real


Descripción

Esto devolverá el valor de un número multiplicado por sí mismo "n" número de veces. Por ejemplo, power(5,3) multiplicará 5 por sí mismo 3 veces y devolverá 125, que es lo mismo que decir 5 * 5 * 5 = 125. Tenga en cuenta que el valor "x" (el número a cambiar) no puede ser un valor negativo.


Ejemplo:

score += power(dmg, 3);

Esto agregará el valor de sostenido en la variable "dmg" a la potencia de 3 a la variable de puntaje.
