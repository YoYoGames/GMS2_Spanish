Devuelve un valor fijado entre el mínimo y el máximo especificados.

Sintaxis:

clamp(val, min, max)


Argumento Descripción
val El valor para sujetar.
min El valor mínimo para fijar entre.
max El valor máximo para fijar entre.


Devoluciones: Real.


Descripción

Con esta función puede mantener un valor de entrada entre un rango especificado.


Ejemplo:

speed = clamp(speed, 1, 10);

El código anterior sujetará la velocidad para que nunca caiga por debajo de 1 o supere los 10.
