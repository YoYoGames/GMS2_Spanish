aleatorio


Descripción

Esta función es buena para las probabilidades en las que no es necesario devolver un número entero (número entero). Por ejemplo, random(100) devolverá un valor de 0 a 99, pero ese valor puede ser 22.56473! También puedes usar números reales y no enteros en esta función como esta: random(0.5), que devolverá un valor entre 0 y 0.4999999.

NOTA: esta función devolverá el mismo valor cada vez que el juego se ejecute de nuevo debido al hecho de que GameMaker Studio 2 genera la misma inicialización aleatoria inicial cada vez que la tarea de depurar el código es mucho más fácil. Para evitar este comportamiento utiliza. randomise al comienzo de tu juego.


Sintaxis:

random(n);


Argumento Descripción
n El rango superior desde el cual se seleccionará el número aleatorio.


Devoluciones:

Real


Ejemplo:

if random(10) >= 9
   {
   score += 100;
   }

Esto producirá aproximadamente una de cada diez posibilidades de agregar 100 a la puntuación.
