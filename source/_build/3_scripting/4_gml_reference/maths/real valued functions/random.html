aleatorio

Devuelve un número real aleatorio entre 0 y n.

Sintaxis:

random(n);


Argumento Descripción
n El rango superior desde el cual se seleccionará el número aleatorio.


Devoluciones: Real


Descripción

Esta función es buena para las probabilidades donde no es necesario devolver un número entero (número entero). Por ejemplo, random(100) devolverá un valor de 0 a 99, pero ese valor puede ser 22.56473! También puede usar números reales y no enteros en esta función como esta: random(0.5), que devolverá un valor entre 0 y 0.4999999.

NOTA: Esta función devolverá el mismo valor cada vez que se vuelva a ejecutar el juego debido al hecho de que GameMaker Studio 2 genera la misma semilla aleatoria inicial cada vez para hacer que el código de depuración sea una tarea mucho más sencilla. Para evitar este comportamiento, use randomise al comienzo de tu juego


Ejemplo:

if random(10) >= 9
   {
   score += 100;
   }

Esto producirá aproximadamente una probabilidad de uno en diez de sumar 100 puntos.
