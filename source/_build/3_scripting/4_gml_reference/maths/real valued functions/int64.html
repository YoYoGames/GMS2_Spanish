int64


Descripción

Esta función intentará convertir un valor dado en un entero de 64 bits, donde el valor debe ser real, una cadena, un int64, un int32 o un ptr. Cualquier otra cosa hará que el juego se bloquee con un mensaje de error. Puede verificar si una variable contiene un int64 usando la función is_int64.


Sintaxis:

int64(val);


Argumento Descripción
val El valor a convertir.


Devoluciones:

64bit integer


Ejemplo:

steam_handle = int64(global.fileReadString);

El código anterior convierte el valor contenido en la variable global en un entero de 64 bits.
