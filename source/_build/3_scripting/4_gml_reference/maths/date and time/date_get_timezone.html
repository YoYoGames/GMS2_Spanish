date_get_timezone

Establece la zona horaria base a usar para todas las funciones de fecha y hora.

Sintaxis:

date_get_timezone();


Devoluciones: constante


Descripción

Obtiene la zona horaria base que se utiliza para el resto de las funciones de fecha y hora. Esta zona horaria puede ser local (según lo establecido por el sistema) o UTC, y la función devolverá una de las siguientes constantes:

Constante Descripción
timezone_local utilice la zona horaria local según lo establecido por el sistema
timezone_utc utilizar el tiempo universal coordinado


Ejemplo:

if date_get_timezone() != timezone_utc
   br>    date_set_timezone(timezone_utc);
   }

Este código verifica la configuración de la zona horaria base para el juego y, si no es UTC, la cambia.
