json_encode

Sintaxis:

json_encode(map)


Argumento Descripción
map un ds_map con la información para codificar


Devoluciones: cadena


Descripción

JSON (JavaScript Object Notation) es un formato liviano de intercambio de datos que es fácil de leer y escribir, tanto para personas como para máquinas. Está construido sobre dos estructuras básicas:

  • Una colección de pares nombre / valor, llamada ds_map en GameMaker Studio 2 pero también conocido como "diccionario" u "objeto".
  • Una lista ordenada de valores, llamada ds_list en GameMaker Studio 2 pero esto también se puede llamar una "matriz" o "secuencia".

json_encode toma un ds_map que ha creado y codificado previamente como una cadena JSON que luego puede usar como (por ejemplo) parte de http_post_string() llamada, o eso se puede almacenar externamente, escrito en un archivo.

NOTA: La funcionalidad jerárquica de JSON está disponible a través de funciones especiales ds_map y ds_list, por lo que puede codificar sublistas y mapas.


Ejemplo:

var hiscore_map, i, str;
hiscore_map = ds_map_create();
for (i = 0; i < 10; i ++;)
   {
   ds_map_add(name[i], score[i]);
   }
str = json_encode(hiscore_map);
get[0] = http_post_string("http://www.angusgames.com/game?game_id=" + string(global.game_id), str)
ds_map_destroy(hiscore_map);

El código anterior crea un ds_map y luego recorre las matrices de nombre y puntuación, y agrega cada par clave / valor al nuevo mapa. A continuación, este mapa está codificado usando json_encode y almacenado como una cadena en la variable "str". Esta cadena se envía a un servidor web usando http_post_string y el ds_map se destruye para evitar una pérdida de memoria ya que ya no se necesita.
