directorio_temp


Descripción

Esto se puede usar para devolver el directorio temporal creado para tu juego cada vez que se ejecuta (la raíz no contiene la "\" final). Este directorio contendrá archivos y se podrá acceder a ellos mientras se ejecuta el juego, pero se eliminará (junto con todos los archivos que contiene) cuando se cierre el juego.

¡ADVERTENCIA! ¡Es posible que esta función no funcione como espera debido a que GameMaker Studio 2 está en un espacio aislado! Consulte la sección sobre Límites del sistema de archivos para obtener más información.


Sintaxis:

temp_directory


Devoluciones:

String


Ejemplo:

ini_open(temp_directory + "\temp_ini.ini");

Esto abrirá un archivo ini en el directorio temporal del juego (creándolo si aún no existe).
