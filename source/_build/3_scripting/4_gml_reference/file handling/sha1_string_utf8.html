sha1_string_utf8

Sintaxis:

sha1_string_utf8(string)


Argumento Descripción
string La cadena para hash.


Devoluciones: cadena


Descripción

En criptografía, SHA-1 es una función de cifrado hash diseñada por la Agencia de Seguridad Nacional de los Estados Unidos y se emplea en varias aplicaciones y protocolos ampliamente utilizados, como el popular Git, donde se utiliza para verificar los cambios de archivos. Esta función tomará una cadena de entrada UTF-8 (que tiene un número variable de bytes por carácter) y devolverá un resumen de mensaje de 160 bits en formato ASCII, por ejemplo:

sha1 input string = "The quick brown fox jumps over the lazy dog"
sha1 output hash = "2fd4e1c6 7a2d28fc ed849ee1 bb76e739 1b93eb12"


De esta forma, puede generar una clave segura que puede almacenarse y usarse para verificar la integridad de la información que se envía (o recibe de) un servidor externo (por ejemplo).

NOTA: hay dos formatos para la codificación sha1, UTF-8 y Unicode. Ambos se proporcionan para facilitar la comunicación con diferentes configuraciones de servidor, pero la más común de usar es Unicode.


Ejemplo:

var hash, str;
str = json_encode(hiscore_map);
hash = sha1_string_utf8(str);
ini_open("local.ini");
ini_write_string("info", "0", hash);
ini_close();
get[0] = http_post_string("http://www.MacSweeney Games.com/CatchTheHaggis?game_hiscores=" + string(global.game_id), str)

El código anterior codificará un ds_map en una cadena JSON. A continuación, se genera un hash sha1 y se almacena en un archivo ini para que luego se pueda usar para verificar la integridad del JSON en caso de que se reciba la misma información más adelante desde el servidor. El JSON luego se envía.
