sha1_string_unicode


Descripción

En criptografía, SHA-1 es una función criptográfica hash diseñada por la Agencia de Seguridad Nacional de los Estados Unidos y se emplea en varias aplicaciones y protocolos ampliamente utilizados, como el popular Git, donde se usa para verificar cambios de archivos. Esta función tomará una cadena de entrada de Unicode y devolverá un resumen de mensaje de 160 bits en formato ASCII, por ejemplo:

sha1 input string = "The quick brown fox jumps over the lazy dog"
sha1 output hash = "2fd4e1c6 7a2d28fc ed849ee1 bb76e739 1b93eb12"


De esta manera, puede generar una clave segura que puede almacenarse y usarse para verificar la integridad de la información que se envía a (o se recibe de) un servidor externo (por ejemplo).

NOTA: Hay dos formatos para la codificación sha1, UTF-8 y Unicode. Ambos se proporcionan para facilitar la comunicación con diferentes configuraciones de servidor, pero el más común de usar es Unicode.


Sintaxis:

sha1_string_unicode(string)


Argumento Descripción
string La cadena de hash.


Devoluciones:

String


Ejemplo:

var hash, str;
str = base64_encode(game_data);
hash = sha1_string_unicode(str);
http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?hash=" + hash); http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?data=" + str);

El código anterior codificará en base64 una cadena y luego generará un hash sha1. Finalmente, tanto el hash como la cadena codificada se envían a un servidor.
