sha1_file

Sintaxis:

sha1_file(filename)


Argumento Descripción
filename El archivo para generar el hash sha1 para.


Devoluciones: cadena


Descripción

En criptografía, SHA-1 es una función hash criptográfica diseñada por la Agencia de Seguridad Nacional de los Estados Unidos y se emplea en varias aplicaciones ampliamente utilizadas, popular Git, donde se usa para verificar cambios de archivos, y los protocolos TLS y SSL, PGP, SSH, S / MIME e IPsec. Esta función tomará un archivo de entrada y devolverá un resumen de mensaje de 160 bits en formato ASCII exclusivo de ese archivo para ser utilizado para la verificación de integridad en cualquier fecha posterior.


Ejemplo:

hash = sha1_file(working_directory + "\game_data.ini")

El código anterior generará un hash sha1 para el archivo especificado y almacenará el valor devuelto en la variable "hash".
