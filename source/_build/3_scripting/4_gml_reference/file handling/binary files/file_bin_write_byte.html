file_bin_write_byte

Sintaxis:

file_bin_write_byte(binfile, byte);


Argumento Descripción
binfile El ID del archivo para escribir.
byte Los datos para escribir


Devoluciones: real.


Descripción

Esta función escribirá un byte de datos en el archivo identificado por el ID del archivo en la posición de escritura actual. Usted proporciona el valor de ID de archivo, tal como lo devuelve la función file_bin_open y el byte de datos para escribir.

NOTA: Estas funciones no funcionan cuando el módulo de destino es HTML5.


Ejemplo:

file_bin_write_byte(file, data);

Esto escribiría un byte en el archivo seleccionado.
