file_bin_open

Sintaxis:

file_bin_open(fname, mode);


Argumento Descripción
fname El nombre del archivo para leer.
mode el indicador de lo que se puede hacer con el archivo.


Devoluciones: Real


Descripción

Esta función abrirá el archivo binario con el nombre indicado. El modo indica qué se puede hacer con el archivo:

  • 0 = lectura
  • 1 = escritura
  • 2 = lectura y escritura

Cuando el archivo no existe, se crea y la función devuelve el id. Del archivo que se debe usar en las otras funciones. Puede abrir varios archivos al mismo tiempo (32 max), pero no olvide cerrarlos una vez que haya terminado con ellos.

¡ADVERTENCIA! ¡Es posible que esta función no funcione como espera debido a que GameMaker Studio 2 está siendo protegido! Consulte la sección sobre Límites del sistema de archivos para obtener más información.

NOTA: Estas funciones no funcionan cuando el módulo de destino es HTML5.


Ejemplo:

file = file_bin_open('myfile.bin', 2);

Esto abriría un archivo del mismo directorio que el juego y asignaría su índice a la variable "archivo".
