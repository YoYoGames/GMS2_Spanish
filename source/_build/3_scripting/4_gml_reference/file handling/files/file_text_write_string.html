file_text_write_string

Sintaxis:

file_text_write_string(fileid, str);


Argumento Descripción
fileid El id del archivo a editar.
str La cadena para escribir en el archivo.


Devoluciones: Real


Descripción

Con esta función puede escribir una cadena en un archivo de texto abierto previamente. Si el archivo ya contiene información, esta información se borrará y la cadena se escribirá al principio del archivo, a menos que haya abierto el archivo con el file_text_open_append. También puede evitar esto usando el file_text_readln funciona junto con el file_text_eof función para recorrer el contenido del archivo hasta que llegue al final y luego comience a escribir.


Ejemplo:

var i, file;
file = file_text_open_write(working_directory + "hiscore.txt");
for (i = 0; i < 10; i += 1)
   {
   file_text_write_real(file, scr[i]);
   file_text_writeln(file);
   file_text_write_string(file, scr_name[i]);
   file_text_writeln(file);
   }
file_text_close(file);

El código anterior abre un archivo para escribir y luego pasa por dos matrices, escribiendo cada valor de matriz en una nueva línea del archivo. El archivo se cierra cuando el bucle ha terminado.
