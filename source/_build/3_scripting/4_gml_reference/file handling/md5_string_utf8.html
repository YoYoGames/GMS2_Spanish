md5_string_utf8

Sintaxis:

md5_string_utf8(string)


Argumento Descripción
string La cadena para hash.


Devoluciones: cadena


Descripción

En criptografía, MD5 (Message-Digest algorithm 5) es una función hash criptográfica ampliamente utilizada con un valor hash de 128 bits y se ha empleado en una amplia variedad de aplicaciones de seguridad. También se usa comúnmente para verificar la integridad de archivos y cadenas. Esta función tomará una cadena de entrada UTF-8 (que tiene un número variable de bytes por carácter) y devolverá el hash hexadecimal MD5 de 32 caracteres que es exclusivo de esa cadena. De esta forma, puede generar una clave segura que puede almacenarse y usarse para verificar la integridad de la información que se envía (o recibe de) un servidor externo (por ejemplo).

NOTA: hay dos formatos para la codificación MD5, UTF-8 y Unicode. Ambos se proporcionan para facilitar la comunicación con diferentes configuraciones de servidor, pero la más común de usar es Unicode.


Ejemplo:

var hash, str;
str = json_encode(hiscore_map);
hash = md5_string_utf8(str);
ini_open("local.ini");
ini_write_string("info", "0", hash);
ini_close();
get[0] = http_post_string("http://www.MacSweeney Games.com/CatchTheHaggis?game_hiscores=" + string(global.game_id), str)

El código anterior codificará un ds_map en una cadena JSON. Luego se genera un hash MD5 y se almacena en un archivo ini para que luego se pueda usar para verificar la integridad del JSON en caso de que se reciba la misma información más adelante del servidor. El JSON luego se envía.
