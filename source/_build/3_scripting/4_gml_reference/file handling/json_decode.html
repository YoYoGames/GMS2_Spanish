json_decode

Sintaxis:

json_decode(string)


Argumento Descripción
string La cadena de formato JSON que está pasando a la función para decodificar


Devoluciones:

ds_map id or -1 if it fails


Descripción

JSON (JavaScript Object Notation) es un formato liviano de intercambio de datos que es fácil de leer y escribir, tanto para personas como para máquinas. Está construido sobre dos estructuras básicas:

  • Una colección de pares nombre / valor, llamada ds_map en GameMaker Studio 2 pero también conocido como "diccionario" u "objeto".
  • Una lista ordenada de valores, llamada ds_list en GameMaker Studio 2 pero esto también se puede llamar una "matriz" o "secuencia".

Con esta función, puede decodificar una pieza de JSON y convertirla en ds_map, listo para usar en GameMaker Studio 2. Si el JSON a decodificar requiere una jerarquía de listas y mapas dentro del ds_map central, estos también se decodifican y también se crean para usted, utilizando las siguientes reglas (tenga en cuenta que estas reglas se aplican únicamente a la estructura de nivel superior):

  • Json es un valor único: devuelve un ds_map con una sola entrada "predeterminada" que es el valor
  • Json es una matriz de objetos o valores - devuelve un ds_map con una sola entrada "default" que es una ds_list de los objetos o valores
  • Json es un objeto - devuelve un ds_map que tiene las entradas de objeto en él
NOTA: Al decodificar matrices, hay un mapa con la clave "predeterminada" SÓLO cuando una matriz es la estructura de nivel superior, y SÓLO para esa matriz de nivel superior. Las listas internas se decodifican directamente en ds_lists sin estar incluidas en un ds_map.

Normalmente sabría a qué teclas descifra JSON, pero si no, puede usar el ds_map_size, ds_map_find_first y ds_map_find_next funciones para analizar el mapa y obtener la información necesaria.



NOTA: GameMaker Studio 2 crea los mapas ds_maps necesarios y las listas del JSON, y para la limpieza solo necesita eliminar el mapa o lista de nivel superior y GameMaker Studio 2 eliminará automáticamente de la memoria todos los mapas y listas que se encuentran debajo.


Ejemplo:

var resultMap = json_decode(requestResult);
var list = ds_map_find_value(resultMap, "default");
var size = ds_list_size(list);
for (var n = 0; n < ds_list_size(list); n++;)
   {
   var map = ds_list_find_value(list, n);
   var curr = ds_map_find_first(map);
   while (is_string(curr))
      {
      global.Name[n] = ds_map_find_value(map, "name");
      curr = ds_map_find_next(map, curr);
      }
   }
ds_map_destroy(resultMap);

El código anterior decodificará una cadena JSON y lo analizará para generar una matriz global.
