load_csv

Sintaxis:

load_csv(filename)


Argumento Descripción
filename El nombre del archivo para abrir (como una cadena)


Devuelve: Índice de cuadrícula de DS (real)


Descripción

Esta función cargará un archivo de formato CSV y lo convertirá en una grilla de DS, devolviendo el valor único de ID para la grilla creada.


Ejemplo:

file_grid = load_csv("spreadsheet.csv");
var ww = ds_grid_get_width(file_grid);
var hh = ds_grid_get_height(file_grid);
var xx = 32;
var yy = 32;
for (var i = 0; i < ww; i++;)
    {
    for (var j = 0; j < hh; j++;)
        {
        draw_text(xx, yy, string(file_grid[# i, j]));
        yy += 32;
        }
    yy = 32;
    xx += 32;
    }

El código anterior abrirá el archivo CSV dado y almacenará la cuadrícula DS devuelta en la variable "file_grid". Esta grilla es luego analizada en un par de for lops y los contenidos dibujados a la pantalla.
