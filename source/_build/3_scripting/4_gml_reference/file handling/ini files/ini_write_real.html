ini_write_real


Descripción

Puede utilizar esta función para escribir un valor (numérico) en un archivo de datos ini. Los archivos Ini se dividen en secciones y luego cada sección se divide posteriormente en pares clave - valor. Así que un archivo ini típico se vería así:


Sintaxis:

ini_write_real(section, key, value);


Argumento Descripción
section La sección del.ini para escribir.
key La clave dentro de la sección correspondiente del.ini para escribir.
value El valor real para escribir en el destino correspondiente.


Devoluciones:

N/A


Ejemplo:

score = 1000;
ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
score2 = ini_read_real("save1", "Score", 0 );
ini_close();

Esto establecerá la puntuación en 1000, luego abrirá "savedata.ini" y escribirá este valor en "save1"> "Score". Luego, establecerá score2 en el valor real debajo de "save1"> "score" en él, 1000, y cerrará el.ini nuevamente. Si no hubiera encontrado este valor, score2 hubiera terminado establecido en 0.
