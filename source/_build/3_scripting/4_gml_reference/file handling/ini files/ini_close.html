ini_close

Sintaxis:

ini_close();


Devoluciones: cadena


Descripción

Esta función debe llamarse en el momento en que termine de leer o escribir en cualquier archivo ini abierto. Si no usa la función después de haber utilizado alguna de las funciones de escritura interna, entonces no se escribirá nada en el disco, ya que la información del archivo se guarda en la memoria hasta que se llame a esta función, lo que obliga a escribir. Si intentas abrir una ini sin haber cerrado previamente otra (o la misma), también obtendrás un error.

La función también devolverá una cadena con el archivo ini codificado en ella. Esta cadena se puede guardar en un servidor y / o usar de nuevo junto con la función ini_open_from_string para volver a crear el ini.


Ejemplo:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

Esto abrirá 'savedata.ini' y establecerá el puntaje en el valor bajo la sección "save1" con la clave "score" en él, luego cierre el.ini nuevamente. Si no hay ningún valor en "save1", "score" o no hay presente el archivo "savedata.ini", la puntuación se establecerá en 0 y se creará un nuevo archivo ini_file.
