ini_read_string

Sintaxis:

ini_read_string(section, key, default);


Argumento Descripción
section La sección de.ini para leer.
key La clave dentro de la sección relevante de.ini para leer.
default La cadena a devolver si una cadena no se encuentra en el lugar definido (o el archivo.ini no existe). Debe ser una cadena.


Devoluciones: cadena


Descripción

Puede usar esta función para leer una cadena (texto) de un archivo de datos ini. Los archivos Ini se dividen en secciones y luego cada sección se divide posteriormente en pares clave - valor. Entonces, un archivo ini típico se vería así:


Ejemplo:

ini_open( 'savedata.ini' );
global.name = ini_read_string( 'save1', 'Name', 'Player1' );
ini_close();

Esto abrirá 'savedata.ini' y establecerá global.name en la cadena bajo 'save1'> 'Name' en él, luego cerrará el.ini nuevamente. Si no hay una cadena debajo de 'guardar1'> 'Nombre', o si no hay un archivo 'savedata.ini' presente, global.name se establecerá en 'Player1'.
