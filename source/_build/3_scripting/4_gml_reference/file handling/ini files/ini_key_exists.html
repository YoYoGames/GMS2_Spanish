ini_key_exists


Descripción

Esta función verifica si existe una clave en el ini actualmente abierto. Esta no es una verificación necesaria para evitar errores, ya que, cuando no existe una clave, la lectura de una clave inexistente solo devolverá un valor predeterminado. Puede ser útil ver si un archivo ini ha guardado datos específicos y algunas otras cosas, sin embargo.


Sintaxis:

ini_key_exists(section, key);


Argumento Descripción
section La sección en el archivo.ini abierto para registrar una clave.
key La clave para comprobar.


Devoluciones:

N/A


Ejemplo:

ini_open("savedata.ini");
if !ini_key_exists("save1", "name")
   {
   global.name = "Player1";
   }
ini_close();

Esto establecerá la variable global.name en "Player1" si no existe una clave como "nombre".
