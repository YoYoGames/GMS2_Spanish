file_find_first

Sintaxis:

file_find_first(mask, attr);


Argumento Descripción
mask La máscara a usar para buscar.
attr El atributo de archivo específico a buscar.


Devoluciones: cadena


Descripción

Esta función devolverá el nombre del primer archivo que satisfaga la máscara y los atributos. Si no existe dicho archivo, se devuelve una cadena vacía. La máscara puede contener una ruta y puede contener comodines, por ejemplo 'C: \ temp \ * doc'. Los atributos proporcionan los archivos adicionales que desea ver, por lo que siempre se devuelven los archivos normales cuando satisfacen la máscara. Puede agregar las siguientes constantes para ver el tipo de archivos que desea (si no desea agregar ningún atributo, use 0):

  • fa_readonly - archivos de solo lectura
  • fa_hidden - archivos ocultos
  • fa_sysfile - archivos de sistema
  • fa_volumeid - archivos de id de volumen
  • fa_directory - directorios
  • fa_archive - archivos archivados


¡ADVERTENCIA! ¡Es posible que esta función no funcione como espera debido a que GameMaker Studio 2 está siendo protegido! Consulte la sección sobre Límites del sistema de archivos para obtener más información.

NOTA: Esta función no funcionará en absoluto en las plataformas de destino JS (Windows8, Tizen y HTML5), y los indicadores de filtro solo funcionarán en Windows y WinPhone (use 0 para todas las demás plataformas).


Ejemplo:

if directory_exists("\User Content")
   {
   file = file_find_first("\User Content\*.doc", fa_readonly);
   }

Esto verificará si el directorio especificado existe, si lo hace, vaya allí y devuelva el primer "solo lectura" doc archivo encontrado.
