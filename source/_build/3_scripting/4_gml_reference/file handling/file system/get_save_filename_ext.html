get_save_filename_ext

Sintaxis:

get_save_filename_ext(filter, fname, directory, caption);


Argumento Descripción
filter El filtro y / o nombre del tipo de archivo.
fname El nombre de archivo sugerido para usar.
directory El directorio para comenzar la búsqueda.
caption El título de la ventana.


Devuelve: Cadena.


Descripción

Esta función abre un diálogo y le pide al jugador un nombre de archivo para guardar con el filtro dado. El filtro tiene la forma " nombre1 | máscara1 | nombre2 | máscara2 |... ", donde la máscara puede contener las diferentes opciones con un punto y coma entre ellas y también puede usar un "*" para referirse a cualquier cadena. Por ejemplo: " bitmaps | *.bmp; *. Wmf ", le pediría al usuario que seleccione el nombre de un archivo de mapa de bits del sistema, pero solo aquellos que tienen las dos extensiones especificadas ( bmp y png ) esta función no guarda el archivo en sí! Solo devuelve una cadena con el nombre completo y la ruta del archivo. Si el usuario presiona " Cancelar ", se devuelve una cadena vacía "". También puede agregar un directorio para comenzar la búsqueda y un título que se mostrará en la parte superior de la ventana en lugar de la opción predeterminada.

Una cosa importante a tener en cuenta al usar esta función es que le concede ciertos permisos para ese archivo, durante la duración de su juego. Entonces, una vez que obtiene la ruta del archivo del usuario, puede acceder a ella una y otra vez sin tener que preguntar, y el permiso se aplica a la lectura y escritura por igual.

NOTA: esta es una función solo de Windows.


Ejemplo:

var file;
file = get_save_filename_ext("screenshot|*.png", "", working_directory, "Save a level here");
if file != ""
   {
   screen_save(file);
   }

El código anterior le pedirá al usuario que seleccione un archivo de nombre de archivo de imagen para guardarlo en el directorio de trabajo y luego verifique el nombre y la ruta del archivo devuelto. Si no es una cadena vacía, toma una captura de pantalla y la guarda usando el nombre de pila.
