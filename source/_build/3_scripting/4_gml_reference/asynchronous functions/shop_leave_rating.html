shop_leave_rating


Descripción

Esta función abre un diálogo dependiente del sistema operativo donde puede solicitar al usuario que publique una calificación o comentario en una página en particular. Puede definir el texto que aparecerá en el diálogo, así como el texto que desea que aparezca en los dos botones y la URL donde se debe publicar el comentario.

NOTA: esta función es solo para objetivos de Android e iOS.
¡IMPORTANTE! En iOS, cuando se usa TestFlight, en desarrollo se mostrará el mensaje pero no podrá hacer nada. Una vez que su aplicación está en la App Store, este botón se activa y el usuario tiene la opción de escribir una opinión o dejar una calificación. Tenga en cuenta que si distribuye el juego a través de TestFlight, esta ventana puede que ni siquiera se muestre.


Sintaxis:

shop_leave_rating(text, yes_string, no_string, url)

Argumento Descripción
text Texto que aparece en el diálogo.
yes_string Texto que aparece en el botón "sí".
no_string Texto que aparece en el botón "no".
url La URL completa donde se debe enviar el comentario.


Devoluciones:

N/A


Ejemplo:

if timer <= 0
   {
   shop_leave_rating("Thanks for playing the game! Why not leave a comment?", "Okay!", "Not Today!", "http://MacSweeney/comments");
   timer = 10000000;
   }
else
   {
   timer -= 1;
   }

El código anterior le pedirá al usuario que deje un comentario si la variable "temporizador" ha contado a 0.
