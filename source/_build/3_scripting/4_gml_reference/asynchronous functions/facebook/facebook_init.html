facebook_init


Sintaxis:

facebook_init(appKey)


Devoluciones:

N/A


Descripción

Antes de usar cualquiera de las funciones de Facebook, debe llamar a esta función para inicializar la API de Facebook y decirle a GameMaker Studio 2 que va a utilizar las funciones específicas de Facebook. Esto se debe hacer solo una vez en el juego y se puede hacer en cualquier momento, pero se recomienda llamar a esta función desde el principio antes que nada. Tenga en cuenta también que incluso si usa facebook_logout en algún punto de tu juego no necesitarás usar esta función nuevamente para iniciar sesión.


Ejemplo:

facebook_init();
var permissions = ds_list_create();
ds_list_add(permissions,"public_profile", "user_friends");
facebook_login(permissions);
ds_list_destroy(permissions);

El código anterior inicializa Facebook, luego crea un ds_list para mantener los permisos específicos que desea solicitar, y finalmente inicia sesión con estos permisos.
