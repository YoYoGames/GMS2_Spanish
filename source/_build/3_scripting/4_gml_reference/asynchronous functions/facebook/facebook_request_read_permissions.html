facebook_request_read_permissions


Sintaxis:

facebook_request_read_permissions(permissions)

Argumento Descripción
permissions Los permisos que quieres tener del usuario.


Devoluciones:

Real


Descripción

Con esta función puede solicitar permisos de lectura adicionales para Facebook. Estos permisos se agregan (como cadenas) a un archivo previamente creado ds_list que luego se usa en la función para enviar la solicitud. El usuario debe haber iniciado sesión correctamente (es decir, han llamado facebook_login ) y el facebook_status debe estar "AUTORIZADO" antes de llamar a esta función. el functon devuelve un valor ID único para la llamada de solicitud o -1 si la solicitud no se envió (por ejemplo, el usuario no ha iniciado sesión).

Esta función desencadenará un evento social asíncrono, que contiene el ds_map async_load. Este ds_map se completará con varios pares clave / valor para informar el éxito o el fracaso de la solicitud.


Ejemplo:

var permissions = ds_list_create();
if !facebook_check_permission("user_likes")
   {
   ds_list_add(permissions, "user_likes");
   }
if !facebook_check_permission("user_interests")
   {
   ds_list_add(permissions, "user_interests");
   }
if !ds_list_empty(permissions))
   {
   request_ID = facebook_request_read_permissions(permissions);
   }
ds_list_destroy(permissions);

El código anterior verifica algunos de los permisos de lectura de Facebook, y si el juego no los ha solicitado, se envía una solicitud.
