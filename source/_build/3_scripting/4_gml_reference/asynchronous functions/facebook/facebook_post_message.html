facebook_post_message


Sintaxis:

facebook_post_message(title, caption, message, picture_url, redirect_url, actionname, action_url)

Argumento Descripción
title El "nombre" del enlace.
caption El título del enlace (aparece debajo del nombre del enlace).
message El mensaje para incluir con la publicación.
picture_url Un enlace a la imagen incluida con esta publicación.
redirect_url El enlace adjunto a esta publicación.
actionname Una acción que se aplicará para esta publicación.
action_url El enlace para usar para la acción anterior.


Devoluciones:

N/A


Descripción

Esta función publicará un mensaje directo al feed del usuario (es decir, una publicación "/ me / feed" en la API gráfica de Facebook). Usted proporciona el nombre y el título, así como un mensaje y una imagen para incluir, luego la url a la que enlazar. Los dos últimos argumentos, "actionname" y "actionurl" son opcionales y pueden dejarse como cadenas vacías ("") o pueden establecerse en una de las acciones permitidas de Facebook, como comentar o agradecer, en cuyo caso también debe proporcione una URL adecuada (para obtener más información, vea Facebook Developers Api Post ).

NOTA: No se muestra el diálogo de Facebook para esta función ya que la publicación se envía "silenciosamente" sin interacción del usuario. Por este motivo, debe haber solicitado los permisos correctos al iniciar sesión ("publish_actions").

NOTA: El usuario debe haber iniciado sesión para que funcione la publicación del mensaje.


Ejemplo:

if mouse_check_button_pressed(mb_left)
   {
   facebook_post_message("Catch The Haggis", "MacSweeneygames.com", "I just got a hi-score playing Catch The Haggis!", "http://MacSweeneygames.com/Clown.jpg", "http://MacSweeneygames.com/CatchTheHaggis.html", "", "");
   }

El código anterior publicará de forma silenciosa un mensaje en el feed de pared de los usuarios.
