achievement_show_achievements


Sintaxis:

achievement_show_achievements()


Devoluciones:

N/A


Descripción

Esta función abrirá la página de logros para la plataforma elegida. Tenga en cuenta que esta es una función asincrónica, es decir: su juego continuará ejecutándose en segundo plano mientras se muestra la página de logros. Como tal, debes tener cuidado donde lo usas y asegúrate de pausar el juego o solo permitir que se muestre en áreas de tu juego donde no interfiera con el juego.


Ejemplo:

if achievement_available()
   {
   global.Pause = true;
   instance_create_layer(0, 0, "Controllers", obj_Pause);
   achievement_show_achievements();
   }

El código anterior comprueba si la tabla de clasificación y el servicio de logros elegidos están disponibles y, de ser así, establece una variable global y crea una instancia de "pausa" del controlador antes de llamar a la función para mostrar los logros en la pantalla.
