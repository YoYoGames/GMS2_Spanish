achievement_load_progress


Sintaxis:

achievement_load_progress()


Devoluciones:

N/A


Descripción

Esta función enviará una solicitud al servidor para obtener información sobre todos los logros disponibles. Se desencadenará un evento asincrónico social de devolución de llamada que contiene el async_load mapa poblado con los pares clave / valor relevantes. La clave de identificación de esto ds_map se usa para identificar la devolución de llamada correcta (puede haber más de una función de activación para cualquier evento asíncrono dado), y se emparejará con la constante achievement_achievement_info así como una cantidad de otros pares clave / valor para cada jugador. Los contenidos exactos del mapa se muestran a continuación:

  1. " id " - Para esta función, debe ser achievement_achievement_info.

  2. " numentries ": la cantidad de logros en la lista.

  3. " AchN ": el nombre del logro, donde "N" es un valor entero que corresponde a su lugar en la lista de entradas.

  4. " AchCompN ": cuán completo es el logro "N" como un valor porcentual de 0 a 100 (una cadena).
NOTA: esta función es solo para iOS.


Ejemplo extendido:

El siguiente código solicitaría el progreso del rendimiento cuando el usuario está en un dispositivo con iOS:

if os_type == os_ios
   {
   achievement_load_progress();
   }

Esto enviará una solicitud de información sobre los logros y generará una devolución de llamada asincrónica con el async_load ds_map que contiene los siguientes datos:

var ident = ds_map_find_value(async_load, "id");
if ident == achievement_achievement_info
   {
   var numentries = ds_map_find_value(async_load, "numentries");
   for(var i = 0; i < numentries; i++;)
      {
      ach_id[i] = ds_map_find_value(async_load, "Ach" + string(i));
      comp[i] = ds_map_find_value(async_load, "AchComp" + string(i));
      }
   }

El código anterior verifica el resultado ds_map en el evento asincrónico social y si su "id" coincide con la constante que se está verificando, recorre las entradas en el mapa para cada uno de los logros y almacena su nombre y porcentaje de finalización.
