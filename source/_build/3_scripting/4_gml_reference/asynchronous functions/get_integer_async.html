get_integer_async


Descripción

Esta función abre una ventana y muestra un mensaje así como también un espacio para que el usuario ingrese un valor (que contendrá el valor predeterminado provisto para comenzar). Esta es una función asíncrona, y como tal, GameMaker Studio 2 no bloquea el dispositivo que se está ejecutando mientras se espera la respuesta, sino que más bien continúa ejecutando eventos de forma normal. Una vez que el usuario ha escrito su cadena y presionado el botón "Aceptar", se desencadena un evento de diálogo asíncrono que, durante la duración de ese solo evento, tendrá un ds_map almacenado en la variable async_load.

Este mapa contendrá las tres claves, " id ", " status " y " value ". " id " es el valor que devolvió la función cuando se llamó, el " status " será cualquiera true para que se presione el botón "Aceptar", o false si el mensaje fue cancelado (donde corresponda, ya que no todas las plataformas de destino permiten cancelar los mensajes). Finalmente, " value " devolverá el número entero que ingresó el usuario (o 0 si no se proporcionó ninguno).


Sintaxis:

get_integer_async(string, default);

Argumento Descripción
String El mensaje para mostrar en el cuadro de diálogo.
default El valor predeterminado


Devoluciones:

Real


Ejemplo extendido:

El evento de prensa del mouse izquierdo (por ejemplo) del objeto que muestra el mensaje tendrá el siguiente código:

msg = get_integer_async("How old are you?", 0);

Lo anterior mostrará un mensaje solicitando que el usuario ingrese una cadena y presione "Aceptar". La id de la función se almacena en la variable "msg" y se usará en el evento Dialogs asíncrono como se muestra a continuación:

var i_d = ds_map_find_value(async_load, "id");
if i_d == msg
   {
   if ds_map_find_value(async_load, "status")
      {
      global.Age = ds_map_find_value(async_load, "value");
      }
   }

El código anterior verifica la clave " id " de la respuesta ds_map contra el valor almacenado en la variable "msg". Si son iguales, verifica si se presionó "Okay" (en lugar de que la ventana se cierre / cancela) y si vuelve true luego establece una variable global a partir del número entero devuelto.
