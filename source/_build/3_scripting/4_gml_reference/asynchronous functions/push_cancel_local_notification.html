push_cancel_local_notification


Descripción

Esta función cancela la notificación de inserción dada. Puede obtener la identificación de la notificación para cancelar usando las funciones push_get_first_local_notification() y push_get_next_local_notification() y la función regresará true en el éxito o false de lo contrario (por ejemplo, si la notificación no existe o ya se ha activado).

NOTA: Esta función está limitada a los módulos de destino de iOS y Android.


Sintaxis:

push_cancel_local_notification(ID);

Argumento Descripción
ID El ID de la notificación para cancelar.


Devoluciones:

Real


Ejemplo:

var map = ds_map_create();
var ntf = push_get_first_local_notification(map);
while(ntf >= 0)
   {
   var data = ds_map_find_value(map, "data");
   if data == "Daily_Reward"
      {
      push_cancel_local_notification(ntf);
      }
   ntf = push_get_next_local_notification(map);
   }
ds_map_destroy(map);

El código anterior crea un ds_map y luego solicita datos para la primera notificación disponible en la cola. Si existe una notificación, el mapa se rellena y el código luego verifica la clave de "datos" para una cadena específica. Si se encuentra la cadena, la notificación se cancela y se mueve para verificar la siguiente notificación hasta que se haya verificado todo en la cola.
