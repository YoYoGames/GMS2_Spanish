cloud_string_save


Descripción

Esta función enviará una cadena al servicio en la nube elegido para el almacenamiento. La función devolverá un valor único de identificación que luego debería usarse en el evento asincrónico apropiado para identificar el ds_map que se devuelve como una "devolución de llamada" desde el servicio en la nube. La cadena debe contener toda la información que necesita guardar para su juego, ya que solo puede almacenar una única "acumulación de datos" en la nube, y al ejecutar esta función nuevamente se sobrescribirán los valores previamente almacenados (como lo hará el uso del cloud_file_save función). La descripción debe ser una breve cadena de información que describa el guardado, por ejemplo: "Nivel2, Etapa2".

Para obtener más información sobre los datos asíncronos devueltos, consulte la función cloud_synchronise.


Sintaxis:

cloud_string_save(string, description);

Argumento Descripción
string La cadena de datos que se cargará.
description Una breve descripción de los datos que se almacenan.


Devoluciones:

Real


Ejemplo:

var t_str = "";
for (var i = 0; i < 10; i++;)
   {
   t_str += string(global.Highscore[i]) + "|"
   }
save_check = cloud_string_save(t_str, "Current Highscores");
var file = file_text_open_write("Highscores.txt");
file_text_write_string(file, t_str);
file_text_close(file);

El código anterior crea una cadena a partir de los valores almacenados en la matriz global "Highscores" y luego escribe esta cadena en el servicio en la nube para su almacenamiento, y también la escribe en un archivo para almacenamiento local.
