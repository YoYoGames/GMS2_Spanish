número_instancia


Descripción

Con esta función puede averiguar cuántas instancias activas del objeto especificado existe en la sala. Al verificar el uso de esta función, si el objeto es un elemento primario, todos los objetos secundarios también se incluirán en el valor de retorno, y también tenga en cuenta que las instancias que se han desactivado con el instance_deactivate Las funciones no se incluirán en esta comprobación.


Sintaxis:

instance_number(obj);


Argumentos:

Argumento Descripción
obj El objeto para sumar el número de instancias de.


Devoluciones:

Real


Ejemplo:

if instance_number(object_index) < 50    {
   instance_create_layer(random(room_width), random(room_height), "Instances", object_index);
   }

El código anterior verificará el número de instancias que se crean del mismo objeto que la instancia actual y luego, si hay menos de 50, cree otro en una posición aleatoria dentro de la sala.
