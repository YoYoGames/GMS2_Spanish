instance_position_list


Descripción

Con esta función puede verificar una posición para una colisión con todas las instancias de otro objeto. Cuando usa esto, está marcando un solo punto en la sala para todas las instancias de un objeto, y la verificación se realizará contra el cuadro delimitador de las instancias o contra la máscara de las instancias si ese objeto tiene marcadas las colisiones precisas. La función llenará una lista de DS (creada previamente) con el único id los valores de todas las instancias del objeto que se considera que están en colisión con la instancia que llama, y ​​tiene la opción de ordenar la lista en función de la distancia desde la posición que se verifica hasta el origen de las instancias encontradas en colisión. Tenga en cuenta que la función también acepta la palabra clave especial all, en cuyo caso se listarán todas las instancias encontradas en colisión. La función devuelve el número de instancias encontradas, o 0 si no se encuentra ninguna.


Sintaxis:

instance_position_list(x, y, obj, list, ordered);

Argumento Descripción
x La posición x para comprobar instancias.
y La posición y para comprobar las instancias.
obj El objeto para comprobar instancias de.
list La lista de DS que se utilizará para almacenar los ID de las instancias en colisión.
ordered Si la lista debe ser ordenada por distancia ( true ) o no ( false ).


Devoluciones:

Int (The number of instances found to be in collision)


Ejemplo:

var _list = ds_list_create();
var _num = instance_position_list(mouse_x, mouse_y, all, _list, false);
if _num > 0
    {
    for (var i = 0; i < _num; ++i;)
        {
        instance_destroy(_list[| i]);
        }
    }
ds_list_destroy(_list);

El código anterior comprobará una colisión con todas las instancias encontradas en la posición del mouse. Se agregarán a una lista de DS, que luego se repite para destruir cada una de las instancias en colisión.
