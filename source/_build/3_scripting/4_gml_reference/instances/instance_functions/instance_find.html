instance_find


Descripción

Todas las instancias tienen un identificador único ( id ) que puede usarse para modificarlos y manipularlos mientras se está ejecutando un juego, pero es posible que no siempre sepas cuál es el id. de una instancia específica, por lo que esta función puede ser útil ya que puedes usarla para recorrer todas ellas y encontrar qué necesitas. Usted especifica el objeto para el que desea encontrar la instancia y un número, y si hay una instancia en esa posición en la lista de instancias, entonces la función devuelve el id de esa instancia, y si no, devuelve la palabra clave especial noone. También puede usar la palabra clave all para iterar a través de todas las instancias en una sala, así como un objeto principal para iterar a través de todas las instancias que forman parte de esa jerarquía principal / secundaria e incluso puede especificar una instancia en sí misma (si tiene su id) como un control para ver si realmente existe en la sala actual. Tenga en cuenta que como las instancias se ordenan de forma arbitraria, no hay un orden específico de cómo esta función verifica las instancias y cualquier instancia puede estar en cualquier posición.

El valor máximo para "n" en esta función sería

  • Para la palabra clave all: instance_count - 1
  • Para un índice de objeto: instance_number(OBJ) - 1


Sintaxis:

instance_find(obj, n);


Argumentos:

Argumento Descripción
obj El objeto para encontrar la enésima instancia de
n El número de la instancia para encontrar.


Devoluciones:

Real (either an instance ID value or the keyword noone)


Ejemplo:

var i;
for (i = 0; i < instance_number(obj_Enemy); i += 1)
   {
   enemy[i] = instance_find(obj_Enemy,i);
   }

El código anterior usará un for iterar a través de todas las instancias de "obj_Enemy" y almacenar su id en la matriz "enemigo []".
