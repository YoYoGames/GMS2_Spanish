instance_copy


Descripción

Con esta función, puede "clonar" una instancia, ya que creará una nueva versión de la instancia que ejecuta el código en su misma posición. El argumento "perf" se utiliza para indicar a esta nueva instancia que realice el evento create o no. Esta función devuelve el id de la nueva instancia que luego puede almacenarse en una variable o utilizarse para acceder a esa instancia.

NOTA: Si decide no realizar el evento create, puede encontrar errores si la instancia depende de cualquier variable inicializada en este evento.


Sintaxis:

instance_copy(perf);


Argumentos:

Argumento Descripción
perf Si se debe realizar el evento Create de la nueva instancia (verdadero) o no (falso).


Devoluciones:

Real (instance ID value)


Ejemplo:

var inst;
inst = instance_number(object_index);
if inst < 10
   {
   instance_copy(true);
   }

El código anterior crea una variable local y la usa para almacenar el número de instancias del objeto que ejecuta el código en la sala. Si el número es menor que 10, la instancia hace una copia de sí mismo.
