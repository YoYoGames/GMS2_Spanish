instance_destroy


Descripción

Usted llama a esta función siempre que desee "destruir" una instancia, normalmente desencadenando un evento de destrucción y también un evento de limpieza. Esto lo eliminará de la sala hasta que se reinicie la sala (a menos que la habitación sea persistente). Llamar a la función sin argumentos simplemente destruirá la instancia que está actualmente en alcance y ejecutando el código, pero puede proporcionar un argumento "id" opcional y dirigirse a una instancia específica utilizando el valor de ID de instancia, o puede orientar todas las instancias de un objeto particular mediante el uso de una object_index. Por ejemplo:

instance_destroy(other); // destroy the other instance in a Collision Event
instance_destroy(obj_Bullet); // destroy ALL instances of the object "obj_Bullet"

La segunda bandera opcional le permite "desactivar" el evento destruir para la instancia que se destruye. Por defecto, siempre se realizará el Destruir evento, pero si configura este indicador para false entonces puede hacer que la instancia se destruya a sí misma y omita realizar ese evento.

¡IMPORTANTE! Esto omitirá el evento destruir, pero el evento de limpieza seguirá siendo llamado.


Vale la pena señalar que cuando destruyes una instancia, su evento de destrucción se invoca inmediatamente después del código o acción que llama a la destrucción. En segundo lugar, aunque se realiza el evento de destrucción, la instancia no se elimina inmediatamente del juego y continuará ejecutando el código contenido en el evento actual. Solo cuando el evento actual termine se eliminará del juego.

Entonces, si tiene, por ejemplo, este código:

if hp <= 0 instance_destroy();
score += 10;

La variable "puntuación" se incrementará a pesar de que el instance_destroy se ha llamado a la función, y la instancia finalmente se eliminará de tu juego al final del evento. Tenga cuidado con esto, como si (por ejemplo) hubiera creado un recurso dinámico para la instancia, como una estructura de datos, y luego hubiera destruido ese recurso en el evento de destrucción, si hay alguna referencia a él después de la función o acción de destrucción. se ha realizado, entonces obtendrás un error de "recurso desconocido", ya que el evento de destrucción lo eliminó del juego.


Sintaxis:

instance_destroy([id, execute_event_flag]);


Argumentos:

Argumento Descripción
id ID de instancia u object_index para destruir (opcional, por defecto es la instancia de llamada)
execute_event_flag Ajustado a true o false para realizar el evento destruir o no (opcional, por defecto es true )


Devoluciones:

N/A


Ejemplo:

if bbox_right < 0 || bbox_left > room_width || bbox_bottom < 0 || bbox_top > room_height
    {
    instance_destroy(id, false);
    }

El código anterior destruirá la instancia que ejecuta el código sin llamar al Destroy Event si la instancia está fuera de los límites de la sala.
