instance_exists


Descripción

Esta función se puede usar de dos maneras dependiendo de lo que desee verificar. Puede darle un object_index para verificar, en cuyo caso esta función regresará true si existen instancias activas del objeto especificado en la sala actual, o también puede proporcionarle una ID de instancia, en cuyo caso se devolverá esta función true si esa instancia específica existe y está activa en la sala actual.

Tenga en cuenta que esta función no tiene en cuenta aquellas instancias que se han desactivado utilizando el instance_deactivate funciones.


Sintaxis:

instance_exists(obj);


Argumentos:

Argumento Descripción
obj El objeto o instancia para verificar la existencia de.


Devoluciones:

Boolean


Ejemplo:

if !instance_exists(obj_Enemy)
   {
   score += 200;
   room_goto(rm_hiscores);
   }

El código anterior comprueba si existen instancias del objeto "obj_Enemy" y si no se agrega a la variable "puntuación" y cambia de sala.
