instance_create_depth


Descripción

Con esta función puede crear una nueva instancia del objeto especificado en cualquier punto dado dentro de la sala y en la profundidad especificada. La profundidad puede ser cualquier valor, mientras menor sea la profundidad, más "cerca" de la cámara se dibujarán las cosas y mayor será la profundidad cuanto más alejada, por lo que se dibujará una instancia a profundidad -200 sobre una instancia a profundidad +300 ( por ejemplo). Tenga en cuenta que esta función creará una capa de habitación para la instancia, ya que todas las instancias deben estar en una capa de la sala, pero dado que se trata de una capa administrada (es decir: no una que haya creado mediante código o en la sala, pero uno que GameMaker Studio 2 ha creado automáticamente) no puede acceder a él, y el layer la variable de instancia devolverá -1.

¡IMPORTANTE! Hay una profundidad de capa mínima y máxima de -16000 a 16000. Cualquier cosa colocada en una capa fuera de ese rango no se dibujará, aunque todos los eventos seguirán ejecutándose de forma normal.

Esta función devuelve el id de la nueva instancia que luego puede almacenarse en una variable o utilizarse para acceder a esa instancia. Tenga en cuenta que esta función también llamará al evento Crear de la instancia que se está creando antes de continuar con el código o las acciones para el evento que llamó a la función.


Sintaxis:

instance_create_depth(x, y, depth, obj);


Argumentos:

Argumento Descripción
x La posición x en la que se creará el objeto
y La posición y en la que se creará el objeto en
depth La profundidad para asignar la instancia creada a
obj El índice de objeto del objeto para crear una instancia de


Devoluciones:

Real (instance ID value)


Ejemplo:

var inst;
inst = instance_create_depth(x, y, -10000, obj_Bullet);
with (inst)
   {
   speed = other.shoot_speed;
   direction = other.image_angle;
   }

El código anterior crea una nueva instancia del objeto "obj_Bullet" y almacena el ID de instancia en una variable. Esta variable se usa para asignar speed y direction a la nueva instancia.
