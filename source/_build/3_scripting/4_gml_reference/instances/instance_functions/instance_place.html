instance_place


Descripción

Con esta función puede verificar una posición para una colisión con otra instancia o todas las instancias de un objeto utilizando la máscara de colisión de la instancia que ejecuta el código para la verificación. Cuando utilizas esto, efectivamente le pides a GameMaker Studio 2 que mueva la instancia a la nueva posición, verifique una colisión, retroceda y le diga si se encontró una colisión o no. Esto funcionará para colisiones precisas, pero solo si tanto la instancia como el objeto que se está comprobando tienen máscaras de colisión precisas, de lo contrario solo se aplicarán colisiones de cuadro delimitadoras. esta función devolverá la instancia única id del objeto que se colisionó, pero si eso no es necesario, es un poco más rápido para usar la función place_meeting. Esta función también acepta las palabras clave especiales all y other y devolverá la palabra clave noone si no se produce una colisión, o el valor único de ID de instancia de la instancia encontrada si se produce una colisión.

Tenga en cuenta que las coordenadas x / y dadas se redondearán al número entero más cercano antes de realizar la verificación, por lo tanto, si esto no es lo que necesita (o si ha estado utilizando un producto heredado de GameMaker ), debe floor las coordenadas en el cheque: instance_place(floor(x), floor(y), obj_Enemy).


Sintaxis:

instance_place(x, y, obj);


Argumentos:

Argumento Descripción
x La posición x para verificar instancias.
y La posición y para verificar instancias.
obj El objeto para verificar si hay instancias de.


Devoluciones:

Real (either an instance ID value or the keyword noone)


Ejemplo:

var inst;
inst = instance_place(x, y, obj_Enemy);
if inst != noone
   {    hp -= inst.dmg;
   with (inst) instance_destroy();
   }

El código anterior verificará si hay una colisión con instancias de "obj_Enemy" y si hay una, reducirá la variable "hp" en la cantidad almacenada en la variable "dmg" de la instancia en colisión y luego destruirá la instancia en colisión.
