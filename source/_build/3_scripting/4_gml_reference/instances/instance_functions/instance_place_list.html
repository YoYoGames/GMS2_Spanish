instance_place_list


Descripción

Con esta función puede verificar una posición para una colisión con todas las instancias de un objeto usando la máscara de colisión de la instancia que ejecuta el código para la verificación. Cuando usa esto, le está pidiendo efectivamente a GameMaker Studio 2 que mueva la instancia a la nueva posición, verifique si hay una colisión, luego retroceda y le diga si se encontró una colisión o no. Esto funcionará para colisiones precisas, pero solo si tanto la instancia como las instancias del objeto que se está verificando tienen máscaras de colisión precisas seleccionadas, de lo contrario, solo se aplicarán colisiones de recuadros de límites. La función llenará una lista de DS (creada previamente) con el único id los valores de todas las instancias del objeto que se considera que están en colisión con la instancia que llama, y ​​tiene la opción de ordenar la lista en función de la distancia desde el origen de la instancia que realiza la comprobación hasta el origen de las instancias que se encuentran en colision Tenga en cuenta que la función también acepta la palabra clave especial all, en cuyo caso se listarán todas las instancias encontradas en colisión. La función devuelve el número de instancias encontradas, o 0 si no se encuentra ninguna.

Tenga en cuenta que las coordenadas x / y proporcionadas se redondearán al número entero más cercano antes de que se realice la comprobación, por lo que si esto no es lo que necesita (o si ha estado utilizando un producto de GameMaker heredado), debería floor Las coordenadas en el cheque: instance_place_list(floor(x), floor(y), obj_Enemy, list, true).


Sintaxis:

instance_place_list(x, y, obj, list, ordered);

Argumento Descripción
x La posición x para comprobar instancias.
y La posición y para comprobar las instancias.
obj El objeto para comprobar instancias de.
list La lista de DS que se utilizará para almacenar los ID de las instancias en colisión.
ordered Si las instancias deben ser ordenadas por distancia desde el origen o no.


Devoluciones:

Int (The number of instances found to be in collision)


Ejemplo:

var _list = ds_list_create();
var _num = instance_place_list(x, y, obj_Enemy, _list, false);
if _num > 0
    {
    for (var i = 0; i < _num; ++i;)
        {
        instance_destroy(_list[| i]);
        }
    }
ds_list_destroy(_list);

El código anterior verificará una colisión con todas las instancias encontradas en la posición de las instancias de llamada. Se agregarán a una lista de DS, que luego se repite para destruir cada una de las instancias en colisión.
