instance_id


Descripción

Esta matriz de solo lectura contiene todos los id s de cada instancia activa dentro de la habitación. Esto significa que si ha utilizado cualquiera de los Instance Deactivate funciones esas instancias que se han desactivado no se incluirán en esta matriz (si ha utilizado previamente un valor de esta matriz, ahora devolverá la palabra clave noone ).


Sintaxis:

instance_id[num]);


Devoluciones:

Real (instance ID value)


Ejemplo:

for (var i = 0; i < instance_count; i ++;)
   {
   with (instance_id[i]) speed += 0.1;
   }

El código anterior recorrerá todas las instancias dentro de la habitación y agregará 0.1 a su velocidad.
