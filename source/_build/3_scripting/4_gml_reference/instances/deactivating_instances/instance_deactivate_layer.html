instance_deactivate_layer


Sintaxis:

instance_deactivate_layer(obj);


Argumento Descripción
layer La cadena de ID de capa (o valor) que se utilizará


Devoluciones: N / A


Descripción

Con esta función, puede desactivar todas las instancias asignadas a una capa específica. Debe proporcionar la Id. De capa, que puede ser el nombre de la capa tal como está escrita en el editor de código (como una cadena) o el valor de ID de capa real que devuelve el layer_create, y tenga en cuenta que solo puede desactivar capas de instancia con esta función. Tenga en cuenta que si ha desactivado una capa que tiene instancias de objetos señalados como Persistentes, entonces deberá reactivar la capa nuevamente con la función instance_activate_layer antes de cambiar de habitación, de lo contrario, las instancias persistentes en la capa no se transferirán y se descartarán.

NOTA: Si desactiva una instancia en el inicio de sala (es decir, desde el código de creación de sala o desde una instancia crea evento de una instancia dentro de la sala) todas las instancias que se colocan dentro de la sala desde el editor de sala seguirán ejecutando su evento de creación antes de ser desactivado

ADVERTENCIA: la desactivación de instancias que tienen habilitada la física NO evitará que sus dispositivos interactúen dentro de la simulación física. Para eso debes establecer su phy_active variable a true o false a medida que activa / desactiva las instancias.


Ejemplo:

instance_deactivate_layer("Enemy Layer");
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

El código anterior desactiva todas las instancias asignadas a la capa "Enemy_Layer" y luego activa una región dentro de la sala.
