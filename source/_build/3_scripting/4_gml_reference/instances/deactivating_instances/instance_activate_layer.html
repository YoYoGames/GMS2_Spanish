instance_activate_layer


Sintaxis:

instance_activate_layer(layer_id);


Argumento Descripción
layer_id La cadena de ID de capa (o valor) que se utilizará


Devoluciones: N / A


Descripción

Con esta función, puede activar una capa que ha sido desactivada previamente. Debe proporcionar la Id. De capa, que puede ser el nombre de la capa tal como está escrita en el editor de código (como una cadena) o el valor de ID de capa real que devuelve el layer_create y todas las instancias desactivadas en esa capa se activarán una vez más. Tenga en cuenta que si ha desactivado una capa que tiene instancias persistentes, deberá reactivar la capa nuevamente con esta función antes de cambiar de sala; de lo contrario, las instancias persistentes no se transferirán y se descartarán.


Ejemplo:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
instance_deactivate_region(view_xview[0] - 64, view_yview - 64, _vx + 128, _vy + 128, false, false);
instance_activate_layer("Player_Layer");

El código anterior activa todas las instancias dentro de la habitación y luego desactiva aquellas que están fuera de los límites de la vista actual de la cámara, a excepción de las instancias en la capa "Player_Layer" que se vuelven a activar al final.
