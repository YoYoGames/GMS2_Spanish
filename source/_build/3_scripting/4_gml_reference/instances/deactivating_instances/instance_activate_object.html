instance_activate_object


Sintaxis:

instance_activate_object(obj);


Argumento Descripción
obj El objeto o instancia para activar (la palabra clave all también se puede usar).


Devoluciones: N / A


Descripción

Con esta función puede activar una sola instancia o todas las instancias de un objeto específico de todas las que se han desactivado previamente. Tenga en cuenta que si ha desactivado una instancia u objeto que se ha marcado como persistente, deberá reactivarlo nuevamente con esta función antes de cambiar de sala; de lo contrario, no se transferirá y se descartará en su lugar.


Ejemplo:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false);
instance_activate_object(obj_Lights);

El código anterior activa todas las instancias dentro de la habitación y luego desactiva aquellas que están fuera de los límites de la vista actual de la cámara, a excepción del objeto "obj_Lights" que se reactivan nuevamente al final.
