Puntuación


Sintaxis:

score;


Devoluciones:

Real


Descripción

Esta construida en instancia de variable se utiliza para contener un valor numérico para la puntuación de objeto (tenga en cuenta que esto no es necesariamente siempre el caso - sólo porque se llama "puntuación", no significa que tiene que ser utilizado para la puntuación que... puede contener cualquier valor real o entero que desee para su juego).


Ejemplo:

if place_meeting(x, y, obj_Bullet)
   {
   score += 10;
   instance_destroy();
   }

El código anterior verifica la instancia actual para ver si hay una colisión con la instancia del objeto indexado en "obj_Bullet", y si la hay, agrega 10 a la puntuación y luego se destruye.
