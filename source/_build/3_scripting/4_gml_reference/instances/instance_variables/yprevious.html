anterior


Descripción

Esta variable incorporada devuelve la posición y anterior para la instancia. Esta variable se establecerá en la posición x actual justo antes del inicio del evento del paso de inicio, pero también se puede configurar a través del código en cualquier momento, lo que significa que puede asignarle su propio valor personalizado si fuera necesario.


Sintaxis:

yprevious;


Devoluciones:

Real


Ejemplo:

if x != xprevious || y != yprevious
   {
   moved = true;
   }

El código anterior verifica si hay alguna diferencia entre la xprevious y yprevious valores y los valores actuales de x y y. Si existe establece una variable en verdadero.
