bbox_bottom


Descripción

Esta variable de solo lectura devuelve la posición y (dentro de la sala) de la parte inferior del cuadro delimitador para la instancia, donde el cuadro delimitador está definido por el ancho y alto máximo de la máscara para la instancia (tal como lo establece el sprite o el índice de máscara). Incluso cuando un sprite tiene una máscara de colisión precisa, el cuadro delimitador existe y se usa para ciertas cosas, por lo que puede usar esta variable para encontrarlo. Tenga en cuenta que cuando la instancia no tiene sprite asignado, bbox_ las variables siempre devolverán 1.


Sintaxis:

bbox_bottom;


Devoluciones:

Real


Ejemplo:

if bbox_bottom > room_height
   {
   y = room_height - sprite_yoffset;
   }

El código anterior comprueba si el cuadro delimitador de la instancia está fuera de la sala y, si lo está, establece la instancia en una nueva posición.
