bbox_right


Descripción

Esta variable de solo lectura devuelve la posición dentro de la sala (a lo largo del eje x) del lado derecho del cuadro delimitador para la instancia, donde el cuadro delimitador está definido por el ancho y alto máximos de la máscara para la instancia (como establecido por el sprite o por el índice de máscara). Incluso cuando un sprite tiene una máscara de colisión precisa, el cuadro delimitador existe y se usa para ciertas cosas, por lo que puede usar esta variable para encontrarlo. Tenga en cuenta que cuando la instancia no tiene sprite asignado, bbox_ las variables siempre devolverán 1.


Sintaxis:

bbox_right;


Devoluciones:

Real


Ejemplo:

if bbox_right > room_width
   {
   x = room_width - sprite_xoffset;
   }

El código anterior comprueba si el cuadro delimitador de la instancia está fuera de la sala y, si lo está, establece la instancia en una nueva posición.
