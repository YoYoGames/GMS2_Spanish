image_alpha


Descripción

Esta variable se usa para obtener o establecer el valor alfa para el sprite. Alpha siempre se calcula como un valor entre 0 y 1, donde 0 es completamente transparente y 1 es completamente opaco. Tenga en cuenta que los cambios en esta variable sea visible, la instancia debe tener ya sea ningún caso sorteo (y por lo GameMaker Studio 2 por defecto dibujar el sprite) o ser dibujado usando una de las funciones de dibujo extendidos como draw_self() o draw_sprite_ext().

NOTA: en las plataformas HTML5 y Windows8 (JavaScript), esta variable no está sujeta. Por lo tanto, si tiene un valor mayor que uno, se "ajustará" de nuevo a 0, dando resultados de parpadeo. Del mismo modo, si usted está por debajo de 0, el alfa se "ajustará" nuevamente a 1. Puede usar la función clamp para esto.


Sintaxis:

image_alpha;


Devoluciones:

Real (between 0 and 1)


Ejemplo:

image_alpha = clamp(image_alpha - 0.01, 0, 1);

El código anterior reducirá lentamente la imagen_alpha hasta que llegue a 0.
