X


Descripción

El valor x de una instancia es la posición horizontal en la sala actual, medida en píxeles. Este valor puede ser 0, positivo o negativo, donde 0 es el lado izquierdo de la sala y moverse hacia la derecha aumenta x, moverse hacia la izquierda disminuye x (un valor negativo para x significa que la instancia se ha salido del lado izquierdo de la sala ) Incluso puede dar xa un valor real como 12.345, pero como no puede obtener.345 de un píxel, la instancia puede no parecer haberse movido, aunque GameMaker Studio 2 mantendrá y usará estos valores internamente y son perfectamente válidos.

Al establecer los valores x (y y) puede hacer que el objeto salte por la habitación hasta la posición que elija, o puede sumarlos y restarlos en cantidades más pequeñas para dar la ilusión de movimiento sin utilizar realmente la función incorporada. speed y direction funciones. Esto debe tenerse en cuenta al hacer sus juegos, ya que a menudo un tipo de movimiento se presta a un estilo particular de juego más que otro.


Sintaxis:

x;


Devoluciones:

Real


Ejemplo:

if keyboard_check(vk_left) x += 5;
if keyboard_check(vk_right) x -= 5;

El código anterior usará las teclas de flecha para sumar o restar a la posición x y así crear la ilusión de movimiento.
