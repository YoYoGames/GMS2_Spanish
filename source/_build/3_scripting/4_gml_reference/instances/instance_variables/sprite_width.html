sprite_width


Descripción

Esta variable de solo lectura devuelve el ancho del sprite que se ha asignado a la instancia. Este ancho se devuelve en píxeles y dependerá de la image_xscale. Si necesita el ancho sin escala, debe usar sprite_get_width.

Para funciones relacionadas con sprites, mira aquí.


Sintaxis:

sprite_width;


Devoluciones:

Real


Ejemplo:

if sprite_width != sprite_get_width(sprite_index)
   {
   image_xscale = 1;
   }

El código anterior comprueba el ancho del sprite asignado a la instancia que ejecuta el código con respecto al ancho del recurso de sprite y, si no es el mismo, restablece el image_xscale a 1,
