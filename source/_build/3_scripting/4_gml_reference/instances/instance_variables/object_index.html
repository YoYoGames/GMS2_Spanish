object_index


Descripción

Esta variable de solo lectura devuelve el índice del objeto desde el que se creó la instancia. Este no es el mismo que el nombre del objeto, que es una cadena y se puede encontrar usando object_get_name, ya que esta función devuelve el número de índice, que es un valor único que GameMaker Studio 2 asigna a cada objeto en el momento de la creación.


Sintaxis:

object_index;


Devoluciones:

Object Index Value (Real)


Ejemplo:

obj_name = object_get_name(object_index);

El código anterior utilizará el object_index para buscar el nombre del objeto desde el que se creó la instancia actual.
