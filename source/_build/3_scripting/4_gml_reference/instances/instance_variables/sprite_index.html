sprite_index


Descripción

Esta variable devuelve el índice del sprite actual para la instancia, o -1 si la instancia no tiene ningún sprite asociado. Puedes cambiarlo para darle a la instancia un sprite diferente dándole el nombre de un sprite del árbol de recursos o usando una variable que tenga un sprite cargado externamente indexado. Cambiar el sprite no cambia el índice del fotograma actualmente visible, por lo que si cambias el sprite en el fotograma 3, el nuevo sprite se dibujará con ese fotograma visible (suponiendo que tenga el mismo número de fotogramas).

Para funciones relacionadas con sprites, mira aquí.


Sintaxis:

sprite_index;


Devoluciones:

Real


Ejemplo:

with (obj_Check)
   {
   if !collision_line(x, y, other.x, other.y, obj_Wall, false, true)
      {
      sprite_index = spr_spotted;
      }
   else
      {
      sprite_index = spr_clear;
      }
   }

El código anterior recorrerá todas las instancias de "obj_Check" verificando una línea de colisión entre ellos y la instancia que ejecuta el código. El sprite de esas instancias cambiará dependiendo del valor de retorno (verdadero o falso) para la línea de colisión.
