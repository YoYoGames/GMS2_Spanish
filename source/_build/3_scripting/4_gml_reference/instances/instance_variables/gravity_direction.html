direccion_gravedad


Descripción

Gravity_direction es una de las propiedades "integradas" que todas las instancias tienen y puede usarse para establecer la dirección del movimiento cuando las instancias gravity es mayor que 0. Tenga en cuenta que las direcciones en GameMaker Studio 2 generalmente se calculan como 0 ° con la derecha, 90 ° arriba, 180 ° hacia la izquierda y 270 ° hacia abajo.


Sintaxis:

gravity_direction;


Devoluciones:

Real


Ejemplo:

if place_meeting(x, y, obj_switch)
   {
   gravity_direction += 180;
   }

El código anterior cambiará la gravedad_dirección si el objeto jugador se encuentra con una instancia de "obj_switch".
