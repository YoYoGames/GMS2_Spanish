capa


Descripción

Esta variable incorporada se crea para cada instancia en una sala y contiene el valor de ID de capa de la capa a la que está asignada la instancia. Este valor se puede usar en otras funciones como layer_get_depth o puede cambiarse para mover la instancia a otra capa, pero tenga en cuenta que si la capa que se está asignando no existe en la sala actual, entonces obtendrá un error que forzará el cierre de su juego. Al asignar una capa, debe proporcionar la ID de capa única devuelta por la función layer_get_id() (cuando se utilizan capas de habitaciones con nombre), o según lo devuelto por la función layer_create() (cuando creas tus propias capas en tiempo de ejecución).

¡IMPORTANTE! si ha creado la instancia usando el instance_create_depth() función, o ha cambiado manualmente el depth variable, la capa asignada a la instancia se convierte en una capa "administrada", que es la que GameMaker Studio 2 controla y administra automáticamente. En estos casos, layer la variable devolverá -1.


Sintaxis:

layer;


Devoluciones:

Layer ID value (Real)


Ejemplo:

layer = layer_create(-1000);

El código anterior creará una nueva capa con una profundidad de -1000 y luego establecerá la instancia layer variable a la ID de capa devuelta, moviendo la instancia desde la capa en la que se encuentra actualmente a la nueva capa que se está creando.
