carné de identidad


Descripción

Esta variable de solo lectura contiene el número único de identificación para la instancia. A cada instancia que crea se le asigna un número que se usa internamente para identificar esta instancia y esta variable es lo que puede usar para hacer referencia a ella. La identificación también se devuelve (y se puede almacenar en una variable) cuando se crea una instancia usando instance_create_layer.
NOTA: El valor de esta variable no es el mismo que el identificador que la instancia se proporciona en el editor de sala y también es diferente del instance_id array que contiene todos los ids de todas las instancias activas actualmente.


Sintaxis:

id;


Devoluciones:

Real (instance ID value)


Ejemplo:

for(var i = 0; i < instance_count; i++;)
   {
   if instance_id[i] != id
      {
      instance_id[i].scr += 5;
      }
   }

El código anterior agrega 5 a la variable "scr" para cada instancia activa en la sala, excepto la que ejecuta el código. Lo hace al recorrer TODAS las instancias activas (utilizando la matriz "instance_id" para devolver cada ID de instancias activas) y compararlas con la variable "id" incorporada, que es la ID de la instancia original que ejecuta el código.
