window_get_cursor

Obtiene el cursor actual que se utiliza en la ventana del juego.

Sintaxis:

window_get_cursor();


Devoluciones: constante


Descripción

Con esta función, puede obtener el cursor actual en la ventana del juego, con el valor de retorno como una de las constantes enumeradas a continuación (para establecer el cursor actual que se está utilizando, puede usar la función window_set_cursor que también toma estas constantes):

Constante Cursor
cr_none

cr_default
cr_arrow
cr_cross
cr_beam
cr_size_nesw
cr_size_ns
cr_size_nwse
cr_size_we
cr_uparrow
cr_hourglass
cr_drag
cr_appstart
cr_handpoint
cr_size_all


Ejemplo:

if mouse_check_button_pressed(mb_left)
   {
   if window_get_cursor() != cr_drag window_set_cursor(cr_drag);
   }

El código anterior cambiará el cursor de la ventana al cursor estándar de arrastrar ventanas si se ha presionado el botón izquierdo del mouse y no se ha cambiado previamente.
