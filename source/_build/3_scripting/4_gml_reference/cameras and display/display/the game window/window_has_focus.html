window_has_focus

Vuelve si la ventana del juego está enfocada o no.

Sintaxis:

window_has_focus();


Devoluciones: Boolean


Descripción

Con esta función, puede sondear el estado de la ventana (o pestaña) y si pierde el enfoque, la función volverá false de lo contrario volverá true. En la mayoría de los casos, simplemente puede utilizar el os_is_paused funciona para probar esto, pero en algunos casos muy específicos (por ejemplo, juegos en Chrome Apps) esa función no se activará, en cuyo caso deberías usar esta función.

NOTA: Esta función solo es válida para HTML5, Windows y MacOS.


Ejemplo:

if !window_has_focus()
   {
   scr_PauseGame();
   }

El código anterior verificará si la ventana del juego está enfocada o no, y si la función regresa false Se llamará un guión.
