window_set_size

Establece el tamaño de la ventana en píxeles.

Sintaxis:

window_set_size( w, h );


Argumento Descripción
w El nuevo ancho de la ventana.
h La nueva altura de la ventana.


Devoluciones: N / A


Descripción

Con esta función puedes cambiar el tamaño de la ventana del juego. Si usa esta función para cambiar el tamaño de la ventana del juego, el contenido de la ventana se ajustará para ajustarse con una relación 1: 1 al lienzo de fondo que mantendrá la imagen nítida y clara (aunque escalada si la ventana es otra que la habitación o el tamaño de la vista). Por ejemplo, si tiene una ventana de 200 x 150 píxeles, y el jugador estira la ventana arrastrando los bordes, el tamaño de la ventana final puede ser de 400 x 300 píxeles, pero el lienzo de dibujo sigue STILL 200 x 150, lo que da una imagen borrosa como la proporción de píxeles es ahora 1: 2. Sin embargo, si usa esta función para establecer el tamaño de la ventana, se mantiene la proporción de 1: 1, aunque la escala de la imagen se ajuste al tamaño de la nueva ventana.


Ejemplo:

if mouse_check_button_pressed(mb_left)
   {
   global.scale += 1;
   window_set_size(200 * global.scale, 150 * global.scale);
   }

El código anterior cambiará el tamaño de la ventana según el valor de la variable "global.scale" cuando se presiona el botón del mouse.
