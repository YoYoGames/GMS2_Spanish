window_handle

Devuelve el id de la ventana del juego.

Sintaxis:

window_handle();


Devoluciones: Puntero


Descripción

Con esta función puede obtener el valor interno de la ID de << span class = "notranslate"> Windows (el HWND, un puntero ). Esta función es realmente útil solo para los escritores de extensión que necesitan el window handle para llamar a la Windows API's en el código DLL (el puntero devuelto debe convertirse en una cadena y luego en C ++ simplemente convertirlo en un HWND). La siguiente tabla muestra las plataformas compatibles junto con lo que devuelven:

Plataforma Devoluciones
Windows Ventana HWND
macOS NSWindow class
Ubuntu (Linux) XWindow
HTML5 Canvas ID


Ejemplo:

win_id = windows_handle();

El código anterior almacenará la identificación de la ventana del juego en una variable.
