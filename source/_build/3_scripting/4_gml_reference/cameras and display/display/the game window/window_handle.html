window_handle

Devuelve la identificación de la ventana del juego.

Sintaxis:

window_handle();


Devoluciones: Puntero


Descripción

Con esta función puede obtener el valor interno de identificación de Windows (HWND, un puntero ). Esta función solo es útil para los redactores de extensiones que necesitan window handle para llamar a las API de Windows en el código DLL (el puntero devuelto se debe convertir en una cadena y luego en C ++ simplemente echarlo a un HWND). La tabla a continuación muestra las plataformas compatibles con lo que devuelven:

Plataforma Devoluciones
Windows Ventana HWND
Mac OS Clase NSWindow
Ubuntu (Linux) Manejar XWindow
HTML5 ID de lienzo


Ejemplo:

win_id = windows_handle();

El código anterior almacenará la identificación de la ventana del juego en una variable.
