window_set_max_width

Establece un ancho máximo para la ventana del juego.

Sintaxis:

window_set_max_width(width);


Argumento Descripción
width El ancho máximo en píxeles para la ventana del juego.


Devoluciones: N / A


Descripción

Esta función se puede usar para establecer un ancho de ventana máximo para su juego. Si habilita la opción de cambiar el tamaño de la ventana en las Opciones del juego para la plataforma objetivo, entonces el jugador puede redimensionar la ventana del juego a cualquier tamaño que desee, sin embargo, al usar esta función puede limitar el ancho máximo al tamaño que especifique. Si desea volver al comportamiento predeterminado (es decir, no mínimo), utilice un valor de -1.

NOTA: Esta función solo está disponible en las plataformas Windows, Mac OS y Linux.


Ejemplo:

window_set_min_width(640);
window_set_min_height(480);
window_set_max_width(1280);
window_set_max_height(960);

El código anterior establecerá el ancho y alto mínimo y máximo para la ventana del juego.
