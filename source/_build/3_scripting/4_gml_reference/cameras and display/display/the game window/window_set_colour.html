window_set_colour

Establece el color de la región dentro de la ventana del juego pero fuera de las vistas visibles (el valor predeterminado es negro).

Sintaxis:

window_set_colour(colour);


Argumento Descripción
colour El color para establecer la región.


Devoluciones: N / A


Descripción

Esta función puede establecer el color de fondo de la ventana del juego. Este color representa el que se usará para aquellas áreas de la ventana del juego que no están ocupadas por ninguna vista. La siguiente imagen ilustra esto:

La imagen de arriba tiene dos vistas con dos puertos de vista, cada uno dibujado en diferentes posiciones. Esto estira la ventana del juego para acomodar ambos puertos y usa el color de la ventana para colorear el fondo donde no se muestra ninguna vista.


Ejemplo:

if window_get_colour() != c_black
   {
   window_set_colour(c_black);
   }

El código anterior verificará el color de la ventana para ver si está configurado como negro o no, y si no lo está, lo configura en negro.
