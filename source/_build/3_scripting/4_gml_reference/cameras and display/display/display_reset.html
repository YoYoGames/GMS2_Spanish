display_reset


Descripción

Esta función restablece la configuración de pantalla a las que se establecieron cuando se inició el juego, pero también le permite cambiar el nivel actual de anti-alias de pantalla completa que se está utilizando y si se debe usar la sincronización vertical. Los niveles de suavizado disponibles son 0,2,4 y 8, con el valor de inicio predeterminado establecido en 0, y la configuración de v-sync predeterminada es falsa (desactivada). La activación de v-sync puede brindar una experiencia de juego más fluida, pero también necesitará más potencia de procesamiento, por lo que su impacto debe considerarse cuidadoso antes de su uso, y lo mismo ocurre con el suavizado en el que, cuanto mayor sea el número, mayor será el procesamiento requerido..

Como no todos los dispositivos de destino son iguales, algunos pueden no ser compatibles con el suavizado 8x o 4x, por lo que hay una variable de solo lectura disponible para obtener los diferentes niveles de AA que el dispositivo que ejecuta el juego puede mostrar:

display_aa

Esta variable devolverá un valor basado en la configuración de bits para los diferentes niveles. Entonces, solo para 2xAA, esto reportará 2, para disponibilidad 2x y 4x reportará 6. Para 8 y 4 reportará 12. Para los 3 niveles (2,4 y 8) reportará 14.


Sintaxis:

display_reset(aa, vsync);

Argumento Descripción
aa El nivel de filtrado anti-aliasing (0, 2, 4 u 8).
vsync Alternar la sincronización vertical a activado (verdadero) o desactivado (falso).


Devoluciones:

Real


Ejemplo:

if display_aa > 12 display_reset(8, true);

El código anterior establecerá el nivel de suavizado en 8 si es compatible y activará v-sync.
