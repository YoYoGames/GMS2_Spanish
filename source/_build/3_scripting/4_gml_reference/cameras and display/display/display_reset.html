display_reset

Restablece la pantalla y cambia el filtrado anti-aliasing y vsync.

Sintaxis:

display_reset(aa, vsync);


Argumento Descripción
aa El nivel de filtrado anti-aliasing (0, 2, 4 u 8).
vsync Conmute la sincronización vertical a on (true) o off (false).


Devoluciones: Real


Descripción

Esta función restablece la configuración de la pantalla a las que se establecieron cuando se inició el juego, pero también le permite cambiar el nivel actual de anti-aliasing de pantalla completa que se utiliza y si se utiliza la sincronización vertical. Los niveles de anti-aliasing disponibles son 0,2,4 y 8, con el valor de inicio predeterminado establecido en 0, y la configuración de sincronización v predeterminada es falsa (desactivada). El cambio de v-sync puede proporcionar una experiencia de juego más fluida, pero también necesitará más potencia de procesamiento, por lo que su impacto debe considerarse cuidadoso antes de su uso, y lo mismo ocurre con el antialiasing, donde cuanto mayor sea el número, más procesamiento se requiere.

Como no todos los dispositivos de destino son iguales, es posible que algunos no sean compatibles con el anti-aliasing 8x o 4x, por lo que existe una variable de solo lectura disponible para obtener los diferentes niveles de AA que puede mostrar el dispositivo que ejecuta el juego:

display_aa

Esta variable devolverá un valor basado en la configuración de bits para los diferentes niveles. Entonces para solo 2xAA, esto reportará 2, para disponibilidad 2x y 4x informará 6. Para 8 y 4 informará 12. Para los 3 niveles (2,4 y 8) reportará 14.


Ejemplo:

if display_aa > 12 display_reset(8, true);

El código anterior establecerá el nivel de antialiasing en 8 si es compatible y activará v-sync.
