display_set_ui_visibility


Sintaxis:

display_set_ui_visibility(flags);


Argumento banderas
flags El Android señala para usar (como un valor entero).


Devoluciones: N / A


Descripción

Esta función se puede usar para mostrar u ocultar la interfaz de usuario del sistema en Android solamente. La función requiere que suministre uno o más indicadores del sistema como un valor entero. Cuando se usa más de un indicador, estos deben fusionarse utilizando el "o" bit a bit, como se muestra en el ejemplo a continuación. Aquí puede encontrar una lista de indicadores del sistema Android.


Ejemplo:

var flags = 1024|4096;
display_set_ui_visibility(flags);

El código anterior utilizará Android banderas del sistema de Android "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" y "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN" para establecer la visibilidad de la pantalla.
