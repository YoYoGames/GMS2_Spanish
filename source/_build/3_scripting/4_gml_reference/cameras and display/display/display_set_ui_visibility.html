display_set_ui_visibility


Descripción

Esta función se puede usar para mostrar u ocultar la IU del Android solo en Android. La función requiere que suministre uno o más indicadores del sistema como un valor entero. Cuando se usa más de una sola bandera, se deben combinar utilizando el bit o "bit", como se muestra en el siguiente ejemplo. Puedes encontrar una lista de banderas del sistema Android aquí.


Sintaxis:

display_set_ui_visibility(flags);

Argumento banderas
flags Las banderas del SISTEMA de Android a usar (como un valor entero).


Devoluciones:

N/A


Ejemplo:

var flags = 1024|4096;
display_set_ui_visibility(flags);

El código anterior utilizará los indicadores del sistema Android "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" y "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN" para establecer la visibilidad de la pantalla.
