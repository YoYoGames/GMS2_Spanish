display_set_gui_size

Escala la GUI al ancho o alto dados.

Sintaxis:

display_set_gui_size(width, height);


Argumento Descripción
width El ancho de la GUI (-1 es igual a 1: 1 con la pantalla o la aplicación_superficie).
height La altura de la GUI (-1 es igual a 1: 1 con la pantalla o la aplicación_superficie).


Devoluciones: N / A


Descripción

Esta función solo afectará a las cosas dibujadas en el evento Draw GUI y se puede usar para establecer un ancho y alto específico para todos los componentes de la GUI en ese evento, y no importa qué tamaño de pantalla o ventana se use para mostrar la GUI, será escalado para caber correctamente. Donde se coloca la GUI (es decir, la posición (0,0)) dependerá de si ha utilizado la función display_set_gui_maximise() o no. Si no usa esta función (la configuración predeterminada), la capa de la GUI se colocará con su punto (0,0) en la misma posición que la superficie de la aplicación (el área donde se está dibujando su juego) y el ancho y la altura se ampliará para ajustarse a esta superficie, de modo que establecer un ancho de GUI de 256px con esta función cuando la superficie de la aplicación sea de 512px estirará esos 256 píxeles para que quepan, haciendo que los píxeles sean dos veces más grandes. Esto es ideal para aquellos juegos que tienen una corrección de relación de aspecto establecida en la Configuración global del juego.

Si has usado el display_set_gui_maximise() función entonces la posición (0,0) para la capa GUI irá predeterminada a la esquina superior izquierda de la ventana del juego, o pantalla (cuando esté en pantalla completa o en dispositivos), con el factor de escala establecido por esa función, y el ancho y alto establecido por este. Puede restablecer la capa de la GUI en cualquier momento a la configuración predeterminada llamando a esta función usando -1 como ancho y alto; esto establecerá la posición (0, 0) en la superficie de la aplicación y el ancho y alto se restablecerán a 1: 1 con la superficie también.


Ejemplo:

display_set_gui_size(768, 1024);

El código anterior bloqueará el evento de GUI del sorteo a la anchura y la altura especificadas, escalando todos los componentes para que quepan en la superficie de la aplicación o en la pantalla, usando esa proporción.
