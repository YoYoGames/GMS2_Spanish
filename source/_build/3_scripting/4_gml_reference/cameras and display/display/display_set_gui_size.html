display_set_gui_size


Descripción

Esta función solo afectará a los elementos dibujados en el Evento GUI de Draw y se puede usar para establecer un ancho y una altura específicos para todos los componentes de la GUI en ese evento, y no importa qué tamaño de pantalla o ventana se use para mostrar la GUI, será Escalado para encajar correctamente. El lugar donde se coloca la GUI (es decir, la posición (0,0)) dependerá de si ha utilizado la función. display_set_gui_maximise() o no. Si no usa esta función (la configuración predeterminada), la capa GUI se posicionará con su punto (0,0) en la misma posición que la superficie de la aplicación (el área donde se está dibujando su juego) y el ancho y la altura se estirará para ajustarse a esta superficie, de modo que establecer un ancho de GUI de 256px con esta función cuando la superficie de la aplicación sea 512px estiraría esos 256 píxeles para que se ajusten, haciendo que los píxeles sean dos veces más grandes. Esto es ideal para aquellos juegos que tienen la corrección de la relación de aspecto establecida en la configuración global del juego.

Si ha utilizado el display_set_gui_maximise() La función (luego) la posición (0,0) para la capa GUI se ubicará por defecto en la esquina superior izquierda de la ventana del juego, o se mostrará (cuando esté en pantalla completa o en dispositivos), con el factor de escala establecido por esa función, y el ancho y alto establecido por éste. Puede restablecer la capa GUI en cualquier momento a la configuración predeterminada llamando a esta función usando -1 como ancho y alto; esto establecerá la posición (0, 0) en la superficie de la aplicación y el ancho y alto se restablecerán a 1: 1 con la superficie también.


Sintaxis:

display_set_gui_size(width, height);

Argumento Descripción
width El ancho de la GUI
height La altura de la GUI


Devoluciones:

N/A


Ejemplo:

display_set_gui_size(768, 1024);

El código anterior bloqueará el evento de la GUI del dibujo en el ancho y la altura dados, escalando todos los componentes para que se ajusten a la superficie de la aplicación, o la pantalla, utilizando esa proporción.
