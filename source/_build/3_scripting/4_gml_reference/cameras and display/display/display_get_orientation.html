display_get_orientation


Descripción

Esta función devolverá una de las dos constantes que GameMaker Studio 2 tiene para decirle si el dispositivo que ejecuta el juego se mantiene en modo horizontal o vertical (consulte la tabla a continuación). Tenga en cuenta que esta función puede no detectar correctamente la orientación del dispositivo cuando se utiliza en el módulo de destino HTML5. Sin embargo, esto es fácilmente imitado por el uso de la siguiente secuencia de comandos:

return (browser_width < browser_height);

Volverá true para retrato y false para el paisaje.


Sintaxis:

display_get_orientation()


Devoluciones:

Constant

Constante Descripción
display_landscape El dispositivo se sostiene horizontalmente, es decir: el borde más largo es de izquierda a derecha, y el botón de menú está a la derecha.
display_landscape_flipped Como arriba, solo que ahora el botón de menú está a la izquierda.
display_portrait El dispositivo se sostiene verticalmente, es decir: el borde más largo es de arriba a abajo y el botón de menú está en la parte inferior.
display_portrait_flipped Como arriba, solo que ahora el botón de menú está en la parte superior.


Ejemplo:

if display_get_orientation() = display_landscape
   {
   global.Config = 0;
   }
else
   {
   global.Config = 1;
   }

El código anterior verifica la orientación del dispositivo y establece una variable global según el valor devuelto.
