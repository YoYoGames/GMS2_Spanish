display_set_sleep_margin


Descripción

Esta función se puede usar para establecer el valor del margen de reposo usado para el tiempo de procesamiento de su juego, y requiere un valor de milisegundos. Los valores por defecto son los siguientes:

Plataforma Milisegundos
Windows 20
Mac OS 20
Ubuntu 20
HTML5 20
Android / Amazon Fire 4
iOS 4
Windows UWP 20
Xbox 20
PS4 20
Interruptor de nintendo 20


Tenga en cuenta que incluso si el método de sincronización está configurado para usar la sincronización vsync, el margen de suspensión tendrá algún efecto sobre la representación del juego, por lo que debe tener cuidado al establecer este valor. Para obtener más información sobre el tiempo de visualización, consulte display_set_timing_method().


Sintaxis:

display_set_sleep_margin(milliseconds);

Argumento Descripción
milliseconds El número de milisegundos para el margen de reposo.


Devoluciones:

N/A


Ejemplo:

if display_get_timing_method() != tm_sleep
{
display_set_timing_method(tm_sleep);
if display_get_sleep_margin() != 20
    {
    display_set_sleep_margin(20);
    }
}

El código anterior verificará el método de sincronización y luego, si no está configurado para tm_sleep se establecerá en eso y el arsenal de reposo se establecerá en 20.
