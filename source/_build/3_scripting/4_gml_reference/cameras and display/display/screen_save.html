guardar pantalla


Descripción

Con esta función puedes guardar una captura de pantalla del juego tal como aparece actualmente en la ventana del juego. El archivo se guardará en el directorio de trabajo del juego, que en Windows se encuentra en <drive>:/Users/YOURUSERNAME/AppData/Local/[Game Name]/ y en Mac sería ~/Library/Application Support/[Game Name]/. La imagen capturada será el render final de la superficie de aplicación del juego, o (si la superficie de la aplicación ha sido desactivada) será una imagen de toda la pantalla o ventana. Es importante tener en cuenta que la función está diseñada para ser llamada en el Evento Final de la GUI de Draw, que debería dar resultados consistentes en todas las plataformas. El uso en cualquier otro evento puede no funcionar como se espera y puede dar resultados diferentes en diferentes objetivos e incluso dispositivos. Si necesita más control sobre lo que se está guardando, debe utilizar la función screen_save_part.

NOTA: Esta función no funciona en el destino HTML5.


Sintaxis:

screen_save(fname);


Argumento Descripción
fname El nombre de archivo para la captura de pantalla para guardar


Devoluciones:

N/A


Ejemplo:

if keyboard_check_pressed(ord("S"))
   {
   screen_save(working_directory + "Screens\Screen_" + string(num++) + ".png")
   }

El código anterior toma una captura de pantalla del juego cuando el usuario presiona la tecla "S". Esta captura de pantalla se guarda en un directorio específico y se numera utilizando una variable que luego se incrementa.
