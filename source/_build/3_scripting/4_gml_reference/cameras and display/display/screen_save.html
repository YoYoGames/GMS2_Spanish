screen_save

Guarda una captura de pantalla del juego como *.png.

Sintaxis:

screen_save(fname);


Argumento Descripción
fname El nombre de archivo de la captura de pantalla para guardar


Devoluciones: N / A


Descripción

Con esta función, puede guardar una captura de pantalla del juego tal como aparece actualmente en la ventana del juego o en el navegador. El archivo se guardará en el directorio de trabajo del juego, que en Windows se encuentra en "C: \ Users \ YOURUSERNAME \ AppData \ Local \ YOURGAMENAME \" Esta es una imagen de toda la región de visualización y no solo una sección específica o ve y estás guardando lo que se dibuja en el búfer posterior en ese momento, lo que significa que dependiendo de dónde llames a la función, los resultados serán diferentes. Si no desea la capa de la GUI, puede guardar la pantalla antes de representar la GUI (es decir, en el evento Draw ), pero si desea que se incluyan los elementos de la GUI, entonces la guarda después del evento Draw GUI. Si necesita más control, debe usar la función screen_save_part.

NOTA: esta función no funciona en el destino HTML5.


Ejemplo:

if keyboard_check_pressed(ord("S"))
   {
   screen_save(working_directory + "\Screens\Screen_"+string(num)+".png")
   num += 1;
   }

El código anterior toma una captura de pantalla del juego cuando el usuario presiona la tecla "S". Esta captura de pantalla se guarda en un directorio específico y se numera usando una variable que luego se incrementa.
