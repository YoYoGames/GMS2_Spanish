camera_get_proj_mat


Descripción

Esta función se puede usar para recuperar la matriz de proyección. La función devuelve el valor de la ID de la matriz que luego puede usarse en otras funciones de la matriz o para establecer la matriz de proyección de otra cámara (usando camera_set_proj_mat )


Sintaxis:

camera_get_proj_mat(camera_id)


Argumentos:

Argumento Descripción
camera_id El valor único de ID de cámara devuelto cuando creó la cámara


Devoluciones:

Matrix ID (real)


Ejemplo:

mat = camera_get_proj_mat(camera_view[0]);

El código anterior obtiene la matriz de proyección para la cámara asignada para ver el puerto [0].
