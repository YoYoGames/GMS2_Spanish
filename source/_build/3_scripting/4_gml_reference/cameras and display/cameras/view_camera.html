view_camera


Descripción

Esta matriz contiene la identificación de cámara única asignada al puerto de vista dado, y puede configurarse en una cámara nueva o leer para obtener la cámara actual, devolviendo -1 si no se asigna ninguna cámara. Puede tener hasta 8 puertos de visualización activos en una sala (valores de matriz de 0 a 7) y puede asignar una cámara a cualquiera de ellos simplemente configurando esta variable en el valor de ID de la cámara (como lo devuelven las funciones) camera_create() o camera_create_view() ) Si ha configurado la cámara para un puerto de visualización a través del Editor de sala, esta variable contendrá la ID de cámara para ese puerto.


Sintaxis:

view_camera[0 ... 7];


Devoluciones:

Camera index or -1 if no camera is assigned


Ejemplo:

if view_camera[0] == -1
   {
   view_camera[0] = camera_create_view(0, 0, room_width, room_height);
   }

El código anterior comprueba si el puerto de visualización [0] tiene una cámara conectada y si no lo está, se crea una y se establece en ella.
