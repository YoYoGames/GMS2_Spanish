camera_destroy


Descripción

Con esta función puedes destruir cualquier cámara. Al llamar a la función, usted proporciona el valor único de la identificación de la cámara, que obtiene de la camera_create_* funciones o desde el view_camera array si usa el Editor de sala para configurar el puerto de vista y la vista. Nunca se debe destruir una cámara que se encuentra actualmente asignado a una vista visible, a menos que se está asignando una nueva cámara a la vista en el mismo paso, y siempre se debe destruir todas las cámaras que se han creado a través de código cuando ya no sea requerido por su juego para evitar fugas de memoria, y también puede destruir las cámaras predeterminadas si tiene alguna asignada en el Editor de habitaciones, pero debe asignar una nueva cámara al puerto de visualización (o deshabilitarla) de lo contrario obtendrá resultados extraños.


Sintaxis:

camera_destroy(camera_id)


Argumentos:

Argumento Descripción
camera_id El valor único de ID de cámara que se devolvió cuando creó la cámara.


Devoluciones:

Real (unique camera ID value)


Ejemplo:

camera_destroy(view_camera[0]);
view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

El código anterior destruye la cámara actualmente asignada para ver el puerto [0], luego crea una nueva cámara y asigna su ID a ese puerto.
