camera_set_view_pos


Descripción

Puede usar esta función para actualizar la posición de la vista de cámara dentro de la habitación. Usted proporciona el valor único de ID de la cámara (como lo devuelven los diferentes camera_create funciones) y luego dé las posiciones xey para configurar la cámara.


Sintaxis:

camera_set_view_pos(camera_id, x, y)


Argumentos:

Argumento Descripción
camera_id El valor único de ID de cámara que se devolvió cuando creó la cámara.
x La posición x para colocar la vista en (en la sala)
y La posición y para colocar la vista en (en la habitación)


Devoluciones:

N/A


Ejemplo:

camera_set_view_pos(view_camera[0], x - (view_wport[0] / 2), y - (view_hport[0] / 2));

El código anterior establecerá la posición de la cámara de visualización para la cámara asignada para ver el puerto [0].
