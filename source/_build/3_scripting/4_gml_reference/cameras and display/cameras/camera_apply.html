camera_apply


Descripción

Esta función aplicará la configuración de la cámara dada a la vista actual de la cámara que se está procesando de inmediato. Si bien puede configurar cualquier cámara de visualización utilizando el específico camera_set_* funciones, la nueva configuración no se usará para renderizar hasta el próximo cuadro de juego, pero con esta función puede hacer esos cambios inmediatamente. Si está utilizando vistas múltiples, entonces la función debe usarse en el Evento de Dibujo y puede usar la variable view_current para aplicar selectivamente la cámara dada solo a la vista de cámara que se está renderizando actualmente. Tenga en cuenta que esto NO cambia el valor de ID de la cámara, solo los ajustes para esa cámara, y que si usa esta función en la cámara predeterminada (es decir, la cámara interna que GameMaker Studio 2 usa cuando no hay cámaras de visualización activas) los cambios serán sobreescribió el siguiente marco de juego por la configuración interna para dibujar la habitación.


Sintaxis:

camera_apply(camera_id)


Argumentos:

Argumento Descripción
camera_id El valor único de ID de cámara que se devolvió cuando creó la cámara.


Devoluciones:

N/A


Ejemplo:

if view_current == 0
   {
   camera_apply(cutscene_cam);
   }

El código anterior (en un evento de Draw) comprueba para ver qué cámara se está visualizando actualmente y si es la vista de la cámara [0], entonces se le aplica la configuración de la cámara a la que hace referencia la variable "cutscene_cam".
