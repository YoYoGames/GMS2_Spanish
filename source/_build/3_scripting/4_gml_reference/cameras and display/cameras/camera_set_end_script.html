camera_set_end_script


Descripción

Esta función se puede usar para establecer un script que se llamará al final de cada cuadro de juego que la cámara tiene asignada a un puerto de vista visible y activo, después de que se haya procesado todo para esa cámara de vista. Usted proporciona el valor único de ID de la cámara (como lo devuelven los diferentes camera_create funciones) y el ID del script que se llamará. El orden en que se adjuntan las secuencias de comandos a las cámaras y la representación real de la vista de la cámara es la siguiente:

  • Las cámaras para todos los puertos de vista activos y visibles tienen su script de actualización llamado

  • Luego, para cada vista:
    • El guión de inicio de la cámara para esa vista se llama
    • Los eventos de dibujo se ejecutan para esa vista
    • El script final para la cámara se llama
  • Pase a la siguiente vista visible activa y repita...


Sintaxis:

camera_set_end_script(camera_id, script)


Argumentos:

Argumento Descripción
camera_id El valor único de ID de cámara que se devolvió cuando creó la cámara.
script El script para ejecutar cada marco del juego


Devoluciones:

N/A


Ejemplo:

camera_set_end_script(view_camera[0], endCamera);

El código anterior establece el script final para la cámara asignada para ver el puerto [0].
