camera_get_default


Descripción

Esta función se puede utilizar para recuperar el valor único de ID de cámara de la cámara predeterminada (la cámara que GameMaker Studio 2 usa cuando no hay vistas de cámara o puertos activos en una sala de juegos).


Sintaxis:

camera_get_default()


Devoluciones:

Camera index (real).


Ejemplo:

var def = camera_get_default();
view_camera[0] = def;

El código anterior obtiene la identificación de la cámara para la cámara predeterminada y configura la cámara de visualización para el puerto [0] para usarla.
