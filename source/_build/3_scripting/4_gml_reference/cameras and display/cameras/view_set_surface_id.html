view_set_surface_id


Descripción

Con esta variable puede establecer el contenido de un puerto de vista para dibujar en una superficie. Cuando se trabaja con superficies, a menudo se requiere capturar toda la región visible de la pantalla hacia la superficie, y así asignarla a un puerto de visualización usando esta función. Esto significa que todo lo que se muestra en la vista de puerto elegida ahora se dibujará en la superficie asignada y el contenido de ese puerto de vista ya no se mostrará, lo que significa que tendrá que:

  • Habilite una nueva vista y dibuje la superficie solo en esa vista (usando view_current para verificar qué vista se está dibujando)

  • Dibuje la superficie en el evento Dibujar GUI de una instancia, ya que la capa de la GUI es independiente de las vistas.

Al usar esta función, usted proporciona el índice de puerto de vista (de 0 a 7) y un índice de superficie (el application_surface o el valor de índice único devuelto por la función surface_create ) o, si una superficie ha sido asignada previamente y desea eliminarla, un valor de -1. Para obtener más información sobre cómo configurar el puerto de visualización en una superficie, consulte también la variable view_surface_id.


Sintaxis:

view_set_surface_id(view_port, surf)


Argumentos:

Argumento Descripción
view_port El puerto de visualización al objetivo (0 - 7)


Devoluciones:

Real


Ejemplo:

if view_get_surface_id(0) == -1
   {
   view_set_surface_id(0, global.vSurf);
   }

El código anterior verificará si se ha asignado una superficie al puerto de vista [0] y, si no, se asignará una.
