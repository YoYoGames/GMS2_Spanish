camera_set_view_size


Descripción

Puede usar esta función para actualizar el tamaño de la cámara de visualización dentro de la sala. Usted proporciona el valor único de ID de la cámara (como lo devuelven los diferentes camera_create funciones) y luego proporcione el ancho y el alto (en píxeles) para configurar la cámara.


Sintaxis:

camera_set_view_size(camera_id, width, height)


Argumentos:

Argumento Descripción
camera_id El valor único de ID de cámara que se devolvió cuando creó la cámara.
width El ancho de la vista de la cámara en píxeles
height El alto de la vista de la cámara en píxeles


Devoluciones:

N/A


Ejemplo:

camera_set_view_size(view_camera[0], view_wport[0], view_hport[0]);

El código anterior configurará el tamaño de la cámara de visualización para la cámara asignada para ver el puerto [0].
