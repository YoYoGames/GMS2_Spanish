camera_get_view_y


Descripción

Esta función se puede utilizar para recuperar la posición y de la vista de una cámara determinada. Tenga en cuenta que esta función solo es válida para cámaras creadas usando camera_create_view() o para los añadidos en el editor de sala.


Sintaxis:

camera_get_view_y(camera_id)


Argumentos:

Argumento Descripción
camera_id El valor de ID de cámara único que se devuelve cuando creaste la cámara


Devoluciones:

Real


Ejemplo:

var vx = camera_get_view_x(view_camera[0]);
var vy = camera_get_view_y(view_camera[0]);
draw_text(vx + 5, vy + 5, "SCORE" + string(score));

El código anterior recupera la posición de la cámara asignada para ver el puerto [0] y luego dibuja el texto relativo a esa posición.
