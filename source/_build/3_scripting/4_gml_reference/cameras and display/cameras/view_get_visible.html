view_get_visible


Descripción

Esta función se puede usar para verificar la visibilidad del puerto de vista dado. La función volverá true si es visible y false si no es.


Sintaxis:

view_get_visible(view_port)


Argumentos:

Argumento Descripción
view_port El puerto de visualización al objetivo (0 - 7)


Devoluciones:

Boolean


Ejemplo:

if !view_get_visible(0)    {
   view_set_visible(0, true);
   }

El código anterior verificará si el puerto de vista [0] es visible y, si no lo está, se configurará como visible.
