camera_create_view


Descripción

Esta función creará una nueva cámara y configurará su vista en una habitación, luego devolverá su valor único de identificación. La cámara se creará utilizando los valores que establezca en esta función para la posición, el tamaño y otros detalles que se correlacionan con los valores que estableció en el Editor de la sala (consulte aquí ). Tenga en cuenta que esta función puede tomar un número variable de argumentos con solo la posición x / y y los argumentos de ancho / alto siendo obligatorios. Cualquier argumento no proporcionado usará sus argumentos predeterminados.

Tenga en cuenta que las cámaras son como todos los recursos dinámicos y ocuparán la memoria cuando se creen. Esto significa que debe tener cuidado de no crear cámaras innecesariamente y destruir aquellas que no necesita llamando a la función camera_destroy(). Esto también significa que las cámaras persistirán de una habitación a otra si se asignan a un global variable (persistirán cuando se asignen también a variables de instancia o locales, pero no podrá acceder a ellas después de cambiar de sala, de ahí la fuga de memoria), de modo que pueda, por ejemplo, crear una nueva cámara, asignarla a una variable global, y luego, al inicio de cada habitación, use camera_set_* funciones para establecer las propiedades antes de asignarlo a un puerto de visualización.


Sintaxis:

camera_create_view(room_x, room_y, width, height, [angle, object, x_speed, y_speed, x_border, y_border])


Argumentos:

Argumento Descripción
room_x La posición x (dentro de la habitación) del lado izquierdo del área que captura la vista de la cámara.
room_y La posición y (dentro de la habitación) de la parte superior del área que captura la vista de la cámara.
width El ancho del área que captura la vista.
height La altura del área que captura la vista.
angle El ángulo para configurar la vista de la cámara (de 0 ° a 360 °, por defecto es 0).
object El ID de objeto (o ID de instancia) para la vista de cámara a seguir (el valor predeterminado es -1, sin objeto).
x_speed La velocidad horizontal máxima a la que se puede mover la vista (establecida en -1 para el seguimiento "instantáneo", el valor predeterminado es -1).
y_speed La velocidad vertical máxima a la que se puede mover la vista (establecida en -1 para el seguimiento "instantáneo", el valor predeterminado es -1).
x_border El espacio horizontal mínimo entre el borde del área de visualización y la instancia que está configurado para seguir (antes de que la vista comience a moverse y si se configura para seguir una instancia, el valor predeterminado es 0).
y_border El espacio vertical mínimo entre el borde del área de visualización y la instancia que está configurado para seguir (antes de que la vista comience a moverse y si está configurado para seguir una instancia, el valor predeterminado es 0).


Devoluciones:

Real (unique camera ID value)


Ejemplo:

view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

El código anterior crea una nueva cámara, define sus propiedades de vista y luego asigna su ID para ver el puerto [0].
