Dibujo


Esta sección contiene todas las funciones relacionadas con el dibujo dentro de la sala de juegos, así como también para controlar cómo se dibujarán las cosas (mezcla, alfa, eliminación, etc.). Hay un gran número de funciones diferentes para el dibujo, y se dividen en las siguientes categorías:

  1. Color y Alfa
  2. Control de GPU
  3. Mipmapping
  4. Dibujo de formularios básicos
  5. Dibujo de Sprites y Azulejos
  6. Texto de dibujo
  7. Dibujar y crear primitivos
  8. Superficies
  9. Iluminación
  10. Partículas
  11. Texturas
  12. Shaders

También hay dos funciones especiales disponibles, una que se usa para habilitar o deshabilitar el evento de sorteo para su proyecto, y la otra para enjuagar todo el ducto de sorteo:

  • draw_enable_drawevent
  • draw_flush
