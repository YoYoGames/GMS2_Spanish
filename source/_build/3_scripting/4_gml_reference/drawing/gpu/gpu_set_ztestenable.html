gpu_set_ztestenable


Descripción

Esta función se puede usar para activar o desactivar las pruebas de z-buffer, lo que afecta la forma en que se dibujarán las cosas (en general, solo se usa cuando se trabaja con proyectos en 3D). Básicamente, de forma predeterminada, cuando la prueba z está desactivada y tiene dos objetos que dibujan en el mismo espacio, ambos objetos se representarán independientemente de si uno sobreextraerá el otro, lo que provocará llamadas de sorteo innecesarias. Si activa esto, se prueba el z-buffer para ver si un objeto será "visible" y no dibujado si no lo es. Tenga en cuenta que este es el comportamiento predeterminado, pero puede cambiar esto cambiando el tipo de comparación utilizado para la prueba z-buffer (consulte la función gpu_set_zfunc.

Por defecto, la prueba z-buffer está desactivada ( false )


Sintaxis:

gpu_set_ztestenable(enable);


Argumentos:

Argumento Descripción
enable Habilitar o deshabilitar la prueba z-buffer ( true o false )


Devoluciones:

N/A


Ejemplo:

gpu_set_ztestenable(true);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

El código anterior activa la prueba z-buffer para dibujar un fondo sprite y luego lo apaga de nuevo para continuar dibujando.
