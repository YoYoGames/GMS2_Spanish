gpu_set_alphatestref


Descripción

Esta función establecerá el valor de referencia para la prueba alfa (cuando se ha habilitado usando el gpu_set_alphatestenable ) Este es el umbral de "corte" en el que no se dibujarán píxeles con alfa.

Básicamente, si tiene este valor establecido en 0 y tiene activada la interpolación lineal (ya sea en las Opciones del juego o usando gpu_set_texfilter ), aún puede obtener errores visuales en los bordes de los objetos superpuestos. Esto se debe a que puede tener áreas casi completamente transparentes que bloquean las cosas que se dibujan detrás de ellas, pero si establece este valor de referencia, por ejemplo, 254 (es decir, uno por debajo del máximo), esto garantiza que solo se dibujarán 100% de píxeles opacos. También puede desactivar la interpolación de textura, lo que normalmente le permitiría usar los valores predeterminados sin problemas y sin necesidad de utilizar esta función, sin embargo, mucho depende de la obra de arte fuente.

NOTA: Esta es una función de dibujo avanzada, por lo que si no está seguro de si la necesita o no, o no sabe exactamente lo que hace, es mejor que no la use, ya que puede dañar seriamente la forma en que se dibuja su juego..


Sintaxis:

gpu_set_alphatestref(val);


Argumentos:

Argumento Descripción
val Establece el valor de referencia para la prueba alfa de 0 a 255 (el valor predeterminado es 0)


Devoluciones:

N/A


Ejemplo:

if !gpu_get_alphatestenable()
   {
   gpu_set_alphatestenable(true);
   gpu_set_alphatestref(128);
   }

El código anterior verificará si la prueba alfa está habilitada y si no, activará las pruebas alfa y establecerá el umbral de prueba en 128 (solo se dibujarán los píxeles con un alfa por encima de 0,5).
