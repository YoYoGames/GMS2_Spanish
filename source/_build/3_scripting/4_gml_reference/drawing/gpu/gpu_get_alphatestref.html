gpu_get_alphatestref


Descripción

Puede usar esta función para encontrar el valor actual para la referencia de prueba alfa (el valor predeterminado es 0, pero puede usar gpu_set_alphatestref para establecer este valor en algo diferente a esto).


Sintaxis:

gpu_get_alphatestref();


Devoluciones:

Real (from 0 - 255)


Ejemplo:

if gpu_get_alphatestenable()
   {
   if gpu_get_alphatestref() < 254
      {
      gpu_set_alphatestref(254);
      }
   }

El código anterior comprueba si las pruebas alfa están habilitadas, y si lo está, comprueba el valor de referencia de la prueba alfa actual y lo establece si es menor que 254.
