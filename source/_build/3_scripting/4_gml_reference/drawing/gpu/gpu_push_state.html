gpu_push_state


Descripción

Con esta función, puede insertar el estado actual de la GPU en una pila para usarla más adelante. En general, usaría esto si quiere "guardar" el estado actual de la GPU (cosas como el modo de fusión, escritura alfa, descarte, etc... serán empujadas a la pila), luego dibuje algo con diferentes configuraciones, y luego reinicie la pila GPU a lo que era antes (llamando gpu_pop_state )


Sintaxis:

gpu_push_state();


Devoluciones:

N/A


Ejemplo:

gpu_push_state();
gpu_set_blendmode(bm_add);
gpu_set_blendenable(false);
gpu_set_cullmode(true);
with (obj_Effect_Parent)
   {
   draw_self();
   }
gpu_pop_state();

El código anterior "guardará" el estado actual de la GPU en la pila, luego cambiará ciertas configuraciones de la GPU y dibujará un grupo de instancias antes de restablecer el estado de la GPU a lo que era anteriormente.
