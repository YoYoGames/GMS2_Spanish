gpu_get_colorwriteenable


Descripción

Esta función se puede usar para recuperar los valores de activación de escritura de color actuales. La función devuelve una matriz 1D de 1 elemento con los siguientes elementos que serán true (habilitado) o false (discapacitado). Por defecto, toda la escritura de color está configurada para true:

  • [0] = Canal rojo habilitado / deshabilitado
  • [1] = canal verde habilitado / deshabilitado
  • [2] = canal azul habilitado / deshabilitado
  • [3] = canal alfa habilitado / deshabilitado


Sintaxis:

gpu_get_colorwriteenable();


Devoluciones:

4 element 1D array (See above for details)


Ejemplo:

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

El código anterior obtiene los valores de escritura de color actuales y luego establece el componente alfa para false.
