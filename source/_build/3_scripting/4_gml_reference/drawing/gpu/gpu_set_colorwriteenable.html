gpu_set_colorwriteenable


Descripción

Con esta función, puede activar o desactivar los canales de color y el canal alfa para todos los demás dibujos. Por ejemplo, puede usar esta función para crear superficies solo de canal alfa (apague la escritura RGB antes de usar la superficie), o para crear efectos especiales mientras dibuja en la pantalla. El valor predeterminado para cada uno de los componentes es true, y se puede suministrar como cuatro argumentos únicos o como una matriz de 4 elementos 1D con los siguientes elementos que serán true (habilitado) o false (discapacitado):

  • [0] = Canal rojo habilitado / deshabilitado
  • [1] = canal verde habilitado / deshabilitado
  • [2] = canal azul habilitado / deshabilitado
  • [3] = canal alfa habilitado / deshabilitado


Sintaxis:

gpu_set_colorwriteenable(red, green, blue, alpha);


Argumentos:

Argumento Descripción
red Habilitar / deshabilitar el canal rojo
green Habilitar / deshabilitar el canal verde
blue Habilitar / deshabilitar el canal azul
alpha Habilitar / deshabilitar el canal alfa


Devoluciones:

N/A


Ejemplo:

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

El código anterior obtiene los valores de escritura de color actuales y luego establece el componente alfa para false.
