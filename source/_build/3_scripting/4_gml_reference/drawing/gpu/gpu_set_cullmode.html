gpu_set_cullmode


Descripción

Esta función se puede usar para configurar el modo de eliminación de fondos ocultos. Un polígono (primitivo) tiene un frente y un lado trasero. El lado frontal se dice que es el lado donde los vértices se definen en el sentido contrario a las agujas del reloj. Normalmente ambos lados están dibujados, pero si haces una forma cerrada (como una pirámide), entonces esto es un desperdicio de poder de procesamiento porque el lado posterior del triángulo nunca se puede ver. En estos casos, puede activar la eliminación de fondos ocultos, ya que esto ahorra aproximadamente la mitad del tiempo de dibujo, aunque eso significa que tiene la tarea de definir sus polígonos de la forma correcta para evitar problemas.

El valor de modo de exclusión utilizado debe ser una de las siguientes constantes (el valor predeterminado es cull_noculling )

  • cull_noculling
  • cull_clockwise
  • cull_counterclockwise


Sintaxis:

gpu_set_cullmode(cullmode);


Argumentos:

Argumento Descripción
cullmode El modo de eliminación para usar (consulte la lista anterior)


Devoluciones:

N/A


Ejemplo:

if gpu_get_cullmode() != cull_clockwise
   {
   gpu_set_cullmode(cull_clockwise);
   }

El código anterior obtiene el modo de descarte actual y si no es cull_clockwise está configurado a esa constante.
