gpu_set_texrepeat


Descripción

Esta función se puede utilizar para indicar a GameMaker Studio 2 si las texturas deben repetirse ( true ) o no ( false ) cuando se usa para modelos y primitivas. Normalmente, los anchos y alturas de textura se encuentran en el rango 0 - 1, pero si especifica un valor mayor a 1 por defecto, el resto no se dibuja (toda la textura se dibujará con una sección "vacía" correspondiente al porcentaje sobre 1 que es Así que un valor de 2 para un ancho o alto de textura dibujaría la textura en la mitad del espacio y dejaría la otra mitad vacía. Sin embargo, al usar esta función para establecer la repetición en verdadero, la textura se repite automáticamente en todos los modelos y primitivas.

NOTA: Los sprites y fondos se dibujan siempre sin repetir, por lo que una vez que dibujas un sprite o fondo, este valor se restablece a false. Este comportamiento predeterminado no afectará la configuración de la etapa de textura (es decir, llamar a la función lo hará, pero la configuración interna a falso no).
NOTA: Esta configuración anulará cualquier repetición de escenario de textura establecida para los sombreadores cuando se usa la función gpu_set_texrepeat_ext.


Sintaxis:

gpu_set_texrepeat(enable);


Argumentos:

Argumento Descripción
enable Habilitar o deshabilitar el filtrado de texturas ( true / false )


Devoluciones:

N/A


Ejemplo:

if gpu_get_texrepeat()
   {
   gpu_set_texrepeat(false);
   }
else
   {
   gpu_set_texrepeat(true);
   }

El código anterior comprueba si la repetición de textura está activada o desactivada y la cambia en consecuencia.
