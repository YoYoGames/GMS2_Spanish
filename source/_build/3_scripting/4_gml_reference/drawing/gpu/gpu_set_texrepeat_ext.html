gpu_set_texrepeat_ext


Descripción

Esta función se puede usar para establecer si una "ranura" de una sola muestra repite la textura dada al usar Shaders en GameMaker Studio 2. Poniéndolo en true repetirá la textura si las coordenadas uv están fuera del rango 0-1, mientras que una configuración de false significará que no se repite.

NOTA: Esta configuración será anulada por el valor establecido al llamar a la función gpu_set_texrepeat.


Sintaxis:

gpu_set_texrepeat_ext(sampler_id, enable);


Argumentos:

Argumento Descripción
sampler_id La identificación de la muestra del sombreador.
enable Habilitar o deshabilitar el filtrado de texturas ( true / false )


Devoluciones:

N/A


Ejemplo:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
   {
   gpu_set_texfilter_ext(s_tex, false);
   }
else
   {
   gpu_set_texfilter_ext(s_tex, true);
   }

El código anterior comprueba si el filtrado de texturas está activado o desactivado para una ID de muestreador específica (almacenada en una variable local) y lo cambia en consecuencia.
