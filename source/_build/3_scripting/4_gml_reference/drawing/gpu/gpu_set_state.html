gpu_set_state


Descripción

Esta función configurará el estado actual de la GPU utilizando el ds_map. El mapa suministrado se puede crear usando la función gpu_get_state


Sintaxis:

gpu_set_state(ds_map);


Argumentos:

Argumento Descripción
ds_map El estado de GPU para establecer como ds_map.


Devoluciones:

N/A


Ejemplo:

gpu_set_state(gpu_map);

El código anterior establece el estado de la GPU utilizando el mapa proporcionado en la variable "gpu_map".
