gpu_get_blendmode


Descripción

Esta función se puede usar para recuperar el modo de mezcla actual que se usa para dibujar. El valor devuelto será una de las siguientes constantes (el valor predeterminado es bm_normal )

Constante Descripción Modo de mezcla extendida
bm_normal Mezcla normal (el modo de mezcla predeterminado). ( bm_src_alpha, bm_inv_src_alpha )
bm_add Mezcla aditiva Los valores de luminosidad de las áreas claras se agregan. ( bm_src_alpha, bm_one )
bm_subtract Mezcla sustractiva. Los valores de luminosidad de las áreas claras se restan. ( bm_zero, bm_inv_src_colour )
bm_max Mezcla máxima. Similar a la mezcla aditiva. ( bm_src_alpha, bm_inv_src_colour )


Sintaxis:

gpu_get_blendmode();


Devoluciones:

Constant (see above for constants)


Ejemplo:

if gpu_get_blendmode() != bm_normal
   {
   gpu_set_blendmode(bm_normal);
   }

El código anterior obtiene el modo de fusión actual y, si no es bm_normal está configurado a esa constante.
