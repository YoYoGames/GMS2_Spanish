gpu_set_texfilter


Descripción

Esta función se puede usar para establecer el filtrado de texturas (interpolación lineal) de todas las imágenes dibujadas en la pantalla del juego. Cuando esto está habilitado ( true ) todas las texturas se suavizarán cuando se dibujen (esto incluye sprites ya que también se consideran texturas), lo que significa que cuando se escala o se mueve si no hay una relación de píxel 1: 1, habrá una "mancha" en varios píxeles que puede las imágenes aparecen borrosas según el estilo artístico utilizado. Si esto está deshabilitado ( false ) entonces las imágenes se dibujarán basadas en el píxel más cercano cuando se escala o se mueve, lo que puede conducir a imágenes "en bloques". El valor predeterminado es false, y esto también se puede cambiar en las Global Game Options para plataformas de destino individuales.

NOTA: Esta configuración anulará cualquier conjunto de interpolación de etapa de textura para sombreadores que usen la función gpu_set_texfilter_ext.


Sintaxis:

gpu_set_texfilter(enable);


Argumentos:

Argumento Descripción
enable Habilitar o deshabilitar el filtrado de texturas ( true / false )


Devoluciones:

N/A


Ejemplo:

if gpu_get_texfilter()
   {
   gpu_set_texfilter(false);
   }
else
   {
   gpu_set_texfilter(true);
   }

El código anterior comprueba si el filtrado de texturas está activado o desactivado y lo cambia en consecuencia.
