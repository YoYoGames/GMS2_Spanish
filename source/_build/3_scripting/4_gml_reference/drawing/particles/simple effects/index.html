Efectos simples

La forma más fácil de crear partículas en tu juego con GameMaker Studio 2 es utilizar el mecanismo de efectos integrado. Los efectos se crean utilizando un sistema interno de partículas, que es básicamente un método muy rápido para dibujar efectos gráficos solamente y, como tal, no tiene que preocuparse por todos los detalles (como la gestión de la memoria) cuando utiliza estas funciones. Simplemente especifique el tipo de efecto, la posición donde debe crearse, el tamaño que desea que sea y, finalmente, su color, luego permita que GameMaker Studio 2 haga todo el trabajo.

Las siguientes funciones existen para crear los efectos de partículas incorporados:

  • effect_create_below
  • effect_create_above
  • effect_clear

También hay varias constantes diferentes para cada uno de los efectos predefinidos que las funciones anteriores pueden usar:

  • ef_cloud
  • ef_ellipse
  • ef_explosion
  • ef_firework
  • ef_flare
  • ef_rain
  • ef_ring
  • ef_smoke
  • ef_smokeup
  • ef_snow
  • ef_spark
  • ef_star

Aunque estos efectos tienen un alcance y personalización limitados, se pueden usar para crear efectos simples y grandes con muy poco esfuerzo. Por ejemplo, creando una pequeña bocanada de humo gris debajo de un misil en movimiento en cada paso, se crea un sendero de humo bastante convincente, por lo que incluso cuando eres un experto en partículas vale la pena recordar que estos efectos existen, ya que aún pueden salvarte un poco hora.
