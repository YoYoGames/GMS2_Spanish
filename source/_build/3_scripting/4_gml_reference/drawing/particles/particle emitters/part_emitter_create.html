part_emitter_create

Crea un nuevo emisor en el sistema de partículas dado.

Sintaxis:

part_emitter_create(ps);


Argumento Descripción
ps El sistema de partículas para crear el emisor en.


Devoluciones: Real


Descripción

Esta función debe usarse para crear un nuevo emisor y asignarlo a un sistema de partículas dado. La función devolverá el valor de índice para el nuevo emisor que debe almacenarse en una variable y se utilizará en todas las demás funciones que hacen referencia al emisor, y el propio emisor debe destruirse cuando ya no se utilice para evitar fugas de memoria (esto se puede lograr utilizando las funciones específicas de destrucción del emisor o destruyendo todo el sistema de partículas al que pertenece el emisor).


Ejemplo:

p_emit1 = part_emitter_create(Sname);

Esto creará un nuevo emisor de partículas y almacenará su índice en la variable "p_emit".
