part_system_automatic_draw

Establece si las partículas en el sistema de partículas deben dibujarse o no automáticamente.

Sintaxis:

part_system_automatic_draw(ind, automatic);


Argumento Descripción
ind El índice del sistema de partículas para cambiar.
automatic Si el dibujo automático está activado (verdadero) o no (falso).


Devoluciones: N / A


Descripción

Esta función se puede usar para desactivar el dibujo de un sistema de partículas, de modo que no se verán las actualizaciones hechas al sistema (automático o de otro tipo). Esta es una opción puramente visual y cuando se establece en false no podrá ver las partículas ya que no están dibujadas, pero aún existen y están cambiando de posición, color, etc., como lo harían normalmente. Cuando el dibujo automático está desactivado, puede ordenar explícitamente a GameMaker Studio 2 que dibuje el estado actual del sistema de partículas usando la función part_system_drawit, y si configura esta función para true nuevamente puedes volver a activar el dibujo automático.

Una cosa a tener en cuenta es que si está utilizando los efectos simples creados por las funciones effect_create_above o effect_create_below entonces puede usar los valores de 0 (para los efectos inferiores) o 1 (para los efectos anteriores) como índice del sistema de partículas y así desactivar el sorteo automático para estos también (esto también funcionará para alternar el dibujo para los efectos de partículas DnD™).


Ejemplo:

part_system_automatic_draw(global.Sname, false);

El código anterior desactiva el dibujo automático para el sistema de partículas indexado en la variable global "Sname".
