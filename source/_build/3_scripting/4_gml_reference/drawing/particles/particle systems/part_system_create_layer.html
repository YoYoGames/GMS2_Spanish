part_system_create_layer


Descripción

Esta función creará un nuevo sistema de partículas en una capa determinada. Usted otorga la ID de capa única que devuelve la función layer_create() o el nombre de la capa para usar como una cadena, por ejemplo "instancia_capa", y luego marcar el sistema como persistente o no. Si el sistema se marca como persistente, persistirá en todas las salas después de que se haya creado (y se le haya asignado una capa administrada si la capa inicial no existe en salas subsiguientes), pero si la persistencia no está marcada, entonces el sistema será se destruye automáticamente al final de la sala en la que se creó (esto es lo mismo que si llamaras a la función part_system_destroy y también destruirá cualquier emisor asociado con el sistema). La función devolverá un valor de ID único para el sistema de partículas que se debe usar en todas las llamadas a funciones adicionales en las que necesite dar una ID de sistema.

¡IMPORTANTE! Si marca el sistema de partículas como persistente, entonces (y cualquier emisor asignado a él) tendrá que limpiarse manualmente utilizando las funciones de destrucción apropiadas cuando no esté en uso, de lo contrario correrá el riesgo de una pérdida de memoria que afectará negativamente a su juego final.


Sintaxis:

part_system_create_layer(layer_id, persistent);

Argumento Descripción
layer El valor de ID de capa o cadena para asignar el sistema de partículas a (puede ser cualquier tipo de capa)
persistent Marcar el sistema de partículas como persistente (configurado en true ) o no (configurado en false )


Devoluciones:

Real (unique particle system ID value)


Ejemplo:

global.p_sys = part_system_create_layer("effects_layer", true);

El código anterior creará un nuevo sistema de partículas en la capa dada y lo marcará como persistente en salas posteriores. La ID para el sistema de partículas se almacena en una variable de ámbito global para referencia futura.
