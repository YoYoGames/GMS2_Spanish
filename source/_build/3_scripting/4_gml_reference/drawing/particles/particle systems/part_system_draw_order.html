part_system_draw_order

Establece el orden en que se dibujan las nuevas partículas.

Sintaxis:

part_system_draw_order(ind, oldtonew);


Argumento Descripción
ind El índice del sistema de partículas para cambiar.
oldtonew Si las partículas viejas deben dibujarse detrás de las más nuevas (verdadero) o viceversa (falso).


Devoluciones: N / A


Descripción

Con esta función puede establecer la forma en que se dibujan las partículas cuando se crean en la pantalla. El sistema predeterminado usa un viejo> nuevo aspecto (la función está configurada para true ), donde las viejas partículas se dibujan a una profundidad mayor que las más nuevas y aparecen "debajo" de las nuevas partículas, pero al establecer esta función en false puede invertir este orden y hacer que las nuevas partículas se dibujen más arriba y aparezcan "debajo" de las anteriores. Las imágenes a continuación ilustran esto, con la imagen de la izquierda siendo el valor predeterminado de true y la imagen a la derecha siendo false:


NOTA: Cuando las partículas se dibujan con un modo de combinación aditiva, el efecto de esta función puede no ser siempre obvio.


Ejemplo:

mysystem = part_system_create();
part_system_clear( mysystem, 1 );

Esto creará un nuevo sistema de partículas con el índice mysystem. Luego establecerá sus partículas para dibujar partículas más nuevas sobre las más antiguas.
