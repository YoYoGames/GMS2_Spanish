part_system_clear

Borra un sistema de partículas a su configuración predeterminada.

Sintaxis:

part_system_clear(ind);


Argumento Descripción
ind El índice del sistema de partículas para borrar.


Devoluciones: N / A


Descripción

Con esta función puede borrar el sistema indexado a su estado predeterminado, eliminar todos los emisores y restablecer la profundidad y la posición (si se han cambiado). Tenga cuidado al usar esta función como si tuviera cualquier configuración de instancia o explosión o cualquier otra acción usando un emisor asociado con este sistema, obtendrá un error a menos que esté usando el part_emitter_exists función para verificar Tampoco es necesario llamar al part_emitter_destroy función ya que esto también se soluciona automáticamente.

NOTA: esta función eliminará las partículas visibles en la sala, pero no borrará las propiedades de la partícula, ni las eliminará de la memoria. Para eso deberías usar las funciones part_type_clear y part_type_destroy.


Ejemplo:

part_system_clear(global.Sname);

El código anterior borrará el sistema de partículas indexado en la variable global "Sname" a su estado predeterminado.
