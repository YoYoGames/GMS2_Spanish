part_particles_clear

Borra todas las partículas en el sistema.

Sintaxis:

part_particles_clear(ind);


Argumento Descripción
ind El índice del sistema de partículas.


Devoluciones: N / A


Descripción

Con esta función, puede eliminar todas las partículas creadas actualmente por el sistema de la sala. No restaurar o eliminar los tipos de partículas en sí, sólo su representación visual, y si usted tiene cualquier objeto la transmisión de partículas de un emisor, estas partículas desaparecen, pero comenzará a aparecer de nuevo el siguiente paso después de llamar a este código.


Ejemplo:

if lives = 0
   {
   part_particles_clear(global.Sname);
   room_goto(rm_intro);
   }

El código anterior verificará el valor de la variable "vidas" y si es igual a 0, borra todas las partículas del sistema y luego cambia de habitación.
