part_particles_create

Crea partículas desde un sistema en una posición dada.

Sintaxis:

part_particles_create(ind, x, y, parttype, number);


Argumento Descripción
ind El índice del sistema de partículas.
x La coordenada x de dónde crear las partículas.
y La coordenada y de dónde crear las partículas.
parttype El índice (tipo) de las partículas que se crearán.
number La cantidad de partículas para crear.


Devoluciones: N / A


Descripción

Esta función es ideal para aquellos efectos que no requieren ninguna de las funciones ofrecidas por los emisores de partículas (por ejemplo, para crear humo a partir de un misil, o un simple efecto de explosión) ya que le permite crear partículas rápida y fácilmente en cualquier posición en la sala de juegos. Tenga en cuenta que debe haber creado el sistema de partículas y el tipo de partícula previamente para esta función que se utilizará.


Ejemplo:

if mouse_check_button(mb_left)
   {
   part_particles_create(sname, mouse_x, mouse_y, p_CursorEffect, 5);
   }

El código anterior comprueba si se presiona el botón del mouse y si vuelve true genera 5 partículas en la posición del ratón.
