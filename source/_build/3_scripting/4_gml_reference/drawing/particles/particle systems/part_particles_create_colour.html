part_particles_create_colour

Crea partículas desde un sistema en una posición dada y con un color dado.

Sintaxis:

part_particles_create_colour(ind, x, y, parttype, colour, number);


Argumento Descripción
ind El índice del sistema de partículas.
x La coordenada x de dónde crear las partículas.
y La coordenada y de dónde crear las partículas.
parttype El índice (tipo) de las partículas que se crearán.
colour La combinación de colores para las partículas.
number La cantidad de partículas para crear.


Devoluciones: N / A


Descripción

Esta función es ideal para aquellos efectos que no requieren ninguna de las funciones ofrecidas por los emisores de partículas (por ejemplo, para crear humo a partir de un misil, o un simple efecto de explosión) ya que le permite crear partículas rápida y fácilmente en cualquier posición en la sala de juegos. También puede colorear las partículas "sobre la marcha", ya que se crean con esta función, y este color anulará el color predefinido de la partícula, pero no combina este color con la vida útil de las partículas. Tenga en cuenta que debe haber creado el sistema de partículas y el tipo de partícula previamente para esta función que se utilizará.


Ejemplo:

if speed > 1
   {
   part_particles_create_colour(sname, x, y, p_MagicSmoke, choose(c_aqua, c_lime, c_fuschia, c_orange), 1);
   }

El código anterior generará una sola partícula en cada paso que la instancia con el código tenga una velocidad mayor a uno. Estas partículas serán de un color aleatorio.
