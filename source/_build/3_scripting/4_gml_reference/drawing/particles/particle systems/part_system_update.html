part_system_update

Actualiza las posiciones de las partículas en un sistema dado.

Sintaxis:

part_system_update(ind);


Argumento Descripción
ind El índice del sistema de partículas para actualizar.


Devoluciones: N / A


Descripción

Esta función hará avanzar el sistema de partículas dado un paso en el tiempo de juego y puede usarse para simular el comportamiento del sistema de partículas cuando la actualización automática está desactivada, o puede usarse para avanzar un sistema de partículas a un punto específico, todo en un solo paso. Por ejemplo, si tiene un sistema de partículas para nieve y desea que parezca que la nieve cae desde el momento en que el jugador entra en la habitación, entonces puede avanzar las partículas de nieve durante unos segundos en el evento de creación de una instancia para que la primera vez que se dibujen, haya partículas en toda la pantalla (el siguiente ejemplo de código muestra esto). También puede usar esta función cuando la función de actualización automática del sistema está desactivada part_system_automatic_update para mover el sistema.


Ejemplo:

repeat (room_speed * 3)
   {
   part_system_update(global.RainSys);
   }

El código anterior hará avanzar el sistema de partículas indexado en la variable global "RainSys" en 3 segundos.
