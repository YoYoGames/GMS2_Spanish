part_type_gravity

Establece la gravedad del tipo de partícula dado.

Sintaxis:

part_type_gravity(ind, grav_amount, grav_direction);


Argumento Descripción
ind El índice del tipo de partícula para cambiar.
grav_amount Fuerza de la gravedad
grav_direction La dirección de la gravedad


Devoluciones: N / A


Descripción

Esta función establecerá la gravedad que afectará a cada partícula del tipo dado que se crea. El valor de la fuerza de gravedad se agrega a la velocidad de la partícula en cada paso y suele ser un valor pequeño como 0.5, mientras que la dirección es la dirección de la gravedad "pull" y sigue las direcciones estándar de GameMaker Studio 2 de 0 °, siendo 90 ° arriba, 180 ° a la izquierda y 270 ° hacia abajo.


Ejemplo:

part_type_shape(global.p1, pt_shape_pixel);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

El código anterior establecerá varios valores de partículas, incluida la gravedad, que agregará 0,2 a la velocidad de cada paso con una dirección de 270 ¿, por lo que arrastrará la partícula "hacia abajo" hacia la parte inferior de la pantalla.
