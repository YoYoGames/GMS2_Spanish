Tipos de partículas

Las partículas son los efectos gráficos que pueden crearse mediante el uso de sistemas de partículas en un juego. Las partículas que define utilizando estas funciones se pueden usar en cualquier sistema que cree y el procedimiento normal sería definir todas sus partículas una vez al comienzo de un juego y almacenar sus valores de índice en variables globales. Estas variables globales se pueden usar para reventar, emitir o crear partículas en cualquier momento del juego. También puede crear, usar y luego destruir partículas según sea necesario si la memoria es un problema o si desea muchos tipos diferentes, pero solo en puntos específicos de su juego. ¡Elija lo que elija, las partículas son un recurso muy versátil!

Las partículas deben crearse y tener su índice almacenado en una variable para que todas las demás rutinas de partículas puedan acceder a ellas correctamente. Las siguientes funciones se usan para crear, verificar, borrar y destruir tipos de partículas:

  • part_type_exists
  • part_type_create
  • part_type_destroy
  • part_type_clear

Una vez que haya creado su tipo de partícula, debe darle una forma. Esta puede ser una de las 14 formas incorporadas que son sprites de 64x64 píxeles con mezcla alfa, o puede usar sus propios sprites que pueden ser animados, estáticos o usar subimágenes aleatorias. Las siguientes funciones definen la forma de la base visual de cada partícula:

  • part_type_shape
  • part_type_sprite

Una vez que haya elegido el sprite para el tipo de partícula (ya sea una forma predeterminada o la suya) puede indicar el tamaño y la escala de la misma. El tamaño se puede especificar como un rango y todas las partículas se crearán con un tamaño elegido dentro del rango dado. La escala es el valor "base" para la escala del eje xey de la partícula antes de que se agreguen las variaciones de tamaño y las siguientes funciones se utilizan para controlar estos valores:

  • part_type_size
  • part_type_scale

Las partículas también pueden tener una orientación y movimiento. Al igual que con la mayoría de las funciones de partículas, la velocidad, la dirección y la orientación se resuelven como un valor aleatorio que se selecciona de un rango determinado y se puede configurar con las siguientes funciones:

  • part_type_speed
  • part_type_direction
  • part_type_gravity
  • part_type_orientation

Una vez que haya ordenado el estilo y el movimiento de las partículas, puede establecer sus valores de color y alfa, que se pueden establecer como un valor absoluto durante toda la vida útil de la partícula, o como una escala de valores que va desde uno al otro gradualmente Tenga en cuenta que también hay una función especial que le permite configurar la mezcla de partículas y sus alrededores, y esto puede ser normal o aditivo y proporciona algunos efectos fantásticos: las funciones que rigen el color, el alfa y la fusión son:

  • part_type_colour_mix
  • part_type_colour_rgb
  • part_type_colour_hsv
  • part_type_colour1
  • part_type_colour2
  • part_type_colour3
  • part_type_alpha1
  • part_type_alpha2
  • part_type_alpha3
  • part_type_blend

Las partículas existen en la pantalla durante un tiempo limitado y esto se llama su "vida útil". La vida útil de las partículas se mide en pasos y (al igual que las funciones previas) esto se expresa como un rango de valores a partir de los cuales a la partícula se le asignará una duración específica. También hay un par de funciones interesantes que pueden usarse para hacer que las partículas generen otras partículas, ya sea en cada paso o cuando "mueren" (es decir: se eliminan de la pantalla), pero tenga mucho cuidado con estas funciones, ya que pueden crear rápidamente miles de partículas y rompen el juego! Las funciones que establecen la duración y la emisión de pasos o muerte son las siguientes:

  • part_type_life
  • part_type_step
  • part_type_death

Vale la pena señalar que la creación de partículas a través del código a menudo puede ser un proceso laborioso y lento debido a la gran cantidad de parámetros que pueden modificarse y ajustarse. Por esa razón, se recomienda que busque y controle algún tipo de Diseñador de partículas (¡o cree uno usted mismo!) Para eliminar la carga de trabajo de la creación de partículas.
