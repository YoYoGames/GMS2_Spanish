part_type_exists

Verifica si existe un tipo de partícula dado.

Sintaxis:

part_type_exists(ind);


Argumento Descripción
ind El índice del tipo de partícula a verificar.


Devoluciones: Boolean


Descripción

Con esta función puedes verificar si el tipo de partícula dado indexado existe en el juego o no. Tenga en cuenta que si la variable que se está verificando es una variable no inicializada (a la que un tipo de partícula tendría asignado su índice), se generará un error.


Ejemplo:

if !part_type_exists(global.p1)
   {
   global.p1 = part_type_create();
   }

El código anterior comprueba si la variable global "p1" almacena el índice de un tipo de partícula y, si no lo hace, crea uno.
