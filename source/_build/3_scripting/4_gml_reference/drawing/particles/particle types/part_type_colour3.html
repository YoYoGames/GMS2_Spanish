part_type_colour3

Establece un tipo de partícula para fundirse entre tres colores a lo largo de su vida útil.

Sintaxis:

part_type_colour3( ind, colour1, colour2, colour3 );


Argumento Descripción
ind El índice del tipo de partícula para cambiar.
colour1 El color en el que comenzará la partícula.
colour2 El color de la partícula será a la mitad de su vida útil.
colour3 El color al que terminará la partícula.


Devoluciones: N / A


Descripción

Esta función se puede usar para establecer un degradado de tres colores para cada partícula creada del tipo dado. El primer color es aquel con el que comenzarán todas las partículas, y el segundo color es el que se mezclará a la mitad de su duración y el tercer color con el que terminará la partícula. Se producirá un cambio de gradiente suave a través de los colores durante la vida útil de las partículas de un color a otro.


Ejemplo:

part_type_colour3( part_Health, c_red, c_white, c_maroon);

El código anterior hará que todas las partículas creadas del tipo indexado en la variable "part_Health" tengan una combinación de colores de rojo a blanco a marrón durante su vida útil.
