part_type_direction

Establece las propiedades de dirección para el tipo de partícula dado.

Sintaxis:

part_type_direction(ind, dir_min, dir_max, dir_incr, dir_wiggle);


Argumento Descripción
ind El índice del tipo de partícula para cambiar.
dir_min La dirección mínima en la que puede comenzar la partícula.
dir_max La dirección máxima en la que puede comenzar la partícula.
dir_incr Cuánto la dirección de la partícula debe aumentar o disminuir por paso.
dir_wiggle Cuánto debe agregarse o quitarse aleatoriamente de la dirección de la partícula por paso.


Devoluciones: N / A


Descripción

Esta función se usa para determinar la dirección de la partícula cuando se crea y también se puede usar para hacer que la partícula aumente o disminuya su dirección durante su vida útil. Los valores mínimos y máximos de la dirección predeterminados en 0, lo que significa que la partícula se moverá hacia la derecha sin variación, pero estos valores se pueden cambiar para limitar la dirección de movimiento siguiendo las direcciones estándar de 0 de GameMaker Studio 2, siendo 90 arriba, 180 siendo izquierda y 270 abajo. Si los configura al mismo valor, las partículas solo se moverán en una dirección.

También puede establecer un valor de incremento que agregará (si es un número positivo) o restará (si es un número negativo) una cantidad de grados a la dirección durante su vida útil. Este valor puede ser un mínimo de (+/-) 0.01.

Finalmente, puede establecer el factor "meneo". Este es un valor que se agregará o restará al azar en la dirección de cada paso de la vida de las partículas. Obviamente, los valores más grandes son más pronunciados que los más pequeños, y este valor puede ser incluso negativo, con un rango máximo entre -20 y 20.


Ejemplo:

part_type_shape(particle1, pt_shape_pixel);
part_type_size(particle1, 0.19, 0.79, 0.01, 0);
part_type_scale(particle1, 5, 5);
part_type_colour3(particle1, 32768, 8454016, 8454016);
part_type_alpha3(particle1, 0, 0.80, 0);
part_type_speed(particle1, 8, 10, -0.50, 1);
part_type_direction(particle1, 0, 359, 1, 20);
part_type_gravity(particle1, 0, 270);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 50);

El código anterior establecerá varios valores de partículas, incluida la dirección, que será un valor aleatorio entre 0 y 359 grados. También subirá 1 grado cada paso y se le agregará una cantidad aleatoria de entre 0 y 20 en cada paso.
