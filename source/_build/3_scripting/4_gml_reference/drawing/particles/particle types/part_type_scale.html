part_type_scale

Establece la escala horizontal y vertical del tipo de partícula.

Sintaxis:

part_type_scale(ind, xscale, yscale);


Argumento Descripción
ind El índice del tipo de partícula para cambiar.
xscale La escala horizontal de la partícula base.
yscale La escala vertical de la partícula base.


Devoluciones: N / A


Descripción

Esta función establecerá la escala horizontal y vertical de la partícula antes de que se realicen otros cambios de tamaño. Esta función funciona en la proporción del tamaño del sprite, por lo que un valor de 1 (el valor predeterminado) es una relación de 1: 1, lo que significa que 0.5 sería la mitad y 2 dobles.


Ejemplo:

part_type_shape(particle1, pt_shape_flare);
part_type_size(particle1, 2, 2, 0.10, 1);
part_type_scale(particle1, 8, 0.50);
part_type_colour3(particle1, 8388608, 4227327, 8454143);
part_type_alpha3(particle1, 0.10, 0.10, 0);
part_type_speed(particle1, 0, 1, 0.18, 20);
part_type_direction(particle1, 0, 359, 0, 0);
part_type_orientation(particle1, 0, 0, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 40);

El código anterior establecerá varios valores de partículas, incluida la escala que establece la escala x ocho veces el tamaño y la escala y a la mitad del tamaño.
