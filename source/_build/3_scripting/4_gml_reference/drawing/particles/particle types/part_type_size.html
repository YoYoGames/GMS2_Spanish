part_type_size

Establece el tamaño de inicio mínimo y máximo del tipo de partícula.

Sintaxis:

part_type_size(ind, size_min, size_max, size_incr, size_wiggle);


Argumento Descripción
ind El índice del tipo de partícula para cambiar.
size_min El tamaño mínimo en el que puede comenzar la partícula.
size_max El tamaño máximo en el que puede comenzar la partícula.
size_incr Cuánto debe aumentar o disminuir la partícula por paso.
size_wiggle Cuánto debe agregarse o quitarse al azar del tamaño de las partículas por paso.


Devoluciones: N / A


Descripción

Esta función se usa para determinar el tamaño de la partícula cuando se crea y también se puede usar para hacer que la partícula aumente o disminuya en tamaño durante su vida útil. Los valores mínimo y máximo de tamaño predeterminado en 1 (lo que significa que el tamaño es la misma proporción que la partícula base después de la part_type_scale escala se ha establecido) y si se establecen en el mismo valor, el tamaño inicial para todas las partículas será el mismo, pero si se establece en valores diferentes, cada partícula comenzará con un tamaño aleatorio dentro del rango, por ejemplo, si tener un mínimo de 0.5 y un máximo de 2, todas las partículas se crearán con un tamaño entre la mitad y el doble de la escala base.

También puede establecer un valor de incremento que hará que el sprite de la partícula crezca (si es un número positivo) o se encoja (si es un número negativo) a lo largo de la vida de la partícula, añadiéndose la cantidad dada al tamaño de partícula en cada paso. Este valor puede ser un mínimo de (+/-) 0.01.

Finalmente, puede establecer el factor "meneo". Este es un valor que se agregará o restará aleatoriamente al tamaño de cada paso de la vida de las partículas. Obviamente, los valores más grandes son más pronunciados que los más pequeños, y este valor puede ser incluso negativo, con un rango máximo entre -20 y 20.


Ejemplo:

part_type_shape(particle1, pt_shape_explosion);
part_type_size(particle1, 2, 2, 0.10, 0.01);
part_type_scale(particle1, 1, 1);
part_type_colour2(particle1, 16777215, 12632256);
part_type_alpha3(particle1, 0, 0.20, 0);
part_type_speed(particle1, 0.20, 0.20, 0, 0);
part_type_direction(particle1, 0, 359, 0, 20);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 0);
part_type_life(particle1, 40, 60);

El código anterior establecerá varios valores de partículas, incluido el tamaño que se establece para duplicar el tamaño cuando se crea cada partícula, y luego incrementar el tamaño en 0,01 en cada paso con un movimiento de 0,1 también.
