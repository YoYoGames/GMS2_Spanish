surface_getpixel_ext


Sintaxis:

surface_getpixel_ext(surface_id, x, y);

Argumento Descripción
surface_id ID de la superficie a usar
x La coordenada x del píxel para verificar
y La coordenada y del píxel para verificar


Devoluciones:

Real


Descripción

Con esta función puede obtener el valor abgr completo de 32 bits de cualquier píxel de una superficie (creada previamente).

NOTA: Esta función es lenta y solo debe usarse cuando sea absolutamente necesario.


Ejemplo:

col = surface_getpixel_ext(mouse_x, mouse_y);
alpha = (col >> 24) & 255;
blue = (col >> 16) & 255;
green = (col >> 8) & 255;
red = col & 255;

El código anterior obtendrá el valor del color de 32 bits en la posición del mouse y luego lo dividirá en sus valores de componentes, almacenándolos en variables.
