surface_free


Sintaxis:

surface_free(surface_id);

Argumento Descripción
surface_id La identificación de la superficie que se liberará.


Devoluciones:

N/A


Descripción

Cuando trabaje con superficies, siempre debe usar esta función cuando haya terminado de usarlas. Las superficies ocupan espacio en la memoria y necesitan ser removidas, normalmente al final de una habitación, pero se puede hacer en cualquier momento dependiendo del uso que se les dé. De lo contrario, pueden producirse fugas de memoria que eventualmente ralentizarán y colapsarán tu juego.

NOTA: Al trabajar con superficies existe la posibilidad de que dejen de existir en cualquier momento debido a que se almacenan en la memoria de textura. SIEMPRE debe verificar que exista una superficie usando surface_exists antes de hacer referencia a ellos directamente. Para más información, consulte Superficies.


Ejemplo:

if keyboard_check_pressed(vk_escape)
   {
   surface_free(surf);
   room_goto(rm_Menu);
   }

El código anterior comprueba si se presiona una tecla y, si detecta una, libera la memoria reservada para la superficie indexada en la variable "navegar" y luego cambia de habitación.
