application_surface_draw_enable


Sintaxis:

application_surface_draw_enable(flag);

Argumento Descripción
flag La bandera será cualquiera true (habilitado, el valor predeterminado) o false (discapacitado).


Devoluciones:

N/A


Descripción

Puede usar esta función para habilitar o deshabilitar el dibujo automático de la superficie de la aplicación. De manera predeterminada, esto está habilitado, pero en muchos casos querrá tomar el control de cuándo y cómo se dibuja la superficie (cuando se usan sombreadores, por ejemplo), en cuyo caso usaría esta función para establecerlo en falso para que pueda dibujarlo usted mismo cuándo y cómo lo necesita.


Ejemplo:

application_surface_draw_enable(false);

El código anterior desactivará el dibujo automático de la superficie de la aplicación.
