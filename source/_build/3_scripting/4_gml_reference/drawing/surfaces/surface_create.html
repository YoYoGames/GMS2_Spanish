surface_create


Sintaxis:

surface_create(w, h);

Argumento Descripción
w El ancho de la superficie que se creará
h La altura de la superficie que se creará.


Devoluciones:

Real


Descripción

Esta función se usa para crear una superficie y devolverá el índice de la superficie que se debe almacenar en una variable para futuras llamadas a funciones. Cuando se crea la superficie por primera vez, puede contener "ruido" ya que básicamente es solo un área de memoria que se deja de lado para ese propósito (y esa memoria puede contener información), por lo que es posible que desee despejar la superficie antes de usarla. una función como draw_clear_alpha.

Es muy recomendable que todas las superficies se creen con un tamaño que sea una potencia de 2, es decir, 16, 128, 512 o 1024 píxeles de tamaño, por ejemplo. Esto no es exactamente necesario en ciertas plataformas (como Windows y MacOS), pero sin duda aumentará la compatibilidad en esos objetivos, mientras que para HTML5 y dispositivos es esencial y es muy importante que recuerde esto o puede tener problemas más adelante.


Ejemplo:

if !surface_exists(surf)
   {
   surf = surface_create(1024, 1024);
   surface_set_target(surf);
   draw_clear_alpha(c_black, 0);
   surface_reset_target();
   view_surface_id[0] = surf;
   }

El código anterior comprueba si existe una superficie y, si no, creará una superficie de 1024 píxeles de ancho y 1024 píxeles de alto, asignando el índice a la variable "navegar". El objetivo del dibujo se establece en la nueva superficie, que se borra y se hace transparente antes de que el objetivo del dibujo se reinicie en la pantalla. Finalmente, la superficie se asigna a una vista.
