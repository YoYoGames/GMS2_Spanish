surface_exists


Sintaxis:

surface_exists(surface_id);

Argumento Descripción
surface_id La identificación de la superficie para verificar


Devoluciones:

Boolean


Descripción

Esta función es esencial cuando se trabaja con superficies debido a su naturaleza volátil. Las superficies siempre se guardan en la memoria de texturas, lo que significa que pueden destruirse de un momento a otro (por ejemplo, cuando un salvapantallas se inicia en Windows o cuando se minimiza en un dispositivo Android ), por lo que siempre debe verificar que exista una superficie antes haciendo algo con eso (esto incluye dibujarlo en la pantalla). El siguiente código de ejemplo muestra un uso típico de este comando en el evento draw de una instancia para verificar una superficie y volver a crearla si se ha eliminado (tenga en cuenta que la superficie se habrá creado originalmente en el evento create del objeto) )


Ejemplo:

if !surface_exists(surf)
   {
   surf = surface_create(1024, 1024);
   }
else
   {
   if view_current = 1
      {
      draw_surface(surf,0,0);
      }
   }

El código anterior verificará si existe una superficie indexada en la variable "navegar", y si no lo hace, la volverá a crear. Si existe, luego verifica qué vista se está dibujando actualmente y si es vista [1] dibuja la superficie.
