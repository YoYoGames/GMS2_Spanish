surface_set_target


Sintaxis:

surface_set_target(surface_id);

Argumento Descripción
surface_id ID de la superficie para establecer como destino del dibujo.


Devoluciones:

Boolean


Descripción

Con esta función, configura todo el dibujo adicional en la superficie del objetivo en lugar de hacerlo en la pantalla y de esta manera puede decirle a GameMaker Studio 2 que solo dibuje cosas específicas en la superficie especificada. Tenga en cuenta que si no llama surface_reset_target una vez que haya terminado, no se dibujará nada en la pantalla ya que todo el dibujo posterior (incluso en otras instancias) se realizará en la superficie. También debe saber que no se verá nada si la superficie no se dibuja en la pantalla en el evento de dibujo de una instancia. También puede verificar el valor de retorno de esta función como una herramienta de depuración para verificar si el objetivo de superficie fue establecido o no, con un valor de retorno de 0 que no permite establecer el objetivo y que cualquier otro valor positivo sea exitoso.

Una cosa que debe tenerse en cuenta es que las superficies están apiladas por lo que no puede saltar de un objetivo a otro y luego restablecer al objetivo de extracción normal al final, sino que debe abrir y cerrar los objetivos de representación. Por ejemplo, esto no funcionará correctamente:

surface_set_target(surf1);
draw_text(32, 32, "surface1");
surface_set_target(surf2);
draw_text(32, 64, "surface2");
surface_reset_target();

En su lugar, debe restablecer el objetivo para cada una de las superficies que configure, al igual que debe usar los corchetes de apertura y cierre {} para los bloques de código. Entonces, lo anterior debe escribirse como esto:

surface_set_target(surf1);
draw_text(32, 32, "surface1");
surface_reset_target();
surface_set_target(surf2);
draw_text(32, 64, "surface2");
surface_reset_target();

o como esto:

surface_set_target(surf1);
draw_text(32, 32, "surface1");
surface_set_target(surf2);
draw_text(32, 64, "surface2");
surface_reset_target();
surface_reset_target();

NOTA: Al trabajar con superficies existe la posibilidad de que dejen de existir en cualquier momento debido a que se almacenan en la memoria de textura. SIEMPRE debe verificar que exista una superficie usando surface_exists antes de hacer referencia a ellos directamente. Para más información, consulte Superficies.


Ejemplo:

if view_current = 0
   {
   surface_set_target(surf);
   with (obj_Effect)
      {
      draw_self();
      }
   surface_reset_target();
   }
else
   {
   draw_surface(surf, 0, 0);
   }

El código anterior verificará qué vista se está dibujando actualmente, y si es view [0] establece el objetivo de dibujo en una superficie y dibuja todas las instancias del objeto "obj_Effect" antes de restablecer el objetivo de dibujo nuevamente. Si la vista no es vista [0], la superficie se dibuja en la pantalla.
