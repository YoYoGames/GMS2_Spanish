surface_save_part


Sintaxis:

surface_save_part(surface_id, fname, x, y, width, height);

Argumento Descripción
surface_id ID de la superficie para establecer como destino del dibujo.
fname El nombre del archivo de imagen guardado.
x La posición x inicial dentro de la superficie.
y La posición inicial de y dentro de la superficie.
width El ancho de la parte para guardar.
height La altura de la parte para guardar.


Devoluciones:

N/A


Descripción

Esta función guardará una parte de una superficie en un disco usando el nombre de archivo dado. La superficie debe guardarse como una *.png archivo de formato, y la posición (x, y) debe darse como coordenadas locales a la superficie, teniendo en cuenta que la esquina superior izquierda de la superficie es siempre (0,0).


Ejemplo:

if keyboard_check_pressed(ord("S")
   {
   surface_save_part(surf, "test.png", 0, 0, 100, 100);
   }

El código anterior verificará si el usuario presiona la tecla "S" en el teclado y, si lo hace, guardará una parte de la superficie indexada en la variable "navegar" en el disco.
