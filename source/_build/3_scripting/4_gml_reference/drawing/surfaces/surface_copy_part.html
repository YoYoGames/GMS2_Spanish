surface_copy_part


Sintaxis:

surface_copy_part(destination, x, y, source, xs, ys, ws, hs);

Argumento Descripción
destination El ID de la superficie para copiar la otra superficie a.
x La posición x para copiar.
y La posición y para copiar a.
source El ID de la superficie a copiar.
xs La posición x en la superficie de origen para copiar.
ys La posición y en la superficie de origen para copiar.
ws El ancho del área en la superficie de origen para copiar.
hs La altura del área en la superficie de origen para copiar.


Devoluciones:

N/A


Descripción

Esta función simplemente toma la imagen de una superficie y la copia en otra en la posición local especificada dentro de esa superficie (donde la posición (0,0) es la esquina superior izquierda de la superficie de destino). Puede especificar una posición x y y local para copiar, así como el ancho y la altura de la sección. Tenga en cuenta que estas coordenadas se basan en el tamaño de la superficie y no en la posición en la que se dibuja la superficie en la sala (consulte la página principal en Superficies para obtener más información). Si la superficie de destino ya tiene información, la copia la sobrescribirá y la función no cambiará la superficie de origen de ninguna manera.

NOTA: Cuando se trabaja con superficies, existe la posibilidad de que dejen de existir en cualquier momento debido a que se almacenan en la memoria de textura. SIEMPRE debe verificar que exista una superficie usando surface_exists Antes de referenciarlos directamente. Para más información ver Superficies.


Ejemplo:

if view_current == 0
    {
    surface_copy_part(surf, 0, 0, temp_surf, 0, 0, view_xview[1] - mouse_x, view_yview[1] - mouse_y);
    }
else
    {
    draw_surface(surf, 0, 0);
    }

El código anterior verificará la vista actual que se está dibujando y si es vista [0] copia la superficie indexada en la variable "temp_surf" en la superficie indexada en la variable "surf". El área copiada corresponde a un rectángulo formado por la posición relativa del ratón dentro de la superficie tal como se dibujaría en la vista [1]. Si la vista actual es diferente a la vista [0], la superficie "navegar" se dibuja en la pantalla.
