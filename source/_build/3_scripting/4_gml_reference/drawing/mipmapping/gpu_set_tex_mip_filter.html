gpu_set_tex_mip_filter


Descripción

Con esta función, puede establecer el modo de filtro mip actual en uno de los tres tipos admitidos. Proporciona la constante que se refiere al modo de filtrado de mip que necesita, donde puede elegir entre filtrado de puntos (configuración predeterminada, es decir, sin filtrado), filtrado lineal (también conocido como filtrado trilineal ) o filtrado aniostrópico (consulte la tabla de constantes abajo).

Constante Descripción
tf_point Esto deshabilitará el filtrado entre los niveles de mipmap, lo que puede causar transiciones de textura visibles, pero ofrece el mejor rendimiento.
tf_linear Esto permitirá el filtrado lineal entre los niveles de mipmap (esto también se conoce como filtrado trilineal ), que suaviza las transiciones de textura, pero dará un impacto menor al rendimiento.
tf_anisotropic Esto permitirá el filtrado anisotrópico entre los niveles de mipmap, lo que mejora en gran medida la calidad de transición de la textura y puede reducir el desenfoque visible con otros modos de filtrado, pero tiene el mayor impacto en el rendimiento.


Sintaxis:

gpu_get_tex_mip_filter(filter);

Argumento Descripción
filter El modo de filtro mip a utilizar (una constante, por defecto: tf_point ).


Devoluciones:

N/A


Ejemplo:

if keyboard_check(ck_enter)
    {
    switch(gpu_get_tex_mip_filter())
        {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
        }
    }

El código anterior verifica el teclado y, si se presiona la tecla especificada, obtendrá el filtro mip actual y cambiará el valor al siguiente, alternando entre los diferentes modos.
