gpu_get_tex_max_mip_ext


Descripción

Con esta función, puede obtener el nivel de mipmap máximo actualmente configurado que se utilizará para una muestra de sombreador determinada. Usted proporciona el valor del índice para la muestra del sombreador (como lo devuelve la función shader_get_sampler_index(), y la función devolverá el nivel de mipmap máximo actual, donde un valor de 0 es la resolución más alta, 1 es usar el primer mipmap, 2 es el segundo, etc.... Tenga en cuenta que esto puede ser bastante útil para evitar artefactos de sangrado cuando renderizar texturas, por ejemplo, establecer el borde de la página de textura en 8px y luego establecer el nivel máximo de mipmap en 3 asegurará que no se produzcan problemas de sangrado a mayores distancias de renderizado.


Sintaxis:

gpu_get_tex_max_mip_ext(sampler_index);


Argumento Descripción
sampler_index El índice de la muestra del sombreador para obtener


Devoluciones:

Real (default: 16)


Ejemplo:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_max_mip_ext(_sampleIndex) != 4
    {
    gpu_set_tex_max_mip_ext(_sampleIndex, 4);
    }
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

El código anterior establece el nivel máximo de mipmap en 4 para la muestra de textura de sombreador dada si aún no se ha establecido en 4.
