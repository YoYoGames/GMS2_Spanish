draw_set_valign


Descripción

Esta función se utiliza para alinear el texto a lo largo del eje vertical y cambiar la alineación vertical cambiará la posición y la dirección en la que se dibuja todo el texto adicional, con el valor predeterminado fa_top. Se aceptan las siguientes constantes:

Constante Alineación
fa_top
fa_middle
fa_bottom


Sintaxis:

draw_set_valign(valign);

Argumento Descripción
valign Alineamiento vertical.


Devoluciones:

N/A


Ejemplo:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text(100, 32, "Score: " + string(score));

El código anterior dibujará la puntuación centrada alrededor del centro del texto.
