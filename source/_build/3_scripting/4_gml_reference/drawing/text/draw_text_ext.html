draw_text_ext

Dibuja una cuerda en una posición dada con un espaciado específico y dentro de un área limitada.

Sintaxis:

draw_text_ext(x, y, string, sep, w);


Argumento Descripción
x La coordenada x de la cadena dibujada.
y La coordenada y de la cuerda dibujada.
string La cuerda para dibujar.
sep La distancia en píxeles entre líneas de texto.
w El ancho máximo en píxeles de la cadena antes de un salto de línea.


Devoluciones: N / A


Descripción

Esta función dibujará el texto de una manera similar a draw_text solo ahora puede establecer el espacio entre cada línea de texto (si el texto ocupa más de una línea) y limitar el ancho (en píxeles) de la cadena por línea, de modo que si cualquier línea supera este valor, GameMaker Studio 2 se dividirá automáticamente el texto a la siguiente línea. Un valor de -1 para el argumento de separación de línea se establecerá por defecto en una separación basada en la altura del carácter "M" en la fuente elegida.


Ejemplo:

draw_text_ext(100, 50, keyboard_string, 3, 300);

El código anterior dibujará cualquier texto que el usuario escriba en el teclado, dividiéndolo en nuevas líneas cada vez que la longitud de la cadena para esa línea exceda los 300 píxeles. el código también mantendrá una separación de 3 píxeles entre líneas si esto ocurriera.
