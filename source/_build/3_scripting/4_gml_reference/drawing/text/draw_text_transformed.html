draw_text_transformado

Dibuja una cuerda con escala y rotación.

Sintaxis:

draw_text_transformed(x, y, string, xscale, yscale, angle);


Argumento Descripción
x La coordenada x de la cadena dibujada.
y La coordenada y de la cuerda dibujada.
string La cuerda para dibujar.
xscale La escala horizontal (por defecto 1).
yscale La escala vertical (por defecto 1).
angle El ángulo del texto.


Devoluciones: N / A


Descripción

Esta función dibujará el texto de una manera similar a draw_text solo ahora puedes elegir escalar el texto a lo largo del eje horizontal o vertical (estirando o achicando de manera efectiva) y también hacer que GameMaker Studio 2 dibuje en ángulo (donde 0 es normal y cada grado sobre 0 gira el texto en sentido antihorario).


Ejemplo:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed(room_width / 2, room_height / 2, "GAME OVER!", 2, 2, image_angle);

El código anterior dibujará el texto dado en el medio de la sala, girando y escalando al doble de su tamaño original.
