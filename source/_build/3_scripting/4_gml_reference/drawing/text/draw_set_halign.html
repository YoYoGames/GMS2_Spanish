draw_set_halign

Alinea cualquier texto dibujado posteriormente horizontalmente.

Sintaxis:

draw_set_halign(halign);


Argumento Descripción
halign Alineación horizontal.


Devoluciones: N / A


Descripción

Esta función se usa para alinear texto a lo largo del eje horizontal y al cambiar la alineación horizontal se cambiará la posición y dirección en la que se dibuja todo el texto adicional con el valor predeterminado fa_left. Se aceptan las siguientes constantes:

Constante Alineación
fa_left
fa_center
fa_right


Ejemplo:

draw_set_halign(fa_left);
draw_text(100, 32, "Score: " + string(score));
draw_set_halign(fa_right);
draw_text(room_width - 100, 32, "Health: " + string(health));

El código anterior dibujará dos cadenas en la misma línea, con el puntaje alineado a la izquierda y el estado alineado a la derecha.
