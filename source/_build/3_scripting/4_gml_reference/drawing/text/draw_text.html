draw_text


Descripción

Con esta función, puede dibujar cualquier cadena en cualquier posición dentro de la sala (para dibujar números reales debe usar el string Función para convertirlos en texto). Para combinar cadenas puede usar + (vea el ejemplo a continuación) y también puede usar \ n dentro de una cadena para agregar un salto de línea para que se dibuje en varias líneas (para obtener información sobre cómo dar formato a una cadena y qué caracteres de escape puede usar). uso, por favor ver aquí ). El color del texto y el alfa se rigen por la base actual y los valores de color establecidos por draw_set_alpha y draw_set_colour.

NOTA: La posición real del texto se verá influenciada por los valores de alineación establecidos por draw_set_halign y draw_set_valign.


Sintaxis:

draw_text(x, y, string);

Argumento Descripción
x La coordenada x de la cadena dibujada.
y La coordenada y de la cadena dibujada.
string La cuerda para dibujar.


Devoluciones:

N/A


Ejemplo:

draw_text(x, y, "Hello, " + global.Name + "!\nI hope you are well!");

El código anterior dibujará una cadena en la posición x / y de la instancia, que utilizará la cadena almacenada en la variable global "Nombre" y la dividirá en dos líneas.
