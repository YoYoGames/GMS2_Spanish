draw_text_colour

Dibuja una cuerda en una posición determinada con un degradado de color.

Sintaxis:

draw_text_colour(x, y, string, c1, c2, c3, c4, alpha);


Argumento Descripción
x La coordenada x de la cadena dibujada.
y La coordenada y de la cuerda dibujada.
string La cuerda para dibujar.
c1 El color de la esquina superior izquierda del texto dibujado.
c2 El color para la esquina superior derecha del texto dibujado.
c3 El color de la esquina inferior derecha del texto dibujado.
c4 El color de la parte inferior izquierda del texto dibujado.
alpha El alfa para el texto.


Devoluciones: N / A


Descripción

Esta función dibujará el texto de una manera similar a draw_text solo ahora puede elegir los colores que se utilizarán para colorear el texto así como el valor alfa, y estos nuevos valores se usarán en lugar del color de dibujo base y alfa.

NOTA: La combinación de degradado no está disponible para el destino HTML5 a menos que WebGL esté habilitado, aunque aún puede establecer los colores de fusión y combinará la fuente con el primer color determinado. Sin embargo, todas las mezclas de esta manera crean una fuente duplicada que luego se almacena en la memoria caché y se utiliza cuando es necesario, lo que dista mucho de ser óptimo y si utiliza varios cambios de color, se ralentizará el rendimiento de su juego. Puede configurar el tamaño de la memoria caché de fuentes para intentar limitar esto si fuera necesario utilizando la función font_set_cache_size.


Ejemplo:

draw_set_colour(c_white);
draw_text(100, 100, "Health");
draw_text_colour(100, 200, string(health), c_lime, c_lime, c_green, c_green, 1);

El código anterior dibujará dos secciones de texto en la misma línea, con el primer texto dibujado en blanco (ya que es el color de dibujo base) y el segundo texto se dibujará con un verde lima a un degradado verde normal.
