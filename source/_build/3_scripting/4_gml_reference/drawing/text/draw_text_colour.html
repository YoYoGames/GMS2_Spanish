dibujar_texto_color


Descripción

Esta función dibujará texto de manera similar a draw_text solo ahora puede elegir los colores a usar para colorear el texto y el valor alfa, y estos nuevos valores se usarán en lugar del color de dibujo base y alfa.

NOTA: La fusión de degradado no está disponible para el destino HTML5 a menos que WebGL esté habilitado, aunque aún puede establecer los colores de fusión y fusionará la fuente con el primer color dado. Sin embargo, toda la combinación de esta manera crea una fuente duplicada que luego se almacena en el caché y se usa cuando es necesario, lo que está lejos de ser óptimo y, si usa múltiples cambios de color, ralentizará el rendimiento de sus juegos. Puede configurar el tamaño de la memoria caché de la fuente para intentarlo y limitarlo si fuera necesario usando la función font_set_cache_size.


Sintaxis:

draw_text_colour(x, y, string, c1, c2, c3, c4, alpha);

Argumento Descripción
x La coordenada x de la cadena dibujada.
y La coordenada y de la cadena dibujada.
string La cuerda para dibujar.
c1 El color para la parte superior izquierda del texto dibujado.
c2 El color para la parte superior derecha del texto dibujado.
c3 El color para la parte inferior derecha del texto dibujado.
c4 El color para la parte inferior izquierda del texto dibujado.
alpha El alfa para el texto.


Devoluciones:

N/A


Ejemplo:

draw_set_colour(c_white);
draw_text(100, 100, "Health");
draw_text_colour(100, 200, string(health), c_lime, c_lime, c_green, c_green, 1);

El código anterior dibujará dos secciones de texto en la misma línea, con el primer texto que se dibuja en blanco (ya que es el color de dibujo base) y el segundo texto con un degradado verde lima a verde normal.
