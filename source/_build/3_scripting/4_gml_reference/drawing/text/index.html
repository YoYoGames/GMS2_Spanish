Texto

Hay varias funciones disponibles para dibujar texto en varias formas. Cabe señalar que algunas de estas funciones cambian los colores y tamaños de texto de forma dinámica cuando se dibujan, pero cuando se trabaja con HTML5 o dispositivos, estas funciones pueden ralentizar el juego o parecer pobre, por lo que puede ser mejor tener varios dimensione los recursos de fuente en lugar de usar un dibujo a escala, o tenga recursos de fuente grandes y luego haga que se reduzcan en lugar de subir. Además, si necesita colorear bloques de texto grandes, es mejor usar draw_set_colour y draw_set_alpha funciona primero junto con el simple draw_text función, en lugar de aquellas funciones extendidas que tienen un componente de color.

Las siguientes funciones son para configurar la fuente que se utilizará al dibujar, así como la posición del texto cuando se dibuja:

  • draw_set_font
  • draw_set_halign
  • draw_set_valign

Estas funciones se pueden usar para dibujar texto en la pantalla, ya sea simplemente o con transformaciones y color:

  • draw_text
  • draw_text_ext
  • draw_text_colour
  • draw_text_transformado
  • draw_text_ext_colour
  • draw_text_ext_transformado
  • draw_text_transformed_colour
  • draw_text_ext_transformed_colour

También existe la siguiente función que es específica para dibujar las puntuaciones máximas creadas utilizando el sistema incorporado de puntuación más alta:

  • draw_highscore
