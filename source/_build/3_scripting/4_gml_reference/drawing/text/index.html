Texto


Hay una serie de funciones disponibles para dibujar texto en varias formas. Debe tenerse en cuenta que algunas de estas funciones cambian los colores y tamaños del texto de forma dinámica cuando se dibujan, pero cuando se trabaja con HTML5 o dispositivos, estas funciones pueden ralentizar el juego o verse deficientes, por lo que puede ser mejor tener varias opciones. dimensione los recursos de fuente en lugar de usar dibujos a escala, o tenga recursos de fuentes grandes y luego dibuje en escala reducida en lugar de subir. Además, si necesita colorear grandes bloques de texto, es mejor usar el draw_set_colour y draw_set_alpha Funciona primero junto con el simple draw_text función, en lugar de aquellas funciones extendidas que tienen un componente de color.

Las siguientes funciones son para configurar la fuente que se usará al dibujar, así como la posición del texto cuando se dibuja:

  • draw_set_font
  • draw_set_halign
  • draw_set_valign
  • draw_get_font
  • draw_get_halign
  • draw_get_valign

Estas funciones se pueden usar para dibujar texto en la pantalla, ya sea simplemente o con transformaciones y color:

  • draw_text
  • draw_text_ext
  • dibujar_texto_color
  • draw_text_transformed
  • draw_text_ext_colour
  • draw_text_ext_transformed
  • draw_text_transformed_colour
  • draw_text_ext_transformed_colour

También existe la siguiente función que es específica para dibujar puntuaciones más altas creadas usando el sistema integrado de calificaciones más altas:

  • dibujar_puntuación más alta
