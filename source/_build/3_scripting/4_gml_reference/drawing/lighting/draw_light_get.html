draw_light_get


Descripción

Esta función obtendrá los parámetros de luz especificados como una matriz con los siguientes 6 elementos:

  • [0] = habilitado / deshabilitado ( true / false )
  • [1] = tipo de luz (ver constantes abajo)
  • [2] = posición x
  • [3] = posición y
  • [4] = posición z
  • [5] = radio de luz (solo para luces puntuales, será 0 para direccional)
  • [6] = color claro (real)

El tipo de luz puede ser una de las siguientes dos constantes:

Constante Descripción
lighttype_dir La luz es una luz direccional
lighttype_point La luz es una luz de punto


Sintaxis:

draw_light_get(ind)


Argumentos:

Argumento Descripción
ind El número de índice de la luz (de 0 a 7)


Devoluciones:

Real


Ejemplo:

light_a = draw_light_get(1);
if light_a[5] < 200
   {
   light_a[5] += 5;
   draw_light_define_point(1, 200, 123, 50, light_a[5], c_white);
   }

El código anterior obtendrá los valores utilizados para definir la luz indexada como "1", luego verifica el radio y si es menor de 200 se incrementará y el radio de la luz se establecerá en el nuevo valor.
