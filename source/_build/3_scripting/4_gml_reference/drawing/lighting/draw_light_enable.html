draw_light_enable


Descripción

Esta función se usa para habilitar una luz definida. Cuando defina una luz de posición o una luz direccional, debe asignarle un número de índice que luego usará esta función para encender o apagar la luz. El valor predeterminado está deshabilitado ( false )

NOTA: solo hay 8 luces de hardware disponibles, por lo que solo se pueden habilitar 8 luces definidas en cualquier momento (aunque se pueden definir más).


Sintaxis:

draw_light_enable(ind, enable)


Argumentos:

Argumento Descripción
ind El número de índice de la luz (de 0 a 7)
enable Habilitar o deshabilitar toda la iluminación ( true o falso)


Devoluciones:

N/A


Ejemplo:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

El código anterior habilitará la iluminación para toda la escena, luego definirá una luz direccional blanca en el espacio de la habitación y, finalmente, encenderá esa luz.
