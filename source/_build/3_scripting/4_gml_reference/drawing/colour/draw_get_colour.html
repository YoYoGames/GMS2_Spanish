draw_get_colour

Obtiene el color de dibujo actual.

Sintaxis:

draw_get_colour()


Devoluciones: Real


Descripción

Esta función devuelve el color de dibujo actual que se utiliza para dibujar formularios, texto, primitivas y modelos 3D sin textura. Esto se puede configurar con el draw_set_colour función.


Ejemplo:

if draw_get_colour() != c_black
   {
   draw_set_colour(c_black);
   }

El código anterior primero verifica el color del dibujo actual y luego, si no está configurado para c_black lo establece en c_black.
