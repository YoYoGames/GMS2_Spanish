draw_set_alpha


Descripción

Con esta función puedes establecer el sorteo base alfa para el juego. Este valor se puede establecer de 0 a 1, siendo 0 totalmente transparente y 1 totalmente opaco (el valor predeterminado), y afectará a todos los dibujos posteriores, incluidos fondos, sprites, fuentes, primitivos y 3D.

Tenga en cuenta que si está dibujando alguno de los activos y suministrando un argumento alfa como parte de la función (por ejemplo, al usar draw_sprite_ext() ), entonces el valor alfa establecido aquí se anulará, por lo que dibujar un sprite usando draw_sprite_ext con un argumento alfa de 1 dibujará el sprite completamente opaco independientemente del conjunto alfa con draw_set_alpha (Esto funcionará igual cuando se usa draw_self() y configurando el image_alpha de la instancia). También tenga en cuenta que cuando una instancia es un dibujo predeterminado (es decir, tiene un sprite pero no tiene Evento de dibujo), entonces el image_alpha variable se fijará al conjunto alfa con draw_set_alpha(), tal que un image_alpha de 1 y un sorteo alfa de 0.5 dibujará el sprite en alfa 0.5, mientras que un image_alpha de 0.25 y un sorteo alfa de 0.5 dibujará el sprite en 0.25 alfa.


Sintaxis:

draw_set_alpha(alpha);


Argumento Descripción
alpha El alfa a configurar (entre 0 y 1)


Devoluciones:

N/A


Ejemplo:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

El código anterior dibujará un texto en la posición especificada con un efecto de sombra creado por el alfa y el color del dibujo modificado.
