draw_set_alpha

Establece la mezcla alfa básica para las funciones de dibujo.

Sintaxis:

draw_set_alpha(alpha);


Argumento Descripción
alpha El alfa para establecer (entre 0 y 1)


Devoluciones: N / A


Descripción

Con esta función puedes establecer el alfa base del juego. Este valor se puede configurar de 0 a 1, siendo 0 completamente transparente y 1 totalmente opaco (el valor predeterminado), y afectará a todos los demás dibujos, incluidos fondos, sprites, fuentes, primitivas y 3D.

Tenga en cuenta que si está dibujando cualquiera de los activos y suministrando un argumento alfa como parte de la función (por ejemplo, cuando usa draw_sprite_ext() ), entonces el valor alfa establecido aquí será anulado, por lo que dibujar un sprite usando draw_sprite_ext con un argumento alfa de 1 dibujará el sprite completamente opaco independientemente del conjunto alfa con draw_set_alpha (Esto funcionará igual cuando se usa draw_self() y establecer el image_alpha de la instancia). También tenga en cuenta que cuando una instancia es un dibujo predeterminado (es decir: tiene un sprite pero no Draw Event), entonces el image_alpha la variable se fijará al conjunto alfa con draw_set_alpha(), tal que image_alpha de 1 y un draw alpha de 0.5 dibujará el sprite en alpha 0.5, mientras image_alpha de 0.25 y un draw alpha de 0.5 dibujará el sprite en 0.25 alpha.


Ejemplo:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

El código anterior dibujará texto en la posición especificada con un efecto de sombra creado por draw alpha y color modificado.
