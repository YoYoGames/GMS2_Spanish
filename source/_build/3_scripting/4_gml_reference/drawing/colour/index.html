Color

GameMaker Studio 2 tiene una amplia selección de funciones para manipular el color y el alfa de lo que se dibuja en la pantalla (para mezclar imágenes usando modos de fusión especiales y otras funciones específicas de la PGU, consulte la sección sobre el control de la GPU.) en sus partes componentes, crea nuevos colores y manipula el dibujo alfa para cambiar la forma en que se combinan las diferentes cosas de tu juego. Además de estas funciones, también hay varias constantes que se pueden usar como colores base en estos (y otros) funciones:

Tenga en cuenta que también puede crear colores a partir de su valor hexadecimal utilizando el símbolo "$" de antemano, y estos valores también se pueden usar en las funciones de color para obtener matices de componentes o para mezclar, etc. El valor hexadecimal de estos colores se divide en tres partes con el formato Azul / Verde / Rojo para que, por ejemplo, defina un color violeta que podría usar:

col = $983c95

Donde 98 es el componente azul, 3c es el componente verde y 95 es el componente rojo.

Las siguientes funciones se pueden usar para obtener los tonos componentes, el tono compuesto, la saturación y la luminosidad de un color seleccionado, así como las propiedades alfa y de otro tipo:

  • colour_get_blue
  • colour_get_green
  • colour_get_red
  • colour_get_hue
  • colour_get_saturation
  • colour_get_value
  • draw_getpixel
  • draw_getpixel_ext
  • draw_get_colour
  • draw_get_alpha

Estas funciones se pueden usar para crear colores a partir de valores de entrada sin procesar para componentes de color:

  • make_colour_hsv
  • make_colour_rgb
  • merge_colour

Las siguientes funciones se pueden usar para configurar las diferentes opciones para dibujar en la pantalla, incluyendo alfa, color y combinación:

  • draw_clear
  • draw_clear_alpha
  • draw_set_alpha
  • draw_set_colour
