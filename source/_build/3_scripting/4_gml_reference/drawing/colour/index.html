Color


GameMaker Studio 2 tiene una amplia selección de funciones para manipular el color y el alfa de lo que se dibuja en la pantalla (para combinar imágenes usando modos de mezcla especiales y otras funciones específicas de PGU, consulte la sección Control de GPU. Con estas funciones puede separar los colores en sus componentes, cree nuevos colores y manipule el sorteo alfa para cambiar la forma en que se dibujan las diferentes cosas de su juego. Además de estas funciones, también hay una serie de constantes que se pueden usar como colores base en estas (y otras) funciones:

Tenga en cuenta que también puede crear colores a partir de su valor hexadecimal utilizando el símbolo "$" de antemano, y estos valores también se pueden usar en las funciones de color para obtener matices de componentes o para mezclar, etc. El valor hexadecimal para estos colores se divide en tres partes con el formato Azul / Verde / Rojo, por ejemplo, para definir un color púrpura que podría usar:

col = $983c95

Donde 98 es el componente azul, 3c es el componente verde y 95 es el componente rojo.

Las siguientes funciones se pueden utilizar para obtener los tonos de los componentes, el matiz compuesto, la saturación y la luminosidad de un color seleccionado, así como las propiedades alfa y otras:

  • colour_get_blue
  • colour_get_green
  • colour_get_red
  • colour_get_hue
  • colour_get_saturation
  • colour_get_value
  • draw_getpixel
  • draw_getpixel_ext
  • draw_get_colour
  • draw_get_alpha

Estas funciones se pueden usar para crear colores a partir de valores de entrada sin procesar para componentes de color:

  • make_colour_hsv
  • make_colour_rgb
  • fusionar color

Las siguientes funciones se pueden usar para configurar las diferentes opciones para dibujar en la pantalla, incluyendo alfa, color y fusión:

  • draw_clear
  • draw_clear_alpha
  • draw_set_alpha
  • draw_set_colour
