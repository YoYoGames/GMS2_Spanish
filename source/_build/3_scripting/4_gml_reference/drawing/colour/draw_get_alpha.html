draw_get_alpha

Obtiene el valor alfa del sorteo actual.

Sintaxis:

draw_get_alpha()


Devoluciones: Real


Descripción

Esta función devuelve el valor actual del dibujo alfa, que oscilará entre 0 (totalmente transparente) y 1 (totalmente opaco). El dibujo alfa afecta la transparencia de todas las funciones de dibujo, y se puede configurar con el draw_set_alpha función.


Ejemplo:

if draw_get_alpha() != 1
   {
   draw_set_alpha(1);
   }

El código anterior primero comprueba el dibujo actual alfa y luego, si no está configurado en 1, lo establece en 1.
