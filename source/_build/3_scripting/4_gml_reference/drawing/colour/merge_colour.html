merge_colour

Fusiona dos colores por una cantidad determinada.

Sintaxis:

merge_colour(col1, col2, amount);


Argumento Descripción
col1 El primer color en fusionarse
col2 El segundo color para fusionarse
amount Cuánto de cada color debe combinarse. Por ejemplo, 0 devolverá col1, 1 devolverá col2 y 0.5 devolverá una combinación de ambos colores por igual


Devoluciones: valor


Descripción

Con esta función puede tomar dos colores y luego combinarlos para crear un nuevo color. La cantidad de cada uno de los colores componentes se puede definir cambiando el argumento "cantidad", donde un valor de 0 devolverá el primer color (col1), un valor de 1 devolverá el segundo color (col2) y un valor entre ellos devolverá la mezcla correspondiente. Por ejemplo, un valor de 0.5 mezclará los dos colores por igual. La siguiente imagen ilustra cómo funciona al combinar los colores rojo y azul juntos:


Ejemplo:

col = merge_colour(c_lime, c_orange, 0.5);

El código anterior usa la función para crear un color combinando lima y naranja 50/50 y luego almacena su valor en la variable "col" para su uso posterior.
