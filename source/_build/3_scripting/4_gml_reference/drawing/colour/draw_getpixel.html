draw_getpixel

Devuelve el color del píxel que se muestra en una determinada coordenada.

Sintaxis:

draw_getpixel(x, y);


Argumento Descripción
x La coordenada x del píxel para verificar
y La coordenada y del píxel para verificar


Devoluciones: Real


Descripción

Con esta función, puede obtener el valor de color de cualquier píxel que se dibuje en el objetivo de renderización actual. Esto significa que los resultados dependerán del evento en el que se invoque la función y también de la superficie objetivo que se utilice. Tenga en cuenta que esto no devolverá ningún valor alfa, para eso debe usar draw_getpixel_ext.

NOTA: Esta función es increíblemente lenta y solo debe usarse cuando sea absolutamente necesario.


Ejemplo:

draw_set_colour(draw_getpixel(500,200))

El código anterior establecerá el color del dibujo para todos los textos, formularios y primitivas al color del píxel en (500,200) en la pantalla del juego.
