draw_clear_alpha

Borra toda la pantalla con un color dado mezclado alfa.

Sintaxis:

draw_clear_alpha(col, alpha);


Argumento Descripción
col El color con el que se borrará la pantalla
alpha La transparencia del color con el que se borrará la pantalla


Devoluciones: N / A


Descripción

Esta función se puede utilizar para limpiar toda la pantalla con un color determinado y el componente alfa del destino se establece en el valor que ha establecido; esta función no hace ninguna fusión, ya que funciona, pero cualquier operación de mezcla posterior puede configurarse para usa el destino alfa que has establecido. Esto es solo para usar en el evento de dibujo de una instancia (no se mostrará si se usa en cualquier otro evento), y también puede ser muy útil para borrar superficies cuando se crean nuevamente.

Vale la pena señalar que al usar el módulo HTML5, puede utilizar esta función para hacer que el fondo del lienzo sea transparente, lo que permitirá que las imágenes que se usen en el HTML del servidor se vean con el juego corriendo sobre ellas. Para lograr este efecto, primero debe ingresar al editor de sala y eliminar las capas de fondo del Editor de sala, luego desmarcar la opción Borrar fondo de vista que se encuentra en Propiedades de vista. Finalmente, deberá crear un nuevo objeto, colocarlo en la primera sala del juego y asignarle una profundidad muy alta (por ejemplo, 10000). Luego, en el evento Pre Draw, utiliza esta función para establecer el lienzo alfa para el color y la transparencia que deseas que tenga.


Ejemplo:

draw_clear_alpha(c_red, 0.5);

Esto borrará la pantalla con una transparencia del 50% (0.5) con el color rojo.
