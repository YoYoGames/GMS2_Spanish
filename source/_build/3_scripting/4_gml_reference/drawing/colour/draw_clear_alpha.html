draw_clear_alpha


Descripción

Esta función se puede utilizar para borrar toda la pantalla con un color determinado y el componente alfa del destino se establece en el valor ajustado - esta función no hace ninguna mezcla, ya que funciona pero cualquier operación de mezcla posterior puede ser configurado para Usa el alfa de destino que has establecido. Esto es solo para usar en el evento draw de una instancia (no se mostrará si se usa en cualquier otro evento), y también puede ser muy útil para limpiar superficies cuando se crean de nuevo.

Vale la pena señalar que al usar el módulo HTML5, puede usar esta función para hacer que el fondo del lienzo sea transparente, lo que permitirá que cualquier imagen que se use en el HTML del host se vea con el juego corriendo sobre ellos. Para lograr este efecto, primero debe ingresar al editor de sala y eliminar cualquier Capa de fondo del Editor de sala, luego desmarque la opción Borrar fondo de vista que se encuentra en las Propiedades de vista. Finalmente, deberá crear un nuevo objeto, colocarlo en la primera sala del juego y asignarle una profundidad muy alta (por ejemplo, 10000). Luego, en el evento Pre Draw, use esta función para configurar el lienzo alfa del color y la transparencia que desea que tenga.


Sintaxis:

draw_clear_alpha(col, alpha);


Argumento Descripción
col El color con el que se borrará la pantalla.
alpha La transparencia del color con el que se borrará la pantalla.


Devoluciones:

N/A


Ejemplo:

draw_clear_alpha(c_red, 0.5);

Esto borrará la pantalla con una transparencia del 50% con el color rojo.
