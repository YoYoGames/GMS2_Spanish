draw_enable_drawevent


Descripción

Con esta función puede elegir activar ( true ) o desactivar ( false ) el evento de sorteo para todas las instancias del juego, lo que le permite controlar cómo y cuándo se dibujan las cosas, lo cual es útil si desea implementar una técnica de "salto de cuadro". Tenga en cuenta que esto no solo evita que las instancias se dibujen en la pantalla, sino que suprime completamente el evento de dibujo, lo que significa que debe tenerse cuidado ya que ninguna lógica de juego que esté presente en ese evento tampoco se ejecutará.


Sintaxis:

draw_enable_drawevent(enable);

Argumento Descripción
Enable Ajustado a true o false.


Devoluciones:

N/A


Ejemplo:

frame_skip ++;
if (frame_skip mod 5) == 0
   {
   draw_enable_drawevent(true);
   }
else
   {
   draw_enable_drawevent(false);
   }

El código anterior verifica una variable y si regresa true, habilita el evento de sorteo; de lo contrario, el evento de sorteo queda deshabilitado.
