draw_sprite

Dibuja un sprite en una posición dada.

Sintaxis:

draw_sprite(sprite, subimg, x, y);


Argumento Descripción
sprite El índice del sprite para dibujar.
subimg La subimagen (marco) del sprite para dibujar (image_index o -1 se correlaciona con el cuadro actual de animación en el objeto).
x La coordenada x de dónde dibujar el sprite.
y La coordenada y de dónde dibujar el sprite.


Devoluciones: N / A


Descripción

Esta función dibuja el sprite y la subimagen dados en una posición dentro de la sala de juegos. Para el sprite puedes usar la variable de instancia sprite_index para obtener el sprite actual asignado a la instancia que ejecuta el código, o puede usar cualquier otro elemento de sprite. Lo mismo se aplica a la subimagen, ya que también se puede establecer en la variable de instancia image_index que establecerá la subimagen a la seleccionada para el sprite de la instancia actual (tenga en cuenta que puede dibujar un sprite diferente y aún usar el valor de la subimagen para la instancia actual), o puede usar cualquier otro valor para dibujar una sub-imagen específica del sprite elegido. Si el valor es mayor que el número de imágenes secundarias, entonces GameMaker Studio 2 repetirá automáticamente el número para seleccionar la imagen correspondiente (por ejemplo, si el sprite que se está dibujando tiene 5 subimágenes numeradas de 0 a 4 y establecemos el índice valor a 7, luego la función dibujará la subimagen 3, numerada 0). Finalmente, la posición xey es la posición dentro de la sala donde se dibujará el sprite, y se centra en el sprite x offset y y offset.


Ejemplo:

draw_sprite(sprite_index, image_index, x, y);
draw_sprite(spr_Halo, 0, x, y-32);

Esto dibujará las instancias asignadas sprite (sprite_index) con la sub-imagen actual en la posición xey de la instancia dentro de la habitación. Luego dibujará el primer fotograma del sprite indexado por "spr_Halo" en la misma posición xey, pero a 32 píxeles 'por encima'.
