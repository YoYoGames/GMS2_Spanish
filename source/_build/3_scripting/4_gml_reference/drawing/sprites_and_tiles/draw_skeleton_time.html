draw_skeleton_time

Dibuja el sprite de animación esquelético en un marco de tiempo específico.

Sintaxis:

draw_skeleton_time(sprite, animname, skinname, time, x, y, xscale, yscale, rot, colour)


Argumento Descripción
sprite El índice del sprite para dibujar.
animname El nombre de la animación para obtener el marco de (una cadena).
skinname El nombre de la máscara que se utilizará para dibujar (una cadena).
time El marco de tiempo para dibujar la animación en (de 0 a la duración final, en segundos).
x La coordenada x de dónde dibujar el sprite.
y La coordenada y de dónde dibujar el sprite.
xscale La escala horizontal del sprite, como un multiplicador: 1 = escala normal, 0.5 es mitad, etc.
yscale La escala vertical del sprite, como un multiplicador: 1 = escala normal, 0.5 es mitad, etc.
rot La rotación del sprite. 0 = normal, 90 = girado 90 grados en sentido antihorario, etc.
colour El color con el que mezclar el sprite.


Devoluciones: N / A


Descripción

Esta función dibujará la animación dada usando la máscara determinada en un momento específico de la animación. El valor del tiempo debe estar entre 0 (el comienzo) y la duración final de la animación, que puede encontrar utilizando la función skeleton_animation_get_duration(). Puede establecer el valor de tiempo a valores superiores a la duración total de la animación y el bucle de animación voluntad de nuevo al principio, pero se corre el riesgo de perder la precisión de coma flotante como el tiempo acumulado se hace más grande.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

var time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

El código anterior dibujará el sprite de animación esquelético dado usando delta-time para establecer el marco que se dibuja.
