draw_sprite_stretched_ext

Dibuja un sprite en una posición dada, estirado, y permite una combinación personalizada y alfa. Esta función ignora el origen del sprite.

Sintaxis:

draw_sprite_stretched_ext(sprite, subimg, x, y, w, h, colour, alpha);


Argumento Descripción
sprite El índice del sprite para dibujar.
subimg El subimg (frame) del sprite para dibujar (image_index o -1 se correlaciona con el cuadro actual de animación en el objeto).
x La coordenada x de dónde dibujar el sprite.
y La coordenada y de dónde dibujar el sprite.
w El ancho del área ocupará el sprite estirado.
h La altura del área que ocupará el sprite estirado.
colour El color con el que mezclar el sprite. c_white es para mostrarlo normalmente.
alpha El alfa del sprite (de 0 a 1 donde 0 es transparente y 1 opaco).


Devoluciones: N / A


Descripción

Esta función hace exactamente lo mismo que draw_sprite_stretched función con la capacidad añadida de establecer la mezcla de color y el valor alfa para el sprite cuando se dibuja (similar a la función draw_sprite_ext )

NOTA: La combinación de colores solo se recomienda para el destino HTML5 cuando WebGL está habilitado, aunque aún puede establecer el color de fusión si no está habilitado y combinará el sprite de la forma habitual. Sin embargo, todas las mezclas de esta manera crean un elemento gráfico duplicado que luego se almacena en la memoria caché y se utiliza cuando es necesario. Esto está lejos de ser óptimo y si usa múltiples cambios de color, reducirá el rendimiento de su juego a menos que active WebGL. si no desea usar WebGL, puede configurar el tamaño de la memoria caché de fuentes para intentar limitar esto si fuera necesario utilizando la función sprite_set_cache_size.


Ejemplo:

draw_sprite_stretched_ext(sprite_index, image_index, x, y, sprite_width / 2, sprite_height / 2, c_white, 0.5);

Esto dibujará las instancias asignadas sprite y su subimagen con la esquina izquierda en la posición x / y de la instancia. Su ancho se establece al mismo que el sprite, y la altura es la altura del sprite dividida por dos. También se mezclará normalmente pero tendrá un valor alfa parcialmente transparente.
