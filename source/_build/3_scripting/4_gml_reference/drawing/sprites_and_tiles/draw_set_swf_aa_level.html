draw_set_swf_aa_level

Establezca el nivel de antialiasing para sprites de vector de formato SWF.

Sintaxis:

draw_set_swf_aa_level(AA);


Argumento Descripción
AA El valor de anti-aliasing para usar de 0 a 1.


Devoluciones: N / A


Descripción

Esta función se puede usar para establecer el nivel de antialiasing (AA) para sprites de vectores de formato SWF. Este puede ser un valor real de 0 a 1 y "suavizará" los bordes de estos sprites. Tenga en cuenta que para ver este efecto, primero debe haber habilitado AA utilizando la función draw_enable_swf_aa().

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.



Ejemplo:

if draw_get_swf_aa_level() == 0
   {
   draw_enable_swf_aa(true);
   draw_set_swf_aa_level(0.5);
   }

El código anterior verificará el valor de AA para sprites de formato SWF, y si es 0 habilita AA y establece el valor en 0.5.
