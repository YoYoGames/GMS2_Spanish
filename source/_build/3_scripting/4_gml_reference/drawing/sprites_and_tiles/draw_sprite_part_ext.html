draw_sprite_part_ext

Dibuja parte de un sprite en una posición dada con escala, mezcla y opciones alfa.

Sintaxis:

draw_sprite_part_ext(sprite, subimg, left, top, width, height, x, y, xscale, yscale, colour, alpha);


Argumento Descripción
sprite El índice del sprite para dibujar.
subimg El subimg (frame) del sprite para dibujar (image_index o -1 se correlaciona con el cuadro actual de animación en el objeto).
left La posición x en el sprite de la esquina superior izquierda del área para dibujar.
top La posición y en el sprite de la esquina superior izquierda del área para dibujar.
width El ancho del área para dibujar.
height La altura del área para dibujar.
x La coordenada x de dónde dibujar el sprite.
y La coordenada y de dónde dibujar el sprite.
xscale La escala horizontal del sprite, como un multiplicador: 1 = escala normal, 0.5 es mitad, etc.
yscale La escala vertical del sprite, como un multiplicador: 1 = escala normal, 0.5 es mitad, etc.
colour El color con el que mezclar el sprite. c_white es para mostrarlo normalmente.
alpha El alfa del sprite (de 0 a 1 donde 0 es transparente y 1 opaco).


Devoluciones: N / A


Descripción

Esta función dibujará una parte del sprite elegido en la posición dada siguiendo las mismas reglas de draw_sprite_part, solo ahora puedes escalar la parte, combinar un color con ella o cambiar su alfa cuando la dibujas en la pantalla (lo mismo que cuando dibujas un sprite con draw_sprite_ext ) Debe tener en cuenta que si la página de texturas permite un recorte automático, es posible que esta función no funcione como se esperaba, ya que el espacio extra "vacío" alrededor del elemento se habrá eliminado para crear la página de textura. Para resolver este problema, deberá establecer la configuración de la página de texturas (en el Editor de grupos de texturas ) para desactivar el recorte automático.

NOTA: Al dibujar con esta función, el desplazamiento de sprite x y el desplazamiento y se ignoran y la parte de sprite se dibujará con la esquina superior izquierda en la posición x / y especificada en la sala.
NOTA: Esta función solo es útil para sprites de mapa de bits y no funcionará con sprites SWF o JSON (Spine).
NOTA: La combinación de colores solo se recomienda para el destino HTML5 cuando WebGL está habilitado, aunque aún puede establecer el color de fusión si no está habilitado y combinará el sprite de la forma habitual. Sin embargo, todas las mezclas de esta manera crean un elemento gráfico duplicado que luego se almacena en la memoria caché y se utiliza cuando es necesario. Esto está lejos de ser óptimo y si usa múltiples cambios de color, reducirá el rendimiento de su juego a menos que active WebGL. si no desea usar WebGL, puede configurar el tamaño de la memoria caché de fuentes para intentar limitar esto si fuera necesario utilizando la función sprite_set_cache_size.


Ejemplo:

draw_sprite_part_ext(sprite_index, image_index, 8, 8, sprite_width-16, sprite_height-16, x, y, 2, 0.5, c_black, 1);

Esto dibujará las instancias asignadas sprite (sprite_index) y su actual marco de animación (image_index), sin embargo, afeitará un margen de 8px de los cuatro lados del sprite. Luego se estirará para duplicar su ancho habitual pero la mitad de su altura habitual, y aunque el alfa todavía es 1, se mezclará con negro (convirtiéndolo en una silueta).
