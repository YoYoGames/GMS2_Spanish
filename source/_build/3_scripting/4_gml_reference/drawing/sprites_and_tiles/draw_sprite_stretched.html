draw_sprite_stretched

Dibuja un sprite en una posición determinada, estirado.

Sintaxis:

draw_sprite_stretched(sprite, subimg, x, y, w, h);


Argumento Descripción
sprite El índice del sprite para dibujar.
subimg El subimg (frame) del sprite para dibujar (image_index o -1 se correlaciona con el cuadro actual de animación en el objeto).
x La coordenada x de dónde dibujar el sprite.
y La coordenada y de dónde dibujar el sprite.
w El ancho del área ocupará el sprite estirado.
h La altura del área que ocupará el sprite estirado.


Devoluciones: N / A


Descripción

Esta función simplemente toma un recurso de sprite y lo estira sobre el ancho y la altura determinados para que ocupe el área. Al igual que con draw_sprite puede especificar un sprite y una imagen secundaria para dibujar, luego la posición x / y en la sala para que se dibuje el sprite y finalmente un ancho y una altura (que deben ser valores de píxel). La imagen a continuación ilustra cómo funciona esta función:

NOTA: Al dibujar con esta función, el desplazamiento de sprite x y el desplazamiento y se ignoran y el sprite se dibujará con la esquina superior izquierda en la posición x / y especificada en la sala.


Ejemplo:

draw_sprite_stretched(sprite_index, image_index, x, y, sprite_width / 2, sprite_height / 2);

Esto dibujará las instancias asignadas sprite y su subimagen con la esquina izquierda en la posición x / y de la instancia. Su ancho se establece en el mismo que el sprite, y la altura es la altura del sprite dividida por dos.
