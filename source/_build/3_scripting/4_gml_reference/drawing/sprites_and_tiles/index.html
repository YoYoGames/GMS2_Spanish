Dibujo Sprites Y Azulejos

Aunque puedes configurar fondos en el editor de sala y puedes asignar sprites a instancias, a menudo esto no es lo suficientemente flexible para el juego que estás haciendo. Por ejemplo, si tiene una instancia que tiene que dibujar un menú, es posible que desee mostrar varios sprites de botones a la vez, o si tiene un juego con fondos de paralaje con desplazamiento, deberá poder configurarlos a través del código en una instancia de controlador de algún tipo. En estas situaciones, debe usar las siguientes funciones que pueden dibujar elementos de sprite y fondo en cualquier lugar de la sala con algunas de las funciones que también le permiten manipular la forma, el tamaño, el color y el alfa del sprite para lograr efectos especiales.

Puede asignar un sprite a una instancia y, de forma predeterminada, GameMaker Studio 2 lo dibujará con las propiedades dadas por las distintas variables de sprite. Sin embargo, si agrega algo al evento de sorteo, está "tomando el control" y debe informar a GameMaker Studio 2 exactamente. que dibujar Esto significa que si debe usar las siguientes funciones para definir exactamente de qué se trata la instancia es dibujar y en qué lugar de la sala es dibujarla:

  • draw_self
  • draw_sprite
  • draw_sprite_ext
  • draw_sprite_general
  • draw_sprite_part
  • draw_sprite_part_ext
  • draw_sprite_stretched
  • draw_sprite_stretched_ext
  • draw_sprite_pos
  • draw_sprite_tiled
  • draw_sprite_tiled_ext

Si está utilizando sprites de formato SWF, también hay opciones adicionales para controlar el suavizado utilizado al dibujar (consulte Importación de Sprites que no son de mapa de bits para obtener más información sobre estos tipos de sprites). Tenga en cuenta que estas funciones funcionan a nivel global y establecerán el AA en todos los sprites SWF vectoriales dibujados después de que se hayan llamado.

  • draw_enable_swf_aa
  • draw_set_swf_aa_level
  • draw_get_swf_aa_level

Además de las funciones enumeradas anteriormente, también hay dos funciones especiales diseñadas para usar con animaciones esqueléticas (consulte Importación de Sprites que no son de mapa de bits para obtener más información sobre estos tipos de sprites, y la sección Animaciones esqueléticas para funciones específicas relacionadas con estos tipos de sprites):

  • draw_skeleton
  • draw_skeleton_instance
  • draw_skeleton_collision
  • draw_skeleton_time

Los mosaicos se dibujan automáticamente en función de los datos que contienen, el conjunto de mosaicos utilizados y las propiedades de mapa de páginas. Sin embargo, puede dibujar mosaicos individuales o mapas en forma utilizando las siguientes funciones:

  • draw_tile
  • tilemap_draw
