draw_tilemap

Descripción

Esta función se puede usar para dibujar un mapa de mosaicos dado en cualquier parte de la habitación. Le das el ID del elemento de mosaico (que obtienes cuando creas un elemento de mosaico usando layer_tilemap_create() o cuando usas la función layer_tilemap_get_id() ), luego da la posición x / y para dibujar (en la habitación). Tenga en cuenta que esto simplemente dibujará el mapa de mosaicos en el punto especificado, utilizando la profundidad de capa de la instancia que llama a la función. No se mueve el tilemap - ni cambiar de ninguna manera - y qué importa si el tilemap está marcado como visible o no.


Sintaxis:

draw_tilemap(tilemap_element_id, x, y);


Argumentos:

Argumento Descripción
tilemap_element_id El valor ID único del elemento tilemap para obtener la posición y de la celda
x La posición x dentro de la habitación para dibujar en
y La posición y dentro de la habitación para dibujar en


Devoluciones:

N/A


Ejemplo:

var lay_id = layer_get_id("Tiles_Effects");
var map_id = layer_tilemap_get_id(lay_id);
draw_tilemap(map_id, mouse_x, mouse_y);

El código anterior obtiene el ID de capa y luego lo usa para obtener un ID de mosaico específico que a su vez se usa para dibujar el mapa de mosaicos en la posición del mouse.
