draw_skeleton_instance


Descripción

Esta función solo se puede usar con instancias que tengan sprites asignados creados con el programa de animación esquelética Spine. Mientras que puedes dibujar estos sprites usando las funciones de dibujo normales (como draw_self() ), hay momentos en los que es posible que desee dibujar un solo cuadro o mostrar un cambio de textura de la piel sin cambiar realmente las propiedades de los sprites para una instancia. En estos casos, puede dibujar el sprite que se ha asignado a la instancia con esta función, donde se proporciona la ID de instancia única y luego el nombre de la animación para obtener el marco (una cadena, como se establece dentro del programa utilizado para hacer el sprite). A continuación, indica el nombre de la máscara que se utilizará (de nuevo, como se configuró cuando se creó el sprite) y un valor de fotograma. El valor del marco es para decirle a GameMaker Studio 2 qué parte de la animación mostrar y es un valor entre 0 y el image_number. El resto de los argumentos son los mismos que los utilizados para el dibujo de sprites normal y cambiarán la escala, el ángulo, el color de fusión y el alfa del cuadro de animación final que se está dibujando.

¡ADVERTENCIA! Debido a la forma en que las animaciones esqueléticas se interpolan entre varios cuadros "clave", hay una sobrecarga adicional asociada con esta llamada y, por lo tanto, se recomienda que evite dibujar los esqueletos basados ​​en esqueletos utilizando este método a menos que sea absolutamente necesario.
¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Sintaxis:

draw_skeleton_instance(instance, animname, skinname, frame, x, y, xscale, yscale, rot, colour, alpha);


Argumento Descripción
instance El índice de la instancia desde la que dibujar el sprite.
animname El nombre de la animación para obtener el marco de (una cadena).
skinname El nombre de la máscara para usar (una cadena).
frame El marco de la animación para dibujar (de 0 a image_number - 1).
x La coordenada x de dónde dibujar el sprite.
y La coordenada y de dónde dibujar el sprite.
xscale La escala horizontal del sprite, como un multiplicador: 1 = escala normal, 0.5 es mitad, etc.
yscale La escala vertical del sprite, como un multiplicador: 1 = escala normal, 0.5 es mitad, etc.
rot La rotación del sprite. 0 = normal, 90 = girado 90 grados en sentido antihorario, etc.
colour El color con el que mezclar el sprite.
alpha El alfa del sprite (de 0 a 1 donde 0 es transparente y 1 opaco).


Devoluciones:

N/A


Ejemplo:

draw_skeleton_instance(obj_Player, "jump", "skin1", 0, x, y, 1, 1, 0, c_white, 1);

El código anterior dibujará el primer fotograma del sprite de animación esquelético asignado a la instancia dada, utilizando el conjunto de animación "saltar" y el aspecto definido como "skin1".
