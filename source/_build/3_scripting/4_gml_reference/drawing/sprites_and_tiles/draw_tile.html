draw_tile

Descripción

Con esta función, puedes dibujar cualquier ficha de un conjunto de fichas - competir con rotaciones, volteretas y espejos - configurar el marco (si está animado) y la posición dentro de la habitación. El valor de ID del conjunto de mosaicos es el índice de conjunto de mosaicos establecido en el IDE y los datos de mosaico son el "blob" de datos que contiene toda la información sobre el índice del mosaico y las transformaciones que se han aplicado. Puedes usar la función tilemap_get para obtener los datos del mosaico que luego pueden manipularse utilizando las funciones de mosaico, o puede construirlo usted mismo utilizando las siguientes constantes de mosaico junto con el valor de índice del mosaico para crear:

Constante Descripción
tile_rotate Se usa para establecer / obtener el bit de rotación de un blob de datos de mosaico.
tile_mirror Se usa para establecer / obtener el bit de espejo de un blob de datos de mosaico.
tile_flip Se usa para establecer / obtener el bit de una ficha de datos de mosaico.
tile_index_mask Una constante especial que es para "y" -ing con el blob de datos de mosaico para extraer el índice del mosaico.


Haría bit a bit "o" estas constantes con el valor del índice del mosaico para obtener los datos del mosaico final:

var tiledata = 8 | tile_rotate | tile_flip;


El código anterior simplemente crea una variable local para almacenar los datos de mosaico para el mosaico con el índice de 8 en el conjunto de mosaicos, y se gira y voltea. Esto se puede usar para dibujar el mosaico en ese índice del conjunto de mosaicos que especifique. esto no está vinculado a ninguna capa de mosaico y el mosaico se dibujará a la profundidad de capa de la instancia que realiza el dibujo, en la posición x / y que especifique en la sala. Tenga en cuenta que también puede configurar el marco para dibujar también. Si el mosaico indexado es parte de un conjunto de animación, puede establecer el número de cuadro aquí (de 0 a la cantidad de cuadros - 1) y el mosaico indexado será simplemente el indicador de qué conjunto de mosaicos animados está usando mientras lo hará ser el marco dado que se dibuja. Si el mosaico no forma parte de un conjunto animado, entonces el cuadro debe establecerse en 0.


Sintaxis:

draw_tile(tile set, tiledata, frame, x, y)


Argumentos:

Argumento Descripción
tile set La ID del mosaico configurado para usar
tiledata Los datos de los azulejos para usar
frame El número de cuadro para el mosaico animado a usar (valor predeterminado 0 para los mosaicos no animados)
x La posición x dentro de la habitación para dibujar en
y La posición y dentro de la habitación para dibujar en


Devoluciones:

Real


Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
tile_set_flip(data, true);
draw_tile(set_Wall, data, 0, x, y);

El código anterior obtendrá los datos en un mosaico del mapa de mosaicos dado, luego establece el indicador "flip" y usa estos datos para dibujar el mosaico invertido en la posición de la instancia.
