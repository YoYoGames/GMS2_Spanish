texture_global_scale

Se usa para controlar la escala de las páginas de textura cargadas desde el archivo WAD.

Sintaxis:

texture_global_scale(pow2integer);


Argumento Descripción
pow2integer El factor de escala a usar (1, sin escala, 2, media escala, 4, escala trimestral, etc.)


Devoluciones: N / A


Descripción

Esta función le permite controlar el escalado de las páginas de textura cargadas desde el archivo WAD que forma parte del ejecutable final de su juego. El valor de entrada debe ser una potencia de dos valores y funcionará de tal manera que:

  • Para configurar las páginas de textura para que sean de uso 1: 1 texture_global_scale(1)
  • Para medias texturas todo el tiempo use texture_global_scale(2)
  • Para cuarto uso texture_global_scale(4), etc...

De esta forma, puede controlar mejor el uso de la memoria de la página de textura en plataformas con problemas de poca memoria. Tenga en cuenta que las páginas de texturas se crean a petición del WAD, por lo que puede llamarlas al inicio del juego para cargar todas las texturas escaladas, o puede llamarlas en momentos específicos para cargar texturas específicas, en cuyo caso Probablemente quiera llamar draw_texture_flush antes de llamar a esta función, utilice las funciones de búsqueda previa para obtener las páginas de textura correctas en la memoria.


Ejemplo:

draw_texture_flush();
texture_global_scale(2);
sprite_prefetch(spr_Trees);

El código anterior eliminará todas las texturas de la memoria, luego establecerá la escala de texturas en 2 (para que las páginas de textura sean de la mitad de tamaño) y finalmente preseleccionará la página de texturas en la que está contenido el sprite "spr_Trees".
