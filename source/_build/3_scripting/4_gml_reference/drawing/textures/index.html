Texturas


Las siguientes funciones están relacionadas con "texturas", es decir: imágenes que se almacenan en VRAM para ser usadas como sprites u otras cosas dentro de tu juego. El primer conjunto de funciones disponibles se usa para recuperar información sobre imágenes específicas y su posición o tamaño en la Página de textura, así como para establecer ciertas características de textura:

  • texture_get_height
  • texture_get_width
  • texture_get_texel_height
  • texture_get_texel_width
  • texture_get_uvs
  • texture_set_stage
  • texture_global_scale

GameMaker Studio 2 también tiene una serie de funciones relacionadas con la gestión de la VRAM asociada a texturas individuales, así como a grupos de texturas completos:

  • draw_texture_flush
  • sprite_flush
  • sprite_flush_multi
  • sprite_prefetch
  • sprite_prefetch_multi
  • textura_prefetch
  • textura_flush

Finalmente, tenemos las siguientes funciones que se utilizan principalmente para depurar el proyecto y garantizar un uso eficiente de la memoria de textura:

  • textura_debug_messages
  • textura_is_ready
  • texturegroup_get_textures
  • texturegroup_get_sprites
  • texturegroup_get_fonts
  • texturegroup_get_tilesets
