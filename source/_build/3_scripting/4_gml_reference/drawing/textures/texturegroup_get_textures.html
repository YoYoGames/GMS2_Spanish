texturegroup_get_textures


Descripción

Esta función se puede utilizar para recuperar los ID de página de textura de las páginas individuales que forman un grupo de textura. Proporciona la cadena de ID del grupo de textura (como se define en el Editor de grupo de textura), y la función devolverá una matriz 1D, donde cada entrada de la matriz es una ID de página de textura única. Si la función falla, es decir, se proporciona un grupo no válido o si el grupo no tiene una textura asignada, la matriz estará vacía (longitud 0).


Sintaxis:

texturegroup_get_textures(tex_id);

Argumento Descripción
tex_id El nombre del grupo de texturas a verificar (una cadena)


Devoluciones:

1D array


Ejemplo:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length_1d(_tex_array); ++i;)
    {
    if texture_is_ready(_tex_array[i])
        {
        texture_prefetch(_tex_array[i]);
        }
    }

El código anterior recuperará los identificadores de la página de textura para el grupo de textura "Menú principal", luego verificará si están desempaquetados y, si los hay, se colocarán en VRAM.
