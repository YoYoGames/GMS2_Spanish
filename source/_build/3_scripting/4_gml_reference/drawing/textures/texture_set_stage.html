texture_set_stage


Descripción

Esta función establecerá la etapa dada "slot" una textura para ser utilizada. El número de "ranuras" de etapa disponibles dependerá de la plataforma que esté compilando, con un máximo de 8 disponibles para Windows, Mac y Linux, pero en dispositivos Android gama baja (por ejemplo) este número puede ser tan bajo como 2 También vale la pena tener en cuenta que GameMaker Studio 2 siempre utiliza la "ranura" de la primera etapa.

NOTA: ¡Esta función no hará nada fuera del contexto de un sombreador en ejecución! Ver Shaders para más información.


Sintaxis:

texture_set_stage(stage, tex);

Argumento Descripción
stage La textura "slot" a utilizar.
tex La textura a utilizar.


Devoluciones:

N/A


Ejemplo:

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

El código anterior obtendrá el identificador de la muestra dentro del sombreador indexado como "GlassShader" y luego establecerá esa constante de sombreado para la textura de sprite dada.
