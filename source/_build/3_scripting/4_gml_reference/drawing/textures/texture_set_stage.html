texture_set_stage

Indica qué textura "ranura" para agregar una textura a cuando se trabaja con sombreadores.

Sintaxis:

texture_set_stage(stage, tex);


Argumento Descripción
stage La textura "ranura" para usar.
tex La textura a usar


Devoluciones: N / A


Descripción

Esta función configurará la "ranura" de la etapa dada como una textura para ser utilizada. El número de "ranuras" de escenario disponibles dependerá de la plataforma en la que esté compilando, con un máximo de 8 disponibles para Windows, Mac y Linux, pero en dispositivos Android inferior (por ejemplo) este número puede ser tan bajo como 2 También vale la pena señalar que GameMaker Studio 2 siempre utiliza automáticamente la "ranura" de la primera etapa.

NOTA: ¡ Esta función no hará nada fuera del contexto de un sombreador en ejecución! Ver Shaders para más información.


Ejemplo:

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

El código anterior obtendrá el identificador de la muestra dentro del sombreador indexado como "GlassShader" y luego establecer esa constante de sombreado a la textura de sprite dada.
