texturegroup_get_tilesets


Descripción

Con esta función, puede recuperar el índice de conjunto de mosaicos de cada uno de los conjuntos de mosaicos asignados a las páginas de texturas dentro del grupo de texturas dado. Proporciona la cadena de ID del grupo de textura (como se define en el Editor de grupo de textura) y la función devolverá una matriz 1D donde cada entrada contiene el índice de conjunto de mosaicos para un recurso de conjunto de mosaicos. Si la función falla, es decir, se proporciona un grupo no válido o si el grupo no tiene una textura asignada, la matriz estará vacía (longitud 0).


Sintaxis:

texturegroup_get_tilesets(tex_id);

Argumento Descripción
tex El nombre del grupo de texturas a verificar (una cadena)


Devoluciones:

1D array


Ejemplo:

var _tex_array = texturegroup_get_tilesets( "MainMenu");
for (var i = 0; i < array_length_1d(_tex_array); ++i;)
    {
    show_debug_message("Tileset " + string(i) + " Index:" + string(tex_array[i]));
    }

El código anterior recuperará los índices de conjunto de mosaicos para el grupo de textura "MainMenu", luego mostrará esos ID en la ventana de resultados de la consola.
