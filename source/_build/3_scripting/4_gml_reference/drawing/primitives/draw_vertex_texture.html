draw_vertex_texture


Sintaxis:

draw_vertex_texture(x, y, xtex, ytex)


Argumento Descripción
x La coordenada x del vértice.
y La coordenada y del vértice.
xtex El ancho de la muestra de textura a usar.
ytex La altura de la muestra de textura a usar.


Devoluciones:

N/A


Descripción

Esta función define la posición de un vértice texturizado para una primitiva. El aspecto final de la primitiva dependerá del tipo primitivo elegido para dibujar (ver draw_primitive_begin para obtener más información), el orden con el que agrega los vértices y la posición del punto de inicio que estableció para la textura. Para finalizar y dibujar el primitivo debe llamar draw_primitive_end.

También puede optar por dibujar solo una parte de la textura sobre la superficie de la primitiva configurando la posición xtex / ytex adecuada para muestrear (estas coordenadas también se conocen como puntos UV ). Este punto se define por un valor entre 0 y 1, siendo la esquina superior izquierda de la imagen el punto (0,0) y la esquina inferior derecha el (1,1) punto. De esta forma, puede texturizar una primitiva con solo una parte del elemento de imagen de origen.

Al dibujar una primitiva texturada, la textura se mezclará con el color establecido por draw_set_colour. Esto significa que para dibujar la textura sin mezclar esto debe configurarse para c_white, como se muestra en el ejemplo a continuación.

NOTA: Para que una textura se repita, debe tener una potencia de dos, es decir: 32x32, 128x128, etc.


Ejemplo:

draw_set_colour(c_white);
var tex = background_get_texture(background0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 480, 1, 1);
draw_vertex_texture(640, 0, 1, 0);
draw_primitive_end();

El código anterior dibujará una franja triangular de 4 vértices (haciendo un rectángulo) texturizada con la textura contenida en la variable "tex", y toda la textura se usará para cubrir la primitiva completada.
