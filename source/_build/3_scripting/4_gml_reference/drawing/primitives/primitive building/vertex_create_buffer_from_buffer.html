vertex_create_buffer_from_buffer

Crea un buffer de vértice desde un buffer regular.

Sintaxis:

vertex_create_buffer_from_buffer(buffer, format);


Argumento Descripción
buffer El búfer para crear el búfer de vértices desde.
format El formato de vértice primitivo a usar.


Devoluciones: Puntero


Descripción

Como con la función vertex_create_buffer, esta función creará un nuevo búfer de vértice, solo que ahora los datos de vértice que almacena se copian del búfer regular que se especifica como fuente. El búfer creado es un búfer de crecimiento especial que está preformateado con el formato de vértice para construir primitivas para usar con sombreadores (por ejemplo). Esta función requiere que suministre el puntero a un búfer regular previamente creado y un formato de vértice que se debe aplicar a los datos copiados.


Ejemplo:

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat);

El código anterior creará un nuevo formato de vértice y luego creará un nuevo búfer de vértice a partir de un búfer regular creado previamente, aplicando el formato de vértice personalizado a él.
