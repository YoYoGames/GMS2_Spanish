vertex_texcoord

Establezca coordenadas UV para la textura que se usa para el vértice.

Sintaxis:

vertex_texcoord(buffer, u, v);


Argumento Descripción
buffer El buffer para escribir la información a.
u La primera coordenada de textura a usar (0 - 1).
v La segunda coordenada de textura a usar (0 - 1).


Devoluciones: N / A


Descripción

Esta función establecerá las coordenadas de textura que se usarán para el vértice que se está definiendo actualmente para la primitiva personalizada. Proporciona el búfer para escribir los datos, así como la posición de UV dentro de la textura a usar.


Ejemplo:

vertex_texcoord(buff, 0, 0);

El código anterior establecerá los valores UV del vértice que se está definiendo.
