draw_vertex_colour


Sintaxis:

draw_vertex_colour(x, y, col, alpha)


Argumento Descripción
x La coordenada x del vértice.
y La coordenada y del vértice.
col El color para dibujar este vértice.
alpha El alfa para dibujar este vértice con (0-1).


Devoluciones:

N/A


Descripción

Esta función define la posición de un vértice para una primitiva, con su propio ajuste de color y alfa. El aspecto final de la primitiva dependerá del tipo primitivo elegido para dibujar y del orden con el que se le agreguen los vértices (ver draw_primitive_begin para más información) y los vértices con diferentes colores y alfas se mezclarán suavemente de uno a otro. Para finalizar y dibujar el primitivo debe llamar draw_primitive_end.


Ejemplo:

draw_primitive_begin(pr_trianglelist);
draw_vertex_colour(100, 100, c_blue, 0.1);
draw_vertex_colour(100, 200, c_red, 0.1);
draw_vertex_colour(150, 150, c_green, 1);
draw_primitive_end();

El código anterior dibujará un triángulo semitransparente con cada vértice coloreado de un color diferente.
