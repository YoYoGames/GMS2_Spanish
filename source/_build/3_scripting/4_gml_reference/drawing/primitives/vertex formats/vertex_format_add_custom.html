vertex_format_add_custom

Agregue valores personalizados al formato de vértice.

Sintaxis:

vertex_format_add_custom(type, usage);


Argumento Descripción
type El tipo de datos que contendrá esta información de vértice personalizada (consulte las constantes de tipo enumeradas a continuación).
usage El uso que obtendrán los datos (consulte las constantes de uso que figuran a continuación).


Devoluciones: N / A


Descripción

Indique a GameMaker Studio 2 que acepte un valor (o valores) personalizado como parte del nuevo formato de vértice que se está creando. Los valores disponibles para usar están definidos por la constante de tipo de datos que elija, que se enumeran a continuación:

Constante Descripción
vertex_type_float1 Un único valor de coma flotante
vertex_type_float2 Dos valores de coma flotante
vertex_type_float3 Tres valores de coma flotante
vertex_type_float4 Cuatro valores de coma flotante
vertex_type_colour Cuatro componentes de valores (r, g, b, a)
vertex_type_ubyte4 Valores de byte sin signo de cuatro componentes (de 0 a 255)

También se debe definir el uso que se dará a estas constantes para que los valores se puedan "enlazar" correctamente dentro del sombreador que se está creando. Esto es necesario debido al hecho de que DX y OpenGL tienen diferentes requisitos, por lo que si no los vincula correctamente, no aparecerán directamente en el sombreado. Las constantes de uso disponibles que puede elegir se enumeran a continuación y las que use dependerán de los detalles del sombreador que se está creando:

Constante Descripción
vertex_usage_position valores de posición (x, y, z)
vertex_usage_colour valores de color (r, g, b, a)
vertex_usage_normal valores normales de vértice (nx, ny, nz)
vertex_usage_textcoord Coordenadas UV (u, v)
vertex_usage_blendweight el peso de mezcla de la matriz de entrada (para la animación esquelética, por ejemplo)
vertex_usage_blendindices los índices de las matrices a usar (para la animación esquelética, por ejemplo)
vertex_usage_depth valor del buffer de profundidad del vértice
vertex_usage_tangent valores tangentes
vertex_usage_binormal valores binormales
vertex_usage_fog valores de niebla
vertex_usage_sample índice de muestra


Ejemplo:

vertex_format_begin();
vertex_format_add_textcoord();
vertex_format_add_custom(vertex_type_float3, vertex_usage_position);
my_format = vertex_format_end();

El código anterior creará un nuevo formato de vértice con solo textura y 3 valores de punto flotante personalizados para la posición. A continuación, almacena el id. De formato en la variable "my_format".
