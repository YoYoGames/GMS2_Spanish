draw_primitive_end


Sintaxis:

draw_primitive_end()


Devoluciones:

N/A


Descripción

Esta función debe llamarse cuando haya terminado de definir los puntos de su primitiva. Si no llama a esta función, no se dibujará nada, ya que esto le indica a GameMaker Studio 2 que ha terminado y que ahora puede dibujar la primitiva definida.


Ejemplo:

var tex;
var tex = sprite_get_texture(spr_Background);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 480, 0, 0);
draw_vertex_texture(640, 480, 1, 0);
draw_vertex_texture(640, 480, 1, 1);
draw_vertex_texture(0, 480, 0, 1);
draw_primitive_end();

El código anterior dibujará una tira triangular de 4 vértices texturizada con la textura contenida en la variable "tex".
