url_open_full

Sintaxis:

url_open_full(url, target, options);


Argumento Descripción
url La URL (dirección del sitio web) a la que enlazar.
target Esta es el área objetivo para abrir la URL en (ver descripción).
options Opciones de navegador estándar (ver descripción).


Devoluciones: N / A


Descripción

Esto abrirá la URL especificada en el navegador del dispositivo de destino elegido o, si está utilizando el módulo de HTML5, en el navegador actualmente abierto. El parámetro "objetivo" que especifique es el mismo que el valor "nombre" de JavaScript estándar cuando usa el open() método (tenga en cuenta que todos menos '_self' pueden resultar en el bloqueo del navegador, o preguntar al usuario si desean permitirlo) y las "opciones" son las mismas que el parámetro JavaScript "specs" para controlar qué propiedades tiene la nueva ventana / tab debería mostrarse (no todos los navegadores pueden admitir todas las características).

Los objetivos válidos son:

"_self": abre el documento vinculado en el mismo cuadro en el que se hizo clic (es el valor predeterminado).
"_blank": abre el documento vinculado en una nueva ventana o pestaña (esto no funcionará si el usuario está bloqueando las ventanas emergentes, en cuyo caso puede usar el clickable_* funciones en su lugar).
"_parent": abre el documento vinculado en el cuadro principal.
"_top": abre el documento vinculado en todo el cuerpo de la ventana.

Las opciones válidas son:

"height = [px]": El alto de la ventana, con un valor mínimo de 100.
"width = [px]": El ancho de la ventana, con un valor mínimo de 100.
"left = [px]": La posición izquierda de la ventana.
"top = [px]": la posición superior de la ventana (solo IE).
"location = [boolean]": si se muestra o no el campo de dirección (el valor predeterminado es 1).
"menubar = [boolean]": si se muestra o no la barra de menús (el valor predeterminado es 1).
"resizable = [boolean]": independientemente de si la ventana es redimensionable (el valor predeterminado es 1).
"scrollbars = [boolean]": si se muestran o no las barras de desplazamiento (el valor predeterminado es 1).
"status = [boolean]": si se agrega o no una barra de estado (el valor predeterminado es 1).
"titlebar = [boolean]": si mostrar o no la barra de título. Esto se ignora a menos que la aplicación de llamada sea una aplicación HTML o un cuadro de diálogo de confianza (el valor predeterminado es 1);
"toolbar = [boolean]": si se muestra o no la barra de herramientas del navegador (el valor predeterminado es sí).

NOTA: En el Windows 8 plataforma de destino (JS), esta función va a funcionar, pero los parámetros adicionales será ignorado y el navegador por defecto será lanzado con la URL dada.


Ejemplo:

url_open_full( 'http://yoyogames.com', '_blank', 'resizable=0, height=200, scrollbars=0');

Esto abriría la página principal de YoYo Games en una nueva ventana que no se puede cambiar de tamaño, tiene una altura de 200 píxeles y no tiene barras de desplazamiento.
