Web

Las funciones enumeradas a continuación son funciones web generales que se pueden usar en todas las plataformas de destino:

  • url_open
  • url_open_ext
  • url_open_full
  • url_get_domain

Además de las funciones generales enumeradas anteriormente, el objetivo HTML5 tiene algunas funciones web adicionales que están específicamente diseñadas para ser utilizadas por los juegos que se dirigen a ella:

  • browser_height
  • browser_width
  • browser_input_capture
  • webgl_enabled
  • clickable_exists
  • clickable_add
  • clickable_add_ext
  • clickable_change
  • clickable_change_ext
  • clickable_set_style
  • clickable_delete
  • analytics_event
  • analytics_event_ext
  • os_browser

Además de estas funciones para el módulo de destino HTML5, también hay una variable especial de JavaScript que se puede usar en las raras ocasiones en que el contenido de la carpeta html5game se almacena en un lugar completamente diferente (URL). Normalmente, esta ruta está codificada en el archivo JavaScript y es relativa a la ubicación desde donde se llama el JavaScript, pero puede establecer la variable g_GameMakerHTML5Dir antes de la llamada al guión del juego y el juego se ejecutará desde la nueva ubicación definida. Por ejemplo:

<script>var g_GameMakerHTML5Dir = "http://mygamealternatedirectory.com/";</script>
