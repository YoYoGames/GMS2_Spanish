url_open_ext

Sintaxis:

url_open_ext(url, target);


Argumento Descripción
url La URL (dirección del sitio web) a la que enlazar.
target Esta es el área de destino para abrir la URL en.


Devoluciones: N / A


Descripción

Esto abrirá la URL especificada en el navegador del dispositivo de destino elegido o, si está utilizando el módulo de HTML5, en el navegador actualmente abierto. El parámetro "objetivo" que especifique es el mismo que el valor "nombre" de JavaScript estándar cuando usa el open() método, y debe tener en cuenta que todo, excepto '_self', puede provocar el bloqueo del navegador o preguntar al usuario si desea permitirlo.

Los objetivos válidos son:

"_self": abre el documento vinculado en el mismo cuadro en el que se hizo clic (es el valor predeterminado).
"_blank": abre el documento vinculado en una nueva ventana o pestaña (esto no funcionará si el usuario está bloqueando las ventanas emergentes, en cuyo caso puede usar el clickable_* funciones en su lugar).
"_parent": abre el documento vinculado en el cuadro principal.
"_top": abre el documento vinculado en todo el cuerpo de la ventana.

NOTA: En el Windows 8 plataforma de destino (JS), esta función va a funcionar, pero los parámetros adicionales será ignorado y el navegador por defecto será lanzado con la URL dada.


Ejemplo:

url_open_ext( 'http://yoyogames.com', '_blank');

Esto abriría la página de inicio de YoYo Games en una nueva ventana.
