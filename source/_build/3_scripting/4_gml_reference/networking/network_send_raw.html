network_send_raw

Sintaxis:

network_send_raw(socket, buffer, size);


Argumento Descripción
socket La identificación del socket a usar.
buffer La identificación del buffer para obtener los datos.
size El tamaño (en bytes) de los datos.


Devoluciones: Real


Descripción

Con esta función puede enviar un paquete de datos "en bruto" a través de la red. La función toma el ID del socket para conectarse y luego debe proporcionar el ID del buffer que contiene los datos que se enviarán (para más información sobre los buffers, consulte Reference - Buffers ) y finalmente el tamaño (en bytes) del paquete de datos. Los datos enviados no están formateados por GameMaker Studio 2 de ninguna manera y los dispositivos receptores recibirán los datos como una secuencia, lo que significa que usted tendrá que manejarlos usted mismo. La función devolverá el número de bytes de datos enviados, o un número menor que 0 si el envío ha fallado.


Ejemplo:

buff = buffer_load("player_save.dat");
network_send_raw(sock, buff, buffer_get_size(buff));

La información anterior carga los datos del búfer previamente guardados en la memoria y devuelve el ID del búfer que se almacenará en la variable "buff". A continuación, este búfer completo se envía como un paquete de datos brutos a través de la red utilizando el socket identificado por la variable "calcetín".
