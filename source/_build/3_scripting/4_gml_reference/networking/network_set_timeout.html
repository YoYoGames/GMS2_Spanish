network_set_timeout


Sintaxis:

network_set_timeout(socket, read_timeout, write_timeout);


Argumento Descripción
socket La identificación del socket a usar.
read_timeout Los milisegundos en los que se agota el tiempo de espera de una transferencia desde un servidor.
write_timeout Los milisegundos en los que se agotarán las transferencias a un servidor.


Devoluciones: N / A


Descripción

Con esta función, puede configurar el tiempo de espera para leer y escribir datos en / desde un servidor a través del socket dado. Tenga en cuenta que el tiempo de espera no genera ningún tipo de evento, por lo que tendrá que lidiar con tiempos de espera usando alarmas (por ejemplo). Tenga en cuenta que este valor solo afecta el envío y la recepción de datos, y si desea cambiar el valor de tiempo de espera de conexión, entonces debe utilizar la función network_set_config.


Ejemplo:

network_set_timeout(sock, 3000, 3000);

El código anterior establecerá el tiempo de espera para leer y escribir datos a través del socket indexado en la variable "calcetín" a 3 segundos.
