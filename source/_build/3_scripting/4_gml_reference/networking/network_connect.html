network_connect

Sintaxis:

network_connect(socket, url, port);


Argumento Descripción
socket La identificación del socket a usar.
url La URL o IP para conectarse (una cadena).
port El puerto para conectarse.


Devoluciones: Real


Descripción

Con esta función, puede enviar una solicitud para conectarse a un servidor. La función toma la identificación del socket para conectarse (ver network_create_socket ) y requiere que especifique la dirección IP a la que se conectará (una cadena) y el puerto a través del cual se conectará, y si la conexión falla, se devolverá un valor inferior a 0. La conexión utiliza un protocolo especial que asegura que solo los juegos de GameMaker Studio 2 conectan entre sí, sin embargo, si necesita conectarse a un servidor que no es un juego de GameMaker Studio 2, puede usar network_connect_raw.


Ejemplo:

client = network_create_socket(network_socket_tcp);
network_connect(client, "192.134.0.1", 6510);

El código anterior creará un nuevo socket TCP y luego intentará conectarse a esa dirección IP en el puerto 6510.
