network_send_packet

Sintaxis:

network_send_packet(socket, buffer, size);


Argumento Descripción
socket La identificación del socket a usar.
buffer La identificación del buffer para obtener los datos.
size El tamaño (en bytes) de los datos.


Devoluciones: Real


Descripción

Con esta función puede enviar un "paquete" de datos a través de la red. La función toma el ID del socket para conectarse y luego debe proporcionar el ID del buffer que contiene los datos que se enviarán (para más información sobre los buffers, consulte Reference - Buffers ) y finalmente el tamaño (en bytes) del paquete de datos. Los paquetes enviados con esta función tienen el formato tal que el juego GameMaker Studio 2 recibe los datos puede "dividir" correctamente los paquetes, y la función devolverá el número de bytes de datos enviados, o un número menor que 0 si el envío falló. Vale la pena señalar que el tamaño final de los datos que se envían devueltos por esta función también incluirá la información del encabezado de GameMaker, que es un adicional de 12 bytes.


Ejemplo:

buff = buffer_load("player_save.dat");
network_send_packet(sock, buff, buffer_get_size(buff));

La información anterior carga los datos de un búfer previamente guardados en la memoria y devuelve el ID del búfer que se almacenará en la variable "buff". Este búfer completo se envía como un paquete a través de la red utilizando el socket identificado por la variable "calcetín".
