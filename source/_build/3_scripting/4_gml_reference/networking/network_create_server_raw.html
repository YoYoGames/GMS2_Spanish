network_create_server_raw

Sintaxis:

network_create_server_raw(type, port, max_client);


Argumento Descripción
type El tipo de servidor para crear (ver las constantes enumeradas a continuación).
port El puerto que utilizará el servidor.
max_client El número máximo de clientes que pueden conectarse a la vez.


Devoluciones: Real


Descripción

Esta función se utiliza para crear un nuevo servidor de red para su juego, utilizando uno de los protocolos de conexión permitidos (consulte las constantes enumeradas a continuación). Proporciona el tipo de servidor, luego le da un puerto para usar y, finalmente, el número de conexiones máximas que se deben permitir en cualquier momento al servidor (tenga en cuenta que este número depende de usted, pero demasiados clientes conectados saturarán la La red o la CPU del dispositivo no podrán manejar el procesamiento de ese número de jugadores, así que utilícelo con cuidado. La función devuelve una identificación única que se debe usar almacenada en una variable y se usa para identificar el servidor en todas las funciones de red adicionales, o un valor inferior a 0 si la conexión falla.

Constante Descripción
network_socket_tcp Crear un socket web utilizando TCP.
network_socket_udp Crear un socket web utilizando UDP.
network_socket_bluetooth Crea un conector Bluetooth ( actualmente no disponible ).

Como esto crea un servidor "sin formato", no aceptará ni utilizará ninguno de los encabezados de datos integrados de GameMaker para la comunicación, por lo que debería usar las funciones network_send_raw y network_send_udp_raw() para enviar datos sin formatear al servidor creado.


Ejemplo:

var port = 6510;
server = network_create_server_raw(network_socket_tcp, port, 32);
while (server < 0 && port < 65535)
   {
   port++
   server = network_create_server(network_socket_tcp, port, 32);
   }

El código anterior intentará crear un servidor que use TCP a través del puerto 32. Si ese puerto no está disponible, entonces recorrerá los puertos para encontrar uno que esté disponible.
