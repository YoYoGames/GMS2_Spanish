network_send_broadcast

Sintaxis:

network_send_broadcast(socket, port, buffer, size);


Argumento Descripción
socket La identificación del socket a usar.
port El puerto que usará el servidor.
buffer La identificación del buffer para obtener los datos.
size El tamaño (en bytes) de los datos.


Devoluciones: Real


Descripción

Con esta función, puede transmitir los datos de un búfer localmente a un rango de direcciones IP (para obtener más información sobre búferes, consulte Referencia - Búferes ). El alcance se limita al del dispositivo que ejecuta el servidor, de modo que si el dispositivo tiene una IP de 92.168.11.130, los datos se emitirán en el rango 92.168.11. *. La función devolverá el número de bytes de datos enviados, o un número menor que 0 si el envío ha fallado.

NOTA: Esta función solo funcionará cuando se use con UDP: su servidor necesita ser TCP y su cliente necesita tener un socket de cliente UDP creado con network_create_socket_ext() para recibir cualquier transmisión enviada desde el servidor.


Ejemplo:

buffer_seek(broadcast_buffer, buffer_seek_start, 0);
buffer_write(broadcast_buffer, buffer_string, global.ServerName);
network_send_broadcast(server, 6511, broadcast_buffer, buffer_tell(broadcast_buffer));

El código anterior escribe la cadena del nombre del servidor actual (almacenado en "global.ServerName"), luego lo escribe en un búfer binario con el id "broadcast_buffer". Estos datos luego se transmiten localmente a un rango de direcciones IP (la IP del dispositivo actualmente está implícita como el rango base de difusión) al puerto 6511.
