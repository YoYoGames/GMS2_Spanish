network_set_config

Sintaxis:

network_set_config(config_value, setting);


Argumento Descripción
config_value El valor de configuración constante para establecer (se enumeran a continuación).
setting El ajuste del valor de configuración.


Devoluciones: cadena


Descripción

Con esta función puede establecer diferentes configuraciones de red. Estas configuraciones se dan como constantes y la configuración dependerá de la constante que haya elegido. La siguiente tabla enumera las constantes disponibles y su configuración:

Constante Descripción Ajuste
network_config_connect_timeout Establecer un valor de tiempo de espera de conexión A value in milliseconds
network_config_use_non_blocking_socket Dile a GameMaker Studio 2 no bloquee al conectar. 0 = block, 1 = don't block



La primera opción ( network_config_connect_timeout ) simplemente establece el tiempo de espera para conectarse a un servidor, pero no cambia el comportamiento de la conexión, aparte del tiempo que tiene que esperar. Se recomienda que no establezcas esto demasiado bajo y lo mantengas en aproximadamente 1000 para un juego de LAN solamente o 4000 o más para Internet, o el juego puede no conectarse aleatoriamente. Si desea establecer un valor de tiempo de espera para enviar / recibir paquetes, utilice la función network_set_timeout().

La segunda opción ( network_config_use_non_blocking_socket ) significa que las funciones de conexión de red devolverán una socket_id instantáneamente, pero no puede enviar o recibir en ese socket hasta que haya recibido un evento de red asíncrono. El evento desencadenado tendrá la clave "tipo" configurada para network_type_non_blocking_connect (Puede encontrar más detalles de la página en el evento asincrónico de red ). Esta es una configuración global ya que GameMaker Studio 2 no admite una mezcla de bloqueo y no bloqueo en una sola aplicación.


Ejemplo:

network_set_config(network_config_connect_timeout, 1000);
network_set_config(network_config_use_non_blocking_socket, 1);

El código anterior configurará el tiempo de espera para la conexión de red a 1000 ms y le indicará a GameMaker Studio 2 no bloquee al conectarse.
