GameMaker Studio 2 tiene una gran cantidad de formas en las que puedes mover instancias dentro de tu sala de juegos, que van desde establecer vectores de velocidad y dirección hasta simplemente "ubicar" una instancia en una posición específica, así como usar métodos más sofisticados como ruta de acceso. hallazgo. También hay un buen número de métodos diferentes para detectar colisiones entre dos instancias, y el que elija dependerá en gran medida del tipo de juego que esté realizando y la situación exacta en la que necesite usar las funciones de colisión.
