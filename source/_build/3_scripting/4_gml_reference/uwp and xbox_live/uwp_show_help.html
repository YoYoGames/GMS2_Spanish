uwp_show_help


Descripción

Esta función inicia la aplicación de ayuda Xbox One, que a su vez mostrará el archivo de ayuda de su juego (que es un archivo en formato HTML que usted ha cargado previamente en el Portal de desarrolladores de XBox). Tenga en cuenta que se especifica una ID de usuario cuando se llama a esta función, de modo que la información de configuración regional del usuario se puede usar cuando se muestra. Puede recuperar el ID de usuario con las funciones xboxlive_get_user(), xboxlive_get_activating_user() o uwp_license_trial_user.

NOTA: esta función solo es aplicable cuando se envían juegos a través de XBox Live para XBox One con el paquete UWP.


Sintaxis:

uwp_show_help(user_id);

Argumento Descripción
user_id El puntero de ID de usuario para abrir el archivo de ayuda para.


Devoluciones:

Boolean


Ejemplo:

if gamepad_button_check_pressed(0, gp_start)
    {
    if uwp_license_trial_version()
        {
        var _uid = uwp_license_trial_user;
        uwp_show_help(_uid);
        }
    }

El código anterior verifica si se presiona un botón y luego verifica la aplicación para ver si se está ejecutando con una licencia de prueba. Si es así, obtiene el ID de usuario para la licencia y abre el archivo de ayuda para ese usuario.
