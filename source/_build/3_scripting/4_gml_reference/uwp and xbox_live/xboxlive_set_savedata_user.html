xboxlive_set_savedata_user


Descripción

Esta función especifica que las futuras operaciones de archivos que operan en el área de guardar juegos (es decir, todas las escrituras de archivos y las lecturas de archivos que no están en el área del paquete) se asociarán con el puntero de ID de usuario especificado. Puede llamarlo con la frecuencia que sea necesaria para redirigir los datos guardados al usuario apropiado, o puede usar la constante pointer_null para guardar en el área de almacenamiento de la máquina genérica.

¡IMPORTANTE! Esta función solo es válida cuando se exporta utilizando el objetivo de UWP para Xbox One y requiere que hayas marcado la opción Habilitar XBox Live en las Opciones de juego de UWP.


Sintaxis:

xboxlive_set_savedata_user(user_id);

Argumento Descripción
user_id La ID de usuario (un puntero) para configurar para guardar, o pointer_null


Devoluciones:

N/A


Ejemplo:

if xboxlive_get_savedata_user() != user_id[0]
    {
    xboxlive_set_savedata_user(user_id[0]);
    }

El código anterior verifica si un usuario está asignado actualmente como destino de guardado, y si no lo están, entonces están asignados.
