UWP y XBox Live


Esta sección describe las funciones específicamente para usar cuando se dirige a la plataforma de destino UWP. El conjunto inicial de funciones es para controlar el comportamiento general del juego cuando se crea una aplicación UWP:

  • uwp_suspend
  • uwp_is_suspending
  • uwp_is_constrained
  • uwp_was_terminated
  • uwp_was_closed_by_user
  • uwp_show_help
  • uwp_license_trial_version
  • uwp_license_trial_user
  • uwp_license_trial_time_remaining
  • uwp_check_privilege

UWP también le permite crear juegos para la plataforma XBox One utilizando la API de XBox Live utilizando las siguientes funciones. Estas funciones solo son válidas cuando se exportan utilizando el objetivo de UWP para Xbox One y requieren que hayas marcado la opción Habilitar XBox Live en las Opciones de juego de UWP.

Algunas de estas funciones requieren que el usuario inicie sesión en XBox Live, que GameMaker Studio 2 intentará hacer automáticamente cuando ejecute el proyecto. Este inicio de sesión "silencioso" generará un Evento Asíncrono del Sistema que puede verificarse para ver si tuvo éxito o no antes de continuar. Tenga en cuenta que, a diferencia de la exportación de la Consola Xbox One, la exportación de UWP para XBox Live solo permite el inicio de sesión de un usuario a la vez.

Las siguientes funciones existen para que las uses con XBox Live en tus proyectos:

  • xboxlive_user_is_signed_in
  • xboxlive_user_is_signing_in
  • xboxlive_gamertag_for_user
  • xboxlive_show_account_picker
