xboxlive_stats_set_stat_int


Descripción

Esta función se puede usar para establecer el valor de una estadística para el ID de usuario dado. Usted suministra el ID de usuario tal como lo devolvió (por ejemplo) la función xboxlive_get_user(), luego la cadena de estadísticas que se establecerá ( si la cadena de estadísticas aún no existe, se creará una nueva estadística y se establecerá el valor dado ) y un valor (un entero) para establecer la estadística en. Tenga en cuenta que el nombre de la estadística debe ser solo alfanumérico, sin símbolos ni espacios.

Al configurar el valor estadístico, cualquier valor anterior se anulará, por lo que depende de usted determinar si el valor estadístico debe actualizarse o no (es decir, verificar que la puntuación más alta sea realmente la más alta) comparando con el valor estadístico actual con el nuevo antes de configurarlo.

La función devolverá -1 si hubo un error o la ID de usuario no es válida, o cualquier otro valor si la función fue llamada con éxito.

¡IMPORTANTE! Esta función solo es válida cuando se exporta utilizando el objetivo de UWP para Xbox One y requiere que hayas marcado la opción Habilitar XBox Live en las Opciones de juego de UWP.


Sintaxis:

xboxlive_stats_set_stat_int(user_id, stat, value);

Argumento Descripción
user_id El puntero de ID de usuario para establecer la estadística para
stat La estadística para establecer (una cadena)
value El valor para establecer la estadística a (un entero)


Devoluciones:

Real


Ejemplo:

var _val = xboxlive_stats_get_stat(p_user_id, "GamesPlayed");
xboxlive_stats_set_stat_int(p_user_id, "GamesPlayed", ++_val);

El código anterior recupera el valor de la estadística "GamesPlayed", luego establece la misma estadística a ese valor más 1.
