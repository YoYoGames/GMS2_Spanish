xboxlive_stats_get_leaderboard


Descripción

Esta función se puede utilizar para recuperar una tabla de clasificación global de rangos para una estadística determinada. Usted proporciona el ID de usuario (como lo devuelve, por ejemplo, la función xboxlive_get_user ), la cadena de estadísticas (como se define cuando la registró como una "Estadística destacada"), y luego especifica una serie de detalles sobre la información de la tabla de clasificación que desea recuperar. Tenga en cuenta que solo puede recuperar una tabla de clasificación global para estadísticas int o reales, pero no para estadísticas de cadena.

¡IMPORTANTE! Las estadísticas utilizadas en las tablas de clasificación mundial deben registrarse como "Estadísticas destacadas" en el XDP / Windows Dev Center, de lo contrario se devolverá un error. Si desea tablas de clasificación locales (sociales), consulte la función xboxlive_stats_get_social_leaderboard().


La función generará una devolución de llamada que activará un evento asíncrono del sistema. Este evento tendrá incorporado el mapa de DS. async_load que luego debe ser analizado para las siguientes claves:

  • " id " - mantendrá la constante achievement_stat_event

  • " eventname " - mantendrá la cadena " GetLeaderboardComplete "

  • " ID de usuario ": el ID de usuario asociado con la solicitud

  • " error " - 0 si es exitoso, algún otro valor si ha habido un error. Los siguientes dos son los errores más comunes devueltos:

    • 2145844844: No se puede encontrar la tabla de clasificación solicitada (la estadística no está registrada como una estadística destacada)
    • -2145844848: solicitud incorrecta (la estadística no es un tipo de tabla de clasificación válida, es decir, una cadena)

  • "ErrorMessage" - Una cadena con un mensaje de error, si lo hay está disponible

  • " display_name ": la ID única para la tabla de clasificación tal como se define en el panel del proveedor.

  • " números de entrada ": el número de entradas en la tabla de clasificación que ha recibido.

El resto del mapa de DS también contendrá los datos de la tabla de clasificación con el siguiente formato (donde "N" es la posición en la información de la tabla de clasificación, de 0 a "números"):

  • " Jugador N ": el nombre del jugador, donde "N" es un valor entero correspondiente a su posición dentro de la lista de entradas de la tabla de clasificación.

  • " Playerid N ": la ID de usuario única del jugador, "N".

  • " Rango N ": el rango del jugador "N" dentro de la tabla de clasificación.

  • " Puntuación N " - La puntuación del jugador "N".

¡IMPORTANTE! Esta función solo es válida cuando se exporta utilizando el objetivo de UWP para Xbox One y requiere que hayas marcado la opción Habilitar XBox Live en las Opciones de juego de UWP.


Sintaxis:

xboxlive_stats_get_leaderboard(user_id, stat, num_entries, start_rank, start_at_user, ascending);

Argumento Descripción
user_id El ID de usuario del usuario para obtener la tabla de clasificación para
stat El stat (como cadena) para crear la tabla de clasificación global de
num_entries El número de entradas de la tabla de clasificación global para recuperar
start_rank El rango en la tabla de clasificación para comenzar (use 0 si el argumento "start_at_user" está establecido en true )
start_at_user Ajustado a true para comenzar en el rango de ID de usuario, false de lo contrario (establecido en false si el argumento "start_rank" es distinto de 0)
ascending Ajustado a true por orden ascendente y false por orden descendente

N/A


Ejemplo extendido:

El siguiente es un ejemplo extendido de cómo se puede utilizar esta función. Para empezar, lo llamaría en algún evento como Room Start o Create:

xboxlive_stats_get_leaderboard(user_id, "GlobalTime", 20, 1, false, true);

Se llamaría al código anterior para obtener una lista de todas las posiciones de la tabla de clasificación social para el juego, y generará una devolución de llamada de evento asíncrono social que trataríamos de la siguiente manera:

if async_load[? "id"] == achievement_stat_event
    {
    if async_load[? "eventname"] == "GetLeaderboardComplete"
        {
        global.numentries = async_load[? "numentries"];
        for(var i = 0; i < numentries; i++;)
            {
            global.playername[i] = async_load[? "Player" + string(i)];
            global.playerid[i] = async_load[? "Playerid" + string(i)];
            global.playerrank[i] = async_load[? "Rank" + string(i)];
            global.playerscore[i] = async_load[? "Score" + string(i]);
            }
        }
    }

El código anterior comprueba el ds_map devuelto en el evento asíncrono social y si su "id" coincide con la constante que se está verificando, luego verifica si el evento ha sido activado por los datos devueltos de la tabla de clasificación antes de recorrer el mapa y almacenar todos los valores en una serie de matrices globales.
