xboxlive_stats_set_stat_string


Descripción

Esta función se puede usar para establecer una estadística en una cadena para el ID de usuario dado. Usted suministra el ID de usuario tal como lo devolvió (por ejemplo) la función xboxlive_get_user(), luego la cadena de estadísticas que se establecerá ( si la cadena de estadísticas aún no existe, se creará una nueva estadística y se establecerá en el valor dado ) y luego la cadena para establecer la estadística en. Tenga en cuenta que el nombre de la estadística debe ser solo alfanumérico, sin símbolos ni espacios, pero el valor de la cadena que está configurando puede tener espacios y caracteres especiales.

Al configurar la cadena de estadísticas, se anulará cualquier cadena anterior, por lo que depende de usted determinar si la cadena de estadísticas debe actualizarse o no comparándola con la cadena de estadísticas actual con la nueva antes de establecerla.

La función devolverá -1 si hubo un error o la ID de usuario no es válida, o cualquier otro valor si la función fue llamada con éxito.

¡IMPORTANTE! Esta función solo es válida cuando se exporta utilizando el objetivo de UWP para Xbox One y requiere que hayas marcado la opción Habilitar XBox Live en las Opciones de juego de UWP.


Sintaxis:

xboxlive_stats_set_stat_string(user_id, stat, value);

Argumento Descripción
user_id El puntero de ID de usuario para establecer la estadística para
stat La estadística para establecer (una cadena)
value El valor para establecer la estadística a (una cadena)


Devoluciones:

Real


Ejemplo:

xboxlive_stats_set_stat_string(p_user_id, "LevelName", global.LevelName);

El código anterior establece la estadística "LevelName" en la cadena almacenada en la variable global dada.
