xboxlive_achievements_set_progress


Descripción

Esta función se puede utilizar para actualizar el progreso de un logro. Usted proporciona el ID de usuario tal como lo devolvió, por ejemplo, la función xboxlive_get_user y luego la cadena de logros (esta es la ID de logros asignada en el XDP / Windows Dev Center cuando se creó). Finalmente, establece el valor de progreso en un valor de 0 a 100. Tenga en cuenta que el sistema de logros rechazará las actualizaciones que sean más bajas que el valor de progreso actual.


Sintaxis:

xboxlive_achievements_set_progress(user_id, achievement, progress);

Argumento Descripción
user_id El ID de usuario del usuario para configurar el logro para
achievement El logro de establecer (una cadena)
progress El valor de progreso a establecer (de 0 a 100).


Devoluciones:

N/A


Ejemplo extendido:

El siguiente es un ejemplo extendido de cómo se puede utilizar esta función. Para empezar, lo llamaría en algún evento como Room Start o Create:

xboxlive_stats_get_social_leaderboard(user_id, "GlobalTime", 20, 1, false, false);

Se llamará al código anterior para obtener una lista de todas las posiciones de la tabla de clasificación mundial para el juego, y generará una devolución de evento asíncrono social que trataríamos de la siguiente manera:

var _progress = (global.Level / global.LevelMax) * 100;
xboxlive_achievements_set_progress(user_id, "Game_Completed", _progress);

El código anterior crea un valor porcentual basado en algunas variables globales y luego lo utiliza para establecer el progreso del logro para el logro "Juego_Completado".
