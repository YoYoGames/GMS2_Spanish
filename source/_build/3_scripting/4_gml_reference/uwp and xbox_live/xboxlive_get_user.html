xboxlive_get_user


Descripción

Con esta función puede recuperar el puntero de ID de usuario para el usuario indexado. Si el usuario no existe, la función devolverá la constante. pointer_null en lugar. Puede encontrar el número de usuarios conectados actualmente con la función xboxlive_get_user_count().

IMPORTANTE: Esta función solo debe ser usada con xboxlive_get_user_count() - No uses esto con los gamepads. Utilice xboxlive_user_for_pad en su lugar.
¡IMPORTANTE! Esta función solo es válida cuando se exporta utilizando el objetivo de UWP para Xbox One y requiere que hayas marcado la opción Habilitar XBox Live en las Opciones de juego de UWP.


Sintaxis:

xboxlive_get_user(index);

Argumento Descripción
index El índice (un entero) para obtener el ID de usuario.


Devoluciones:

Pointer


Ejemplo:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
    {
    user_id[i] = xboxlive_get_user(i);
    }

Lo anterior recorre todos los usuarios que han iniciado sesión y almacena su puntero de ID único en una matriz.
