xboxlive_sprite_add_from_gamerpicture


Descripción

Con esta función, puede obtener la imagen del jugador para un puntero de ID de usuario determinado. La función funciona de forma asíncrona, y activará un evento asíncrono de imagen cargada para informarle que la función ha finalizado (como con HTML5 / URL llamadas a sprite_add() ). El evento Asynchronous Image Loaded tendrá un mapa DS almacenado en la variable incorporada async_load. El mapa contiene la siguiente información:

  • " nombre de archivo ": la ruta completa al archivo que solicitó.
  • " id ": el ID del recurso que ha cargado. Esta será la misma que la variable a la que ha asignado el recurso.
  • " estado ": devuelve un valor inferior a 0 para un error.
NOTA: cuando cargue dinámicamente un sprite en su juego en tiempo de ejecución, debe recordar quitarlo nuevamente usando sprite_delete() cuando ya no sea necesario, de lo contrario, existe el riesgo de una pérdida de memoria que ralentizará y eventualmente bloqueará su juego.
¡IMPORTANTE! Esta función solo es válida cuando se exporta utilizando el objetivo de UWP para Xbox One y requiere que hayas marcado la opción Habilitar XBox Live en las Opciones de juego de UWP.


Sintaxis:

xboxlive_sprite_add_from_gamerpicture(user_id, imagesize, xorig, yorig);

Argumento Descripción
user_id El ID de usuario (un puntero) para obtener la imagen del jugador para
imagesize Tamaño en píxeles del sprite a devolver.
xorig Indica la posición x del origen en el sprite.
yorig Indica la posición y del origen en el sprite.

Ejemplo:

var _a = 0;
var _num = xboxlive_get_user_count();
for (var i = 0; i < _num; ++i;)
    {
    var _uid = xboxlive_get_user(i);
    if _uid != pointer_null
        {
        global.UserName[_a] = xboxlive_gamedisplayname_for_user(_uid);
        global.UserScore[_a] = xboxlive_gamerscore_for_user(_uid);
        global.UserRep[_a] = xboxlive_reputation_for_user(_uid);
        global.UserAvatar[_a] = xboxlive_sprite_add_from_gamerpicture(_uid, 256, 0, 0);
        ++a;
        }
    }

El código anterior recorre los usuarios registrados y almacena sus datos de jugador en varios arreglos globales.
