xboxlive_agegroup_for_user


Descripción

Esta función devolverá una de las cuatro constantes (que se muestra a continuación) para indicar el rango de edad asignado al puntero de ID de usuario especificado.

¡IMPORTANTE! Esta función solo es válida cuando se exporta utilizando el objetivo de UWP para Xbox One y requiere que hayas marcado la opción Habilitar XBox Live en las Opciones de juego de UWP.


Sintaxis:

xboxlive_agegroup_for_user(user_id);

Argumento Descripción
user_id El ID de usuario (un puntero) para comprobar


Devoluciones:

Constant

Constante Descripción
xboxlive_agegroup_unknown El grupo de edad para el usuario es desconocido.
xboxlive_agegroup_child El grupo de edad del usuario es entre 8 y 12 (inclusive)
xboxlive_agegroup_teen El grupo de edad del usuario es entre 13 y 17 (incluido, y entre 13 y 19 en Corea del Sur)
xboxlive_agegroup_adult El grupo de edad del usuario es mayor de 18 años (o mayor de 20 años en Corea del Sur).


Ejemplo:

var _age = xboxlive_agegroup_for_user(xboxlive_get_activating_user());
if _age != xboxlive_agegroup_adult
    {
    global.ContentControl = true;
    }
else global.ContentControl = false;

El código anterior verifica el grupo de edad del usuario que activa y establece una variable global para true o false Dependiendo de la constante devuelta.
