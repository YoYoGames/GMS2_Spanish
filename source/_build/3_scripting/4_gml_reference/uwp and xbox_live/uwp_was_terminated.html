uwp_was_terminated


Descripción

Con esta función, puede averiguar si el sistema cerró previamente el juego (es decir, no es la primera ejecución del juego). Si ha sido así, la función volverá. true, de lo contrario volverá false. Un ejemplo de uso sería verificarlo y ver si el sistema terminó previamente el juego, y luego (si vuelve) true ) para cargar la etapa anterior en la que se encontraba el jugador en el juego. Si se dirige a la XBox One, el comportamiento requerido de los juegos si se detecta esto será diferente entre las diferentes versiones de los documentos de certificación, por lo que vale la pena revisar la última versión para ver qué debe hacer el juego.


Sintaxis:

uwp_was_terminated();


Devoluciones:

Boolean


Ejemplo:

if xboxlivewas_terminated()
    {
    buffer_load("Player_Save.sav");
    global.GameLoaded = true;
    }

El código anterior verifica si la aplicación ha sido terminada previamente por el sistema y, de ser así, carga un búfer y establece una variable global.
