iap_product_details

Agregue detalles de un solo producto a un ds_map.

Sintaxis:

iap_product_details(product_id, ds_map);


Argumento Descripción
purchase_id La cadena de identificación del producto para la compra.
ds_map El ds_map que almacenará la información del producto.


Devoluciones: N / A


Descripción

Con esta función puedes poblar una previamente creada ds_map con una serie de pares clave / valor que describen la compra dada. Los siguientes datos se devuelven dentro del mapa:

  • " id ": la cadena de identificación del producto.

  • " título " - El título del producto.

  • " descripción " - La descripción del producto.

  • " precio ": el precio del producto, cuando esté disponible (no todas las tiendas lo admiten).

  • " tipo ": el tipo de producto, que será "Consumible" o "Durable", donde esté disponible (no todas las tiendas lo admiten).

  • " verificado " - Será cualquiera true o false (es decir: números reales 0 o 1) dependiendo de si su juego ha recibido o no detalles de este producto en la tienda y, por lo tanto, cree que este producto está disponible para su compra.

NOTA: el ds_map used no es creado por la función, lo que significa que debe crearlo previamente y eliminarlo nuevamente de la memoria cuando no esté en uso con la función apropiada.


Ejemplo:

var p_map = ds_map_create();
iap_product_details(ds_map_find_value(iap_data, "index"), p_map);
p_title = " + string(ds_map_find_value(p_map, "title")));
p_description = " + string(ds_map_find_value(p_map, "description")));
ds_map_destroy(p_map);

El código anterior creará un ds_map y luego llenarlo con los detalles del producto tomado del producto especial iap_data ds_map que se crea exclusivamente en el evento IAP. El "título" y la "descripción" se agregan luego a las variables para que luego puedan dibujarse en la pantalla.
