iap_consume

Borra el estado de compra de las compras no administradas.

Sintaxis:

iap_consume(purchase_id);


Argumento Descripción
purchase_id La cadena de identificación del producto para la compra.


Devoluciones: N / A


Descripción

Esta función se puede usar para borrar el estado de compra de cualquier compra no administrada (consumible). El "purchase_id" es la cadena de identificación del producto que especificó al activar el IAP usando iap_activate y debe ser el mismo que usaste para identificar el producto en la tienda objetivo. Esta función desencadenará un evento IAP del "tipo" iap_ev_consume, que contendrá las siguientes claves adicionales:

  • " producto ": la identificación del producto consumido.

Este ID de producto se puede usar para verificar qué compra ha desencadenado el evento y puede tratarlo en consecuencia.

ADVERTENCIA: no puede comprar un producto consumible varias veces seguidas sin consumirlo; debe realizar una operación de consumo entre compras posteriores.


Ejemplo:

if mouse_check_button_pressed(mb_left)
   {
   iap_consume("noads")
   }

El código anterior borrará el estado de la compra cuando se presiona el botón del mouse.
