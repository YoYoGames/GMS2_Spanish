iap_status

Le permite rastrear el estado del sistema de compras en la aplicación.

Sintaxis:

iap_status();


Devoluciones: constante


Descripción

Esta función le permite rastrear el estado de GameMaker Studio 2 en el sistema de compras de la aplicación y devuelve una de las siguientes constantes:

Constante Descripción
iap_status_uninitialised en compras de aplicaciones no se han activado para su juego.
iap_status_unavailable No hay conexión disponible para la tienda objetivo.
iap_status_loading Se cierra una conexión a la tienda objetivo y se está activando.
iap_status_available Las compras se han activado y la tienda objetivo está disponible.
iap_status_processing Actualmente hay comunicación entre el juego y la tienda objetivo, como cuándo (por ejemplo) hacer una compra o restaurar una compra, etc.
iap_status_restoring En el objetivo de iOS, se puede devolver mientras el juego intenta restaurar las compras.


Ejemplo:

status = iap_status();

El código anterior solicita el estado de la tienda para el juego actual y lo almacena en una variable para verificar.
