iap_enumerate_products

Complete una ds_list con todos los productos IAP actualmente disponibles en la tienda de destino.

Sintaxis:

iap_enumerate_products();


Devoluciones: N / A


Descripción

Esta función recupera la lista de identificadores de productos de compras en la aplicación encontrados. En la mayoría de las plataformas, esto simplemente devolverá el conjunto de identificadores de productos proporcionados a través de iap_activate(), pero en las plataformas donde la tienda ofrece la posibilidad de recuperar una lista completa de los productos disponibles para un título, este conjunto de identificadores de productos incluirá todos los identificadores de productos disponibles para el juego. La función requiere que suministre un producto previamente hecho ds_list (que debe borrar primero si se ha utilizado para otros fines) que se completará con la llamada a la función. Independientemente de la plataforma, si la tienda objetivo no está disponible, obtendrá los ID de productos de los que se proporcionaron cuando activó las compras utilizando iap_activate.


Ejemplo:

var status = iap_status();
var iap_list = ds_list_create();
if status == iap_status_available
   {
   iap_enumerate_products(iap_list);
   }

El código anterior verifica el estado de la tienda IAP de destino y, si está disponible, rellenará la lista creada con las ID de productos disponibles para uso futuro.
