font_add_sprite


Descripción

Con esta función, puede usar una "tira de sprites" para crear un nuevo recurso de fuente, donde cada subimagen sería un símbolo o letra individual. El índice es el número de mapa UTF8 para el primer personaje que se va a usar y los intervalos que se usan son los mismos que para una fuente normal, comenzando con "!" pero recuerde que solo debe incluir lo que necesita y que debe usar el ord función para obtener el valor UTF8 correcto para la letra de inicio. La fuente de sprite creada tendrá las subimágenes configuradas en un glifo individual de la fuente, donde el "primer" glifo tendrá el índice de imagen de 0. Puede definir "espacio" como cualquier carácter que desee, por ejemplo, un solo alinee el tamaño que desea que tenga el espacio, y todos los espacios en el texto se representarán en ese ancho (pero la imagen elegida nunca se renderizará), o si no proporciona un espacio para el espacio, entonces el ancho del espacio más ancho carácter en la fuente de sprite se utilizará en su lugar.

También puede configurar el espaciado de la fuente para que sea proporcional ( true ) o no ( false ), donde una fuente proporcional está espaciada en función del ancho real de las letras (por lo que la letra "i" ocupa menos espacio que la letra "w", por ejemplo), mientras que una fuente no proporcional se espaciará según el subtítulo ancho de imagen, como una fuente monoespaciado. Finalmente, puede definir el espacio que debe dejarse entre cada letra al escribir, y este puede ser cualquier valor entero, donde 0 es sin espacio (las letras "tocarán" si son proporcionales). La función devuelve un valor de índice que debe almacenarse en una variable, ya que será necesario en todos los demás códigos que hagan referencia a esta fuente.

También vale la pena señalar que no puede haber espacios en el mapa de caracteres, de modo que si usted, por ejemplo, usa '.' y '0' pero no '/', debe incluir '/' de todos modos o dejar una subimagen en blanco entre los dos caracteres utilizados.

El sprite en sí puede ser un activo de sprite del árbol de recursos o uno cargado desde un archivo usando sprite_add primero para obtener el índice del sprite cargado (que luego se usa en la función). Para obtener más detalles sobre el mapa de caracteres que debe usar para crear fuentes de sprite, consulte la página en las tablas de fuentes. Es importante tener en cuenta que para las funciones de alineación de fuentes (como draw_set_halign() ) para que funcione como se espera al usar una fuente de sprite, el sprite debe tener su origen en la esquina superior izquierda. Si usa otros valores, tendrá que tener en cuenta la compensación de origen al dibujar texto con la fuente.

NOTA: Cuando cargue una fuente en GameMaker Studio 2, debe recordar eliminarla nuevamente (con font_delete ) cuando ya no sea necesario, de lo contrario existe el riesgo de una pérdida de memoria que disminuirá la velocidad y, finalmente, se bloqueará el juego.

Sintaxis:

font_add_sprite(spr, first, prop, sep);


Argumento Descripción
spr El sprite para agregar una fuente basada en
first El índice del primer personaje disponible.
prop Establecer como fuente proporcional o no.
sep El espacio para irse entre cada letra.


Devoluciones:

Real


Ejemplo:

global.Font = font_add_sprite(spr_HUD_Font, ord("!"), true, 2);

El código anterior creará un nuevo recurso de fuente desde un elemento de sprite y almacenará el índice de la nueva fuente en la variable "global.Font" para su uso posterior.
