font_get_uvs


Descripción

Esta función devuelve una matriz con las coordenadas UV para la textura de la fuente en la página de textura, completando la matriz con los siguientes valores: [0] = izquierda, [1] = arriba, [2] = derecha y [3] = abajo. Este valor se puede usar en otras funciones de dibujo, especialmente en el dibujo general cuando se utilizan primitivos y las funciones de sombreado.


Sintaxis:

font_get_uvs(font);


Argumento Descripción
font El índice de la fuente a usar.


Devoluciones:

Array


Ejemplo:

var tex = font_get_uvs(fnt_Main);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

El código anterior almacenará las coordenadas UV para el fondo dado en una matriz local y luego asignará los valores a las variables de instancia.
