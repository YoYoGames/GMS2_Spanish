font_texture_page_size


Descripción

Esta variable incorporada se puede usar para obtener o establecer el tamaño de la página de textura cuando se usa la función font_add. Al agregar una fuente con esa función, GameMaker Studio 2 creará una caché de página de textura de los glifos necesarios hasta el tamaño definido por esta variable. El proceso es el siguiente:

  • Cada personaje que usa se almacena en caché desde la fuente en una página de textura
  • Cuando la página de texturas está llena (porque usa caracteres grandes o muchos caracteres diferentes) GameMaker Studio 2 eliminará los caracteres del caché y los reemplazará con los nuevos que necesite
  • GameMaker Studio 2 intenta no hacer que la textura sea demasiado grande porque no todos los dispositivos pueden manejar texturas grandes, y trata de no hacer la textura demasiado pequeña porque quitar y volver a agregar caracteres a la textura lleva tiempo de CPU y hace que tu juego funcione despacio

Si crees que necesitas un tamaño de página de textura más grande o más pequeño que el tamaño predeterminado de 1024px que usa GameMaker Studio 2, puedes establecer el valor usando esta variable incorporada. Si no estás usando el font_add() función, la configuración de esto no tendrá ningún efecto en el rendimiento de su juego como las fuentes añadidas en el IDE se almacenan en caché en las páginas de textura regulares (que puede controlar desde el Editor de fuentes y las opciones del juego para la plataforma de destino).


Sintaxis:

font_texture_page_size;


Devoluciones:

Real (maximum width/height in pixels)


Ejemplo:

font_texture_page_size = 512;
newfont = font_add( 'Arial', 24, true, true, 32, 128);

El código anterior establece la página de textura del caché de fuentes a un tamaño máximo de 512 px y luego agrega una nueva fuente al juego.
