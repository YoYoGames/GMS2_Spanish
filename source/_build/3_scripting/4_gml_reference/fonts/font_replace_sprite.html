font_replace_sprite


Descripción

Con esta función, puede usar una "tira de sprites" para reemplazar un elemento de fuente de sprite creado previamente, donde cada subimagen sería un símbolo o letra individual. El índice es el número de mapa UTF8 para el primer personaje que se va a usar y los intervalos que se usan son los mismos que para una fuente normal, comenzando con "!" pero recuerde que solo debe incluir lo que necesita y que debe usar el ord función para obtener el valor UTF8 correcto para la letra de inicio.

También puede establecer el espaciado para que la fuente sea proporcional ( true ) o no ( false ), donde una fuente proporcional está espaciada en función del ancho real de las letras (por lo que la letra "i" ocupa menos espacio que la letra "w", por ejemplo), mientras que una fuente no proporcional se espaciará según el subtítulo ancho de imagen, como una fuente monoespaciado. Finalmente, puede definir el espacio que debe dejarse entre cada letra al escribir, y este puede ser cualquier valor entero, donde 0 es sin espacio (las letras "tocarán" si son proporcionales).

También vale la pena señalar que no puede haber espacios en el mapa de caracteres, por lo que si, por ejemplo, usa '.' y '0' pero no '/', debe incluir '/' de todos modos o dejar una subimagen en blanco entre los dos caracteres utilizados. Sintaxis:

font_replace_sprite(ind, spr, first, prop, sep);


Argumento Descripción
ind El índice de la fuente que se reemplazará (es decir, todas las referencias a este recurso después de esta función se referirán a la nueva fuente, no al original).
spr El sprite para agregar una fuente basada en
first El índice del primer personaje disponible.
prop Si se debe eliminar el espacio en blanco alrededor de cada letra (si es falso, la fuente creada tendrá un espacio único, de lo contrario será como la mayoría de las fuentes normales).
sep El espacio entre cada letra, 0 lo que significa que el derecho de una letra solo toca la izquierda de la siguiente.


Devoluciones:

N/A


Ejemplo:

font_replace_sprite(global.Font, working_directory + "\Sprites\spr_Font.png", ord("!"), true, 2);

El código anterior reemplazará al activo de fuente indexado en la variable global "Fuente" con un nuevo recurso de fuente de un archivo de imagen externo.
