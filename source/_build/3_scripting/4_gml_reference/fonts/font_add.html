fuente_add


Descripción

Esta función se puede usar para agregar una fuente a tu juego desde las fuentes que están instaladas en el sistema en el que se está ejecutando. Puede definir el tamaño de la fuente (en puntos), así como si la fuente debe ser negrita o cursiva, y también puede definir el rango de caracteres a incluir (para una tabla completa de caracteres disponibles y su valor UTF8, consulte Fuente mesas ). La función devuelve un valor de índice que debe almacenarse en una variable, ya que será necesario en todos los códigos adicionales que se refieren a esta fuente.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Para el módulo de destino HTML5, esta función solo se puede usar para agregar fuentes web, y solo funcionará si WebGL no está habilitado. Si WebGL está habilitado, debe usar un recurso de fuente general (es decir: usar draw_set_font junto con una fuente del árbol de recursos) o fuentes de mapa de bits (ver font_add_sprite ).

¡IMPORTANTE! Cuando se dirige a HTML5, la cursiva y el texto en negrita solo se admiten si el recurso de fuente o la fuente web tienen estos caracteres incluidos. No se sintetizarán a partir de la fuente base.

En todos los demás destinos (no web), puede utilizar esta función para agregar una fuente desde un archivo. El archivo debe incluirse en el paquete del juego utilizando la funcionalidad de Archivos incluidos de GameMaker Studio 2, y debe ser un *.ttf Formato de archivo de fuente, útil para agregar fuentes no estándar como Asia o Árabe.

ADVERTENCIA!: Si incluyes una fuente en *.ttf Formato con un juego, debe tener licencia para su distribución con el juego.


Al cargar una fuente de un archivo de esta manera, el tamaño de la fuente es en píxeles y los valores primero y último se ignoran, lo que significa que se agregarán todos los glifos de la fuente. Las fuentes agregadas de esta manera se asignan a su propia página de texturas, por lo que se debe tener cuidado al usar esta función, ya que aumentará el número de intercambios de texturas al dibujar. También vale la pena señalar que las fuentes pueden aparecer un poco más grandes cuando se dibujan, ya que los glifos pueden tener partes que se dibujan fuera del cuadro delimitador definido para la fuente. También debe tener en cuenta que hay una limitación de alrededor de 200 glifos que se pueden representar en un solo marco para una fuente en particular.

NOTA: cuando cargue una fuente en GameMaker Studio 2: Studio, debe recordar eliminarla de nuevo (con font_delete ) cuando ya no se necesita, de lo contrario, existe el riesgo de una pérdida de memoria que ralentizará y eventualmente bloqueará su juego.


Sintaxis:

font_add(name, size, bold, italic, first, last);


Argumento Descripción
name El nombre de la fuente que se agregará (por ejemplo, 'Arial'), o la ruta del archivo si la fuente está incluida *.ttf expediente.
size El tamaño de la fuente: puntos para fuentes web, píxeles para *.ttf fuentes
bold Si la fuente es negrita (verdadera) o no (falsa).
italics Si la fuente es cursiva (verdadero) o no (falso).
first El primer carácter a incluir (si no está seguro, vaya por 32).
last El último carácter a incluir (si no está seguro, vaya por 128).


Devoluciones:

Real


Ejemplo:

newfont = font_add( 'Arial', 24, true, true, 32, 128);

Esto creará una nueva fuente que tiene un tamaño de 24 puntos, usa "Arial" que está en negrita y cursiva y el índice de esta nueva fuente se almacena en la variable "newfont". El rango de fuente incluye mayúsculas y minúsculas, números y todos los signos de puntuación comunes.
