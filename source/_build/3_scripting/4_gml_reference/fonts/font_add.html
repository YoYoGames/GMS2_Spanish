font_add


Descripción

Esta función se puede usar para agregar una fuente a su juego de aquellas fuentes que están instaladas en el sistema en el que se está ejecutando. Puede definir el tamaño de la fuente (en puntos), así como si la fuente debe ser negrita o cursiva, y también puede definir el rango de caracteres a incluir (para una tabla completa de caracteres disponibles y su valor UTF8, consulte Fuente tablas ). La función devuelve un valor de índice que debe almacenarse en una variable, ya que será necesario en todos los demás códigos que hagan referencia a esta fuente.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Para el módulo de destino HTML5, esta función solo se puede usar para agregar Fuentes web, pero solo funcionará si WebGL no está habilitado ; si WebGL está habilitado, entonces debe usar un archivo de fuente incluido. La siguiente tabla muestra las fuentes que son estándar en todos los navegadores y que deberían estar disponibles para su uso sin problemas. Cualquier otra fuente puede o no existir en la computadora o dispositivo que ejecuta su juego, por lo que el uso de esta función debe limitarse a estas fuentes estándar:

En todos los demás destinos (que no sean JavaScript), puede usar esta función para agregar una fuente desde un archivo. El archivo debe incluirse en el paquete del juego utilizando la funcionalidad Archivos incluidos de GameMaker Studio 2, y debe ser un *.ttf formato de archivo de fuente, útil para agregar fuentes no estándar como asiático o árabe.

¡ADVERTENCIA!: Si incluye una fuente en *.ttf formato con un juego, debe tener licencia para su distribución con el juego.


Al cargar una fuente desde un archivo de esta manera, el tamaño de la fuente es en píxeles y el primer y último valor se ignoran, lo que significa que se agregarán todos los glifos de la fuente. Las fuentes añadidas de esta manera se asignan a su propia página de texturas, por lo que se debe tener cuidado al usar esta función ya que aumentará el número de intercambios de texturas al dibujar. También vale la pena señalar que las fuentes pueden aparecer un poco más grandes cuando se dibujan, ya que los glifos pueden tener partes que se dibujan fuera del cuadro delimitador definido para la fuente. También debe tener en cuenta que existe una limitación de alrededor de 200 glifos que se pueden representar en un solo cuadro para una fuente en particular.

NOTA: Cuando cargue una fuente en GameMaker Studio 2: Studio, debe recordar eliminarla nuevamente (con font_delete ) cuando ya no sea necesario, de lo contrario existe el riesgo de una pérdida de memoria que disminuirá la velocidad y, finalmente, se bloqueará el juego.


Sintaxis:

font_add(name, size, bold, italic, first, last);


Argumento Descripción
name El nombre de la fuente que se agregará (por ejemplo, 'Arial') o la ruta del archivo si la fuente está incluida *.ttf archivo.
size El tamaño de la fuente: puntos para Fuentes web, píxeles para *.ttf fuentes
bold Si la fuente es negrita (verdadera) o no (falsa).
italics Si la fuente es cursiva (verdadera) o no (falsa).
first El primer personaje para incluir (si no estás seguro, ve por 32).
last El último personaje para incluir (si no estás seguro, ve por 128).


Devoluciones:

Real


Ejemplo:

newfont = font_add( 'Arial', 24, true, true, 32, 128);

Esto creará una nueva fuente de 24 pts de tamaño, usa "Arial", que es negrita y cursiva, y el índice de esta nueva fuente se almacena en la variable "newfont". El rango de fuentes incluye mayúsculas y minúsculas, números y todos los signos de puntuación comunes.
