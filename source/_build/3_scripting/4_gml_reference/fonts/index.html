Fuentes


GameMaker Studio 2 tiene una serie de funciones que se pueden utilizar para obtener información sobre los recursos de fuentes incluidos, así como un conjunto adicional de funciones que se pueden usar para cambiar, agregar o eliminar fuentes de tu juego en tiempo de ejecución. En particular, la función font_add() se puede usar para agregar fuentes no estándar, como fuentes de glifo asiático, en *.ttf formato.

Vale la pena señalar que si agrega una fuente a su juego que no tiene los glifos requeridos por el texto que desea escribir, entonces el carácter Unicode 9647 (▯) se usa para sustituir esos glifos que faltan al representarlo en el dibujar evento. Entonces, si su fuente no tiene, por ejemplo, el símbolo °, escribir 90 ° realmente producirá 90 °.

Las siguientes funciones se pueden usar para obtener información sobre recursos de fuentes:

  • font_get_name
  • font_get_fontname
  • font_get_first
  • font_get_last
  • font_get_italic
  • font_get_bold
  • font_get_size
  • font_get_texture
  • font_get_uvs

Las siguientes funciones se pueden usar para cambiar y manipular fuentes durante un juego:

  • font_set_cache_size
  • font_exists
  • font_add
  • font_add_sprite
  • font_add_sprite_ext
  • font_replace_sprite
  • font_replace_sprite_ext
  • font_delete
  • font_texture_page_size
