font_set_cache_size


Descripción

Al realizar la fusión y el coloreado de imágenes, HTML5 no puede hacerlo dinámicamente de la misma forma que se podría realizar un ejecutable. Por lo tanto, GameMaker Studio 2 tiene que guardar temporalmente una copia combinada de las imágenes y cargarlas cuando sea necesario. Esta función establece cuántas copias combinadas de la fuente determinada pueden almacenarse en caché antes de sobrescribir las antiguas. El valor predeterminado es 4.

NOTA: para las fuentes de sprite debe usar la función equivalente para sprites, sprite_set_cache_size.


Sintaxis:

font_set_cache_size(ind, max);


Argumento Descripción
ind El índice de la fuente para cambiar el tamaño de caché de.
max La cantidad máxima de copias en caché de la fuente que se puede almacenar.


Devoluciones:

N/A


Ejemplo:

font_set_cache_size(fnt_MainMenu, 2);

Esto establecerá el caché de fuentes de la fuente indexada en la variable "fnt_MainMenu" a 2 copias.
