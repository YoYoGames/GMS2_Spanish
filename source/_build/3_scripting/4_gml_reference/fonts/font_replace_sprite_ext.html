font_replace_sprite_ext


Descripción

Esta función reemplazará una fuente de sprite que creaste previamente usando la función font_add_sprite_ext. Utiliza una "tira de sprites", donde cada subimagen es un símbolo o letra individual, y las imágenes se asignan a letras en función del argumento "string_map" de la función. Este argumento es una cadena que puede usar para indicar a GameMaker Studio 2 orden en que se encuentran las subimágenes de la fuente de sprite y las asignará automáticamente al escribir el texto. Entonces, por ejemplo, si tiene un mapa de cadena de "AaBbCcDdEeFfGgHh", su fuente de sprite debe tener las subimágenes ordenadas de esta manera.

También puede configurar el espaciado de la fuente para que sea proporcional ( true ) o no ( false ), donde una fuente proporcional está espaciada en función del ancho real de las letras (por lo que la letra "i" ocupa menos espacio que la letra "w", por ejemplo), mientras que una fuente no proporcional se espaciará según el subtítulo ancho de imagen, como una fuente monoespaciado. Finalmente, puede definir el espacio que debe dejarse entre cada letra al escribir, y este puede ser cualquier valor entero, donde 0 es sin espacio (las letras "tocarán" si son proporcionales). La función devuelve un valor de índice que debe almacenarse en una variable, ya que será necesario en todos los demás códigos que hagan referencia a esta fuente.

NOTA: Esta función requiere que haya agregado previamente una fuente de sprite usando font_add_sprite(), ya que reemplazará eso y no se puede usar en las fuentes que se incluyen como parte de los recursos del juego.
¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Sintaxis:

font_replace_sprite_ext(font, spr, string_map, prop, sep);

Argumento Descripción
font La fuente para reemplazar.
spr El sprite para agregar una fuente basada en
string_map Cadena desde la que se toma el orden de subimagen de sprite.
prop Establecer como fuente proporcional o no.
sep El espacio para irse entre cada letra.


Devoluciones:

N/A


Ejemplo:

font_replace_sprite_ext(global.Font, spr_CalcFont, "0123456789+-*/=", true, 2);

El código anterior reemplazará un elemento de fuente de elemento sprite que está indexado en la variable "global.Font" con imágenes de elemento de elemento indexadas en la variable "spr_CalcFont".
