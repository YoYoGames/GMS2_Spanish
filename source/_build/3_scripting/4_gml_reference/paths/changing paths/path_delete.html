path_delete

Elimina una ruta determinada.

Sintaxis:

path_delete(index);


Argumento Descripción
index El índice de la ruta para eliminar.


Devoluciones: N / A


Descripción

Puede usar este código para eliminar una ruta de la memoria. Si esta ruta ha sido creada dinámicamente usando path_add, la variable que contiene el índice de ruta ya no será válida para acceder a la ruta porque ya no existe, y si la ruta se creó usando el Editor de ruta, ya no se puede acceder a esa ruta en todo el juego, ya que lo elimina permanentemente.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.



Ejemplo:

var t_path;
t_path = path_add();
if mp_grid_path(grid, t_path, x, y, obj_Player.x, obj_Player.y, 1)
   {
   path_assign(mypath, t_path);
   }
path_delete(t_path);

El código anterior creará una ruta y almacenará su índice en una variable local. Esta ruta se usa para almacenar mp_grid_path ruta generada que, si tiene éxito en encontrar su camino hacia el objetivo, entonces se asigna a la ruta indexada en "mypath". Finalmente, se borra la "t_path".
