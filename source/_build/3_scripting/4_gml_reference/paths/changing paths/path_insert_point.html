path_insert_point

Inserta un punto en la ruta dada en una posición establecida.

Sintaxis:

path_insert_point(index, n, x, y, speed);


Argumento Descripción
index El índice de la ruta para insertar el punto en.
n El punto de definición para insertar el nuevo punto ANTES.
x La coordenada x (relativa a la ruta) del nuevo punto.
y La coordenada y (relativa a la ruta) del nuevo punto.
speed El factor de velocidad del punto.


Devoluciones: N / A


Descripción

Con esta función puede insertar un nuevo punto en una ruta (la ruta puede haberse creado en el editor de ruta o mediante el código usando path_add ) El punto se agregará a la ruta antes del punto "n" que se especifica en la función.


Ejemplo:

path_insert_point(mypath, 0, 50, 50, 100);

Esto insertará un punto al comienzo de la ruta indexada en la variable "mypath", en (50,50), con un factor de velocidad del 100%.
