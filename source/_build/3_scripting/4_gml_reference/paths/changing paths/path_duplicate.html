path_duplicate

Duplica una ruta existente y devuelve su índice.

Sintaxis:

path_duplicate(index);


Argumento Descripción
index El índice de la ruta existente para duplicar.


Devoluciones: Real


Descripción

Esta función toma una ruta y la copia en una nueva ruta. La nueva ruta se crea en el proceso y el índice se devuelve para ser utilizado en todas las llamadas posteriores para usar esta nueva ruta.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.



Ejemplo:

mypath = path_duplicate(choose(path_1, path_2, path_3, path_4));

El código anterior elige uno de los cuatro recursos de ruta y lo duplica, almacenando el índice de la nueva ruta en la variable "mypath".
