path_add_point

Agrega un punto a la ruta dada.

Sintaxis:

path_add_point(index, x, y, speed);


Argumento Descripción
index El índice de la ruta para agregar el punto a.
x La coordenada x del nuevo punto.
y La coordenada y del nuevo punto.
speed El factor de velocidad del punto (el valor predeterminado es 100).


Devoluciones: N / A


Descripción

Con esta función puede agregar un punto a la ruta especificada y establecer su factor de velocidad. Este punto se agrega al final de la ruta, independientemente de la posición del punto, y el factor es igual al porcentaje de path_speed que la siguiente instancia realmente se activa cuando alcanza este punto en la ruta. Si desea colocar un punto de ruta en alguna otra posición que no sea el final, debe usar path_insert_point.


Ejemplo:

path = path_add();
var i;
i = 0;
repeat(10)
   {
   path_add_point(path0, x + lengthdir_x(50, i), y+ lengthdir_y(50, i), 100);
   i += 36;
   }

El código anterior creará una ruta circular alrededor de la posición x / y de la instancia que ejecuta el código.
