path_delete_point

Elimina un punto de una ruta existente.

Sintaxis:

path_delete_point(ind, n);


Argumento Descripción
index El índice de la ruta para eliminar un punto de.
n El punto de definición para eliminar.


Devoluciones: N / A


Descripción

Con esta función puede eliminar un punto de la ruta especificada. Los puntos de ruta están numerados de 0, por lo que el primer punto es siempre el 0º y el último punto de ruta será path_get_number() - 1.


Ejemplo:

path_delete_point(mypath, path_get_number(mypath) - 1);

Esto eliminaría el último punto en la ruta indexada en la variable "mypath".
