path_get_name

Devuelve el nombre de una ruta determinada.

Sintaxis:

path_get_name(index);


Argumento Descripción
index El índice de la ruta para verificar


Devuelve: Cadena.


Descripción

Esta función devolverá el nombre de la ruta a la que se hace referencia como cadena. El nombre es lo que se haya usado para definir la ruta en el editor o (si la ruta se ha creado mediante una función de código) devolverá una cadena con el formato "_newpathXX" donde "XX" es el número de la ruta generada, comenzando en 0 e incrementándose en uno cada vez que se crea una nueva ruta. Tenga en cuenta que esto es solo una cadena y no se puede usar para hacer referencia directamente a la ruta, para eso necesitaría el índice de ruta. Sin embargo, puede usar esta cadena para obtener el índice de ruta utilizando la cadena devuelta junto con la función asset_get_index().


Ejemplo:

name = path_get_name(mypath);

Esto establecerá "nombre" al nombre de la ruta indexada en la variable "mypath".
