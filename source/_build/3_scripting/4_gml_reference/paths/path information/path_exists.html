path_exists

Devuelve si existe una ruta con el índice dado o no.

Sintaxis:

path_exists(index);


Argumento Descripción
index El índice de la ruta a verificar.


Devuelve: booleano.


Descripción

Esta función devuelve si existe una ruta con el índice dado o no. Tenga en cuenta que si comprueba la existencia de una ruta a través de una variable que aún no se ha declarado, arrojará un error.


Ejemplo:

if path_exists(path0)
   {
   path_start(path0, 4, path_action_reverse, 0);
   }

Esto iniciará path0, si existe.
