path_get_point_speed

Devuelve la configuración de velocidad del punto de definición dado de una ruta determinada.

Sintaxis:

path_get_point_speed(index, n);


Argumento Descripción
index El índice de la ruta para verificar
n El punto de identidad para verificar


Devoluciones: Real.


Descripción

Con esta función puede obtener la velocidad del punto (como se define en el Editor de ruta o cuando agrega dinámicamente un punto de ruta usando path_add_point() ) expresado como un porcentaje. Por lo tanto, si tiene un punto de ruta establecido en 50 en el editor de ruta, esta función devolvería 50 cuando se use.


Ejemplo:

if path_get_point_speed(c_path, 0) != 100
   {
   path_change_point(c_path, 0, path_get_point_x(c_path, 0), path_get_point_x(c_path, 0), 100);
   }

el código anterior comprueba el punto de ruta "0" de la ruta indexada por "c_path" para ver si la velocidad no es igual a 100. Si no es así, establece la velocidad de ese punto en 100.
