Caminos

GameMaker Studio 2 tiene una amplia colección de funciones y variables integradas para usar rutas dentro de tu juego. Esta sección trata de los básicos que se usan cuando se trata de rutas que se crearon previamente en Path Editor, pero existen otras funciones que le permiten crear sus propias rutas usando código y manipularlas de varias maneras.

Existen las siguientes funciones básicas que se ocupan de las rutas:

  • path_start
  • path_end

Las siguientes variables están incorporadas en todas las instancias cuando se crean y se pueden usar para obtener información sobre una ruta o para cambiar ciertas propiedades:

  • path_index
  • path_position
  • path_positionprevious
  • path_speed
  • path_scale
  • path_orientation
  • path_endaction

Tenga en cuenta que puede encontrar más información sobre variables de instancia aquí: Variables de instancia.

Se pueden encontrar más funciones para las rutas en las siguientes dos secciones:

  1. Información de ruta
  2. Cambiando caminos
