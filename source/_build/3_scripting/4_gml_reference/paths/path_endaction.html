path_endaction

Establece qué hacer cuando la instancia llega al final de la ruta.

Sintaxis:

path_endaction;


Devoluciones: constante.


Descripción

Esta variable se puede usar para obtener o cambiar la reacción de una instancia cuando llega al final de la ruta actual. Normalmente, establecerías esto cuando comiences la ruta usando path_start pero es posible que desee cambiar este comportamiento según la cantidad de eventos en su juego. Los valores disponibles se expresan usando las siguientes constantes:

  • path_action_stop = detener el camino
  • path_action_restart = continuar desde la posición inicial, saltando al inicio si la ruta no está cerrada
  • path_action_continue = iniciar la ruta nuevamente desde la posición actual
  • path_action_reverse = revertir la velocidad de la ruta (ejecutar la ruta en reversa)


Ejemplo:

if path_endaction == path_action_stop
   {
   path_endaction = choose(path_action_restart, path_action_continue, path_action_reverse);
   }

El código anterior verificará la acción final del camino y, si está configurado para detenerse, se seleccionará una nueva acción final al azar.
