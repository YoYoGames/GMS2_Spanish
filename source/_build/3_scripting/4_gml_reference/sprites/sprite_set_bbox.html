sprite_set_bbox


Descripción

Esta función se puede utilizar para establecer los valores del cuadro delimitador para un sprite. Usted suministra el índice de sprites para usar y luego los valores de izquierda, superior, derecha e inferior para las posiciones del cuadro delimitador. Las posiciones son valores absolutos, donde la posición (0, 0) corresponde a la esquina superior izquierda del sprite, independientemente del desplazamiento del sprite, cualquier píxel "vacío" que pueda tener el sprite, o dónde se dibuja en el habitación.

NOTA: Esta función afecta el recurso de sprite , de modo que todas las instancias posteriores con este sprite tendrán el mismo cuadro de límite.


Sintaxis:

sprite_set_bbox(ind, left, top, right, bottom);

Argumento Descripción
ind El índice del sprite para establecer el cuadro delimitador en.
left El lado izquierdo del cuadro delimitador.
top La parte superior del cuadro delimitador.
right El lado derecho del cuadro delimitador.
bottom La parte inferior del cuadro delimitador.


Devoluciones

N/A


Ejemplo:

if sprite_get_bbox_mode(sprite_index) == 2
    {
    left = irandom(sprite_width / 2);
    right = irandom((sprite_width / 2) + irandom(sprite_width / 2));
    top = irandom(sprite_height / 2);
    bottom = irandom((sprite_height / 2) + irandom(sprite_height / 2));
    sprite_set_bbox(sprite_index, left, top, right, bottom);
    }

El código anterior verificará el modo de cuadro delimitador del sprite asignado al sprite_index, y si está configurado a manual entonces tendrá su cuadro de límite cambiado.
