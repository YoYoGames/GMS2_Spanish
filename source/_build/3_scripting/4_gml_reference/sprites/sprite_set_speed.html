sprite_set_speed


Sintaxis:

sprite_set_speed(index, speed, type)

Argumento Descripción
index El índice del sprite para obtener el tipo de animación de


Devoluciones:

Constant (see above)


Descripción

Esta función se puede usar para establecer la velocidad de la animación base y escribir como para el recurso de sprite. Esto normalmente se establece en el Editor de Sprites, pero puede haber momentos en los que se requiera que lo configure manualmente para un recurso de sprite determinado. Debe proporcionar el índice de sprites para establecer, la velocidad de animación y el tipo de velocidad de animación para aplicar. El tipo de velocidad de animación puede ser cualquiera de las siguientes constantes:

Constante Descripción
spritespeed_framespersecond El sprite se definió con animación en cuadros por segundo.
spritespeed_framespergameframe El sprite se definió con animación en cuadros por cuadro de juego.


Seleccionar el tipo de velocidad a usar para calcular animaciones es muy importante ya que hay una gran diferencia entre las dos opciones. Los siguientes dos ejemplos ilustran esta diferencia:

  • Si tiene un objeto que dibuja 1 cuadro por segundo y establece la velocidad de la imagen en 0.5, dibujará a 0.5 cuadros por segundo. Si la velocidad de fotogramas del juego es de 60 fotogramas por segundo, el sprite dibujará 1 fotograma por cada 120 fotogramas del juego.

  • Si tienes un sprite que dibuja 1 cuadro por cuadro de juego y configuras la velocidad de la imagen a 0.5, dibujará 0.5 cuadros por cuadro de juego. Si la velocidad de fotogramas del juego es de 60 fotogramas por segundo, el sprite dibujará 30 fotogramas por cada 60 fotogramas del juego.

Puede averiguar cuál era el valor de velocidad utilizado para la animación del sprite utilizando la función sprite_get_speed, y puedes obtener el tipo de velocidad de animación usando sprite_get_speed_type.

¡IMPORTANTE! Esta función afectará al recurso de sprite , por lo que cualquier cambio que realice con esta función afectará a todas las instancias que estén usando el sprite y todas las instancias futuras también.


Ejemplo:

sprite_set_speed(sprite_index, 10, spritespeed_framespersecond);

El código anterior establece la velocidad de animación base para el sprite asignado a la instancia a 10 cuadros por segundo.
