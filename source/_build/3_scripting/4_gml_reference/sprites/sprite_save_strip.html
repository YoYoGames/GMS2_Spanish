sprite_save_strip


Sintaxis:

sprite_save_strip(ind, filename);

Argumento Descripción
ind El índice del sprite para cambiar el mapa de transparencia de.
filename El nombre de archivo para la tira de sprites guardada.


Devoluciones

N/A


Descripción

Esta función creará una imagen de tira de todas las subimágenes desde un sprite. Puede especificar el nombre del archivo que se guardará, pero el tipo de archivo real debe ser *.png.

NOTA: Dependiendo de la plataforma de destino elegida, está limitado en cuanto a dónde puede guardar y cargar archivos. Consulte Manejo de archivos para obtener más información.


Ejemplo:

var spr, i;
spr = sprite_create_from_surface(0, 0, 32, 32, true, true, 16, 16);
for (i = 1; i < 8; i +=1)
   {
   sprite_add_from_surface(spr, i, 0, 32, 32, true, true, 16, 16);
   }
sprite_save_strip(spr, "Custom_Player_Sprite.png");
sprite_delete(spr);

El código anterior crea un sprite de la superficie indexada en la variable "surf", asignando su índice a la variable local "spr_Custom", y luego usa un for para recorrer la superficie y capturar varias secciones que se agregan al sprite como subimágenes. este nuevo sprite se guarda como una tira png antes de ser eliminado de la memoria.
