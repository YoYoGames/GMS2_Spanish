sprite_get_width


Sintaxis:

sprite_get_width(index);

Argumento Descripción
index El índice del sprite para encontrar el ancho de.


Devoluciones

Real


Descripción

Con esta función puede encontrar el ancho del recurso sprite base, sin transformaciones, en píxeles.


Ejemplo:

if sprite_width != sprite_get_width(sprite_index)
   {
   image_xscale = 1;
   }

El código anterior comprueba el ancho del sprite tal como está en la instancia actual y si hay una diferencia entre ese y el sprite base original, restablece la escala del eje x.
