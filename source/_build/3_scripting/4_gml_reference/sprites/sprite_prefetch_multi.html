sprite_prefetch_multi


Descripción

Esta función se puede utilizar para extraer previamente (colocar en la memoria de texturas) varias páginas de texturas que contienen los sprites dados. Suministra una matriz rellena con los índices de sprite (como se definió cuando creó el activo de sprite) y las páginas de textura en las que se encuentran se cargarán en la memoria. Tenga en cuenta que la función volverá -1 si la recuperación previa no es compatible con el recurso elegido o la plataforma de destino es HTML5, o regresará 0 si todo funcionaba correctamente.

NOTA: hay un impacto en el rendimiento ya que la textura se carga en la memoria de textura en la mayoría de los dispositivos, por lo que se recomienda que llame sprite_prefetch_multi en cualquier gráfico requerido al inicio de un nivel para evitar cualquier pérdida.


Sintaxis:

sprite_prefetch_multi

Argumento Descripción
array Array con los índices de sprite para obtener


Devoluciones:

Int (-1 or 0)


Ejemplo:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_prefetch_multi(spr_a);

El código anterior crea una matriz donde cada elemento contiene un índice de sprite. Esta matriz se utiliza para colocar esas texturas de sprites en la memoria.
