sprite_prefetch_multi


Sintaxis:

sprite_prefetch_multi

Argumento Descripción
array Matriz con los índices de sprites para buscar


Devoluciones:

N/A


Descripción

Esta función se puede usar para captar previamente (colocar en la memoria de textura) varias páginas de textura que contienen los sprites. Suministra una matriz poblada con los índices de sprites (como se definió cuando creó el activo de sprite) y las páginas de textura en las que están se cargarán en la memoria.

NOTA: Hay un golpe de rendimiento a medida que la textura se carga en la memoria de textura en la mayoría de los dispositivos, por lo que se recomienda que llame sprite_preftech en cualquier gráfico requerido al comienzo de un nivel para evitar puestos.


Ejemplo:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_prefetch_multi(spr_a);

El código anterior crea una matriz donde cada elemento contiene un índice de sprites. Esta matriz se usa para colocar esas texturas de sprites en la memoria.
