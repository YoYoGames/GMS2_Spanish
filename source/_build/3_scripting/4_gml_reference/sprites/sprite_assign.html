sprite_assign


Sintaxis:

sprite_assign(index, sprite);

Argumento Descripción
index El índice del sprite que se debe copiar (es decir, sobrescrito).
duende The sprite to be copied from.


Devoluciones

N/A


Descripción

Esta función toma dos índices de sprite previamente creados (o incluidos), y copia la imagen de uno a otro. De esta forma, puede copiar (o "clonar") un elemento en otro índice. Tenga en cuenta que no puede copiar a un recurso del juego. Debes haber creado el sprite para copiarlo previamente usando sprite_add o sprite_duplicate funciones.


Ejemplo:

if sprite_exists(global.Player_Sprite)
   {
   var t_spr = sprite_add("player.png", 16, true, true, 0, 0);
   sprite_assign(global.Player_Sprite, t_spr);
   sprite_delete(t_spr);
   }
else
   {
   global.Player_Sprite = sprite_add("player.png", 16, true, true, 0, 0);
   }

El código anterior comprueba si la variable global "Player_Sprite" contiene un sprite y, si lo hace, utiliza sprite_assign para cambiarlo por uno que se haya cargado desde un archivo externo. Si no contiene un sprite, uno se carga y su índice se almacena en esa variable.
