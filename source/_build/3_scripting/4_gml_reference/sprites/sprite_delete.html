sprite_delete


Sintaxis:

sprite_delete(index);

Argumento Descripción
index El índice del sprite que se eliminará.


Devoluciones

N/A


Descripción

Esta función eliminará un sprite del juego, liberando cualquier memoria que esté reservada para él. Esta es una eliminación permanente, y si el activo que eliminaste se incluyó como parte de los recursos del juego, ni siquiera reiniciar el juego (a menos que cierres primero la aplicación) lo recuperará. Esta es una función muy importante para aquellos momentos en los que necesita crear y cambiar sprites de recursos externos y siempre debe usarse para eliminar aquellos activos que un juego ya no necesita, o para borrar un activo indexado de una variable antes de volver a usarlos. asignando otro activo a esa variable. Esto no se puede usar para eliminar sprites que están incluidos en el juego como parte del árbol de recursos.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

var spr = sprite_create_from_surface(application_surface, 0, 0, 32, 32, false, false, 16, 16);
sprite_merge(spr_Player, spr);
sprite_delete(spr);

El código anterior crea una variable local y luego almacena el índice del sprite creado desde la superficie de la aplicación. Este elemento se fusiona con el elemento indexado en la variable "spr_Player" antes de eliminarse de la memoria nuevamente.
