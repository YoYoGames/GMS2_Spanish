sprite_set_bbox_mode


Descripción

Esta función se puede utilizar para establecer el modo de cuadro delimitador para un sprite. Usted suministra el índice de sprites y el modo a usar, que debe ser una de las siguientes constantes:

Constante Descripción
bboxmode_automatic Automático: el cuadro delimitador se calculará automáticamente, de acuerdo con la configuración de tolerancia para el sprite
bboxmode_fullimage Imagen completa: el cuadro delimitador se configurará para utilizar el ancho y la altura completos del sprite, independientemente de la tolerancia y los píxeles "vacíos"
bboxmode_manual Manual: el cuadro delimitador se configura manualmente según los valores definidos por el usuario, lo que requiere el uso de la función sprite_set_bbox()
NOTA: Esta función afecta el recurso de sprite , de modo que todas las instancias posteriores con este sprite tendrán el mismo modo de cuadro delimitador.


Sintaxis:

sprite_set_bbox_mode(ind, mode);

Argumento Descripción
ind El índice del sprite para cambiar el modo de.
mode El modo de establecer (una constante).


Devoluciones

N/A


Ejemplo:

if sprite_get_bbox_mode(sprite_index) != 0
    {
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
    }

Lo anterior verifica el modo bbox para el sprite actual y si no está automatic, entonces se establece en ese valor.
