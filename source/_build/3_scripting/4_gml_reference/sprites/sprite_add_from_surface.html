sprite_add_from_surface


Sintaxis:

sprite_add_from_surface(index, surface, x, y, w, h, removeback, smooth);

Argumento Descripción
index El índice del sprite para agregar la nueva imagen a.
surface El índice de la superficie desde la que se obtiene la imagen.
x La posición x para copiar de.
y La posición y para copiar de.
w El ancho del área a copiar (desde la posición x).
h La altura del área a copiar (desde la posición y).
removeback Indica si se deben hacer transparentes todos los píxeles con el color de fondo (píxel inferior izquierdo).
smooth Indica si alisar los bordes.


Devoluciones:

N/A


Descripción

Esta función funciona exactamente de la misma manera que sprite_create_from_surface solo en lugar de crear un nuevo sprite desde el área de la superficie indexada que selecciona, agrega el área definida de la superficie como una nueva sub-imagen a un sprite creado previamente (esto significa que no puede agregarlo directamente a un sprite desde El árbol de recursos, sino solo uno creado previamente a partir de una superficie, o uno que ha sido duplicado desde un sprite de recursos usando sprite_duplicate ).

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, true, true, 16, 16);
var i;
for (i = 1; i < 8; i +=1)
   {
   sprite_add_from_surface(spr_Custom, surf, i, 0, 32, 32, true, true, 16, 16);
   }

El código anterior crea un sprite desde la superficie indexada en la variable "surf", asignando su índice a la variable "spr_Custom", y luego usa un for bucle para moverse por la superficie y capturar varias secciones que se agregan al sprite como subimágenes.
