sprite_get_tpe


Sintaxis:

sprite_get_tpe(sprite, index);

Argumento Descripción
sprite El índice del sprite para encontrar la entrada de la página de textura de.
index El índice de imagen del sprite.


Devoluciones

Real


Descripción

Esta función devolverá un valor especial que se relaciona con la posición del sprite y la subimagen dados en la página de texturas utilizada por tu juego. De esta forma puede pasar el valor a extensiones para usar fuera del juego compilado, omitiendo efectivamente las funciones de dibujo de GameMaker Studio 2 y permitiendo que el sprite se use en contenido DOM que luego puede dibujarse en cualquier lugar dentro de la ventana que contiene el lienzo del juego. Esta función es de particular interés para aquellos que desean crear botones y otros medios interactivos fuera del elemento canvas de GameMaker Studio 2 en su página de host usando la función clickable_add.

NOTA: esta función es solo para HTML5.


Ejemplo:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

El código anterior crea un botón DOM en la posición (32, 32) de la página en la que se ejecuta el lienzo del juego. El botón usa el sprite al que se hace referencia desde la página de texturas como "spr_MS_Home" y cuando se presiona el botón se abrirá una nueva ventana para la URL especificada y con las propiedades definidas para esa ventana.
