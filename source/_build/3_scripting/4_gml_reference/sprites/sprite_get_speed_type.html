sprite_get_speed_type


Sintaxis:

sprite_get_speed_type(index)

Argumento Descripción
index El índice del sprite para obtener el tipo de animación de


Devoluciones:

Constant (see above)


Descripción

Esta función se puede usar para recuperar el tipo de animación de sprite como se define para el recurso de sprite en el Editor de Sprite. El valor devuelto será una de las siguientes constantes:

Constante Descripción
spritespeed_framespersecond El sprite se definió con animación en cuadros por segundo.
spritespeed_framespergameframe El sprite se definió con animación en cuadros por cuadro de juego.


Los siguientes dos ejemplos ilustran la diferencia que esto puede causar al calcular las velocidades de animación para el sprite:

  • Si tiene un objeto que dibuja 1 cuadro por segundo y establece la velocidad de la imagen en 0.5, dibujará a 0.5 cuadros por segundo. Si la velocidad de fotogramas del juego es de 60 fotogramas por segundo, el sprite dibujará 1 fotograma por cada 120 fotogramas del juego.

  • Si tienes un sprite que dibuja 1 cuadro por cuadro de juego y configuras la velocidad de la imagen a 0.5, dibujará 0.5 cuadros por cuadro de juego. Si la velocidad de fotogramas del juego es de 60 fotogramas por segundo, el sprite dibujará 30 fotogramas por cada 60 fotogramas del juego.

Puede averiguar cuál era el valor de velocidad utilizado para la animación del sprite utilizando la función sprite_get_speed, y puedes configurar la velocidad y el tipo de animación con la función sprite_set_speed.


Ejemplo:

s_speed = sprite_get_speed(sprite_index);
s_type = sprite_get_speed_type(sprite_index);

El código anterior obtiene la velocidad del sprite y el tipo de animación del sprite y los almacena en variables de instancia para uso futuro.
