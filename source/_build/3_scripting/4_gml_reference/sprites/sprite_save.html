sprite_save


Sintaxis:

sprite_save(ind, subimg, fname);

Argumento Descripción
ind El índice del sprite para guardar.
subimg La sub-imagen del sprite para guardar.
fname El nombre de archivo para el sprite guardado.


Devoluciones

N/A


Descripción

Esta función se puede usar para guardar cualquier subimagen de cualquier elemento en disco, dándole el nombre de archivo especificado. Esta imagen debe ser una *.png archivo.

NOTA: Dependiendo de la plataforma de destino elegida, está limitado en cuanto a dónde puede guardar y cargar archivos. Consulte Manejo de archivos para obtener más información.


Ejemplo:

var surf, spr_custom;
surf = surface_create(32, 32);
surface_set_target(surf);
draw_clear_alpha(c_black, 0);
draw_sprite(spr_Body, 0, 0, 0);
draw_sprite(spr_Clothes, 0, 0, 0);
draw_sprite(spr_Hair, 0, 0, 0);
spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, true, true, 16, 16);
surface_reset_target();
surface_free(surf);
sprite_save(spr_custom, 0, "Player_Custom_Sprite.png");
sprite_delete(spr_Custom);

El código anterior crea una superficie y almacena su índice en la variable local "navegar". Luego apunta a esa superficie, la despeja y dibuja varios sprites uno encima del otro. Crea un nuevo sprite a partir de la imagen compuesta dibujada en la superficie y asigna su índice a la variable local "spr_Custom" antes de liberar la memoria utilizada por la superficie. Finalmente, el sprite resultante se guarda en un archivo y luego se elimina de la memoria también.
