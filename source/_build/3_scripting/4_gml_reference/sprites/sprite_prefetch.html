sprite_prefetch


Descripción

Esta función se puede utilizar para extraer previamente (colocar en la memoria de textura) una página de textura con el elemento sprite dado. Usted suministra el índice de sprites como se definió cuando creó el activo de sprites, y la página de textura en la que se encuentra se cargará en la memoria. Tenga en cuenta que la función volverá -1 si la recuperación previa no es compatible con el recurso elegido o la plataforma de destino es HTML5, o regresará 0 si todo funcionaba correctamente.

NOTA: hay un impacto en el rendimiento ya que la textura se carga en la memoria de textura en la mayoría de los dispositivos, por lo que se recomienda que llame sprite_prefetch en cualquier gráfico requerido al inicio de un nivel para evitar cualquier pérdida.


Sintaxis:

sprite_prefetch(ind)

Argumento Descripción
ind El índice de sprites a buscar


Devoluciones:

Int (-1 or 0)


Ejemplo:

sprite_prefetch(spr_Player_Aura);

El código anterior colocará el sprite referenciado en la memoria de texturas listo para usar.
