skeleton_animation_list


Sintaxis:

skeleton_animation_list(sprite, list);

Argumento Descripción
sprite El índice de sprites de la animación del esqueleto de la columna para obtener la lista.
list El ID de la lista de DS para rellenar con los nombres de la animación.


Devoluciones:

N/A


Descripción

Con esta función puedes poblar un (pre-creado) ds_list con todos los nombres de las animaciones incluidas como parte del sprite de animación esquelética. Los nombres serán cadenas y luego se pueden usar en las otras funciones de animación para estos tipos de sprites.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

var list = ds_list_create();
skeleton_animation_list(sprite_index, list);
for (var i = 1; i < ds_list_size(list); i++;)
   {
   skeleton_animation_mix(list[| 0], list[| i], 0.5);
   }
ds_list_destroy(list);

El código anterior crea un ds_list de todos los nombres de animación para el sprite utilizado por la instancia. Luego pasa a través de estos y establece el valor de mezcla para todos ellos con la primera animación en 0.5.
