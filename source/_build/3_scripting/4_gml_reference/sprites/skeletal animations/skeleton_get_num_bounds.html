skeleton_get_num_bounds


Sintaxis:

skeleton_get_num_bounds();


Devoluciones:

Integer


Descripción

Esta función devolverá el número de cuadros delimitadores asociados con el sprite de animación esqueleto asignado a la instancia que ejecuta el código. Esto se puede usar junto con la función skeleton_get_bounds para recuperar datos sobre cada uno de los cuadros delimitadores.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++;)
    {
    var b_info = skeleton_get_bounds(i);
    if b_info[0] > 0
        {
        var data = b_info[1] + ":";
        for(var j = 0; j < b_info[0]; j++;)
            {
            data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";
            }
        draw_text(20, yy, data);
        yy += 20;
        }
    }

El código anterior recorrerá cada uno de los cuadros delimitadores asociados con el elemento sprite actualmente asignado y luego dibujará esos datos como una cadena en la pantalla.
