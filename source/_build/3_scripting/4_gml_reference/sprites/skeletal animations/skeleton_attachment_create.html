skeleton_attachment_create


Sintaxis:

skeleton_attachment_create(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot);

Argumento Descripción
name El nombre (como una cadena) del archivo adjunto para crear.
sprite El sprite_index para obtener la imagen del archivo adjunto.
ind El image_index para obtener la imagen del archivo adjunto.
xorigin El origen x de la imagen que se está usando.
yorigin El origen y para la imagen que se está usando.
xscale La escala horizontal de la imagen, como un multiplicador: 1 = escala normal, 0.5 es mitad, etc...
yscale La escala vertical de la imagen, como un multiplicador: 1 = escala normal, 0.5 es mitad, etc...
rot La rotación de la imagen. 0 = normal, 90 = girado 90 grados en sentido antihorario, etc.


Devoluciones:

1 if successful, -1 if not


Descripción

Cuando creas tu sprite de animación esquelética, puedes asignar espacios de archivos adjuntos y archivos adjuntos para acceder a ellos. Estas son simplemente imágenes que están separadas de la animación, pero cuando se unen se moverán junto con el hueso al que están unidas. Normalmente asignaría estos archivos adjuntos en su programa de animación (Spine), pero con esta función puede crear los suyos en tiempo de ejecución utilizando un elemento de sprite de su juego. la función requiere que le dé un nombre al archivo adjunto (como una cadena) y luego configure sprite_index y image_index para usar, así como el origen xey (que puede ser diferente al definido por el sprite en las propiedades de sprite), y también puede establecer cualquier transformación que desee aplicar a la imagen cuando se adjunta. Si el archivo adjunto se creó con éxito, la función volverá 1 y si no fuera así (usted suministró un índice de sprites inválido, o el sprite de base no es un sprite de Spine) entonces regresará -1.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

skeleton_attachment_create("sword", spr_Weapons, 0, 0, 80, 1, 1, 90);
skeleton_attachment_create("knife", spr_Weapons, 1, 0, 45, 1, 1, 90);
skeleton_attachment_create("crossbow", spr_Weapons, 0, 10, 30, 1, 1, 0);
skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow"));

El código anterior verificaría el nombre del archivo adjunto actualmente asignado para la ranura llamada "slot_leftHand" y si se devuelve una cadena vacía, se adjunta un nuevo elemento sprite.
