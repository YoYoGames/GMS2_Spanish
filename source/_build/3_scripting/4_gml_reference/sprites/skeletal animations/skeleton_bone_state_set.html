skeleton_bone_state_set


Sintaxis:

skeleton_bone_state_set(bone, map);

Argumento Descripción
bone El nombre (como una cadena) del hueso.
map El ds_map (creado previamente) que almacena los datos del hueso.


Devoluciones:

N/A


Descripción

Su animación esquelética se compone de varios "huesos", que tendrá definidos y nombrados en su programa de animación, y esta función se puede usar para establecer ciertos datos para el hueso nombrado en cualquier momento. Tenga en cuenta que estos datos establecerán la pose actual para el esqueleto, dependiendo del conjunto de animación utilizado, y la función requiere una creación previa. ds_map, que debe tener las siguientes claves y sus valores requeridos:

  • "x": la posición x local del hueso en relación con el hueso original.

  • "y": la posición local y del hueso en relación con el hueso original.

  • "ángulo": la rotación local del hueso en relación con el hueso original.

  • "xscale": la escala horizontal local del hueso.

  • "yscale": la escala vertical local del hueso.

  • "padre": el nombre (una cadena) del hueso principal.

Esta función se proporciona para que pueda "interceptar" datos de animación y modificarlos antes de que se dibujen en la pantalla, y como tal, debería usarlos en el evento Otra actualización de animación, ya que este evento se desencadena justo antes del sorteo. Eventos.

Tenga en cuenta que puede usar el mismo mapa que completó usando el skeleton_bone_state_get función, a pesar de que contiene las claves y valores "World" adicionales. Dado que estos se refieren a valores de solo lectura, establecerlos con esta función no tendrá ningún efecto.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

var map = ds_map_create();
skeleton_bone_state_get("head", map);
var xx = ds_map_find_value(map, "worldX");
var yy = ds_map_find_value(map, "worldY");
var deltax = mouse_x - (x + xx);
var deltay = mouse_y - (y + yy);
var angle = arctan2(deltay, deltax);
var angle = point_direction(xx, yy, mouse_x, mouse_y);
ds_map_replace(map, "angle", -angle * 180 / pi);
skeleton_bone_state_set("head", map);
ds_map_clear(map);

El código anterior crea un ds_map y luego lo rellena con los datos del hueso llamado "cabeza". Luego extrae la posición mundial del hueso y usa esa información para establecer el "ángulo" del hueso para rastrear la posición del mouse en el juego.
