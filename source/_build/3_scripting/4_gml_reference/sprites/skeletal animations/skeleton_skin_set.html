skeleton_skin_set


Sintaxis:

skeleton_skin_set(skinname);

Argumento Descripción
skinname El nombre (una cadena) de la máscara que se usará.


Devoluciones:

N/A


Descripción

Un sprite de animación esquelética se puede dibujar con una variedad de texturas (llamadas "máscaras"), lo que le permite utilizar una sola animación para múltiples propósitos diferentes. Por ejemplo, puede tener una animación de paseo genérica y esto puede ser utilizado por sus personajes NPC, enemigos, etc. en un juego simplemente cambiando la máscara con la que se dibuja. Estas máscaras deben haber sido previamente creadas y nombradas en su programa de animación, y este nombre se pasa a la función (como una cadena) para configurar el sprite para usar esa máscara cuando se dibuja.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

if skeleton_skin_get() == "skin_Enemy1"
   {
   skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
   }

El código anterior verificaría el aspecto asignado actualmente para la animación y, si es "skin_Enemy1", elige y establece un nuevo aspecto de una de las tres opciones.
