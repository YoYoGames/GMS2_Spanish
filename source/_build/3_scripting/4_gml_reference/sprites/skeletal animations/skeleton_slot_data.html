skeleton_slot_data


Sintaxis:

skeleton_slot_data(sprite, list);

Argumento Descripción
sprite El índice de sprites de la animación del esqueleto de la columna vertebral para obtener los datos.
list El ID de la lista de DS para poblar con los mapas de DS.


Devoluciones:

N/A


Descripción

Con esta función puedes poblar un (pre-creado) ds_list con datos para cada una de las ranuras de archivos adjuntos disponibles que contiene una animación. Esta información se devuelve en el foro de un ds_map que contiene los siguientes pares clave / valor:

  • "nombre": el nombre de la ranura.

  • "hueso": el nombre del hueso.

  • "archivo adjunto": el nombre del archivo adjunto si se usa uno o "(ninguno)" si no hay uno asignado.

Los valores para cada clave serán cadenas y luego se pueden usar en las otras funciones de inserción de esqueleto para estos tipos de sprites. Tenga en cuenta que ds_map creado no se destruyen por lo que tendrá que recorrer el ds_list y destruye cada uno de los mapas creados tú mismo.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

var list = ds_list_create();
var open = true;
slot_name = "";
skeleton_slot_data(sprite_index, list);
for (var i = 0; i < ds_list_size(list); i++;)
   {
   var map = list[| i];
   if open
      {
      if map[? "attachment"] == "(none)"
         {
         open = false;
         slot_name = map[? "name"];
         }
      }
   ds_map_destroy(map);
   }
ds_list_destroy(list);

El código anterior crea una lista de DS y luego la rellena con los datos de ranura para el sprite de instancia. Estos datos se analizan para extraer los mapas de DS individuales con los datos de la ranura. Esto se verifica luego para ver si hay un espacio vacío, y si es así, a la variable "nombre_de_transferencia" se le asigna el nombre de la ranura vacía antes de que se destruya el mapa de DS. Finalmente destruimos la lista de DS porque ya no es necesaria.
