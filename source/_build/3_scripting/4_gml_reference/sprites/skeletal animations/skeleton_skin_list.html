skeleton_skin_list


Sintaxis:

skeleton_skin_list(sprite, list);

Argumento Descripción
sprite El índice de sprites de la animación del esqueleto de la columna para obtener la lista.
list El ID de la lista de DS para rellenar con los nombres de la animación.


Devoluciones:

N/A


Descripción

Con esta función puedes poblar un (pre-creado) ds_list con todos los nombres de las máscaras incluidas como parte del sprite de animación esquelética. Los nombres serán cadenas y luego se pueden usar en las otras funciones de máscara de animación de esqueleto para estos tipos de sprites.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

var list = ds_list_create();
skeleton_skin_list(sprite_index, list);
var num = ds_list_size(list);
skeleton_skin_set(list[| irandom(num - 1));
ds_list_destroy(list);

El código anterior crea una lista de DS y luego la rellena con los nombres de máscara. Luego se elige una aleatoria y se aplica a la instancia antes de que se destruya la lista.
