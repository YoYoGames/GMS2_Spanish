skeleton_animation_set


Sintaxis:

skeleton_animation_set(animname);

Argumento Descripción
animname El nombre (una cadena) de la animación configurada para usar.


Devoluciones:

N/A


Descripción

Una vez que haya asignado un sprite de animación esquelética a su instancia, puede definir qué animación debe usar en cualquier momento con esta función. Cuando creaste tu sprite (en Spine) habrás definido múltiples conjuntos de animaciones para diferentes estados y los nombrará a cada uno apropiadamente. Estos nombres son los que usará para indicarle a GameMaker Studio 2 qué animación debe usar en cualquier momento de su juego. Por ejemplo, si su sprite es para un juego de plataforma, entonces puede tener un conjunto de "salto", un conjunto de "ejecución" y un conjunto "inactivo", todo dentro del sprite y puede cambiar entre ellos en su juego usando este función.

Tenga en cuenta que establecer una nueva animación de esta manera restablecerá el image_index a 0, comenzando el nuevo conjunto de animación desde el comienzo de su ciclo de animación. Si este no es el comportamiento deseado, entonces puede establecer un valor de "mezcla" usando skeleton_animation_mix y cuando configura una nueva animación, se interpola sin problemas con el conjunto anterior. También debe asegurarse de tener un fotograma clave al comienzo de cada animación para evitar que los datos se retengan de animaciones anteriores, a menos que planee utilizar pistas de animación separadas.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

if keyboard_check_pressed(vk_space)
   {
   if skeleton_animation_get() != "jump"
      {
      skeleton_animation_set("jump");
      }
   }

El código anterior cambiará el conjunto de animación que se usa para el conjunto de "salto" cuando se presiona la tecla de espacio, pero solo si el conjunto actual que se está utilizando no es "saltar".
