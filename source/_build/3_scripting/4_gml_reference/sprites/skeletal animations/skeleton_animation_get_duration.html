skeleton_animation_get_duration


Sintaxis:

skeleton_animation_get_duration(animname);

Argumento Descripción
animname El nombre (una cadena) de la animación configurada para usar.


Devoluciones:

Real


Descripción

Esta función devolverá el tiempo requerido para que el conjunto de animación dado se ejecute antes de volver al principio. El valor de retorno está en segundos.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo 1:

time += delta_time / 1000000;
var duration = skeleton_animation_get_duration(skeleton_animation_get());
var frame = floor((image_number * (mTime / duration)) + 0.5) % image_number;
image_index = frame;
draw_self();

El código anterior establecerá el image_index al valor correcto para el sprite de animación esquelético actualmente asignado.

Ejemplo2:

time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

El código anterior dibujará el sprite de animación esquelético dado usando delta-time para establecer el marco que se dibuja.
