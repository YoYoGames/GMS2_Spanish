Animaciones esqueléticas


Las funciones que se encuentran en esta sección solo se pueden usar con sprites que se importaron de un archivo de animación esquelético (como los archivos JSON que exporta Spine ) y se pueden usar para obtener información sobre un elemento de animación en su juego, así como ciertas propiedades dentro de una animación. Estas funciones se pueden usar junto con las funciones y variables de sprites regulares, lo que le permite (por ejemplo) mezclar dos animaciones de esqueleto usando estas funciones especiales mientras configura la escala de la imagen usando las variables de instancia de sprites normales. Para obtener más información sobre las variables de instancia de sprite, consulte la sección sobre Variables de instancia.

NOTA: Para obtener más información sobre la importación de sprites de animación esquelética realizados con Spine, consulte la sección Importar sprites que no son de mapa de bits.
¡IMPORTANTE! Estas funciones no están disponibles con la licencia de prueba del producto.


Las funciones disponibles son:

  1. skeleton_animation_get
  2. skeleton_animation_set
  3. skeleton_animation_get_ext
  4. skeleton_animation_set_ext
  5. skeleton_animation_get_duration
  6. skeleton_animation_mix
  7. skeleton_animation_list
  8. skeleton_animation_clear
  9. skeleton_animation_get_frames
  10. skeleton_animation_get_frame
  11. skeleton_animation_set_frame
  12. skeleton_get_minmax
  13. skeleton_get_num_bounds
  14. skeleton_get_bounds
  15. skeleton_skin_get
  16. skeleton_skin_set
  17. skeleton_skin_list
  18. skeleton_attachment_get
  19. skeleton_attachment_set
  20. skeleton_attachment_create
  21. skeleton_bone_data_get
  22. skeleton_bone_data_set
  23. skeleton_bone_state_get
  24. skeleton_bone_state_set
  25. skeleton_slot_data
  26. skeleton_collision_draw_set

También puede usar varias rutinas de dibujo diseñadas específicamente para estos tipos de sprites (aparte de las rutinas normales draw_sprite() funciones). Estas funciones le brindan más control sobre lo que se está dibujando y le permiten cambiar las características específicas de la animación, como el tiempo y las máscaras:

  1. draw_skeleton
  2. draw_skeleton_collision
  3. draw_skeleton_time
