skeleton_animation_get


Sintaxis:

skeleton_animation_get();


Devoluciones:

String


Descripción

Con esta función, puede obtener el conjunto de animación actual que está utilizando su sprite de animación esquelética. El valor de retorno es una cadena, que será el nombre del conjunto como lo definió en su programa de animación esquelética.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

if keyboard_check_pressed(vk_space)
   {
   if skeleton_animation_get() != "jump"
      {
      skeleton_animation_set("jump");
      }
   }

El código anterior cambiará el conjunto de animación que se usa para el conjunto de "salto" cuando se presiona la tecla de espacio, pero solo si el conjunto actual que se está utilizando no es "saltar".
