skeleton_animation_set_frame


Sintaxis:

skeleton_animation_set_frame(track, index);

Argumento Descripción
track La pista de animación para establecer el marco de.
index El nuevo índice para el cuadro de animación.


Devoluciones:

N/A


Descripción

Esta función establecerá la animación asignada a la pista dada al cuadro que proporcione. El índice de cuadro debe ser un valor real entre 0 y el número de cuadros para la animación (que puede obtener usando la función skeleton_animation_get_frames ) Si proporciona un valor fuera de este rango, la animación se enlazará para compensar la diferencia, por ejemplo, si la animación tiene 5 fotogramas y establece el fotograma en 8, el fotograma real que se muestra será 3.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

if global.pause
    {
    skeleton_animation_set_frame(0, -image_speed);
    }

El código anterior establecerá el índice de cuadro para la animación en la pista 0 a la velocidad de imagen negativa cuando la variable global "pausa" es true (esencialmente pausando la animación).
