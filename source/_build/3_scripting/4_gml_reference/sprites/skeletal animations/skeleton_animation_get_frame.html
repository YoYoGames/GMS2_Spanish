skeleton_animation_get_frame


Sintaxis:

skeleton_animation_get_frame(track);

Argumento Descripción
track La pista de animación para obtener el marco de.


Devoluciones:

Integer


Descripción

Esta función devolverá el número de fotograma de la animación en la pista de animación especificada. La función devolverá -1 si no se asigna ninguna animación a la pista específica dada.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

var frame = skeleton_animation_get_frame(0);
if frame > 0 && frame < 2
    {
    var box = skeleton_get_minmax();
    var ww = (box[2] - box[0]) / 2;
    var hh = (box[3] - box[1]) / 2;
    part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10);
    }

El código anterior verificará el fotograma actual de la animación asignada a la pista 0 y luego arrojará algunas partículas de un punto en el medio del cuadro delimitador total para el sprite.
