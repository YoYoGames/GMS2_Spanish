sprite_get_name


Sintaxis:

sprite_get_name(index);

Argumento Descripción
index El índice del sprite para obtener el nombre de.


Devoluciones

String


Descripción

Esta función devolverá el nombre como una cadena del sprite especificado. Este nombre es el que se ha especificado para el sprite en el árbol de recursos de la ventana principal de GameMaker Studio 2. Tenga en cuenta que esto es solo una cadena y no se puede usar para hacer referencia directamente al sprite, para eso necesitaría el índice de sprites. Sin embargo, puede usar esta cadena para obtener el índice de sprites utilizando la cadena devuelta junto con la función asset_get_index().


Ejemplo:

str = sprite_get_name(sprite_index);

El código anterior obtendrá el nombre del índice de sprites para la instancia que ejecuta el código y almacenará el valor de retorno en la variable "str".
