game_get_speed


Descripción

Esta función se puede utilizar para obtener la velocidad del juego, ya sea como la cantidad de cuadros de juego que se ejecutarán por segundo o como la cantidad de micro segundos por cuadro de juego. Tenga en cuenta que este no es el valor de FPS de la velocidad de funcionamiento real (para ese uso el fps_real variable), sino más bien el número de cuadros de juego (FPS) que el juego intentará mantener cada segundo, o la longitud de cada cuadro de juego en microsegundos que el juego intentará mantener (MPF). Cuando utiliza esta función, debe proporcionar una de las siguientes constantes que determinarán el tipo del valor de retorno:

Constante Descripción
gamespeed_fps Obtiene la velocidad del juego usando cuadros por segundo.
gamespeed_microseconds Obtiene la velocidad del juego usando microsegundos por cuadro.


Entonces, por ejemplo, si la velocidad del juego se establece en 30 en las Opciones del juego y usa el tipo FPS, la función devolverá 30, pero si usa el MPF, la función devolverá 33333.


Sintaxis:

game_get_speed(type);

Argumento Descripción
type El tipo de método utilizado para obtener la velocidad del juego (ver las constantes anteriores).


Devoluciones:

Real


Ejemplo:

if game_get_speed(gamespeed_fps) != 60
    {
    game_set_speed(60, gamespeed_fps);
    }

El código anterior comprueba si el juego se está ejecutando con una velocidad del juego de 60 FPS y si no está configurado a 60 FPS.
