juego_restart

Sintaxis:

game_restart();


Devoluciones: N / A


Descripción

Con esta función puedes reiniciar el juego. Esto es esencialmente lo mismo que ejecutar el juego por primera vez, por lo que se activará el evento de inicio del juego, así como el evento final del juego. Sin embargo, debe tenerse en cuenta que las variables globales no se reinicializarán a menos que se codifiquen explícitamente como tales, por ejemplo, la variable global incorporada score no comenzará en cero después de reiniciar un juego si ya se ha modificado en el juego. Este también es el caso para el estado de la GPU (por lo tanto, si ha establecido el color de dibujo o alfa, por ejemplo, permanecerá en el valor modificado) y para la velocidad del juego. También tenga en cuenta que si cualquier recurso del árbol de recursos se ha modificado en el tiempo de ejecución dentro del juego, por ejemplo, si cambia el origen de un recurso de sprite o cambia la posición de un recurso de ruta, tampoco se reiniciarán.


Ejemplo:

if keyboard_check_pressed(ord("R")) game_restart();

Esto reiniciaría el juego cuando el jugador presiona la tecla "R".
