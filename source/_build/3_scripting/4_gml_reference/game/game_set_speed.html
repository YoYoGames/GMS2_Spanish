game_set_speed


Descripción

Esta función se puede usar para establecer la velocidad del juego. Puede configurar esto de una de dos maneras: como marcos de juego por segundo (FPS) o como microsegundos por fotograma de juego (MPF), usando una de las siguientes dos constantes:

Constante Descripción
gamespeed_fps Establece los juegos con los cuadros por segundo.
gamespeed_microseconds Establece la velocidad del juego usando microsegundos por cuadro.


Entonces, por ejemplo, una velocidad del juego de 30 cuadros por segundo sería 33333 microsegundos por cuadro de juego, que luego se expresaría por esta función como:

game_set_speed(30, gamespeed_fps);

o:

game_set_speed(33333, gamespeed_microseconds);


Sintaxis:

game_set_speed(speed, type);

Argumento Descripción
speed La nueva velocidad del juego (como FPS o MPF).
type El tipo de método utilizado para establecer la velocidad del juego (ver las constantes arriba).


Devoluciones:

N/A


Ejemplo:

if os_browser == browser_not_a_browser
    {
    game_set_speed(60, gamespeed_fps);
    }
else
    {
    game_set_speed(30, gamespeed_fps);
    }

El código anterior comprueba si el juego se está ejecutando en un navegador y establece la velocidad del juego como un valor de FPS.
