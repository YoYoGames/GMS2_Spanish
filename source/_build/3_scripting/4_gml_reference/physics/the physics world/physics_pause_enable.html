physics_pause_enable

Pausa o detiene la simulación física.

Sintaxis:

physics_pause_enable(flag)


Argumento Descripción
flag Esto se puede configurar para true pausar la simulación, o false para comenzarlo de nuevo


Devoluciones: N / A


Descripción

Normalmente, para que funcione una simulación de física, debe ser continua y no puede detenerse ni iniciarse, ni pueden trasladar instancias repentinamente de un lugar a otro en la sala. Sin embargo, hay momentos en los que debe pausar la simulación, ya que (por ejemplo) el sistema operativo se ha detenido, por lo que utilizaría esta función. Pausa la simulación si la bandera está configurada para true y no se realizarán más cálculos de física hasta que la bandera esté configurada para false de nuevo.

NOTA: Esto es de uso particular si desea desactivar todas las instancias en una habitación, ya que incluso cuando está desactivado, un cuerpo físico seguirá siendo calculado y simulado en el mundo de la física.


Ejemplo:

if os_is_paused()
   {
   physics_pause_enable(true);
   instance_deactivate_all(true);
   instance_create_layer(x, y, "Controllers", obj_PauseMenu);
   }

El código anterior comprueba si el sistema operativo se ha pausado y, si lo tiene, pausa el mundo de la física, desactiva todo excepto él mismo, y luego crea una instancia de menú de pausa.
