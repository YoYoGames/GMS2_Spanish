physics_draw_debug

Esta función dibuja un esquema básico de las propiedades físicas para la instancia.

Sintaxis:

physics_draw_debug()

Devoluciones: N / A


Descripción

Al crear un sistema de física en GameMaker Studio 2, a menudo es importante tener una representación visual de lo que realmente está sucediendo con una instancia. Esto se puede lograr mediante el uso de physics_draw_debug, que dibuja un esquema de las propiedades físicas asociadas con la instancia que ejecuta el código. Aquí hay una imagen de cómo se ve:

Se debe tener en cuenta que para que esto funcione debe estar en el evento draw de la instancia, y se dibujará usando el color de dibujo definido actualmente.


Ejemplo:

draw_set_colour(c_red);
physics_draw_debug();

El código anterior dibujará una representación de las propiedades físicas asociadas con esa instancia utilizando el color rojo.
