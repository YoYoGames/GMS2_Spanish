physics_world_draw_debug

Esta función extrae algunos, o todos, del sistema mundial de física para la depuración dependiendo del valor del indicador.

Sintaxis:

physics_world_draw_debug(flag)


Argumento Descripción
flag El valor del indicador para mostrar diferentes funciones de depuración


Devoluciones: N / A


Descripción

Esta función dibujará una representación del sistema mundial de física de la sala para ayudar con la depuración cuando tenga problemas para lograr que la escena física funcione correctamente. A veces el problema puede estar en la parte de reproducción de su juego, por ejemplo, un sprite puede ser dibujado en la posición o rotación incorrecta, dando la apariencia de una física incorrecta, o puede ser que sus cálculos no sean correctos. De cualquier manera, esta función te ayudará a encontrar y resolver el problema dibujando lo que está sucediendo en tu sistema de física. Para hacer esto, necesita decirle qué parte del sistema dibujar, y esto se hace pasando un valor creado a partir de los bits almacenados en varias constantes de "marca":

Argumento Descripción
phy_debug_render_aabb Esto muestra el cuadro delimitador absoluto de cada accesorio en relación con el eje de la sala
phy_debug_render_collision_pairs Esto mostrará cualquier accesorio que esté actualmente en colisión
phy_debug_render_coms Esto marca el centro de masa de cada accesorio en la habitación
phy_debug_render_core_shapes Muestra las formas básicas que componen los accesorios en la habitación
phy_debug_render_joints Esto dibujará cada una de las articulaciones de todos los accesorios en la sala
phy_debug_render_obb Esto muestra el cuadro delimitador relativo para los accesorios en la sala
phy_debug_render_shapes Esto muestra las formas reales que componen los accesorios dentro de la habitación

Entonces, para dibujar dos de esas opciones tendremos que hacer un "o" bit a bit (en GameMaker Studio 2 esto está representado por |) en las dos y almacenar la variable resultante para la función a usar. Aquí hay un ejemplo de cómo establecería una variable para usar con la función que dibujará solo los centros de masa y las uniones de todos los accesorios en una habitación:

flag = phy_debug_render_coms | phy_debug_render_shapes

Que le daría algo como esto:

Como puede ver con el código anterior, solo se dibujan las formas y los centros de masa en la pantalla (así como los colores que representan sus estados; para obtener más información, consulte la documentación de Box 2D en línea). De la misma manera, puede visualizar más, o incluso todas, las constantes que desea depurar... ¡Simplemente "o" todas juntas! Una cosa más hay que señalar - esta función debe ser llamada solamente en caso de empate de una instancia que tiene una profundidad menor que aquellos que desea depurar o de lo contrario no será capaz de ver la información de depuración.


Ejemplo:

flag = phy_debug_render_aabb | phy_debug_render_collision_pairs | phy_debug_render_obb;
physics_world_draw_debug(flag);

El código anterior dibujará todos los cuadros de límite absoluto, los cuadros de límite relativos, así como los dispositivos en colisión dentro del sistema de física actual.
