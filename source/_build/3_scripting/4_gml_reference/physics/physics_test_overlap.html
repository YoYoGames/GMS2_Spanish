physics_test_overlap

Comprueba una superposición de dos dispositivos en una posición determinada.

Sintaxis:

physics_test_overlap(xpos, ypos, angle, obj);


Argumento Descripción
xpos La posición x en la habitación para verificar
ypos La posición y en la sala para verificar
angle El ángulo para verificar (de la instancia de llamada)
obj El objeto para verificar


Devuelve: booleano.


Descripción

Esta función se puede usar para verificar y ver si un cuerpo físico (es decir, el accesorio de una instancia) se superpone, o se superpondrá, cuando se gira y se coloca en una posición determinada en la habitación. el argumento "ángulo" es el ángulo de rotación que la instancia llamante tiene (o tendrá) en la posición que se va a verificar, y el argumento "obj" puede ser una identificación de instancia única, e índice de objeto o las palabras clave todas u otras.


Ejemplo:

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
   {
   alarm[0] = room_speed;
   ignited = true;
   }

El código anterior verificará una posición para una superposición de artefactos físicos, y si hay una, establece una variable y una alarma.
