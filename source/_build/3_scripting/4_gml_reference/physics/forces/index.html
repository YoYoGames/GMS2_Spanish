Efectivo

En el mundo de la física de GameMaker Studio 2, las instancias no solo se ven afectadas por la gravedad y las colisiones, también se ven afectadas por las fuerzas que eliges aplicar a ellas, y conocer la fuerza correcta para aplicar es muy importante cuando quieres controlar una Instalar correctamente dentro de la simulación física. Hay dos fuerzas que se pueden usar y aplicar a tus accesorios:

  • Impulsos: un impulso establecerá la dirección y la velocidad inmediatamente después de que se aplique, lo que lo hace ideal para cosas como resortes, disparos de balas, etc.

  • Fuerzas: una fuerza actuará sobre la velocidad y dirección existentes y la cambiará con el tiempo, lo que la hace ideal para cosas como el viento, un imán, etc.

  • Torque: el par rige la probabilidad de que una instancia siga girando alrededor de su eje después de que se haya aplicado una fuerza, y la velocidad a la que dejará de girar. Agregar torque es un proceso gradual, como usar una fuerza para agregar movimiento.

  • Impulso angular: se puede usar un impulso angular para configurar un dispositivo girando instantáneamente, de manera muy similar a como se puede usar un impulso regular para poner un dispositivo instantáneamente en movimiento.

Las siguientes funciones existen para aplicar diferentes tipos de fuerzas a las instancias habilitadas para la física, y cada función ofrece una explicación más completa de cómo se aplicará la fuerza que los esquemas dados anteriormente:

  • physics_apply_force
  • physics_apply_impulse
  • physics_apply_local_force
  • physics_apply_local_impulse
  • physics_apply_torque
  • physics_apply_angular_impulse
