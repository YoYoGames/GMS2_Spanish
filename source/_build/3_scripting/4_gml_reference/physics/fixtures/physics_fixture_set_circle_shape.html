physics_fixture_set_circle_shape

Esta función establece la forma del accesorio a la de un círculo con el radio especificado.

Sintaxis:

physics_fixture_set_circle_shape(fixture, rad)


Argumento Descripción
fixture el índice del accesorio
rad radio del círculo


Devoluciones: N / A


Descripción

Esta función define una forma de círculo para su dispositivo con un radio definido por el argumento "rad".


Ejemplo:

physics_fixture_set_circle_shape(fix_Ball, sprite_get_width(spr_Ball) / 2);

El código anterior aplicará una forma circular al dispositivo indexado en la variable "fix_Ball" con un radio igual al del ancho del sprite "spr_Ball" dividido por 2.
