physics_fixture_delete

Esta función elimina un dispositivo de la memoria.

Sintaxis:

physics_fixture_delete(fixture)


Argumento Descripción
fixture el accesorio que se va a eliminar de la memoria


Devoluciones: N / A


Descripción

Dado que una vez que un accesorio se ha vinculado a una instancia que la instancia mantiene las propiedades de las luminarias, podemos eliminar el dispositivo de la memoria con esta función, especialmente si ya no planeamos usarla de nuevo. Tenga en cuenta que si no quita los dispositivos después de que ya no los necesite puede provocar una pérdida de memoria que se ralentizará y, finalmente, bloqueará el juego, por lo que debe tener cuidado de limpiar correctamente después de definir y vincular los dispositivos.


Ejemplo:

var fix, inst;
fix = physics_fixture_create();
physics_fixture_set_circle_shape(fix, 16);
physics_fixture_set_density(fix, 1.0);
inst = instance_create_layer(x, y, "Instances", genericBodyObject);
physics_fixture_bind(fix, inst);
physics_fixture_delete(fix);

El código anterior creará un accesorio y asignará su índice a la variable "arreglar". A continuación, define la forma y la densidad del dispositivo antes de vincularlo a la instancia que se creó con el índice almacenado en la variable "inst". Finalmente, el accesorio se elimina para evitar fugas de memoria ya que ya no es necesario.
