physics_fixture_set_polygon_shape

Esta función establece la forma del accesorio a la de un polígono.

Sintaxis:

physics_fixture_set_polygon_shape(fixture)


Argumento Descripción
fixture el índice del accesorio


Devoluciones: N / A


Descripción

Esta función establece una forma de polígono para su accesorio, pero deberá usar physics_fixture_add_point para definir realmente la forma de este polígono en relación con el origen del accesorio. El polígono se cierra cuando el dispositivo está vinculado a una instancia. También debe tener en cuenta que se debe llamar a esta función antes de definir cualquier punto, y también debe tener al menos tres puntos definidos para su polígono antes de vincularlo a una instancia o se obtendrá un error.


Ejemplo:

physics_fixture_set_polygon_shape(fix_Ship);
physics_fixture_add_point(fix_Ship, 0,0);
physics_fixture_add_point(fix_Ship, -40, 100);
physics_fixture_add_point(fix_Ship, 40, 100);

El código anterior aplicará una forma de polígono al dispositivo indexado en la variable "fix_Ship" y luego definirá tres puntos para crear una forma triangular.
