physical_set_restitution

Esta función establece el valor de restitución para un accesorio vinculado.

Sintaxis:

physics_set_restitution(fixture, restitution)


Argumento Descripción
fixture la identificación del accesorio atado
restitution el nuevo valor de restitución para aplicar


Devoluciones: N / A


Descripción

Cuando une un accesorio a una instancia usando physics_fixture_bind esto devuelve una "id" para el accesorio vinculado. Puede usar esta identificación para establecer el valor de restitución del accesorio vinculado, no el dispositivo "base", en cualquier momento usando esta función. La restitución generalmente se establece como un valor entre 0 y 1, pero puede usar valores más altos si es necesario, aunque los resultados pueden ser impredecibles.


Ejemplo:

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

El código anterior obtiene el valor de restitución actual para las propiedades físicas enlazadas de la instancia y luego las establece en un valor diferente.
