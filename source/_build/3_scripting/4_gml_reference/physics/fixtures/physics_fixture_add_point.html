physics_fixture_add_point

Esta función agrega un punto para crear una forma de polígono o una forma de cadena para un accesorio.

Sintaxis:

physics_fixture_add_point(fixture, xpos, ypos)


Argumento Descripción
fixture el índice del accesorio
xpos la posición x del punto ( relativa al origen del accesorio para formas de polígono)
ypos la posición y del punto ( relativa al origen del dispositivo para formas de polígono)


Devoluciones: N / A


Descripción

Esta función define los puntos de un polígono que se ha establecido previamente en el dispositivo usando cualquiera physics_fixture_set_polygon_shape o puede usarse para agregar puntos a una cadena de accesorios de borde usando la función physics_fixture_set_chain_shape. Para una forma de polígono, los puntos son relativos al origen del dispositivo, con la posición 0,0 del eje x / y como el centro, y las coordenadas posteriores se calculan como siempre en GameMaker Studio 2 con abajo / derecha siendo + x, + y y arriba / left siendo -x, -y. Aquí hay una ilustración para ayudarlo a visualizar este proceso:

Como puede ver, cualquier polígono convexo simple se puede modelar de esta manera, pero los polígonos cóncavos no son compatibles en este momento. Finalmente, para cerrar el polígono, debe vincular el dispositivo a una instancia. También debe tener en cuenta que también debe tener al menos tres puntos definidos para su polígono antes de vincularlo a una instancia o recibirá un error y agregar un punto a un dispositivo que no tenga una forma de polígono previamente establecida también causará un error. Tenga en cuenta también que la cantidad máxima de puntos permitidos para un dispositivo es ocho.

Para las formas de cadena (que son simplemente formas de borde conectadas en serie), esta función establecerá un punto como una "conexión" para encadenar los bordes a la vez. Por lo tanto, debe haber definido al menos dos puntos antes de vincular el dispositivo a la instancia; de lo contrario, obtendrá un error. Sin embargo, a diferencia de los polígonos, no está limitado a un máximo de ocho puntos y puede agregar tantos como necesite para crear la cadena y los puntos se definen en las coordenadas reales de la sala, en lugar de en relación con la instancia.


Ejemplo:

physics_fixture_set_polygon_shape(fix_Ship);
physics_fixture_add_point(fix_Ship, 0,0);
physics_fixture_add_point(fix_Ship, -40, 100);
physics_fixture_add_point(fix_Ship, 40, 100);

El código anterior aplicará una forma de polígono al dispositivo indexado en la variable "fix_Ship" y luego definirá tres puntos para crear una forma triangular.
