phy_col_normal_x

Devuelve el componente x de la colisión normal para el punto de colisión asociado.

Sintaxis:

phy_col_normal_x;


Devoluciones: Boolean


Descripción

Esta variable de solo lectura devuelve el componente x de la colisión normal correspondiente a la phy_collision_x valor de matriz Para cada punto de contacto existe un contacto asociado normal (que suele ser la misma normal para todos los puntos de contacto en la colisión). Este contacto normal es un vector unitario que apunta de una instancia en la colisión a otra, y puede usarse, por ejemplo, para calcular la dirección correcta de "empuje" para resolver colisiones.

NOTA: Esta variable solo está disponible en el evento de colisión de una instancia habilitada para física.


Ejemplo:

repeat(5 + irandom(5))
   {
   with (instance_create_layer(x, y, "Effects", obj_Debris)
      {
      physics_apply_local_impulse(0, 0, other.phy_col_normal_x[0], other.phy_col_normal_y[0]);
      }
   }

El código anterior usa el contacto normal para establecer la dirección de movimiento de una instancia creada en el evento de colisión entre dos instancias habilitadas para física.
