phy_speed

Devuelve la velocidad de la instancia habilitada física.

Sintaxis:

phy_speed;


Devoluciones: Real


Descripción

Esta variable de solo lectura devuelve la velocidad como un valor en píxeles por segundo de la instancia habilitada para física. Si necesita cambiar este valor, debe hacerlo cambiando los vectores xey por las variables phy_speed_x y phy_speed_y.


Ejemplo:

if phy_speed > 10
   {
   phy_linear_damping += 0.01;
   }
else
   {
   phy_linear_damping = 2;
   }

El código anterior comprueba la velocidad de la instancia habilitada física y luego cambia la amortiguación lineal en función del valor devuelto.
