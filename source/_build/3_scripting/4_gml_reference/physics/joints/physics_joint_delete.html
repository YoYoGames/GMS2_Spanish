physical_joint_delete

Eliminar una articulación del mundo de la física.

Sintaxis:

physics_joint_delete(joint)


Argumento Descripción
joint El índice de la unión que desea eliminar


Devoluciones: N / A


Descripción

Una vez que dos instancias con representaciones de física han sido vinculadas por una unión, esto puede eliminarse de nuevo en cualquier momento. Normalmente, esto sucederá automáticamente cuando se destruya una de las dos instancias o cuando la sala termine, pero hay ocasiones en las que puede desear hacerlo manualmente. En esos casos, usaría esta función.

También se debe tener en cuenta que cuando se destruye una instancia que forma parte de una unión de engranajes, la unión de engranajes debe eliminarse utilizando esta función antes de cualquiera de los casos involucrados en la formación del engranaje (pero las juntas restantes se eliminarán automáticamente), ejemplo en el evento de destrucción de la instancia.


Ejemplo:

var reactionForceX, reactionForceY, reactionForce;
if shipJoint
   {
   reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
   reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
   reactionForce = sqrt((reactionForceX + reactionForceX) + (reactionForceY + reactionForceY));
   if reactionForce > 2
      {
      physics_joint_delete(shipJoint);
      shipJoint = -1;
      }
   }

El código anterior comprueba si la variable "shipJoint" contiene un índice conjunto y, si lo hace, calcula la fuerza que se aplica a esa unión utilizando las dos constantes. Finalmente, si la fuerza total es mayor que 2, la unión se elimina.
