Constantes conjuntas de física

Hay una gran cantidad de constantes incluidas en GameMaker Studio 2 GML que son específicas para las uniones entre los dispositivos en el mundo de la física. Estos pueden usarse junto con muchas de las diferentes funciones conjuntas para establecer u obtener información de ellos en tiempo real mientras se ejecuta la simulación física. Sin embargo, debe tener en cuenta que los cálculos complejos se realizan cuando los llama, por lo que deben usarse con cuidado y solo cuando sea necesario, y tener en cuenta que muchos son exclusivos de un tipo específico de articulación. También tenga en cuenta que si bien puede obtener todos estos valores con la función apropiada, solo puede establecer aquellos que están marcados como no de solo lectura.

Las siguientes constantes se pueden aplicar a cualquiera de los tipos de unión disponibles:

Constante Descripción Solo lectura
phy_joint_anchor_1_x La coordenada x del primer punto de anclaje de la junta en la sala Yes
phy_joint_anchor_1_y La coordenada y del primer punto de anclaje de la articulación en la habitación Yes
phy_joint_anchor_2_x La coordenada x del segundo punto de anclaje de la junta en la sala Yes
phy_joint_anchor_2_y La coordenada y del segundo punto de anclaje de la articulación en la sala Yes
phy_joint_reaction_force_x Esta es la fuerza de reacción que se aplica a la segunda instancia en una articulación en la posición x ancla Yes
phy_joint_reaction_force_y Esta es la fuerza de reacción que se aplica a la segunda instancia en una articulación en la posición de anclaje Yes
phy_joint_reaction_torque Este es el par que se aplica a la segunda instancia en una articulación en la posición de anclaje Yes

Estas constantes son para aquellas uniones que tienen un motor unido a ellas (revoluto, prismático, rueda):

Constante Descripción Solo lectura
phy_joint_max_motor_force El valor especificado cuando la unión se creó para la fuerza máxima del motor No
phy_joint_max_motor_torque El valor especificado cuando la unión se creó para el par motor máximo No
phy_joint_motor_force La fuerza motriz actual Yes
phy_joint_motor_speed La velocidad actual del motor No
phy_joint_motor_torque El torque actual del motor Yes

Para una articulación giratoria puede usar la siguiente constante (así como las constantes del motor si se ha agregado una):

Constante Descripción Solo lectura
phy_joint_angle El ángulo que crea una línea entre los dos puntos de anclaje de la unión. Esto se calcula utilizando las coordenadas del mundo de la física
( no las coordenadas de la sala GameMaker Studio 2 ) en radianes. 		Yes
phy_joint_angle_limits Habilite o deshabilite la limitación de ángulo para la articulación. Establezca el valor en true para habilitar o false deshabilitar. No
phy_joint_upper_angle_limit El límite superior del ángulo para la unión en grados. No
phy_joint_lower_angle_limit El límite de ángulo inferior para la unión en grados. No

Para una articulación prismática puede usar la siguiente constante:

Constante Descripción Solo lectura
phy_joint_translation Obtiene la distancia entre las coordenadas x / y del ancla y las coordenadas x / y locales. Yes
phy_joint_speed La velocidad de movimiento conjunta actual. Yes

Para una unión de polea, puede usar las siguientes constantes (así como las de las uniones a distancia):

Constante Descripción Solo lectura
phy_joint_length_1 Esto devolverá la longitud de la unión desde las primeras coordenadas x / y locales a las coordenadas del primer ancla x / y No
phy_joint_length_2 Esto devolverá la longitud de la unión desde las segundas coordenadas locales x / y a las coordenadas del segundo anclaje x / y Yes

Para una distancia, soldadura y uniones de rueda, puede usar las siguientes constantes (así como las de las uniones de poleas):

Constante Descripción Solo lectura
phy_joint_damping_ratio La relación de amortiguación no es dimensional y define la "elasticidad" de la junta. El valor para esta constante es típicamente entre 0 y 1, pero puede ser mayor, y en 1, la amortiguación es crítica, lo que significa que todas las oscilaciones deben desaparecer. No
phy_joint_frequency Esto devolverá (o establecerá) la frecuencia de oscilación para la articulación, en hertzios, y típicamente la frecuencia debe ser menor que la mitad de la frecuencia del paso de tiempo, como lo establece la función physics_world_update_speed. No

Para una junta de fricción puede usar las siguientes constantes:

Constante Descripción Solo lectura
phy_joint_max_torque El valor de par máximo para la articulación. No
phy_joint_max_force El valor de fuerza máximo para la articulación. No

Para una unión de cable, puede usar la siguiente constante:

Constante Descripción Solo lectura
phy_joint_max_length La extensión máxima para la conexión entre los dos puntos de anclaje. No
