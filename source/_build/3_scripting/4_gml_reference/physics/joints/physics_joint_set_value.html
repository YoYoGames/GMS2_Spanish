physics_joint_set_value

Esta función establece una propiedad conjunta conjunta con el valor especificado.

Sintaxis:

physics_joint_set_value(joint, field, value)


Argumento Descripción
joint El índice de la articulación que desea cambiar
field La constante de la propiedad conjunta que desea cambiar
value El nuevo valor para la propiedad conjunta


Devoluciones: N / A


Descripción

Ciertas propiedades de la junta pueden cambiarse y establecerse incluso después de la creación de la junta, pero solo en aquellas juntas que tienen un motor, es decir, articulaciones giratorias y articulaciones prismáticas. Hay una serie de constantes que se pueden utilizar en esta función y se pueden encontrar aquí: Constantes de Física Conjunta.


Ejemplo:

if physics_joint_get_value(revJoint, phy_joint_max_motor_torque) < 2
   {
   physics_joint_set_value(revJoint, phy_joint_max_motor_torque, 2);
   {

El código anterior comprueba si el par motor máximo de las uniones está configurado a menos de 2 y si lo está, lo establece en 2.
