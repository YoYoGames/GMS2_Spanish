physics_joint_get_value

Esta función prueba una articulación y devuelve un valor según el tipo de articulación que se prueba y la constante que se utiliza.

Sintaxis:

physics_joint_get_value(joint, value)


Argumento Descripción
joint El índice de la articulación que desea probar
value La constante para la propiedad conjunta que desea probar


Devoluciones: Real


Descripción

Al usar una serie de constantes predefinidas, puede pedirle a GameMaker Studio 2 que le cuente una serie de cosas sobre el estado de una articulación determinada. Esto es muy útil ya que le da la capacidad de eliminar uniones o cambiar un comportamiento de instancias dependiendo de cuáles sean sus necesidades en ese momento. Hay una serie de constantes que se pueden usar en esta función y se pueden encontrar aquí: Constantes de Física Conjunta, pero tenga en cuenta que los cálculos complejos se realizan cuando los llama, por lo que deben usarse con cuidado y solo cuando sea necesario y tenga en cuenta que muchos son exclusivos de un tipo específico de articulación.

Si la propiedad no existe (si, por ejemplo, comprueba una articulación de polea para la traducción prismática), entonces 0 será el valor de retorno.


Ejemplo:

var reactionForceX, reactionForceY, reactionForce;
if shipJoint
   {
   reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
   reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_y);
   reactionForce = point_distance(0, 0, reactionForceX, reactionForceY);
   if reactionForce > 2
      {
      physics_joint_delete(shipJoint);
      shipJoint = -1;
      }
   }

El código anterior comprueba si la variable "shipJoint" contiene un índice conjunto y, si lo hace, calcula la fuerza que se aplica a esa unión utilizando las dos constantes. Finalmente, si la fuerza total es mayor que 2, la unión se elimina.
