physics_joint_enable_motor

Activa (o desactiva) el motor de una articulación que permite este comportamiento.

Sintaxis:

physics_joint_enable_motor(joint, motor)


Argumento Descripción
joint La unión que desea activar o desactivar el motor
motor Si desea encender el motor (verdadero) o apagarlo (falso)


Devoluciones: N / A


Descripción

Cuando tiene una articulación con un motor ( prismático o revoluto ), es posible que desee activar o desactivar el motor según las variables y condiciones dentro del juego. Para esto, debe haber almacenado el índice de la unión previamente en una variable y luego puede encender o apagar el motor usando esta función y configurando el argumento "motor" para true o false.


Ejemplo:

var t_fix = physics_fixture_create();
physics_fixture_set_circle_shape(t_fix, sprite_get_width(sprite_index) / 2);
var o_id=instance_create_layer(x+300, y, "Instances", obj_Door);
physics_fixture_bind(t_fix, id);
physics_fixture_bind(t_fix, o_id);
perma_joint = physics_joint_revolute_create(id, o_id, x+25, y, -90, 90, 1, 10, 2, 0, 0);
physics_joint_enable(perma_joint, 1);
physics_fixture_delete(t_fix);

El código anterior crea y define un nuevo dispositivo y luego crea una instancia de "obj_Door", vinculando el dispositivo creado a las dos instancias. A continuación, se unen una articulación giratoria sin motor y los ángulos se limitan a un giro de +/- 90 grados, y almacenamos el índice de articulación en la variable "perma_joint". Luego encendemos el motor usando esta variable, antes de finalmente eliminar el dispositivo de la memoria ya que ya no es necesario.
