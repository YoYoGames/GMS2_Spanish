physics_particle_create

Crea una sola partícula de cuerpo blando con los parámetros dados.

Sintaxis:

physics_particle_create(flags, x, y, xv, yv, col, alpha, category)


Argumento Descripción
flags Las banderas para establecer en la partícula.
x La posición x para crear la partícula en.
y La posición y para crear la partícula en.
xv La velocidad horizontal inicial.
yv La velocidad vertical inicial.
col El color base para usar para la partícula.
alpha La base alfa para usar para la partícula.
category La categoría definida por el usuario a la que pertenece la partícula.


Devoluciones: Real


Descripción

Con esta función puedes crear una sola partícula en cualquier lugar dentro de tu sala de juegos, configurando ciertas banderas y propiedades. Los "indicadores" son el valor de retorno de una combinación de las siguientes constantes:

Constante Descripción
phy_particle_flag_water Las propiedades predeterminadas para una partícula de cuerpo blando.
phy_particle_flag_zombie Una partícula zombie es aquella que se destruirá después de un solo paso con todos los demás marcados de esta manera.
phy_particle_flag_wall Esto define la partícula como estática, esencialmente creándola como un objeto inamovible en la simulación física, ya que permanecerán en una posición fija sin importar lo que colisiona con ellos. Debe usar esta bandera en lugar de establecer la densidad en 0.
phy_particle_flag_spring Las partículas de resorte producen el efecto de estar unidas entre sí, como por un resorte. Las partículas creadas con esta bandera están "conectadas" por parejas, y cada partícula se conecta a la que estaba más cerca de ella en el momento de la creación. Una vez emparejadas, las partículas no cambian a los "compañeros", y cuanto más lejos una fuerza externa los separe, mayor será la potencia con la que colisionarán cuando se elimine esa fuerza externa. Tenga en cuenta que no importa cuán lejos se encuentren las partículas emparejadas una de la otra, la conexión entre ellas no se romperá.
phy_particle_flag_elastic Las partículas elásticas se deforman y también pueden rebotar cuando colisionan con otros cuerpos rígidos en la simulación física.
phy_particle_flag_viscous Una partícula viscosa es aquella que exhibe "adherencia" o "pegajosidad", como el petróleo. Las partículas viscosas se agruparán y se unirán más.
phy_particle_flag_powder Las partículas de polvo producen un efecto de dispersión como el que puede ver con arena o polvo.
phy_particle_flag_tensile Las partículas extensibles se utilizan para producir el efecto de la tensión superficial, o la curvatura tensa en la superficie de un cuerpo de líquido. Se podrían usar, por ejemplo, para crear la tensión superficial que vería en una gota de agua. Una vez que la tensión se rompe, las partículas rebotan como si fueran elásticas, pero también continúan atrayéndose. Como resultado, las partículas tienden a formar grupos cuando rebotan.
phy_particle_flag_colourmixing Las partículas de mezcla de colores adquieren algo del color de otras partículas con las que colisionan. Tenga en cuenta que si solo una de las dos partículas colisionantes es una mezcla de colores, la otra retiene su color previo a la colisión.



Estas banderas usan enmascaramiento de bits para crear un valor de salida final que luego se verifica para establecer las diferentes propiedades básicas de la partícula (siendo la propiedad base siempre la de phy_particle_flag_water ) Por ejemplo, si quiere simular un líquido viscoso con tensión superficial, debería usar el bitwise o "|" marcar los bits apropiados como se muestra en el ejemplo a continuación. De esta forma, puede establecer diferentes propiedades (distintas de las propiedades globales) para cada partícula individual creada.

Además de las banderas, también puede establecer la posición en la sala para crear la partícula en, la velocidad horizontal y vertical inicial, el color inicial y alfa, así como la categoría de usuario. Los valores de color y alfa solo se usan al dibujar la partícula usando la función physics_particle_draw(), pero el valor de la categoría se usa en muchas otras funciones. Este valor es un valor entero arbitrario que le da a cada partícula que desea que tenga propiedades similares, y se puede utilizar para establecer indicadores más adelante para esa categoría específica, así como una serie de otras cosas. Por ejemplo, si ha definido una partícula de "agua" y una de "aceite", le daría a todas esas partículas marcadas como "agua" la categoría de usuario de 1 y todas esas partículas creadas como "aceite" la categoría de usuario de 2, lo que permite para luego dirigirse a una sola categoría para cambiar sus banderas u obtener datos específicos.

NOTA: Puede usar cualquier valor entero para la categoría excepto 0 que está reservada por la simulación y se usa para seleccionar todas las categorías en otras funciones (como para dibujar).

La función devuelve un valor único de índice (o ID) para la partícula que puede almacenarse en una variable para ser utilizada en otras funciones que hacen referencia directamente a la partícula.


Ejemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var x_vel = lengthdir_x(5, image_angle);
var y_vel = lengthdir_y(5, image_angle);
var p = physics_particle_create(flags, x, y, x_vel, y_vel, c_white, 1, 1)

El código anterior creará variables para establecer el tipo de partícula y su velocidad inicial, luego úselas para crear una partícula en la posición x / y de las instancias.
