physics_particle_get_group_flags

Obtiene los indicadores de grupo para un grupo de partículas.

Sintaxis:

physics_particle_get_group_flags(group)


Argumento Descripción
group El grupo de partículas para obtener.


Devoluciones: Real


Descripción

Con esta función puede recuperar los indicadores de grupo para un grupo de partículas. los group valor es el que se devolvió cuando creó el grupo de partículas utilizando la función physics_particle_group_end, y la función devolverá un valor que es el valor combinado de los indicadores actualmente establecidos.


Ejemplo:

var flags = physics_particle_get_group_flags(group, flags);
if flags != phy_particle_group_flag_solid | phy_particle_group_flag_rigid
   {
   flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
   physics_particle_set_group_flags(group, flags);
   }

El código anterior creará una variable para almacenar el valor de los indicadores y luego lo usará para verificar los indicadores del grupo indexado en la variable "grupo". Si no son lo mismo, el grupo se configura con estos indicadores.
