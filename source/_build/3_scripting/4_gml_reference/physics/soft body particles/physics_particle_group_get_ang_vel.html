physics_particle_group_get_ang_vel

Obtenga la velocidad angular de un grupo completo de partículas.

Sintaxis:

physics_particle_group_get_ang_vel(group)


Argumento Descripción
group El grupo de partículas para obtener.


Devoluciones: Real


Descripción

Con esta función puedes recuperar la velocidad angular de un grupo de partículas. los group valor es el que se devolvió cuando creó el grupo de partículas utilizando la función physics_particle_group_end, y la función devolverá un valor que es el valor combinado de los indicadores actualmente establecidos.


Ejemplo:

ang_v = physics_particle_group_get_ang_vel(group1);

El código anterior obtendrá la velocidad angular del grupo de partículas indexada en la variable "grupo1" y la almacenará en una variable.
