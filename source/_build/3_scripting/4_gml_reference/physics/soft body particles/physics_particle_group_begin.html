physics_particle_group_begin

Comience la creación de un grupo de partículas de cuerpo blando con los parámetros dados.

Sintaxis:

physics_particle_group_begin(flags, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category)


Argumento Descripción
flags Las banderas para establecer en la partícula.
groupflags El grupo marca para establecer en la partícula (ver Descripción a continuación).
x La posición x para crear el grupo de partículas en.
y La posición y para crear el grupo de partículas en.
ang El ángulo (en grados) para rotar el grupo para crear.
xv La velocidad horizontal inicial.
yv La velocidad vertical inicial.
ang_velocity La velocidad angular inicial.
col El color base para usar para el grupo de partículas.
alpha La base alfa para usar para el grupo de partículas.
strength Establece la fuerza de cohesión entre las partículas en el grupo.
category La categoría definida por el usuario a la que pertenece el grupo.


Devoluciones: N / A


Descripción

Con esta función puedes crear un grupo de partículas en una habitación. Un grupo de partículas es aquel en el que las partículas necesarias para crear una forma específica se crean todas a la vez como un grupo, lo que le permite crear cuerpos blandos simulados con varias propiedades establecidas por los indicadores utilizados. Los "indicadores" son el valor de retorno de una combinación de las constantes de partículas (ver la función physics_particle_create para detalles de estos), mientras que los "groupFlags" son una combinación de las siguientes constantes de grupo (o 0 para ningún indicador):

Constante Descripción
phy_particle_group_flag_solid Un grupo de partículas sólidas evita que otros dispositivos se alojen en su interior. En caso de que algo penetre en él, el grupo de partículas sólidas empujará el elemento dañino hacia su superficie, haciendo que un grupo de partículas sólidas posea una fuerza de repulsión especialmente fuerte.
phy_particle_group_flag_rigid Los grupos de partículas rígidos son aquellos cuya forma no cambia, incluso cuando colisionan con otros dispositivos.



Estas banderas de grupo usan enmascaramiento de bits para crear un valor de salida final que luego se verifica para establecer las diferentes propiedades básicas del grupo, además de las establecidas por los indicadores de partículas. Puede usar ninguna, una o ambas de estas constantes utilizando bitwise o "|" para enmascarar los bits apropiados (como se muestra en el ejemplo a continuación).

Además de las banderas, también puede establecer la posición en la sala para crear el grupo de partículas en (la posición dada será el centro de masa del grupo terminado), la velocidad horizontal y vertical inicial, así como la velocidad angular de el grupo, lo que le permite crear grupos en movimiento y rotativos.

También puede establecer el color inicial y alfa de todas las partículas en el grupo, con los valores que se utilizan al dibujar las partículas utilizando la función physics_particle_draw(). Después de eso, establece la fuerza cohesiva de las partículas en el grupo, que es otra forma de decir qué tan fuerte se "pegan" entre sí. La fuerza puede ser un valor entre 0 y 1, donde 0 no es cohesión y 1 es cohesión máxima.

Finalmente debes dar la categoría para tu grupo de partículas. Este valor es un valor entero arbitrario que le da a cada grupo de partículas que desea que tenga propiedades similares, y se puede utilizar para establecer indicadores más adelante para esa categoría específica, así como una serie de otras cosas.

NOTA: Puede usar cualquier valor entero para la categoría excepto 0 que está reservada por la simulación y se usa para seleccionar todas las categorías en otras funciones (como para dibujar).

Es importante tener en cuenta que llamar a esta función no crea el grupo en la sala de física. Esta función simplemente comienza el proceso y establece las propiedades de la partícula base para el grupo. Para crear realmente el grupo de partículas, debe darle un formulario (ver el ejemplo a continuación) y luego llamar a la función physics_particle_group_end(), en ese punto será creado.

NOTA: Las banderas de tipo de partículas que establezca también influirán en la cohesión del grupo de partículas, de modo que si señala las partículas como del tipo de agua (por ejemplo), cuando el grupo colisiona con otro dispositivo se romperá, mientras que si marca la partícula como un tipo de resorte, entonces mantendrá su cohesión en la colisión.


Ejemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

El código anterior almacena los indicadores para el tipo de partícula y las propiedades del grupo de partículas en variables y luego los utiliza para crear un grupo de partículas circular con un radio de 100 píxeles en la posición del mouse.
