physics_particle_get_data_particle

Obtenga datos sobre partículas individuales en su simulación de física.

Sintaxis:

physics_particle_get_data_particle(ind, buffer, flags)


Argumento Descripción
ind El índice (ID) de la partícula para obtener los datos.
buffer El búfer (creado previamente) para usar para almacenar los datos.
flags Las banderas para usar para extraer datos sobre tipos de partículas específicos.


Devoluciones: datos de buffer


Descripción

Esta función devuelve varias piezas de información sobre una sola partícula en la simulación física usando los indicadores indicados. El índice de partículas (su ID) es el que fue devuelto por la función physics_particle_create(), y el buffer usado debe haber sido creado previamente usando la función buffer_create(). Debe ser del tipo "crecer", con un tamaño aproximado al de los datos de retorno esperados. Las banderas mismas se configuran usando las constantes que se dan a continuación, y se usaría la bit a bit o "|" para crear un valor de indicador único para obtener la información deseada.

Constante Descripción Tipo de datos
phy_particle_data_flag_typeflags El valor de los indicadores para la partícula. buffer_u32
phy_particle_data_flag_position La posición xey de la partícula. 2 x buffer_f32
phy_particle_data_flag_velocity La velocidad horizontal y vertical. 2 x buffer_f32
phy_particle_data_flag_colour El color y el valor alfa (hexadecimal). buffer_f32
phy_particle_data_flag_category La categoría de partículas (como se definió cuando creó la partícula). buffer_f32




Ejemplo:

var part = physics_particle_create(flags, x, y, x_vel, y_vel, c_white, 1, 1)
var flags = phy_particle_data_flag_position | phy_particle_data_flag_velocity;
var buffer = buffer_create(16, buffer_grow, 4);
physics_particle_get_data_particle(part, buffer, flags);
px = buffer_read(buffer, buffer_f32);
py = buffer_read(buffer, buffer_f32);
pvelx = buffer_read(buffer, buffer_f32);
pvely = buffer_read(buffer, buffer_f32);
buffer_delete(buffer);

El código anterior obtiene la posición y la velocidad de la partícula indexada por la variable "parte" y almacena los datos en una serie de variables.
