Partículas suaves del cuerpo

GameMaker Studio 2 utiliza la biblioteca de física del cuerpo rígido Box2D, y como tal no permite la física del cuerpo blando. Sin embargo, la extensión Liquid Fun agrega partículas al sistema de física, lo que permite la creación de cuerpos blandos simulados utilizando la física rígida del cuerpo. Básicamente, puede crear una forma de partículas que mantenga la cohesión y así simular un cuerpo completo, que, debido a la interacción de las partículas, da la impresión de un cuerpo blando. También puede usar las partículas físicas para simular líquidos y otros sistemas físicos basados ​​en partículas.

Las siguientes secciones están todas relacionadas con el uso de partículas físicas para crear grioups (cuerpos blandos) y otros efectos:

Antes de que comiences a crear tus partículas de física necesitarás establecer las propiedades físicas básicas para ellas, como lo harías con un accesorio en la física regular. Sin embargo, ten en cuenta que establecer estas propiedades afectará a todas las partículas creadas previamente, así como a cualquier nueva partícula que crees, por lo que normalmente solo utilizarías estas funciones al inicio del juego o nivel. Sin embargo, hay funciones específicas disponibles para establecer las banderas de partículas para partículas individuales (y grupos de partículas) que cambiarán sus propiedades como "rebote" o "viscosidad", lo que le permite crear diferentes tipos de partículas dentro de los rangos de estos ajustes globales.

Las siguientes funciones están disponibles para configurar y obtener datos globales del sistema de partículas:

  • physics_particle_count
  • physics_particle_group_count
  • physics_particle_set_max_count
  • physics_particle_set_radius
  • physics_particle_set_density
  • physics_particle_set_damping
  • physics_particle_set_gravity_scale
  • physics_particle_get_max_count
  • physics_particle_get_radius
  • physics_particle_get_density
  • physics_particle_get_damping
  • physics_particle_get_gravity_scale

También tiene las siguientes funciones disponibles para establecer indicadores en partículas individuales (o grupos de partículas), así como para obtener información, como posición o color, de una sola partícula o un solo grupo:

  • physics_particle_set_flags
  • physics_particle_set_category_flags
  • physics_particle_set_group_flags
  • physics_particle_get_group_flags
  • physics_particle_group_get_inertia
  • physics_particle_group_get_centre_x
  • physics_particle_group_get_centre_y
  • physics_particle_group_get_vel_x
  • physics_particle_group_get_vel_y
  • physics_particle_group_get_ang_vel
  • physics_particle_group_get_x
  • physics_particle_group_get_y
  • physics_particle_group_get_angle
  • physics_particle_get_data
  • physics_particle_get_data_particle
  • physics_particle_group_get_data

Puede crear partículas de dos maneras: individualmente o en grupos. Las partículas individuales se comportarán exactamente según lo definido por la configuración global y las banderas que se establecieron cuando se crearon, pero los grupos de partículas pueden mostrar comportamientos diferentes según las banderas de su grupo. Estos indicadores de grupo se pueden establecer en diferentes valores para darle al grupo diferentes propiedades, como penetración y rigidez.

Las siguientes funciones están disponibles para crear partículas:

  • physics_particle_create
  • physics_particle_group_begin
  • physics_particle_group_circle
  • physics_particle_group_box
  • physics_particle_group_polygon
  • physics_particle_group_add_point
  • physics_particle_group_end
  • physics_particle_group_join
  • physics_particle_group_delete

No solo puedes crear partículas, sino que también puedes eliminarlas, ya sea individualmente o en una región. Las siguientes funciones están disponibles para eliminar partículas del sistema:

  • physical_particle_delete
  • physics_particle_delete_region_circle
  • physics_particle_delete_region_box
  • physics_particle_delete_region_poly

A menos que haya activado el dibujo de depuración usando physics_draw_debug(), no verá partículas en la pantalla cuando las cree. Por lo tanto, para ver realmente las partículas, debe dibujarlas en la pantalla del Evento de Draw de una instancia utilizando las siguientes funciones (tenga en cuenta que también puede dibujarlas usando el draw_sprite funciones si obtienes los datos para cada partícula usando la función physics_particle_get_data )

  • physics_particle_draw
  • physics_particle_draw_ext
