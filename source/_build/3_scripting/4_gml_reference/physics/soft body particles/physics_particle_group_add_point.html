physics_particle_group_add_point

Establezca los puntos de la forma del polígono que se usarán para crear un grupo de partículas del cuerpo blando.

Sintaxis:

physics_particle_group_add_point(x, y)


Argumento Descripción
x La posición x en la habitación para el punto.
y La posición y en la habitación para el punto.


Devoluciones: N / A


Descripción

Esta función establece un punto en la habitación para definir la forma de un polígono que se usará para crear un grupo de partículas de cuerpo blando. Debes haber indicado previamente a GameMaker Studio 2 que vas a definir una forma de polígono usando la función physics_particle_group_polygon () y luego usar esta función para definir los puntos individuales del polígono. Debe dar al menos tres puntos al definir la forma del polígono, pero puede otorgar hasta ocho, y la función permitirá la definición de polígonos cóncavos. Sin embargo, si genera un polígono con cavidades, los puntos dentro se ignorarán y se creará una forma convexa para el grupo de partículas.


Ejemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_polygon();
physics_particle_group_add_point(200, 200);
physics_particle_group_add_point(300, 300);
physics_particle_group_add_point(100, 300);
mLastGroup = physics_particle_group_end();

El código anterior almacena los indicadores para el tipo de partícula y las propiedades del grupo de partículas en variables y luego los usa para crear un grupo de partículas poligonales de tres lados en la posición del mouse.
