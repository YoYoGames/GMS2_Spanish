asset_get_index


Sintaxis:

asset_get_index(name);

Argumento Descripción
name El nombre del activo del juego para obtener el índice de (una cadena).


Devoluciones:

Real


Descripción

Puede usar esta función para obtener el índice de identificación exclusivo de un recurso del juego a partir de su nombre. Si no se encuentra el activo, la función devolverá un valor de -1, de lo contrario, devolverá el ID de índice único para el activo que se comprueba. Esta identificación se puede usar en otras funciones como lo haría con cualquier otro valor de índice, como sprite_index o el path_index, por ejemplo. Tenga en cuenta que aunque esta función se puede usar para referenciar recursos de cadenas (ver el ejemplo a continuación) siempre debe asegurarse de que el activo exista antes de usarlo, de lo contrario, puede recibir errores que bloquearán su juego.

NOTA: Los nombres de los scripts no se resolverán como activos en la plataforma de destino HTML5 debido a la ofuscación del código final, lo que puede causar problemas y debe tenerse en cuenta al usar la función.


Ejemplo:

var obj = asset_get_index("obj_Enemy_" + string(global.Level));
if obj > -1
   {
   instance_create_layer(random(room_width), random(room_height), obj, "Enemy_Layer");
   }

El código anterior obtendrá un índice de objeto de una cadena, y si ese índice existe, crea una instancia del objeto en el juego.
