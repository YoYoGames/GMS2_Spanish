mp_linear_path

Sintaxis:

mp_linear_path(path, xgoal, ygoal, stepsize, checkall);


Argumento Descripción
path El índice de la ruta a ser utilizada.
xgoal La posición x objetivo.
ygoal La posición y de destino.
stepsize La velocidad con la que la instancia se mueve en píxeles por paso.
checkall Si se deben verificar las colisiones con todas las instancias (verdadero) o solo con las sólidas (falso).


Devuelve: booleano.


Descripción

Esta función calcula una ruta en línea recta desde la posición actual de la instancia hasta el punto especificado por los valores "xgoal" y "ygoal". Utiliza el tamaño de paso indicado, el mismo que en la función mp_linear_step(). La ruta indicada ya debe existir y será sobrescrita por la nueva ruta y la función volverá si se encontró una ruta completa (verdadera) o no (falsa). Si se devuelve falso, se sigue generando una ruta, pero solo se ejecutará hasta la posición donde la ruta fue bloqueada.

Nota: Esta función no mueve la instancia. Establece una ruta solamente, y debe usar las funciones de ruta para el movimiento.


Ejemplo:

if mp_linear_path(path, obj_Player.x, obj_Player.y, 2, 0) path_start(path, 2, 0, 0);

Esto hace que el objeto compruebe y vea si hay una ruta lineal desde su posición actual a la posición de "obj_Player". Si hay, entonces comienza la ruta.
