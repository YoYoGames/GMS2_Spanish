mp_grid_get_cell

Sintaxis:

mp_grid_get_cell(id, x , y);


Argumento Descripción
id Índice del mp_grid que se va a utilizar
x1 La coordenada x de la cuadrícula para verificar.
y1 La coordenada y de la cuadrícula para verificar.


Devoluciones: Real


Descripción

Con esta función puede verificar cualquier celda dada de la mp_grid para ver si ha sido marcada como ocupada o no, dando el índice de la mp_grid (creada previamente) y las coordenadas x an y de la celda a verificar. Si se ha ocupado o la posición que se está verificando está fuera de los límites de la cuadrícula, la función devolverá -1, de lo contrario, devolverá 0.


Ejemplo:

if mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1
   {
   image_blend = c_red;
   }
else
   {
   image_blend = c_lime;
   }

El código anterior verificará la celda mp_grid que corresponde a la posición del mouse y si está ocupada establece el image_blend variable a rojo, y si no está ocupado, lo pone a verde.
