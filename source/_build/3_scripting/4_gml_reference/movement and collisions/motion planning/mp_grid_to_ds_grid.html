mp_grid_to_ds_grid

Sintaxis:

mp_grid_to_ds_grid(source, destination);


Argumento Descripción
source Índice del mp_grid que se va a utilizar
destination Índice del ds_grid que se usará para copiar los datos de la grilla.


Devuelve: booleano.


Descripción

Con esta función, puede copiar la cuadrícula MP dada en una cuadrícula DS. La grilla DS debe tener el mismo tamaño que la grilla MP, aunque no tiene que ser así (los datos se perderán si es más pequeño, y si es más grande, todas las celdas extra de la grilla serán 0). Las celdas de la cuadrícula DS contendrán el valor -1 si la celda de la cuadrícula MP fue marcada como ocupada, o será 0 si no lo está.


Ejemplo:

motion_grid = ds_grid_create(room_width / 32, room_height / 32); mp_grid_to_ds_grid(mp_grid, motion_grid);

El código anterior creará una nueva grilla DS y luego copiará los datos de la cuadrícula MP contenidos en la variable "mp_grid" en la nueva grilla DS.
