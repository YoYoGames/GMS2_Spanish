place_snapped

Sintaxis:

place_snapped(hsnap, vsnap);


Argumento Descripción
hsnap El ajuste horizontal para verificar.
vsnap El ajuste vertical para verificar


Devoluciones: Boolean


Descripción

Con esta función puede verificar y ver si el origen de una instancia (su posición xey) está alineado con una grilla con los valores hsnap y vsnap especificados por usted.


Ejemplo:

with (obj_Pieces)
   {
   if !place_snapped(32, 32)
      {
      move_snap(32, 32);
      }
   }

El código anterior comprueba todas las instancias de "obj_Pieces" para ver si están ajustadas a una cuadrícula de píxeles de 32x32, y si no lo están, los ajusta a la posición más cercana en esa cuadrícula.
