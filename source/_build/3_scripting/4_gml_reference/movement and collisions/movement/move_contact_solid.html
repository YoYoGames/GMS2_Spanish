move_contact_solid

Sintaxis:

move_contact_solid( dir, maxdist );


Argumento Descripción
dir La dirección para moverse adentro
maxdist La distancia máxima que puede recorrer el objeto (-1 o 0, un valor predeterminado de 1000 píxeles).


Devoluciones: N / A


Descripción

Esta función moverá la instancia que ejecuta el código un número determinado de píxeles en la dirección especificada hasta que se encuentre con una instancia con sólido marcado como verdadero. Puede usar -1 o 0 para que maxdist sea un 1000px predeterminado, es decir: GameMaker Studio 2 moverá la instancia continuamente hasta 1000 píxeles hasta que esté fuera de colisión.


Ejemplo:

if !place_meeting(x, y + 1, obj_Floor)
   {
   move_contact_solid(270, -1);
   }

El código anterior verificará debajo de la instancia una colisión con "obj_Floor", y si no hay ninguna, la moverá hacia abajo hasta que se produzca una colisión o se cubrirán 1000 píxeles.
