move_bounce_all

Sintaxis:

move_bounce_all( adv );


Argumento Descripción
adv Ya sea para habilitar el rebote avanzado (verdadero) o no (falso).


Devoluciones: N / A


Descripción

Con esta función, puede ordenar a una instancia que reboten todas las instancias dentro de la sala, con la única excepción de aquellas que no tienen ningún sprite o índice de máscara asignado. También puede indicarle que use una verificación de colisión precisa cuando esté habilitada, pero tenga en cuenta que esto requiere que todas las instancias tengan máscaras precisas habilitadas y ralentizará en gran medida su juego cuando haya muchas instancias involucradas debido a la cantidad de procesamiento que se debe realizar. Esto normalmente debe ir en el evento de paso de una instancia, pero también se puede usar de forma selectiva en un evento de colisión, como se ilustra en el siguiente ejemplo de código.


Ejemplo:

if other.visible
   {
   move_bounce_all(false);
   }

El código anterior se colocará en un evento de colisión con otro objeto. Primero verifica si el objeto es visible y si lo está, luego realiza el move_bounce_all() acción. Tenga en cuenta que en este caso el rebote es selectivo y solo se calculará para esta colisión, en lugar de para cada instancia, cada paso.
