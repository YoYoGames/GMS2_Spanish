move_random

Sintaxis:

move_random(hsnap, vsnap);


Argumento Descripción
hsnap El ajuste horizontal (el tamaño en píxeles entre 'células').
vsnap El ajuste vertical (el tamaño en píxeles entre 'células').


Devoluciones: N / A


Descripción

Con esta función puede establecer la instancia en una posición en cualquier lugar de la sala, pero alineada a una cuadrícula "invisible". Por lo tanto, un valor de 32 para hsnap y vsnap establecerá la instancia en una posición aleatoria que esté alineada con una cuadrícula de 32x32 cuadrados. puede establecer estos valores en 1 para obtener una posición en cualquier lugar de la sala.

NOTA: Esta función se ajustará a las mismas posiciones cada vez que se vuelva a ejecutar el juego debido al hecho de que GameMaker Studio 2 genera la misma semilla aleatoria inicial cada vez para hacer que el código de depuración sea una tarea mucho más sencilla. Para evitar este comportamiento, use randomise al comienzo de tu juego Esto solo es cierto al probar y depurar el juego, ya que el paquete ejecutable final no mostrará este comportamiento y será aleatorio en cada jugada.


Ejemplo:

if keyboard_check(vk_space) move_random(1,1);

Esto moverá la instancia a una posición aleatoria en cualquier lugar de la sala cuando se presione la tecla espacio.
