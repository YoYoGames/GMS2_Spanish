move_outside_solid

Sintaxis:

move_outside_solid(dir, maxdist);


Argumento Descripción
dir La dirección para moverse adentro
maxdist La distancia máxima que puede recorrer el objeto (-1 o 0, un valor predeterminado de 1000 píxeles).


Devoluciones: N / A


Descripción

Con esta función, puede indicarle a una instancia que salga de una colisión con cualquier instancia marcada como sólida en cualquier dirección y cualquier número de píxeles en cada paso, con un valor de -1 o 0 para que maxdist sea un valor predeterminado de 1000 px, es decir: GameMaker Studio 2 moverá la instancia continuamente hasta 1000 píxeles hasta que esté fuera de colisión.


Ejemplo:

if other.solid
   {
   var pdir;
   pdir = point_direction(other.x, other.y, x, y);
   move_outside_solid(pdir, -1);
   }

El código anterior entraría en un evento de colisión y verificará si la "otra" instancia en la colisión está marcada como "sólida". Si lo es, moverá la instancia fuera de colisión.
