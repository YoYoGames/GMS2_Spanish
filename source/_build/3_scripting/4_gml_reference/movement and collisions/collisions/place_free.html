place_free


Descripción

Puede usar esta función para verificar y ver si la instancia de llamada chocaría con cualquier instancia marcada como sólida en su juego. Ahora, debe tenerse en cuenta que para que esto funcione, la instancia que ejecuta el código debe tener una máscara de colisión válida (ya sea para el sprite en sí o a través de mask_index ) y solo registrará colisiones con aquellas instancias marcadas sólidas que también tienen una máscara válida.

La función en sí misma básicamente funciona tomando la instancia y probando las colisiones con solo las instancias marcadas sólidas cuando se coloca en la posición especificada por los argumentos x / y. La comprobación de colisión puede ser precisa o basarse en el cuadro delimitador de la instancia, según el tipo de máscara de colisión que se haya seleccionado.

Tenga en cuenta que las coordenadas x / y proporcionadas se desplazarán al entero más cercano antes de que se realice la comprobación.


Sintaxis:

place_free(x, y);

Argumento Descripción
x La posición x para comprobar.
y La posición y para comprobar.


Devoluciones:

Boolean


Ejemplo:

if place_free(mouse_x, mouse_y)
   {
   x = mouse_x;
   y = mouse_y;
   }

El código anterior verificará una colisión con cualquier instancia marcada sólida si la instancia de llamada se colocara en la misma posición que el mouse. Si no se detecta una colisión, entonces la instancia tiene sus coordenadas x / y establecidas en las del mouse.
