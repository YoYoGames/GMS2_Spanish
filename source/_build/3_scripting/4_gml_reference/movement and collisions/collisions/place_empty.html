place_empty

Sintaxis:

place_empty(x, y);


Argumento Descripción
x La posición x para verificar.
y La posición y para verificar.


Devuelve: booleano.


Descripción

Puede usar esta función para verificar y ver si la instancia de llamada colisionaría con cualquier otra instancia en su juego. Ahora, se debe tener en cuenta que, para que esto funcione, la instancia que ejecuta el código debe tener una máscara de colisión válida (ya sea para el sprite mismo o a través del mask_index ) y solo registrará colisiones con aquellas instancias que también tengan una máscara válida.

La función en sí funciona básicamente tomando la instancia y probando colisiones cuando se coloca en la posición especificada por los argumentos x / y. La verificación de colisión puede ser precisa o basada en el cuadro delimitador de la instancia, dependiendo del tipo de máscara de colisión que se haya seleccionado.

Tenga en cuenta que las coordenadas x / y dadas se anotarán al número entero más cercano antes de que se realice la verificación.


Ejemplo:

if place_empty(mouse_x, mouse_y)
   {
   x = mouse_x;
   y = mouse_y;
   }

El código anterior comprobará si hay una colisión con cualquier otra instancia si la instancia de llamada se colocara en la misma posición que el mouse. Si no se detecta una colisión, la instancia tiene sus coordenadas x / y establecidas a las del mouse.
