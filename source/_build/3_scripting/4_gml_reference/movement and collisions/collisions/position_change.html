cambio de posición


Descripción

Esta función verificará una posición para una colisión con cualquier instancia en el punto dado, y si hay una, cambiará todas las instancias en colisión para que sean instancias del objeto elegido. Puede establecer el argumento "perf" en true lo que obligará a GameMaker Studio 2 a realizar los eventos Destroy and Clean Up de la instancia encontrada y el evento Create de la nueva instancia, o false Para no realizar ninguno de estos eventos. Tenga en cuenta que si elige no realizar los eventos Destruir, Limpiar y Crear, cualquier instancia creada que use una variable normalmente definida en el evento crear bloqueará el juego ya que esa variable ya no existirá.


Sintaxis:

position_change(x, y, obj, perf);

Argumento Descripción
x La coordenada x de dónde cambiar las instancias en colisión.
y La coordenada y de dónde cambiar las instancias en colisión.
obj El nuevo objeto en el que el objeto llamante cambiará a.
perf Ya sea para realizar el evento Crear de ese nuevo objeto (verdadero) o no (falso).


Devoluciones:

N/A


Ejemplo:

position_change(200, 200, obj_Bird, true);

Esto cambiará todas las instancias que colisionen en el punto (200,200) en una instancia de "obj_Bird", realizando el evento Create de "obj_Bird" para cada una de ellas en el proceso.
