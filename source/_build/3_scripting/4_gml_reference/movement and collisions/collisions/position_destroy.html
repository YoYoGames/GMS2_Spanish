position_destroy


Descripción

Esta función simplemente destruye todas las instancias que se encuentran en colisión con la posición especificada. Las colisiones se basan en la máscara de las instancias, y si alguna parte de la máscara se sobrepasa con el objetivo, la función destruirá esa instancia. Las instancias destruidas de esta manera activarán sus eventos Destroy and Clean Up.


Sintaxis:

position_destroy(x, y);

Argumento Descripción
x La coordenada x de dónde destruir las instancias en colisión.
y La coordenada y de dónde destruir las instancias en colisión.


Devoluciones:

N/A


Ejemplo:

if mouse_check_button_pressed(mb_left)
   {
   position_destroy(mouse_x, mouse_y);
   }

Esto destruirá todas las instancias que chocan con la posición del cursor del mouse cuando se presiona el botón izquierdo del mouse.
