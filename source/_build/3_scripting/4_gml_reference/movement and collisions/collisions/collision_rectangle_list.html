collision_rectangle_list


Descripción

Esta función es la misma que la collision_rectangle() función, solo en lugar de solo detectar una instancia en colisión a la vez, detectará múltiples instancias. Usted suministra las posiciones x / y de la parte superior izquierda y la parte inferior derecha del área para verificar el objeto que se debe verificar, y puede establecer que la verificación sea precisa (en cuyo caso, todas las instancias que se verifiquen deberán tener una colisión precisa). máscaras) y si la comprobación debe incluir la instancia de llamada o no. También proporciona una lista de DS que se rellenará con el único id valores de todas las instancias del objeto que se considera que están en colisión con la instancia que llama, y ​​tiene la opción de ordenar la lista según la distancia desde el centro del área rectangular hasta el origen de las instancias que se encuentran en colisión. Tenga en cuenta que la función también acepta la palabra clave especial all, en cuyo caso se listarán todas las instancias encontradas en colisión. La función devuelve el número de instancias encontradas, o 0 si no se encuentra ninguna.


Sintaxis:

collision_rectangle_list(x1, y1, x2, y2, obj, prec, notme, list, ordered);

Argumento Descripción
x1 La coordenada x del lado izquierdo del rectángulo a verificar.
y1 La coordenada y del lado superior del rectángulo a verificar.
x2 La coordenada x del lado derecho del rectángulo a verificar.
y2 La coordenada y del lado inferior del rectángulo a verificar.
obj El objeto para comprobar las colisiones de instancia.
prec Si la verificación se basa en colisiones precisas ( true, que es más lento) o su caja delimitadora en general ( false, Más rápido).
notme Si la instancia que llama, si es relevante, debe ser excluida ( true ) o no ( false ).
list La lista de DS que se utilizará para almacenar los ID de las instancias en colisión.
ordered Si la lista debe ser ordenada por distancia ( true ) o no ( false ).


Devoluciones:

Int (The number of instances found to be in collision)


Ejemplo:

var _list = ds_list_create();
var _num = collision_rectangle_list(x - 100, y - 100, x + 100, y + 100, obj_Enemy, false, true, _list, false);
if _num > 0
    {
    for (var i = 0; i < _num; ++i;)
        {
        instance_destroy(_list[| i]);
        }
    }
ds_list_destroy(_list);

El código anterior comprobará un área rectangular de 100 píxeles alrededor de la posición de la instancia de llamada para detectar colisiones con las instancias de "obj_Enemy". Si hay colisiones, la lista creada previamente se repite y se destruye cada instancia que estuvo en la colisión.
