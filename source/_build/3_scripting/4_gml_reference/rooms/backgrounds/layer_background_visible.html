layer_background_visible


Descripción

Al usar esta función, puede alternar la visibilidad de los elementos de fondo. Proporciona la ID del elemento de fondo (que obtiene cuando crea un elemento de fondo usando layer_background_create() o cuando usas la función layer_background_get_id() ), y luego establecer el argumento visible para true o false. Cuando se establece en true el elemento se mostrará, y cuando se establece en false, el elemento no se mostrará. Tenga en cuenta que esto depende de la visibilidad de la capa, e incluso si el elemento de fondo está marcado como visible, no se dibujará si la capa en la que se encuentra está marcada como no visible.


Sintaxis:

layer_background_visible(background_element_id, visible)


Argumentos:

Argumento Descripción
background_element_id El valor de ID único del elemento de fondo para cambiar
visible La palanca de visibilidad, que puede ser true o false


Devoluciones:

N/A


Ejemplo:

if mouse_check_button_pressed(mb_left)
   {
   var back = layer_background_get_id(layer);
   if layer_background_get_visible(back)
      {
      layer_background_visible(back, false);
      }
   else
      {
      layer_background_visible(back, true);
      }
   }

El código anterior comprueba la pulsación de un botón del mouse y, si se detecta uno, alternará la visibilidad de fondo del elemento de fondo asignado a la capa en la que está activa la instancia que ejecuta el código.
