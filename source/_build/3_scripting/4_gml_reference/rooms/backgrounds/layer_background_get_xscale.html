layer_background_get_xscale


Descripción

Esta función se puede usar para obtener el valor del multiplicador de escala actual del elemento de fondo. Le das la ID del elemento de fondo (que obtienes cuando creas un elemento de fondo usando layer_background_create() o cuando usas la función layer_background_get_id() ), y la función devolverá el valor real que representa el multiplicador de escala que se usa para dibujar el sprite. El valor predeterminado es 1.


Sintaxis:

layer_background_get_xscale(background_element_id);


Argumentos:

Argumento Descripción
background_element_id El valor de ID único del elemento de fondo para obtener la información de


Devoluciones:

Real


Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_xscale(back_id) != -1
   {
   layer_background_xscale(back_id, -1);
   }

El código anterior obtendrá la ID de capa para la capa denominada "Background_sky" y luego la usará para obtener la ID del elemento de fondo en esa capa. Esta ID se usa para verificar la escala x para el elemento y si no es -1, se establece en -1.
