layer_background_create


Descripción

Con esta función puedes asignar un recurso de sprite a una capa para usar como fondo en tu proyecto. Usted proporciona la identificación de capa (que obtiene cuando crea la capa usando layer_create() ) o el nombre de la capa (como cadena - esto tendrá un impacto en el rendimiento) y un índice de sprites (que sería el nombre del sprite como se muestra en el árbol de recursos), y se agregará a la capa. La función devuelve el valor único de ID para el elemento, que luego se puede usar en otras funciones de capa para fondos.


Sintaxis:

layer_background_create(layer_id, sprite)


Argumentos:

Argumento Descripción
layer_id El valor de ID exclusivo de la capa al objetivo (o el nombre de la capa como una cadena)
sprite El índice de sprites que se utilizará


Devoluciones:

Background element ID (Real)


Ejemplo:

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

El código anterior crea una nueva capa y luego le agrega un nuevo elemento de fondo, configurando un sprite para que sea la imagen de fondo utilizada.
