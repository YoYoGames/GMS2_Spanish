layer_background_get_sprite


Descripción

Esta función se puede usar para obtener el valor actual del índice de sprites del elemento de fondo. Le das la ID del elemento de fondo (que obtienes cuando creas un elemento de fondo usando layer_background_create() o cuando usas la función layer_background_get_id() ), y la función devolverá un valor real que representa el índice de sprites que se muestra. Si el elemento no tiene un sprite asignado, la función devolverá -1.


Sintaxis:

layer_background_get_sprite(background_element_id);


Argumentos:

Argumento Descripción
background_element_id El valor de ID único del elemento de fondo para obtener la información de


Devoluciones:

Sprite Index (Real) or -1


Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
   {
   layer_background_sprite(back_id, spr_Clouds);
   }

El código anterior obtendrá la ID de capa para la capa denominada "Background_sky" y luego la usará para obtener la ID del elemento de fondo en esa capa. Esta ID se usa para verificar el sprite asignado al elemento, configurándolo en el sprite "spr_Clouds" si no lo está.
