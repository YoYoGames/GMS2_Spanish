layer_background_get_id


Descripción

Esta función se puede usar para recuperar el valor único de ID del elemento de fondo en una capa. Usted proporciona la identificación de capa (que obtiene cuando usa el nombre de la capa junto con layer_get_id() ) y la función devolverá el valor de ID asociado con el elemento de fondo en la capa. Tenga en cuenta que esta función está específicamente diseñada para usarse con fondos que se agregaron en el IDE, como si agregara un fondo a una capa a través del código usando la función layer_background_create, luego devolverá la identificación única para el elemento de fondo agregado.


Sintaxis:

layer_background_get_id(layer_id)


Argumentos:

Argumento Descripción
layer_id El valor de ID único de la capa para orientar


Devoluciones:

Background element ID value (Real)


Ejemplo:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_sprite(back_id, bck_Trees_Winter);

El código anterior obtendrá la ID de capa para la capa denominada "Background_trees" y luego la usará para obtener la ID del elemento de fondo en esa capa. Esta identificación se usa para cambiar el elemento sprite.
