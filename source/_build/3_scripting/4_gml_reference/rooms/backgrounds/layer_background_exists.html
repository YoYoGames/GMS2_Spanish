layer_background_exists


Descripción

Puede usar esta función para verificar y ver si un elemento de fondo existe en cualquier capa dada. Usted proporciona la identificación de capa (que obtiene cuando crea la capa usando layer_create() ) o el nombre de la capa (como cadena; esto tendrá un impacto en el rendimiento) y la ID del elemento de fondo (que se obtiene al crear un elemento de fondo usando layer_background_create() o cuando usas la función layer_background_get_id() ) y la función regresará true si el elemento existe, o false si no es así

NOTA: Esta función funciona dentro del alcance de la sala de destino actual, de forma predeterminada, la habitación en la que se llama a la función, que se puede configurar con la función layer_set_target_room().


Sintaxis:

layer_background_exists(layer_id, background_element_id)


Argumentos:

Argumento Descripción
layer_id El valor de ID exclusivo de la capa al objetivo (o el nombre de la capa como una cadena)
background_element_id El valor de ID único del elemento de fondo para verificar


Devoluciones:

Boolean


Ejemplo:

var lay_id = layer_get_id("Background_trees");
if layer_background_exists(lay_id, global.TreesBackground)
   {
   layer_background_destroy(lay_id, global.TreesBackground);
   }

El código anterior comprueba la capa "Background_trees" para ver si el elemento de fondo dado existe y si lo hace, entonces se destruye (pero no la capa).
