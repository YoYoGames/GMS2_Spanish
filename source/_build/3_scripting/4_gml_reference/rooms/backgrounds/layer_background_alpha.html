layer_background_alpha


Descripción

Esta función controla el alfa (transparencia) del sprite de fondo. Le das la ID del elemento de fondo (que obtienes cuando creas un elemento de fondo usando layer_background_create() o cuando usas la función layer_background_get_id() ), y luego establecer el valor alfa para usar. Alpha puede estar entre 0 (totalmente transparente) y 1 (totalmente opaco) con el valor alfa predeterminado para el elemento de fondo de 1. Tenga en cuenta que si la capa a la que se ha asignado el elemento de fondo no está visible, o el elemento se ha creado invisible: no verá ninguna diferencia con esta función hasta que la capa o elemento se haya vuelto visible nuevamente.


Sintaxis:

layer_background_alpha(background_element_id, alpha);


Argumentos:

Argumento Descripción
background_element_id El valor de ID único del elemento de fondo para cambiar
alpha El alfa para sprite de fondo, de 0 a 1 (por defecto es 1)


Devoluciones:

N/A


Ejemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_alpha(back_id, random(1));

El código anterior obtiene el valor ID del fondo asignado a la capa "Background_sky" y luego establece su alfa en un valor aleatorio entre 0 y 1.
