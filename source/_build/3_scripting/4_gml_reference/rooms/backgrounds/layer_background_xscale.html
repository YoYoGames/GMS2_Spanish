layer_background_xscale


Descripción

Esta función se puede usar para establecer la escala a lo largo del eje x de un elemento de fondo. Le das la ID del elemento de fondo (que obtienes cuando creas un elemento de fondo usando layer_background_create() o cuando usas la función layer_background_get_id() ), y luego establecer la velocidad de la animación. El valor de la escala que da es un multiplicador que se utilizará para cambiar la forma en que se muestra el elemento de fondo, donde un valor de 0.5 mostraría el elemento a la mitad de la escala, y un valor de 2 se mostraría en la escala doble. Tenga en cuenta que los valores negativos son válidos, y "volteará" el elemento alrededor de la posición (0,0), por lo que una escala x de -1 mostraría la imagen invertida.


Sintaxis:

layer_background_xscale(background_element_id, xscale);


Argumentos:

Argumento Descripción
background_element_id El valor de ID único del elemento de fondo para cambiar
xscale El valor de escala a usar (1 no es escalar)


Devoluciones:

N/A


Ejemplo:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_xscale(back_id, -1);

El código anterior obtendrá la identificación de capa para la capa denominada "Agua de fondo" y luego la usará para obtener la ID del elemento de fondo en esa capa. Esta ID luego se usa para cambiar el elemento xscale.
