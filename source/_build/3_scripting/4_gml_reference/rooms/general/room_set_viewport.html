room_set_viewport


Descripción

Con esta función puede establecer las propiedades del puerto de visualización para cualquier habitación de su juego, excepto la actual. Proporciona el índice de habitación para establecer el puerto de vista, el índice de puerto de vista (de 0 a 7) y luego si el puerto de vista debe estar visible al inicio de la sala o no (configurado en true hacer el puerto visible, y false de otra manera). Después de eso, establece la coordenada xey (correspondiente a la posición superior izquierda del puerto de visualización) y luego el ancho y alto en píxeles.


Sintaxis:

room_set_viewport(rm, vind, vis, xport, yport, wport, hport);


Argumentos:

Argumento Descripción
rm El índice de la habitación para establecer
vind El índice del puerto de vista para establecer
vis La visibilidad del puerto de visualización ( true es visible, false is invisible )
xport La posición x para el puerto de vista en la sala
yport La posición y del puerto de visualización en la sala
wport El ancho (en píxeles) del puerto de visualización
hport La altura (en píxeles) del puerto de visualización


Devoluciones:

N/A


Ejemplo:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_viewport(global.myroom, 0, true, 0, 0, 640, 480);

Esto establecerá las propiedades del puerto de visualización [0] en la sala indexada en la variable "global.myroom".
