room_set_persistent


Descripción

Con esta función puedes cambiar (o configurar) la persistencia de cualquier habitación de tu juego, excepto la actual. Una sala con persistencia marcada como true mantendrá el estado de todas las instancias dentro de esa habitación si el jugador se va y luego regresa, mientras que si la persistencia está marcada como false se restablecerá al estado inicial todo el tiempo. Debe tener en cuenta que una habitación persistente utiliza mucha más memoria que una habitación normal y no se recomienda tener demasiadas en su juego.

NOTA: ¡ Esta función NO funcionará para desactivar la persistencia si la sala ya ha sido visitada! Una habitación persistente, una vez visitada, se guarda en la memoria y para desactivar la persistencia debe ir a esa habitación y configurar el room_persistent variable a false y luego salir de la habitación de nuevo.



Sintaxis:

room_set_persistent(index, val);


Argumentos:

Argumento Descripción
index El índice de la habitación para establecer la persistencia de.
val Si la habitación debe ser persistente ( true ) o no ( false )


Devoluciones:

N/A


Ejemplo:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, true);

Esto creará una nueva sala y almacenará su índice en la variable "global.myroom". Luego establecerá su ancho a 640 píxeles, su altura a 480 píxeles, su título a 'Sala de juegos' y su persistencia a 'verdadero'.
