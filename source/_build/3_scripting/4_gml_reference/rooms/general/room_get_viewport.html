room_get_viewport


Descripción

Con esta función puede recuperar los detalles de un puerto de visualización en una sala distinta a la actual. Usted proporciona la identificación de la sala y el índice del puerto de vista para recuperar (de 0 a 7) y la función devolverá una matriz de 5 índices, donde:

  • [0] = visible (Booleano: true / false )
  • [1] = posición x (real)
  • [2] = posición y (real)
  • [3] = ancho (real)
  • [4] = altura (real)


Sintaxis:

room_get_viewport(rm, vind);


Argumentos:

Argumento Descripción
rm El índice de la habitación para obtener los datos de la vista de
vind El índice del puerto de vista para obtener


Devoluciones:

Array (5 elements: visible, x, y, width, height)


Ejemplo:

v_vals = room_get_viewport(rm_Game, 0);
if v_vals[0] == false
    {
    room_set_view(rm_Game, true, v_vals[1], v_vals[2], v_vals[3], v_vals[4]);
    }

El código anterior recupera los datos del puerto de visualización para la sala determinada y luego verifica si el puerto está marcado como visible. Si no es así, los datos del puerto de visualización están configurados para hacerlo visible.
