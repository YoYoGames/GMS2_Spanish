room_persistent


Descripción

Esta variable se puede usar para obtener y establecer el indicador persistente para la sala actual. Si está configurado para true la habitación se considera persistente, en cuyo caso, cada vez que salga de la habitación y regrese, se mantendrá el estado de las instancias dentro de esa habitación. Sin embargo, si está marcado como false, cada vez que regrese a la sala, se restablecerá a su estado inicial. Debe tener en cuenta que una habitación persistente utiliza mucha más memoria que una habitación normal y no se recomienda tener demasiadas en su juego.


Sintaxis:

room_persistent;


Devoluciones:

Boolean


Ejemplo:

if lives < 1
   {
   room_persistent = false;
   room_goto(rm_start);
   }

El código anterior verifica la variable "vidas" y si es menor que 1, configurará la persistencia de la sala para false y luego cambiar habitaciones
