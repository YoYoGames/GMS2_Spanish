room_goto


Descripción

Esta función le permite ir a cualquier sala de su proyecto de juego, ya sea que se haya creado utilizando el código o en el árbol de recursos. Proporciona el valor único de ID de sala (almacenado en la variable para el nombre de la sala o como una variable devuelta por la función) room_add. Tenga en cuenta que llamar a esta función no cambia instantáneamente las habitaciones, y la habitación no cambiará hasta el final del cuadro de juego actual (lo que significa que cualquier código después de esta función se ejecutará, como lo harán algunos eventos). Esta función también activará el evento Room End.


Sintaxis:

room_goto(index);


Argumentos:

Argumento Descripción
index El índice de la sala a la que ir.


Devoluciones:

N/A


Ejemplo:

switch (global.rm)
   {
   case 0: room_goto(rm_level1); break;
   case 1: room_goto(rm_level2); break;
   case 2: room_goto(rm_level3); break;
   }

El código anterior verificará una variable global y cambiará la habitación según el valor que contenga.
