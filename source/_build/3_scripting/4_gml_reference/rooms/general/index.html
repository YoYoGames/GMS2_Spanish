Funciones generales de la sala y variables

Las salas en GameMaker Studio 2 son donde todo sucede en tu juego y debes tener al menos una sala en cualquier juego para que se ejecute, pero es posible que necesites muchas más, dependiendo del tipo de juego y de la estructura de tu proyecto. Esta sección tiene todas las funciones generales de GML y las variables relacionadas con las salas, y obtiene información sobre ellas, así como para establecer ciertas propiedades dentro de ellas. Se debe tener en cuenta que las funciones relacionadas con las propiedades de los vestuarios nunca deben ejecutarse desde la habitación que desea cambiar, sino que deben ejecutarse primero desde una instancia en otra sala.

Las siguientes variables globales se utilizan para obtener información básica sobre cualquier habitación dada:

  • room_exists
  • room_first
  • room_last
  • room_next
  • room_previous
  • habitación
  • room_speed
  • room_height
  • room_width
  • room_persistent
También puede usar la siguiente función para obtener el nombre de la sala, tal como aparece en el árbol de recursos:

  • room_get_name

Las siguientes funciones se usan para moverse entre habitaciones:

  • room_goto
  • room_goto_next
  • room_goto_previous
  • room_restart

NOTA: Estas funciones no se pueden usar con la sala actual; de lo contrario, pueden ocurrir errores y comportamientos impredecibles.

  • room_add
  • room_duplicate
  • room_assign
  • room_instance_add
  • room_instance_clear
  • room_set_background_colour
  • room_set_height
  • room_set_width
  • room_set_persistent
  • room_set_view_enabled
  • room_set_viewport
  • room_get_viewport
  • room_set_camera
  • room_get_camera
