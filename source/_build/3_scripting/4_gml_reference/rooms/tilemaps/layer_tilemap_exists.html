layer_tilemap_exists

Descripción

Puede usar esta función para verificar y ver si un elemento de mapa de azulejos existe en cualquier capa dada. Usted proporciona la identificación de capa (que obtiene cuando crea la capa usando layer_create() ) o el nombre de la capa (como una cadena; esto tendrá un impacto en el rendimiento) y la ID del elemento de mapa de mosaico (que se obtiene cuando se crea un elemento de mosaico utilizando layer_tilemap_create() o cuando usas la función layer_tilemap_get_id() ) y la función regresará true si el elemento existe, o false si no es así

NOTA: Esta función funciona dentro del alcance de la sala de destino actual, de forma predeterminada, la habitación en la que se llama a la función, que se puede configurar con la función layer_set_target_room().


Sintaxis:

layer_tilemap_exists(layer_id, tilemap_element_id)


Argumentos:

Argumento Descripción
layer_id El valor de ID exclusivo de la capa al objetivo (o el nombre de la capa como una cadena)
tilemap_element_id El valor ID único del elemento tilemap para verificar


Devoluciones:

Boolean


Ejemplo:

var lay_id = layer_get_id("tilemap_trees");
if layer_tilemap_exists(lay_id, global.Treestilemap)
   {
   layer_tilemap_destroy(lay_id, global.Treestilemap);
   }

El código anterior comprueba la capa "tilemap_trees" para ver si el elemento tilemap dado existe y, si lo hace, se destruye (pero no la capa).
