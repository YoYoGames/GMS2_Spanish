layer_tilemap_create

Descripción

Con esta función, puede asignar un recurso de conjunto de mosaicos a una capa que se utilizará como mapa de página en su proyecto. Usted proporciona el ID de la capa (que obtiene cuando crea la capa usando layer_create() ) o el nombre de la capa (como una cadena; esto tendrá un impacto en el rendimiento) y luego una posición inicial (x, y) para agregar el tilemap a la sala, el recurso de conjunto de azulejos que se usará, y luego el ancho y la altura de la mapa de til en celdas (es decir, un ancho de 20 y una altura de 10 crearán un mapa de til con 200 celdas de 20 celdas de ancho y 10 celdas de alto), con el tamaño de las celdas definidas por el conjunto de mosaicos elegido. Vale la pena señalar que no puede colocar mosaicos en posiciones negativas dentro del mapa de til, por lo que todos los mosaicos deben colocarse dentro de los espacios de celda 0 a ancho, 0 a altura.


Sintaxis:

layer_tilemap_create(layer_id, x, y, tile set, width, height)


Argumentos:

Argumento Descripción
layer_id El valor de ID único de la capa a destino (o el nombre de la capa como una cadena)
x La posición x del tilemap en la sala.
y La posición y del tilemap en la sala.
tile set El índice de conjunto de azulejos que se utilizará
width El ancho tilemap (en celdas)
height La altura del tilemap (en celdas)


Devoluciones:

tilemap element ID (Real)


Ejemplo:

global.back_layer = layer_create(10000);
global.back_tiles = layer_tilemap_create(global.back_layer, 0, 0, tmap_Trees, 16, 32);

El código anterior crea una nueva capa y luego le agrega un nuevo elemento tilemap, estableciendo la posición del tilemap en (0,0), así como el conjunto de mosaicos que se utilizará y el ancho y la altura del tilemap.
