layer_tilemap_create

Descripción

Con esta función puede asignar un recurso de sprite a una capa para usar como un mapa de mosaico en su proyecto. Usted proporciona la identificación de capa (que obtiene cuando crea la capa usando layer_create() ) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y luego una posición inicial (x, y) para agregar el mapa de azulejos a la habitación, el recurso de establecer teselas a usar, y luego el ancho y alto del tilemap en las celdas (es decir, un ancho de 20 y una altura de 10 creará un mapa de mosaico con 200 celdas de 20 celdas de ancho y 10 celdas de altura), con el tamaño de las celdas definidas por el conjunto de teselas elegido. Vale la pena señalar que no puede colocar los mosaicos en posiciones negativas dentro del mapa de mosaicos, por lo que todos los mosaicos deben colocarse dentro de los espacios de celdas 0 al ancho, 0 a la altura.


Sintaxis:

layer_tilemap_create(layer_id, x, y, tile set, width, height)


Argumentos:

Argumento Descripción
layer_id El valor de ID exclusivo de la capa al objetivo (o el nombre de la capa como una cadena)
x La posición x del mapa de mosaicos en la habitación
y La posición y del mapa de mosaicos en la sala
tile set El índice del conjunto de mosaicos que se utilizará
width El ancho tilemap (en celdas)
height La altura del mapa de mosaico (en celdas)


Devoluciones:

tilemap element ID (Real)


Ejemplo:

global.back_layer = layer_create(10000);
global.back_tiles = layer_tilemap_create(global.back_layer, 0, 0, tmap_Trees, 16, 32);

El código anterior crea una nueva capa y luego agrega un nuevo elemento de mapa de azulejos, estableciendo la posición del mapa de mosaicos en (0,0), así como el conjunto de mosaicos que se utilizarán y el ancho y alto del mapa de mosaicos.
