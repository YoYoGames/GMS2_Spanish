tile_get_index

Descripción

Esta función se puede utilizar para obtener el índice de mosaico (la posición del mosaico dentro de la imagen del conjunto de mosaicos) de un conjunto de tiledata. Usted da el tiledata, que puede ser recuperado usando la función tilemap_get, y la función devolverá un valor entero para el índice o -1 si hay un error.


Sintaxis:

tile_get_index(tiledata)


Argumentos:

Argumento Descripción
tiledata el tiledata para comprobar


Devoluciones:

Integer


Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var ind = tile_get_index(data);
data = tile_set_index(data, irandom(23));
tilemap_set(map_id, data, mx, my);

El código anterior obtiene el ID de mapa de til de la capa dada y luego lo usa para obtener el tiledata para la celda debajo de la posición del mouse. Estos datos se utilizan para establecer el índice de mosaico en un número aleatorio.
