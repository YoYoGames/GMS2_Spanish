layer_tilemap_destroy

Descripción

Esta función destruirá el elemento de mosaico dado. Usted proporciona la identificación de mosaico (que obtiene cuando crea el mapa de mosaico usando layer_tilemap_create() o cuando usa la identificación de la capa junto con layer_get_tilemap_id() ) y esto lo eliminará. Tenga en cuenta que esto no elimina la capa, solo el mapa de azulejos de ella, y si el mapa de mosaicos es uno que se ha agregado en el editor de sala, la próxima vez que salga de la habitación y luego regrese, el mapa de mosaico se volverá a crear. Sin embargo, si la sala es persistente, el mapa de mosaico se eliminará a menos que la persistencia de la sala se apague de nuevo.


Sintaxis:

layer_tilemap_destroy(tilemap_element_id)


Argumentos:

Argumento Descripción
tilemap_element_id El valor de ID único del mapa de mosaico que se destruirá


Devoluciones:

N/A


Ejemplo:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if layer_tilemap_exists(lay_id, tile_id)
   {
   layer_tilemap_destroy(tile_id);
   }

El código anterior comprueba la capa "Tiles_trees" para ver si el elemento tilemap dado existe y, si lo hace, se destruye (pero no la capa).
