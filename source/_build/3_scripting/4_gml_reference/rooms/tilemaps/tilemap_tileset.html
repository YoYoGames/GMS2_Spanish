tilemap_tileset

Descripción

Al usar esta función, puede cambiar el recurso de conjunto de mosaicos asignado a un elemento de mosaico determinado en una capa. Le das el ID del elemento de mosaico (que obtienes cuando creas un elemento de mosaico usando layer_tilemap_create() o cuando usas la función layer_tilemap_get_id() ), y luego suministrar un índice de conjunto de mosaicos y al mapa de mosaicos se le dará el nuevo sprite.


Sintaxis:

tilemap_tileset(tilemap_element_id, tileset_index)


Argumentos:

Argumento Descripción
tilemap_element_id El valor ID único del elemento tilemap para cambiar
tileset_index El nuevo índice de conjunto de mosaicos para usar


Devoluciones:

N/A


Ejemplo:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if tilemap_get_tileset(tile_id) != ts_Nighttime
   {
   tilemap_tileset(tile_id, ts_Nighttime);
   }

El código anterior comprueba el conjunto de mosaicos actual asignado al mapa de tiempo en la capa "Tiles_trees" y si no es "ts_Nighttime", ese conjunto de mosaicos se asigna al mapa de mosaicos.
