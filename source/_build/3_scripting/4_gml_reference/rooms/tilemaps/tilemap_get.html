tilemap_get


Descripción

Usando esta función puedes recuperar los datos del mosaico de una celda del elemento tilemap. Le das el ID del elemento de mosaico (que obtienes cuando creas un elemento de mosaico usando layer_tilemap_create() o cuando usas la función layer_tilemap_get_id() ), así como la posición xey de la celda y la función devolverá los datos del mosaico "blob". Estos datos son esencialmente una máscara de bits que contiene el índice de teselas, los elementos booleanos volteados / rotados / espejo y cualquier información de máscara que se haya aplicado (ver tilemap_set_mask para obtener más detalles), y el valor de datos resultante se puede usar en las funciones de mosaico para cambiar las propiedades de un mosaico. Tenga en cuenta que estamos utilizando posiciones de celda x / y basadas en el tamaño de la cuadrícula de mosaico y no como una posición en la habitación, por lo que "cell_x" es un valor de 0 a ancho de mosaico, y "cell_y" es un valor de 0 a altura del mapa de azulejos Si necesita obtener los datos de un mosaico en una posición específica de la habitación, entonces necesita usar la función tilemap_get_at_pixel().

¡IMPORTANTE! Si los mosaicos en el mapa de mosaicos no se han modificado (es decir: no se rotan ni se voltean, etc.), entonces el valor de retorno de los datos del mosaico será exactamente igual al índice del mosaico del juego de fichas. Por lo tanto, puede crear "mapas de colisión" de mosaicos usando un mosaico en el índice 1 en el juego de fichas, por ejemplo, luego use esta función para verificar 1 o 0 (un mosaico vacío) para calcular colisiones.


Sintaxis:

tilemap_get(tilemap_element_id, cell_x, cell_y);


Argumentos:

Argumento Descripción
tilemap_element_id El valor único de ID del elemento tilemap para obtener los datos de mosaico de
x_cell La celda a lo largo del eje x para obtener los datos del mosaico
y_cell La celda a lo largo del eje y para obtener los datos del mosaico de


Devoluciones:

Real (-1 if there is an error)


Ejemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

El código anterior obtiene la ID para el mapa de mosaicos en la capa "Tiles_Sky" y luego la usa para obtener los datos del mosaico en la celda (0, 0). Estos datos de mosaico se vuelven a girar antes de usarse para establecer el mosaico en el mapa de mosaico nuevamente.
