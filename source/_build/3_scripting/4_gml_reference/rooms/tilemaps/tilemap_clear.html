tilemap_clear

Descripción

Usando esta función puedes borrar / establecer todos los mosaicos en un mapa de mosaicos dado. Le das el ID del elemento de mosaico (que obtienes cuando creas un elemento de mosaico usando layer_tilemap_create() o cuando usas la función layer_tilemap_get_id() ), y luego proporcione los datos de los mosaicos con los que desea borrar la capa. Un valor predeterminado de 0 borrará todas las teselas de la capa (esencialmente haciendo que todas las teselas estén "vacías"), mientras que usted puede usar las funciones de teselas dedicadas para crear sus propios datos de teselas con los que borrar el mosaico.


Sintaxis:

tilemap_clear(tilemap_element_id, tiledata)


Argumentos:

Argumento Descripción
tilemap_element_id El valor ID único del elemento tilemap para cambiar
tiledata Los datos de mosaico que se usarán para borrar la capa


Devoluciones:

N/A


Ejemplo:

var layer_id = layer_get_id("Forest");
var tile_id = layer_tilemap_get_id(layer_id);
tilemap_clear(tile_id, 0);

El código anterior obtiene el valor ID de un mapa de mosaico creado en el editor de sala, y luego lo borra usando mosaicos "vacíos".
