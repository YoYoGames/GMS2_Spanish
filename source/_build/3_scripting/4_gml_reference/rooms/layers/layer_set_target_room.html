layer_set_target_room


Descripción

Cuando llamas a esta función, le estás diciendo a GameMaker Studio 2 que todas las funciones de capas adicionales se deben aplicar a la sala determinada. De esta forma, puede cambiar o generar capas de procedimiento en una sala que no es la habitación actual. Cuando haya terminado de agregar capas o elementos de capa a una sala, llame a la función layer_reset_target_room para restablecer el objetivo de la sala (o llamar de nuevo a esta función con un argumento de sala de -1).

¡ADVERTENCIA! Mientras se dirige a otra sala, puede usar todas las funciones de capa normales, excepto que no puede crear instancias usando instance_create_layer o instance_create_depth, ni la función de capa layer_add_instance estar disponible.


Sintaxis:

layer_set_target_room(room)


Argumentos:

Argumento Descripción
room La sala a destino para todas las funciones de capas adicionales


Devoluciones:

N/A


Ejemplo:

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
   {
   layer_sprite_create(l, random(room_get_width(rm_Game)), random(room_get_height(rm_Game)), spr_Trees);
   }
layer_reset_target_room();

El código anterior establece la sala objetivo en la sala "rm_Game" y luego obtiene la ID de la capa llamada "SpriteAssets" en esa sala. Este ID de capa se usa para agregar 50 elementos de sprite aleatorios a la capa, antes de que el objetivo de capa se restablezca a la habitación actual.
