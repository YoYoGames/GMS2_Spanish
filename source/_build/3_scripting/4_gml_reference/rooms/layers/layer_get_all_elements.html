layer_get_all_elements


Descripción

Puede usar esta función para obtener los identificadores de elemento de la capa dada. Usted proporciona la identificación de capa (que obtiene cuando crea la capa usando layer_create() ) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y la función devolverá una matriz de ID, donde cada entrada en la matriz es una ID única para un elemento en esa capa. Por ejemplo, si la capa es una Capa de Activos, la matriz se rellenará con los valores de ID para cada activo de sprite asignado a la capa. Tenga en cuenta que usar código para trabajar con capas significa que puede asignar diferentes tipos de elementos a la misma capa, por lo que puede tener activos de sprites junto con instancias, por ejemplo, en cuyo caso puede usar la función layer_get_element_type para obtener el tipo de elemento al que se relaciona la ID.


Sintaxis:

layer_get_all_elements(layer_id)


Argumentos:

Argumento Descripción
layer_id El valor ID único de la capa para obtener los elementos de (o el nombre de la capa como una cadena)


Devoluciones:

Array (1D, populated with Element IDs)


Ejemplo:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length_1d(a); i++;)
   {
   if layer_get_element_type(a[i]) == layerelementtype_sprite
      {
      layer_sprite_destroy(a[i])
      }
   }

El código anterior obtiene los ID de todos los elementos de instancia asignados a la capa de la instancia que ejecuta el código. El código luego verifica si alguno de los elementos devueltos son activos de sprite y si son destruidos.
