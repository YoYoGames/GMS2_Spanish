layer_get_name


Descripción

Puede usar esta función para obtener el nombre de la capa dada. Usted proporciona el valor único de ID de capa y si la capa es una de las capas nombradas creadas en el editor de sala, la función devolverá una cadena con el nombre de la capa. Si la capa no es una de las del editor de sala (es decir: se creó utilizando layer_create ) luego se devolverá una cadena vacía.


Sintaxis:

layer_get_name(layer_id)


Argumentos:

Argumento Descripción
layer_id El valor ID único de la capa para obtener el nombre de


Devoluciones:

String

var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length_1d(a); i++;)
   {

   i   i{
   i   ids_list_add(layer_list, a[i])
   i   i}
   }

El código anterior obtiene los ID de todas las capas de la sala y luego realiza un bucle para verificar si hay capas con nombre. Si son ellos, entonces su identificación se agrega a una lista.
