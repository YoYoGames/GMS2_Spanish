layer_destroy


Descripción

Esta función destruirá la capa dada. Usted proporciona la identificación de capa (que obtiene cuando crea la capa usando layer_create() ) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y esto lo eliminará de la sala actual. Si la capa ha sido diseñada en el editor de sala, la próxima vez que salga de la sala y luego regrese, la capa se volverá a crear con los contenidos originales; sin embargo, si la sala es persistente, la capa se eliminará a menos que la persistencia de la sala se apaga de nuevo. Cuando destruyes una capa de esta manera, también se eliminan todos sus contenidos, por lo que cualquier ID de referencia para fondos o mapas de mosaico, etc. dejará de ser válida y se destruirán todas las instancias asignadas a la capa (realizando su Destruir Evento) al mismo tiempo, si tienen uno, así como el evento de limpieza).


Sintaxis:

layer_destroy(layer_id)


Argumentos:

Argumento Descripción
layer_id El valor de ID exclusivo de la capa para destruir (o el nombre de la capa como una cadena)


Devoluciones:

N/A


Ejemplo:

if !instance_exists(obj_Bullet_Parent)
   {
   layer_destroy(global.Bullet_Layer);
   }

El código anterior verificará si existe alguna instancia del objeto "obj_Bullet_Parent", y si no lo hace, destruirá la capa con el ID almacenado en la variable global.
