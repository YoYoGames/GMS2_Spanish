layer_add_instance


Descripción

Esta función se puede usar para mover una instancia determinada desde la capa en la que se encuentra actualmente a otra capa. Usted proporciona la identificación de capa (que obtiene cuando crea la capa usando layer_create() ) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y la ID de instancia de la instancia para moverse entre las capas. Por ejemplo, supongamos que tu reproductor está en una capa que está a una profundidad menor que otra capa y quieres que aparezca detrás de las capas entre las dos. Puede usar esta función para cambiar el reproductor de la capa de primer plano a la capa de fondo y hacer que aparezca detrás de las otras capas que se están dibujando.

NOTA: Esta función no se puede usar para agregar una nueva instancia a una capa. Debe haber creado la instancia previamente y haber almacenado su ID en una variable.


Sintaxis:

layer_add_instance


Argumentos:

Argumento Descripción
layer_id El valor único de ID de la capa de instancia para destino (o el nombre de la capa como una cadena)
instance_id El valor único de ID de instancia de la instancia para pasar a la capa de destino


Devoluciones:

N/A


Ejemplo:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

El código anterior primero obtendrá el índice de la instancia más cercana a la posición x / y determinada y lo almacenará en una variable local. A continuación, obtiene la ID de capa de instancia única para la capa denominada "Instance Front" y mueve la instancia encontrada a esa capa.
