layer_get_id


Descripción

Esta función se puede usar para obtener el valor único de ID para una capa determinada. En el IDE, todas las capas tienen un nombre y un tipo, y para poder editarlas o cambiarlas a través del código, debe proporcionar el valor de ID de la capa. Esta función se usa para recuperar esta ID usando el nombre (una cadena) de la capa (como está escrito en el IDE). Si crea una nueva capa a través del código usando la función layer_create() entonces esa función devolverá el valor de ID único en su lugar (las capas dinámicas creadas no reciben nombres). Tenga en cuenta que si proporciona el nombre de una capa que no existe en la sala actual, recibirá un error y el proyecto se bloqueará.


Sintaxis:

layer_get_id(layer_name)


Argumentos:

Argumento Descripción
layer_name El nombre de la capa (una cadena)


Devoluciones:

Layer ID Value (Real), or -1 if the layer specified doesn't exist


Ejemplo:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

El código anterior primero obtendrá el índice de la instancia más cercana a la posición x / y determinada y lo almacenará en una variable local. A continuación, obtiene la ID de capa de instancia única para la capa denominada "Instance Front" y mueve la instancia encontrada a esa capa.
