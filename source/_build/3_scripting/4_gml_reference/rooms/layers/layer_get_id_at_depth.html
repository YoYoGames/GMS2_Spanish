layer_get_id_at_depth


Descripción

Puede usar esta función para obtener los ID de todas las capas asignadas a una profundidad específica. Usted da la profundidad para verificar y la función devolverá una matriz con 1 o más entradas dependiendo de si hay capas en la profundidad dada o no. Si no hay capas en la profundidad dada, la matriz tendrá una sola entrada en la posición [0] con un valor de -1, pero si hay capas en la profundidad, se hará una entrada en la matriz para cada capa encontrada: el valor de entrada será el valor de ID único para una capa.


Sintaxis:

layer_get_id_at_depth(depth)


Argumentos:

Argumento Descripción
depth La profundidad para verificar y recuperar los ID de capa.


Devoluciones:

Array


Ejemplo:

var a = layer_get_id_at_depth(0);
if a[0] != -1
    {
    for (var i = 0; i < array_length_1d(a); i++;)
        {
        layer_destroy(a[i]);
        }
    }

El código anterior recupera datos sobre las capas con una profundidad de 0. Se realiza una comprobación para ver si existe alguna capa a esa profundidad, y si hay, entonces se analiza la matriz devuelta y se destruye cada una de las capas encontradas.
