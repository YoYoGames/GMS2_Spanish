layer_set_visible


Descripción

Con esta función, puede alternar la visibilidad de una capa. Cuando una capa es invisible, no se dibujará nada que esté asignado a la capa, y si se asignan instancias a la capa, entonces ni siquiera ejecutarán su evento de dibujo (más o menos lo mismo que si configurara la variable de instancia) visible a false ) Tenga en cuenta que las instancias que ya están marcadas como invisibles no se marcarán como visibles si la capa en la que se encuentran está configurada como visible, ya que se trata de dos configuraciones independientes, aunque su comportamiento es el mismo.


Sintaxis:

layer_set_visible(layer_id, visible)


Argumentos:

Argumento Descripción
layer_id El valor de ID exclusivo de la capa al objetivo (o el nombre de la capa como una cadena)
visible Si la capa debe ser visible ( true ) o no ( false )


Devoluciones:

N/A


Ejemplo:

var lay_id = layer_get_id("Instances");
if layer_get_visible(lay_id)
   {
   layer_set_visible(lay_id, false);
   }
else
   {
   layer_set_visible(lay_id, true);
   }

El código anterior obtiene el valor de ID para la capa denominada "Instancias" en el editor de sala, luego usa la ID para verificar si la capa es visible o no, alternar la visibilidad de la capa dependiendo del valor devuelto.
