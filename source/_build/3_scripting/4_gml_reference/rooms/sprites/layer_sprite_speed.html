layer_sprite_speed


Descripción

Esta función se puede usar para establecer la velocidad de una imagen animada que se ha asignado a un elemento de sprite. Le das la ID del elemento sprite (que obtienes cuando creas un elemento sprite usando layer_sprite_create() o cuando usas la función layer_sprite_get_id() ), y luego establecer la velocidad de la animación. Si el IDE ha establecido la velocidad de animación del sprite utilizado para calcularse como marcos por cuadro de juego, entonces el valor de velocidad dado es un multiplicador, siendo 1 el valor predeterminado, y configurarlo a 0,5 será la mitad de la velocidad de animación, mientras se establece a 1.5 lo duplicará. Sin embargo, si el IDE ha establecido la velocidad de animación para el sprite que se calcula como marcos por segundo, entonces el valor de velocidad dado es un valor de fotogramas por segundo, por lo que establecerlo en 5, por ejemplo, establecerá la animación en bucle 5 cuadros por segundo que se ejecuta el juego. Si el sprite utilizado no tiene subimágenes, esta función no tendrá efecto.


Sintaxis:

layer_sprite_speed(sprite_element_id, speed);


Argumentos:

Argumento Descripción
sprite_element_id El valor único de ID del elemento sprite para cambiar
speed El multiplicador / valor de velocidad a usar para el elemento


Devoluciones:

N/A


Ejemplo:

var lay_id = layer_get_id("Assets_water");
var spr_id = layer_sprite_get_id(lay_id, "gfc_water");
layer_sprite_speed(spr_id, 0.5);

El código anterior obtendrá la ID de capa para la capa denominada "Assets_water" y luego la usará para obtener la ID del elemento sprite dado en esa capa. Esta ID se usa luego para cambiar la velocidad de la imagen del elemento.
