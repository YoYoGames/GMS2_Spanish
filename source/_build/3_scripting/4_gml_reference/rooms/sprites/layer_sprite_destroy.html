layer_sprite_destroy


Descripción

Esta función destruirá el elemento sprite dado. Proporcionas la ID de sprite (que obtienes cuando creas el sprite usando layer_sprite_create() o cuando usa la identificación de la capa junto con layer_get_sprite_id() ) y esto lo eliminará. Tenga en cuenta que esto no elimina la capa, solo el sprite de ella, y si el sprite es uno que se ha agregado en el editor de sala, la próxima vez que salga de la sala y luego regrese, el sprite se volverá a crear. Sin embargo, si la sala es persistente, el sprite se eliminará a menos que la persistencia de la sala se apague de nuevo.


Sintaxis:

layer_sprite_destroy(sprite_element_id)


Argumentos:

Argumento Descripción
sprite_element_id El valor de ID único del sprite que se destruirá


Devoluciones:

N/A


Ejemplo:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.Asset_spr_trees)
   {
   layer_sprite_destroy(lay_id);
   }

El código anterior verifica la capa "Asset_Trees" para ver si el elemento sprite dado existe y si lo hace, entonces se destruye (pero no la capa).
