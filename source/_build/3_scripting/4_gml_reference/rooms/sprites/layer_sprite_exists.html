layer_sprite_exists


Descripción

Puede usar esta función para verificar y ver si un elemento de sprite existe en cualquier capa dada. Usted proporciona la identificación de capa (que obtiene cuando crea la capa usando layer_create() ) o el nombre de la capa (como una cadena - esto tendrá un impacto en el rendimiento) y la ID del elemento de sprite (que obtienes cuando creas un elemento de sprite usando layer_background_create() o cuando usas la función layer_sprite_get_id() ) y la función regresará true si el elemento existe, o false si no es así

NOTA: Esta función funciona dentro del alcance de la sala de destino actual, de forma predeterminada, la habitación en la que se llama a la función, que se puede configurar con la función layer_set_target_room().


Sintaxis:

layer_sprite_exists(layer_id, sprite_element_id)


Argumentos:

Argumento Descripción
layer_id El valor de ID exclusivo de la capa al objetivo (o el nombre de la capa como una cadena)
sprite_element_id El valor único de ID del elemento sprite para verificar


Devoluciones:

Boolean


Ejemplo:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.TreesSprites)
   {
   layer_sprite_destroy(lay_id, global.TreesSprites);
   }

El código anterior verifica la capa "Asset_trees" para ver si el elemento sprite dado existe y si lo hace, entonces se destruye (pero no la capa).
