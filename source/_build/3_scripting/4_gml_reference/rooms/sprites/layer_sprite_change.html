layer_sprite_change


Usando esta función puedes cambiar el recurso de sprite asignado a un elemento de sprite dado en una capa. Le das la ID del elemento sprite (que obtienes cuando creas un elemento sprite usando layer_sprite_create() o cuando usas la función layer_sprite_get_id() ), y luego suministrar un elemento de elemento de sprite activo será cambiado. Tenga en cuenta que si el sprite tiene subimágenes, entonces también se animará (esto se puede controlar usando el layer_sprite_index y layer_sprite_speed funciones). Tenga en cuenta que puede asignar un valor de -1 como el nuevo índice de sprites y no se mostrará ningún sprite, aunque el elemento seguirá existiendo y aún podrá cambiarse más tarde.


Sintaxis:

layer_sprite_change(sprite_element_id, sprite_index)


Argumentos:

Argumento Descripción
sprite_element_id El valor único de ID del elemento sprite para cambiar
sprite_index El nuevo índice de sprites para usar


Devoluciones:

N/A


Ejemplo:

var s = layer_sprite_get_id(layer, global.Asset_sprite);
if layer_sprite_get_sprite(s) != spr_Nighttime
   {
   layer_sprite_change(s, spr_nighttime);
   }

El código anterior comprueba el índice de sprites del elemento con el ID almacenado en la variable global global.Asset_sprite y si no es "spr_Nighttime", ese sprite se asigna al elemento.
