layer_sprite_create


Con esta función puedes asignar un recurso de sprite a una capa para usar en tu proyecto. Usted proporciona la identificación de capa (que obtiene cuando crea la capa usando layer_create() o cuando usas el nombre de la capa junto con layer_get_id() ), una posición dentro de la sala y un índice de sprites (que sería el nombre del sprite como se muestra en el árbol de recursos), y se agregará a la capa. La función devuelve el valor único de identificación para el elemento, que luego se puede usar en otras funciones de capa para sprites.


Sintaxis:

layer_sprite_create(layer_id, x, y, sprite)


Argumentos:

Argumento Descripción
layer_id El valor de ID único de la capa para orientar
x La posición x para usar
y La posición y para usar
sprite El índice de sprites que se utilizará


Devoluciones:

Sprite element ID (Real)


Ejemplo:

global.asset_layer = layer_create(10000);
for (var i = 0; i< 10; i++;)
   {
   global.asset_spr_trees[i] = layer_sprite_create(global.asset_layer, random(room_width), room_height - 100, spr_Trees);
   }

El código anterior crea una nueva capa y luego agrega 10 nuevos elementos de sprite, almacenando la ID de cada elemento en una matriz.
