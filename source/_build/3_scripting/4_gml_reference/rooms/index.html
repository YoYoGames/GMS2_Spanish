Habitaciones

Al crear salas en GameMaker Studio 2, puede optar por usar el Editor de sala dedicado o puede crear contenido de sala dinámico mediante código, o tal vez un poco de ambos. La mayoría de los aspectos de una sala pueden modificarse, crearse o eliminarse en tiempo de ejecución, lo que significa que puede crear mazmorras de procedimiento o destrucción dinámica fácilmente al hacer su juego.

Puede encontrar una descripción general de todas las funciones disponibles de las diferentes secciones a continuación:

  1. Funciones generales de la sala y variables
  2. Funciones generales de capa
  3. Tilemaps
  4. Antecedentes
  5. Sprites

Una cosa importante a tener en cuenta al crear capas de sala es que puede tener múltiples tipos de elementos diferentes en una capa. Entonces, por ejemplo, puedes tener una capa para enemigos, y en esa capa tienes instancias enemigas así como ciertos activos de sprites que solo están relacionados con ese enemigo. O podría tener un elemento de fondo con un elemento de mosaico para crear el aspecto de una habitación. Esto no está permitido en el editor de sala, pero al crear cosas dinámicamente puede hacerlo, por lo que funciones como layer_get_all_elements existe.
