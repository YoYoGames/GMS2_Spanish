xboxlive_user_is_signing_in


Descripción

Esta función solo es válida cuando se exporta utilizando el destino UWP para Xbox One y requiere que haya marcado la opción Habilitar XBox Live en las Opciones del juego UWP. La función verificará si un usuario está en proceso de iniciar sesión y regresar true si ese es el caso y false de otra manera.


Sintaxis:

xboxlive_user_is_signing_in();


Devoluciones: Boolean


Ejemplo:

if !xboxlive_user_is_signed_in()
    {
    if !xboxlive_user_is_signing_in()
        {
        xboxlive_show_account_picker();
        }
    }
else
    {
    global.GamerTag = xboxlive_gamertag_for_user();
    }

El código anterior comprueba si un usuario ha iniciado sesión en XBox Live y si han iniciado sesión, el código obtendrá el gamertag para el usuario y lo almacenará en una variable global para su uso futuro. Si no han iniciado sesión, el código verifica si están en proceso de inicio de sesión, en cuyo caso no sucederá nada más y, si no lo están, se abrirá el selector de cuenta, lo que les pedirá que firmen. -en.
