shader_is_compiled


Sintaxis:

shader_is_compiled(shader);


Argumento Descripción
shader El sombreador para verificar


Devoluciones:

N/A


Descripción

Esta función verificará un sombreador en tiempo de ejecución para asegurarse de que se haya compilado correctamente. Si tiene, entonces la función regresa true de lo contrario, regresa false. Esta función debe usarse al inicio del juego para asegurarse de que la plataforma que ejecuta su juego haya compilado satisfactoriamente los sombreadores utilizados (particularmente en Windows donde algunas computadoras pueden estar usando DX9 con Shader Level 2.0 y no una versión posterior con shader level 3.0 )

Si tu shader NO ha sido compilado y llamas shader_set el juego se bloqueará, por lo que vale la pena tener algún tipo de control cada vez que esté utilizando algo más que simples sombreadores GLSL ES.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

global.GFX = false;
if shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp)
   {
   global.GFX = true;
   }

El código anterior establecerá una variable global para false, y luego si los dos sombreadores que se están verificando se han compilado correctamente, se establecerá en true.
