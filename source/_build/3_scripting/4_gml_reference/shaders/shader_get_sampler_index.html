shader_get_sampler_index


Sintaxis:

shader_get_sampler_index(shader, uniform);


Argumento Descripción
shader El índice del sombreador a usar.
shader La muestra del sombreador para obtener el mango de (una cadena).


Devoluciones:

Real


Descripción

Como no puede cambiar el valor de una muestra de sombreado dentro del sombreador, debe configurarlo antes de llamar al sombreador utilizando una de las funciones de conjunto uniformes disponibles. Sin embargo, para ser capaz de hacer que primero debe llamar a esta función para obtener el "mango" de la toma de muestras de sombreado que tendrá que ajustar.

NOTA: Aunque un sombreador está formado por dos programas discretos (vértice y fragmento), esta función no diferenciará entre los dos y devolverá el control de la muestra del sombreador de cualquiera de ellos.
¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

El código anterior obtendrá el identificador de la muestra dentro del sombreador indexado como "GlassShader" y luego establecer esa constante de sombreado a la textura de sprite dada.
