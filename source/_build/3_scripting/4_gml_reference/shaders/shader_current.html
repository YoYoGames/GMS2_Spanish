shader_current


Sintaxis:

shader_current();


Devoluciones:

Real (ID value of the shader, or -1 if no shader)


Descripción

Esta función devolverá el valor de ID de índice del sombreador que se está utilizando actualmente para la representación, o devolverá -1 si no se está utilizando ningún sombreador.

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

if shader_current() == -1
   {
   shader_set(sh_warp)
   }

El código anterior comprobará para ver cuál es el sombreador actual y si devuelve -1 (no se utiliza ningún sombreador), entonces se establece un sombreador.
