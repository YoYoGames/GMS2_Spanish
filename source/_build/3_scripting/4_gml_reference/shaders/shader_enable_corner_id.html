shader_enable_corner_id


Sintaxis:

shader_enable_corner_id(enable);


Argumento Descripción
enable Habilitar ( true ) o desactivar ( false ) esta función.


Devoluciones:

N/A


Descripción

Con esta función puede establecer un estado global para todos los sombreadores que se usan donde, cuando está habilitado, el sombreador "roba" 2 bits de los valores de color de entrada. El primero es del bit más bajo del valor de color rojo, y el segundo es del bit más bajo del valor de color azul. Estos valores pueden recuperarse en el sombreador para determinar con qué vértice se trata (es decir, en qué esquina).

¡IMPORTANTE! Esta función no está disponible con la licencia de prueba del producto.


Ejemplo:

shader_enable_corner_id(true);

El código anterior permitirá el uso de bits de color para la ID de esquina para todos los sombreadores.
