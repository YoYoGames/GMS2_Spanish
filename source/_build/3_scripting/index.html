Scripting

Esta sección trata sobre los diferentes lenguajes de scripting disponibles para la programación en GameMaker Studio 2. El idioma o método que utilice para crear sus proyectos dependerá de su habilidad y de su historial anterior, y dado que cada persona es diferente, GameMaker Studio 2 pretende ser lo más adaptable posible a sus diferentes necesidades, ofreciendo dos opciones diferentes para que pueda crear. juegos:

Arrastrar y soltar (DnD ™) es ideal para principiantes o aquellos que están más orientados visualmente o artísticamente (esto a veces se llama un "lenguaje visual de scripting"). Consiste en arrastrar y soltar acciones en el Editor de objetos para crear una lista de funciones que seguirán las instancias del objeto. Puede descubrir cómo usar DnD ™ de las siguientes secciones del manual:
  1. Arrastrar y soltar
  2. Referencia de arrastrar y soltar

Aquí puede obtener una descripción general de la sintaxis del lenguaje de GameMaker junto con ejemplos de uso y estructura del programa, y ​​también puede encontrar una guía de referencia completa para todas las funciones incorporadas de GML. Si no está familiarizado con el uso de un lenguaje de scripts, le recomendamos que comience con la Descripción general de GML; de lo contrario, simplemente use la guía de referencia para verificar los nombres de las funciones cuando sea necesario:
  1. Descripción general de GML
  2. Referencia GML

Vale la pena señalar que puedes "mezclar y combinar" cada una de estas técnicas de programación diferentes cuando creas tus juegos, y seleccionar una u otra cuando creas un nuevo proyecto no te limita a usar solo ese único idioma.
