Editores

GameMaker Studio 2 tiene varios editores diferentes que puedes usar para agregar, eliminar y editar los diferentes recursos que tu juego requiere. En el curso del desarrollo, necesitarás usar algunos (o incluso todos) de estos editores, y recordar lo que hace todo a veces puede ser un poco abrumador. Para ayudar, puede encontrar cada uno de los diferentes editores enumerados a continuación, y hacer clic en cualquiera de ellos lo llevará a una página que explica qué es y cómo usarlo.


  • Sprites


  • Imágenes


  • Juegos de azulejos


  • Sonidos


  • Caminos


  • Guiones


  • Shaders


  • Fuentes


  • Líneas de tiempo


  • Objetos


  • Habitaciones


  • Notas


  • Archivos incluidos


  • Extensiones


  • Opciones


  • Configuraciones

Tenga en cuenta que cuando tiene el Árbol de recursos en foco, el menú superior del IDE tiene un nuevo elemento de menú: Esto simplemente le permite crear un nuevo recurso sin tener que usar el botón derecho del mouse menú en una carpeta de recursos individuales.
