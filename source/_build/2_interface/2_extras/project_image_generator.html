Generador de imágenes del proyecto


Se accede al generador de imágenes del proyecto desde el menú de herramientas en la parte superior del IDE de GameMaker Studio 2. Esta herramienta está diseñada para crear rápidamente todas las diferentes pantallas de inicio, imágenes promocionales e íconos que son requeridas por todas las diferentes plataformas de destino desde un solo conjunto de imágenes. Usted suministra la imagen base y el icono de la base y luego hace clic en el botón Generar, y las Opciones del juego para cada plataforma se actualizarán con las nuevas imágenes.

Para que esto funcione correctamente, debe proporcionar una pantalla de bienvenida de alta resolución y una imagen de icono de alta resolución. Los tamaños recomendados serían:

  • Pantalla de bienvenida: al menos 1920x1080px, o 1080x1920px para retrato, y creada como PNG de 24 bits sin transparencias.

  • Íconos: al menos 1024x1024px (que es el tamaño del ícono más grande requerido), y creado como PNG de 24 bits que puede tener transparencias

Una vez que haya suministrado estas imágenes, se mostrarán en la Vista previa de Windows y luego podrá establecer el tipo de "ajuste" que se utilizará al aplicarlas a las diferentes plataformas. Puede elegir de lo siguiente:

  • Mantenga la relación de aspecto: esto escalará la imagen para que se ajuste manteniendo la relación de aspecto y agregando un color de fondo para rellenar los lados si el ajuste no es 1: 1.

  • Estirar para ajustar: esto estirará las imágenes para ajustarse al tamaño completo de la imagen / icono.

  • Recortar para ajustar: tomará la imagen base y se ajustará mejor a lo largo del eje horizontal o vertical y luego recortará el resto. Por ejemplo, si usted tiene una imagen de 640x480 que tiene que ser utilizado como una pantalla de presentación de 480x800, entonces la imagen se ampliará para ajustarse al eje vertical (480 -> 800) y el resto de la imagen se va a recortar distancia. Del mismo modo, si tiene una imagen de 1080x1920 que debe ajustarse a 800x800, la imagen se reducirá para ajustarse al eje horizontal y el resto se recortará.

Ahora todo lo que tiene que hacer es hacer clic en Generar y GameMaker Studio 2 todas las opciones de imagen del juego y la configuración del icono con imágenes para cada plataforma.

¡IMPORTANTE! Si bien esta herramienta generará todas las imágenes que necesita, ¡eso no significa que sea perfecta! Intentará hacer todo lo posible para que las imágenes se adapten a los diferentes tamaños y resoluciones para cada plataforma, pero a veces puede que esto no se vea como realmente desea, ya que realmente no existe una "talla única". Por lo tanto, después de ejecutar esta herramienta, le recomendamos que revise todas las imágenes en cada una de las opciones de juego de la plataforma y modifique aquellas que tal vez no se vean exactamente como desea manualmente.
