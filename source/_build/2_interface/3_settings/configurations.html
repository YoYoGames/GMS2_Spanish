Configuraciones

Debido a las exigencias del desarrollo multiplataforma y la necesidad de que un producto esté disponible y personalizado para diferentes clientes y / o dispositivos, GameMaker Studio 2 incluye configuraciones para simplificar las cosas. Con este sistema, puede cambiar rápidamente entre diferentes pantallas de inicio, iconos y archivos incluidos y probar o exportar solo aquellos que sean relevantes para la plataforma o cliente de destino en cualquier momento, todo desde el mismo código base.

¡IMPORTANTE! Las configuraciones no están disponibles con la Licencia de prueba del producto.


Para configurar sus configuraciones, se recomienda que primero prepare una configuración "predeterminada" incorporando todos los activos configurables para el proyecto en GameMaker Studio 2, incluyendo iconos y pantallas de inicio (realizadas a través de las Opciones del juego ), Archivos incluidos y cualquier Extensión que es posible que necesite, y también puede ir y cambiar las propiedades de salida de los recursos de sonido o la configuración del grupo de texturas y muchas otras opciones... Casi todo lo que se puede configurar en el IDE se almacenará en la configuración seleccionada actualmente.

NOTA: con los archivos y extensiones incluidos, puede seleccionar una plataforma de destino para exportar. Eso también debe configurarse de modo que cuando cree configuraciones individuales, simplemente se trate de quitar el tic de las que no necesita.


Una vez que haya configurado la configuración predeterminada, puede agregar una nueva al árbol de recursos haciendo clic derecho en la carpeta de configuraciones y seleccionando Agregar. Una vez hecho esto, puede seleccionar la nueva configuración desde la parte superior del árbol de recursos y una vez seleccionada, puede continuar y personalizarlos para que cambien junto con las configuraciones. Para eso, ahora debe revisar las Opciones del juego para cada plataforma objetivo y agregar, eliminar o cambiar iconos, pantallas emergentes y cualquier otra información que sea relevante para la configuración, así como configurar los Grupos de texturas para cada configuración. Si ha configurado grupos de texturas para cada configuración, también necesitará revisar todos los activos gráficos (sprites y fondos) y reasignarlos. Cada cambio que realice en estas configuraciones será recordado por el administrador de configuración y se recordará siempre que cambie las configuraciones.

Vale la pena señalar que puede hacer que GameMaker Studio 2 aplique automáticamente cualquier configuración dada a cualquier plataforma de destino determinada. Esto se hace seleccionando la configuración que necesita para esa plataforma desde dentro de la Ventana de Objetivos. Una vez que haya configurado una configuración para ser utilizada de esta manera, cada vez que compile para ese objetivo, ya sea para probar o para crear un paquete final de activos, se usará la configuración. Para obtener más información sobre la Ventana de objetivos, consulte la sección sobre Compilación.
