Configuraciones

GameMaker Studio 2 permite configurar muchos aspectos diferentes de cómo se mostrará y compilará su proyecto, incluida la configuración de diferentes configuraciones para controles específicos de la plataforma, o el uso de una herramienta SCM (Gestión de control de origen) y mucho más. La mayoría de estas configuraciones se explican en las siguientes secciones:

  1. Preferencias específicas de la plataforma
  2. Opciones de juego
  3. Grupos de texturas
  4. Grupos de audio
  5. Configuraciones
