Micro servidor web


Esta es la función principal de las herramientas pequeñas: conectarse y servir archivos a algunas de las plataformas de destino disponibles para GameMaker Studio 2 (como el destino HTML5, por ejemplo). Cuando ejecuta un juego en un objetivo que requiere el Micro Web Server, se abrirá una nueva ventana a medida que el juego se sirve en el dispositivo elegido:

NOTA: La primera vez que ejecuta un juego en un objetivo que requiere el Micro Web Server, es posible que reciba una alerta de firewall. Debería permitir que el servidor web se comunique a través del firewall si esto sucede.


El servidor web mostrará una cierta cantidad de información de depuración relacionada con los archivos que se han servido, y tiene un botón para borrar esto en la parte inferior. No puede editar la mayor parte de la otra información aquí (ya que se encuentra a través de los dispositivos de destino Gestor de configuración, pero puede cambiar el valor de velocidad de red. Si se establece que no sea el valor máximo efectivamente va a estrangular la velocidad de transferencia nework y permitir nada para probar juegos en un entorno de bajo ancho de banda.
