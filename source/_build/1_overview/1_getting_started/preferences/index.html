preferencias generales

La ventana de Preferencias es una parte esencial de GameMaker Studio 2, ya que es donde puede configurar y personalizar su experiencia usando el IDE, así como configurar cualquier SDK de plataforma específica, etc. En esta sección cubrimos las áreas específicas de IDE de la ventana de preferencias: para las plataformas de destino individuales, consulte la sección sobre Preferencias de la plataforma objetivo. Tenga en cuenta que después de cambiar cualquier preferencia de esta ventana, debe hacer clic en el botón Aplicar para que surta efecto.

Si, después de cambiar las opciones aquí para personalizar GameMaker Studio 2 a tu gusto, deseas hacer una copia de seguridad de las preferencias que has configurado, debes ir a la carpeta %AppData%/GameMakerStudio2/<your_account_name>/ y luego copie los siguientes dos archivos a un lugar seguro:

  • local_settings.json
  • cloud_settings.json

Sin embargo, si desea restablecer las preferencias que ha cambiado, debe hacer clic en el botón Restablecer valores predeterminados en la parte superior de la ventana. Tenga en cuenta que esto solo restaurará los valores predeterminados para la sección que se está editando actualmente.

Puede abrir las Preferencias de GameMaker Studio 2 desde el menú Archivo en la parte superior izquierda del IDE:

Una vez que se abra la ventana de Preferencias, tendrá las siguientes opciones diferentes:

  1. General
  2. Arrastrar y soltar
  3. Editor de imagen
  4. Editor de Sprites
  5. Editor de objetos
  6. Editor de sala
  7. Editor de Spline
  8. Tile Set Editor
  9. Líneas de tiempo
  10. Editor de extensiones
  11. Árbol de recursos
  12. Editores de texto
  13. Idiomas
  14. Mercado
  15. Complementos
  16. Runtime Feeds

Tenga en cuenta que también hay una sección para Preferencias específicas de plataforma que se enumera en esta sección del manual aquí.
