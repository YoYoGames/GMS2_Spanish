¡IMPORTANTE! Si ha estado utilizando una licencia de prueba para GameMaker Studio 2 y luego compra una de las otras licencias (de escritorio, móvil, UWP, web o consola), entonces podrá cerrar sesión en el IDE y luego reiniciar GameMaker Studio 2 y volver a iniciar sesión de nuevo para asegurarse de que la nueva licencia se aplique correctamente (aunque esto debería suceder automáticamente). Puede cerrar sesión en el menú Archivo.
