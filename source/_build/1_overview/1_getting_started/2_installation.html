Instalación


GameMaker Studio 2 está diseñado para su uso en computadoras de escritorio con las siguientes especificaciones mínimas:

Para la plataforma de Windows necesitarás tener la siguiente configuración básica:

  • CPU de doble núcleo compatible con Intel de 64 bits
  • 2GB de RAM
  • Tarjeta gráfica compatible con DX11
  • Microsoft 64bit Windows 7
  • Al menos 3 GB de espacio libre en el disco duro

Sin embargo, las especificaciones recomendadas son:

  • CPU cuádruple compatible con Intel de 64 bits
  • 8GB de RAM
  • Tarjeta gráfica basada en DX11
  • Microsoft 64bit Windows 10
  • Disco duro SSD

Para la plataforma Mac OS necesitarás tener al menos las siguientes especificaciones mínimas:

  • Mac OSX El Capitan o superior
  • Un procesador Intel Dual Core i3
  • 4GB o más de RAM
  • Xcode 7 o superior

Sin embargo, las especificaciones recomendadas son:

  • Mac OS
  • Un procesador Intel Quad Core i5
  • 8GB de RAM
  • Disco duro SSD

Puede descargar el instalador de GameMaker Studio 2 desde el sitio web de YoYo Games. Una vez instalado y la primera vez que ejecutas GameMaker Studio 2, es posible que se te pida que actualices antes de poder usar el producto. Después de que se hayan instalado las actualizaciones, se le solicitará que proporcione los detalles de la licencia de su cuenta YoYo Games (consulte aquí ) antes de poder utilizar el producto, lo que significa que la primera vez que use GameMaker Studio 2 debe tener una conexión a Internet. Después del inicio de sesión inicial, debe poder utilizar el producto fuera de línea, sin embargo, periódicamente se le solicitará que se conecte para verificar la cuenta de inicio de sesión.
