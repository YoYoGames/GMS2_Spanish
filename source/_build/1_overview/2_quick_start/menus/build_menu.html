El menú de compilación


El menú de compilación es donde puedes elegir construir tu proyecto para probarlo, depurarlo o como un ejecutable final. Las opciones disponibles son:

  • Ejecutar: ejecute su proyecto actual en la plataforma de destino elegida para la prueba. Para ver las diferentes opciones de compilación disponibles al compilar, consulte la sección sobre Compilación. Tenga en cuenta que este comando es lo mismo que presionar el botón Ejecutar en el IDE.

  • Depurar: ejecute su proyecto actual para probarlo con el depurador, de modo que pueda ver información detallada de rendimiento y corregir errores o problemas que pueda tener. Para obtener más información sobre la depuración, consulte la sección Depuración. Tenga en cuenta que este comando es lo mismo que presionar el botón Depurar en el IDE.

  • Clean - GameMaker Studio 2 mantiene un caché de archivos para ayudar a acelerar el tiempo de compilación cuando se prueba. Sin embargo, a veces un caché "obsoleto" puede ocasionar errores extraños e inesperados en tu juego, en cuyo caso lo primero que debes hacer es limpiar el caché desde aquí y volver a probarlo. También es una buena idea limpiar el caché antes de crear un ejecutable final para la plataforma de destino. Tenga en cuenta que este comando es lo mismo que presionar el botón Limpiar caché en el IDE.

  • Crear ejecutable: compilará el proyecto actual y creará un archivo ejecutable (o archivos) para la plataforma de destino. Primero abrirá el explorador de archivos para que pueda seleccionar dónde desea guardar el proyecto y luego compilar el juego. Consulte la sección sobre Compilación para obtener más información sobre la configuración de los objetivos de compilación. Tenga en cuenta que este comando es lo mismo que presionar el botón Crear ejecutable en el IDE.

  • Mostrar instalador de trabajador remoto: se abrirá una ventana del explorador de archivos en la ubicación del instalador para la herramienta Trabajador remoto (consulte aquí para obtener más información sobre los trabajadores remotos).
