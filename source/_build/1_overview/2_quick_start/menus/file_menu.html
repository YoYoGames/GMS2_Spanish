El menú Archivo


El menú de archivo es donde puede elegir cambiar, guardar o importar proyectos, así como establecer las preferencias IDE de GameMaker Studio 2. Las opciones disponibles son:

  • Nuevo proyecto: esto es para crear un nuevo proyecto desde cero en GameMaker Studio 2. Abrirá el diálogo "Guardar" y le pedirá que elija una ubicación para guardar el nuevo proyecto. Una vez que haya dado un nombre al proyecto y una ubicación para guardarlo, lo llevarán a un espacio de trabajo nuevo y en blanco para comenzar a trabajar en él. Los archivos de proyecto se guardan con *.yyp extensión de archivo.

  • Abrir proyecto: esto abrirá el explorador de archivos de GameMaker Studio 2 para que seleccione un archivo previamente creado *.yyp archivo de proyecto para cargar

  • Importar proyecto: si tiene un proyecto creado con el software GameMaker Studio 2 1.4 de YoYo Games, puede seleccionar esta opción para importar el *.gmx o el *.gmz archivo.

    NOTA: para importar un proyecto GameMaker previo a GameMaker Studio 2, debe haber sido creado utilizando la versión 1.4, e incluso entonces puede no funcionar directamente en la importación. Hay muchas cosas diferentes entre GameMaker Studio 2 y GameMaker Studio 2 1.4, por lo que es posible que sus proyectos más antiguos necesiten algunos ajustes para que funcionen. Consulte el artículo Porting A GMS Game para GameMaker Studio 2 en YoYo Games Knowledge Base y la sección del manual que trata sobre las funciones obsoletas y los scripts de compatibilidad.
    ¡IMPORTANTE! La importación de proyectos antiguos y proyectos YYZ no está disponible con la licencia de prueba del producto.
  • Proyectos recientes: puede usar esta opción para ver proyectos usados ​​recientemente. Al seleccionarlo, se volverá a abrir la página de inicio (en caso de que la haya cerrado) y podrá seleccionar un proyecto recientemente utilizado de la lista que allí se encuentra.

  • Guardar proyecto: guardará su juego en la misma ubicación que definió cuando creó el proyecto (o desde donde lo abrió).

  • Guardar proyecto como: igual que en el ejemplo anterior, solo usted especifica la ubicación y el nuevo nombre del proyecto.

  • Proyecto de exportación: esto exportará el proyecto como un solo *.yyz archivo. Muy útil para enviar copias a personas o para guardar copias de seguridad.
    ¡IMPORTANTE! La exportación de proyectos YYZ no está disponible con la licencia de prueba del producto.
  • Preferencias: al seleccionar esto, accederá a las Preferencias de Windows, donde puede configurar el aspecto y la sensación de GameMaker Studio 2.

  • Nuevo IDE: cuando seleccione esto, se abrirá una nueva instancia del IDE (se abrirá en la página de inicio).

  • Mi cuenta: esta opción abrirá su navegador web predeterminado y lo llevará a su página de cuenta YoYo.

  • Cerrar sesión: al seleccionar esto, se cerrará la sesión de GameMaker Studio 2. Tendrá que volver a iniciar sesión con la misma licencia (o cualquier otra) para poder utilizar el producto más adelante.
    ¡IMPORTANTE! Si ha estado usando una Licencia de prueba para GameMaker Studio 2 y luego compra una de las otras licencias (Escritorio, Móvil, UWP, Web o Consola), entonces tal vez necesite desconectarse del IDE, reiniciar GameMaker Studio 2 y volver a iniciar sesión. de nuevo para asegurarse de que la nueva licencia se aplique correctamente (aunque esto debería suceder automáticamente).
  • Salir: salga de GameMaker Studio 2 en el escritorio.
